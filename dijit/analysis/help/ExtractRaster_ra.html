Extract Raster

This tool can clip a raster to a boundary, either to a rectangular area or to a shape you define interactively on the screen. You can clip to the extent of the area you currently have displayed on your map or by the study area defined by a polygon.

Choose input data to extract

The input layer to clip.

Specify clipping geometry

Choose the extent type to use when clipping the geometry of the layer:

  • Use current extent—The current map extent will be used to clip out the resulting layer. This is the default.
  • Choose study area—Allows you to define a clipping geometry using the draw tool.
  • Choose layer—Choose a feature layer or imagery layer to define the extent, or choose a polygon feature layer to define the clipping geometry. When a polygon feature layer is selected, a check box is displayed to specify wether to use input features for clipping geometry. By default, it's unchecked. It means the input feature layer is used for define the extent. If it's checked, it means the input features are used for clipping geometry.

Specify clipping type

Specify whether you want to retain the area inside or outside your clip geometry:

  • Data inside the geometry—Imagery inside the clip extents will be extracted. This is the default.
  • Data outside the geometry—Imagery outside the clip extents will be extracted.

Result layer name

The name of the layer that will be created in My Content and added to the map. The default name is based on the tool name and the input layer name. If the layer already exists, you will be prompted to provide another name.

You can specify the name of a folder in My Content where the result will be saved using the Save result in drop-down box. If you have the privileges to create both tiled and dynamic imagery layers, you can specify which layer type to generate in the output using the Save result as drop-down box.
