Apply Raster Function Template

This tool processes your imagery with the function, or chain of functions, as specified by the raster function template.

Inputs

The inputs allow you to select the layers you want to process, and set up any parameters associated with the raster functions in the processing chain.

Result layer name

The name of the layer that will be created in My Content and added to the map. The default name is based on the tool name and the input layer name. If the layer already exists, you will be prompted to provide another name.

You can specify the name of a folder in My Content where the result will be saved using the Save result in drop-down box.

You can specify the layer type of output imagery using the Save result as drop-down box. Two layer types are available:

  • Tiled imagery layer—Provides imagery access as static tiles and associated metadata and supports client-side processing and rendering. This is the default.
  • Dynamic imagery layer—Provides versatile dynamic imagery access capabilities and supports on-demand server-side processing and dynamic mosaicking.
