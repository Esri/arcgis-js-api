Estrai raster

Questo strumento consente di ritagliare un raster lungo il contorno, in un'area rettangolare o in una forma definita in modo interattivo sullo schermo. È possibile ritagliare fino all'estensione dell'area attualmente visualizzata sulla mappa o in base all'area di studio definita da un poligono.

Scegliere dati di input da estrarre

Il layer di input da ritagliare.

Specifica geometria di ritaglio

Scegliere il tipo di estensione da utilizzare quando si ritaglia la geometria del layer:

  • Utilizza estensione corrente: verrà utilizzata l'estensione della mappa corrente per ritagliare il layer risultante. Si tratta dell'impostazione predefinita.
  • Scegli area di studio: consente di definire una geometria di ritaglio utilizzando lo strumento di disegno.
  • Scegli layer: scegliere un feature layer o un layer di immagini per definire l'estensione oppure scegliere un feature layer poligonale per definire la geometria di ritaglio. Quando è selezionato un feature layer poligonale, viene visualizzata una casella di controllo per specificare se utilizzare le feature di input per il ritaglio della geometria. Per impostazione predefinita, è deselezionato. Significa che il Feature Layer di input viene utilizzato per definire l'estensione. Se è selezionato, significa che le caratteristiche di input vengono utilizzate per il ritaglio della geometria.

Specifica tipo di ritaglio

Specificare se si vuole mantenere l'area all'interno o all'esterno della geometria ritagliata:

  • Dati all'interno della geometria: verranno estratte le immagini all'interno delle estensioni della clip. Si tratta dell'impostazione predefinita.
  • Dati al di fuori della geometria: verranno estratte le immagini al di fuori delle estensioni della clip.

Nome layer risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.
