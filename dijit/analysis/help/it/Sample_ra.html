Esempio

Questo strumento crea una tabella o una feature class puntuale che mostra i valori di celle da un raster, o un insieme di raster, per posizioni definite. Le posizioni sono definite da celle raster, feature poligonali, feature polilinee o da un insieme di punti.

I valori dei pixel saranno estratti da tutti i raster di input in ogni posizione. Verrà creata una tabella o una feature class puntuale con campi contenenti i valori dei pixel per ogni raster di input. Gli attributi aggiuntivi della tabella raster di input non saranno inclusi nella tabella di output.

Qualsiasi combinazione di raster, ad esempio a banda singola, multibanda, multidimensionale o multivariato, può essere specificata come raster di input. La struttura della tabella di output cambia quando i raster di input sono multidimensionali. Quando viene specificato un raster multibanda come uno dei raster di input, verranno utilizzate tutte le bande di tale input. Per elaborare una selezione di bande da un raster multibanda di input, creare prima un nuovo dataset raster composto da quelle bande particolari. Utilizzare il risultato nell'elenco dei raster di input.

La posizione di input può essere un raster o una feature puntuale:

  • Raster: i pixel contenenti valori validi (non NoData) vengono utilizzati per estrarre i valori dei pixel da tutti i raster di input, e il centro del pixel viene utilizzato per le posizioni dei punti.
  • Punto: i valori saranno campionati in ogni posizione del punto.
  • Polilinea o poligono: i valori saranno campionati al poligono o al centroide polilinea. Se il raster di input è multidimensionale, i valori saranno aggregati in base al tipo di Statistica.

I raster di input sono campionati nel loro riferimento spaziale e risoluzione nativi. Se ci sono più raster di input con diversi riferimenti spaziali, le posizioni di input sono proiettate al riferimento spaziale di ogni singolo raster da cui vengono estratti i valori. L'ambiente di analisi viene applicato alle posizioni di input, ma i raster di input non rispettano l'ambiente di analisi.

Le posizioni che estraggono i valori dalle celle NoData nel raster di input riceveranno un valore <null> nella tabella di output.

Per eseguire analisi con feature multipuntuali, prima di utilizzarle in questo strumento è necessario convertirle in singole feature puntuali.

Il riferimento spaziale delle coordinate x,y nella tabella di campionamento di output è lo stesso del raster della posizione di input o delle feature puntuali, a meno che non sia specificato nell'ambiente del Sistema di coordinate di output. Se il riferimento spaziale dei dati relativi alle posizioni di input è sconosciuto, anche il riferimento spaziale delle coordinate x,y è sconosciuto.

Il riferimento spaziale della feature class di output è lo stesso del raster della posizione di input o delle feature puntuali, a meno che non sia specificato nell'ambiente del Sistema di coordinate di output.

L'ambiente del Fattore di elaborazione parallela è supportato solo quando l'input è un raster multidimensionale.

Se si seleziona Usa estensione mappa corrente, verranno analizzate le variabili o le bande del layer visibili nell'estensione mappa corrente. Se questa opzione è deselezionata, verranno analizzate le variabili o le bande del layer, anche se esterne all'estensione mappa corrente.

Scegliere il layer o i layer di immagini da campionare

Un singolo layer di immagini o un elenco di layer di immagini i cui valori saranno campionati in base ai dati di posizione di input.

Per campionare tutte le sezioni da un layer di immagini multidimensionale, deve essere l'unico raster di input.

Scegliere il layer della posizione

Dati che identificano le posizioni da campionare.

I dati di posizione di input possono essere un raster, o una feature class puntuale, polilinea o poligonale.

Specificare campo ID univoco

Un campo contenente un valore diverso per ogni posizione o feature nel raster della posizione di input o feature class.

Specificare le informazioni di acquisizione dei dati di posizione (facoltativo)

Specificare il tempo, la profondità, o altri dati di acquisizione associati alle celle di posizione o alle feature.

Per ogni dimensione, ci sono tre opzioni per specificare le informazioni di acquisizione associate alle celle di posizione o alle feature:

  • Singolo campo o valore: quando si seleziona questa opzione, è necessario specificare le seguenti informazioni:
    • Campo o valore dimensionale: selezionare un campo di valori dimensionali dai dati di posizione, oppure immettere un valore dimensionale specifico.
  • Singolo campo o valore con valori relativi: se questa opzione è selezionata, le informazioni sottostanti devono essere specificate insieme per definire l'intervallo entro il quale le statistiche saranno calcolate per le variabili:
    • Campo o valore dimensionale: selezionare un campo di valori dimensionali dai dati di posizione.
    • Valore relativo prima: specificare un numero non negativo usato per sottrarre dal Campo o valore dimensionale per definire l'inizio di un intervallo di valori della dimensione. Questo valore rappresenta il numero di giorni se la dimensione è il tempo.
    • Valore relativo dopo: specificare un numero non negativo usato per aggiungere al Campo o valore dimensionale per definire la fine di un intervallo di valori di dimensione. Questo valore rappresenta il numero di giorni se la dimensione è il tempo.
  • Campi o valori di inizio e fine: se questa opzione è selezionata, le informazioni sottostanti devono essere specificate insieme per definire l'intervallo entro il quale le statistiche saranno calcolate per le variabili:
    • Campo o valori di inizio: selezionare un campo di valori dimensionali dai dati di posizione, oppure immettere un valore dimensionale specifico, per definire l'inizio di un intervallo di valori di dimensione.
    • Campo o valori di fine: selezionare un campo di valori dimensionali dai dati di posizione, oppure immettere un valore dimensionale specifico, per definire la fine di un intervallo di valori di dimensione.

Specificare campo o valore di distanza di buffer

La distanza specificata intorno alle feature dei dati relativi alla posizione. La distanza di buffer è specificata nelle unità lineari del riferimento spaziale della feature di posizione. Se la feature utilizza un riferimento geografico, l'unità sarà costituita da gradi.

Le statistiche saranno calcolate all'interno di questa area di buffer.

Scegliere il tipo di statistiche

Il tipo di statistica da calcolare.

  • Minimo: trova il valore minimo all'interno dell'intervallo specificato.
  • Massimo: trova il valore massimo all'interno dell'intervallo specificato.
  • Mediano: trova il valore mediano all'interno dell'intervallo specificato.
  • Medio: calcola il valore medio dell'intervallo specificato.
  • Somma: calcola la somma delle variabili all'interno dell'intervallo specificato.
  • Maggioranza: trova il valore che si verifica con più frequenza.
  • Minoranza: trova il valore che si verifica con meno frequenza.
  • Deviazione standard: calcola la deviazione standard.
  • Percentile: calcola un percentile definito all'interno dell'intervallo specificato.

Valore percentile

Il percentile da calcolare quando il parametro Tipo di statistiche è impostato su Percentile.

Questo valore può variare da 0 a 100. L'impostazione predefinita è 90.

Specificare la tecnica di ricampionamento

Algoritmo di ricampionamento utilizzato per il campionamento di un raster.

  • Più vicino: vicino più vicino. Questa è l'impostazione predefinita.
  • Bilineare: interpolazione bilineare
  • Cubica: convoluzione cubica

Specificare il layout di output

Specifica se i valori campionati appaiono in righe o colonne nella tabella di output. Questa opzione è disponibile solo quando il raster multidimensionale di input contiene una variabile e una dimensione e ogni sezione è un raster a banda singola.

  • I valori campionati appaiono in riga: i valori campionati appaiono in righe separate nella tabella di output. Questa è l'impostazione predefinita.
  • I valori campionati appaiono in colonna: i valori campionati appaiono in colonne separate nella tabella di output. Questa opzione è valida solo quando il raster multidimensionale di input contiene una variabile e una dimensione, e ogni sezione è un raster a banda singola.

Specificare il tipo di output

Specifica se generare una feature class puntuale con valori campionati nella sua tabella, o solo una tabella con valori campionati.

  • Tabella: generare una tabella con i valori campionati. Questa è l'impostazione predefinita.
  • Feature: generare una feature class puntuale con valori campionati nella sua tabella. Se le feature di posizione di input sono polilinee o poligonali, ogni punto di output è il centroide di ogni geometria delle feature di posizione di input.

Nome del layer dei risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.

È possibile specificare il nome di una cartella in I miei contenuti in cui salvare il risultato usando la casella a discesa Salva risultato in.
