Spartiacque

Questo strumento determina l'area contribuente su una serie di celle in un raster.

Un punto d'ingresso sul fiume Mississippi in Louisiana creerà uno spartiacque che si estende fino ad includere zone dal New York al Montana e al Canada.

Scegli layer di direzione flusso

Il raster di input che mostra la direzione del flusso fuori da ogni cella.

Il raster di direzione del flusso può essere creato usando la funzione di raster Direzione Flusso, con il tipo di direzione flusso predefinita D8.

Specifica posizioni del punto di versamento di input

Le posizioni del punto di versamento di input.

Questo può essere un raster o una feature di punti. Per un raster, ciò rappresenta le celle oltre le quali verranno determinate l'area o il bacino pluviale di contribuzione. Tutte le celle che non sono NoData verranno utilizzate come celle di origine. Per un input di punto, ciò rappresenta le posizioni oltre le quali verranno determinate l'area o il bacino pluviale di contribuzione.

Scegli campo punto versamento

Il campo usato per assegnare valori alle posizioni del punto di versamento.

Se il layer di punti di versamento di input è un raster, Value viene usato. Se è una feature, viene utilizzato un campo numerico. Se il campo contiene valori a virgola mobile, verranno troncati in numeri interi.

Nome layer risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.

È possibile specificare il nome di una cartella in I miei contenuti in cui salvare il risultato usando la casella a discesa Salva risultato in.
