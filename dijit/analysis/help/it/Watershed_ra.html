Spartiacque

Questo strumento determina l'area contribuente su una serie di celle in un raster.

Uno spartiacque è l'area di pendenza a monte che contribuisce al flusso, generalmente di acqua, a uno sbocco comune come drenaggio concentrato. Può far parte di uno spartiacque più grande e può anche contenere piccoli spartiacque, denominati sottobacini idrografici.

Il valore di ogni spartiacque sarà preso dal valore dell’origine nel raster di input o nel feature layer di punti di versamento. Quando il punto di versamento è un dataset raster, i valori di cella vengono utilizzati per impostazione predefinita. Quando il punto di versamento è un dataset di feature puntuale, i valori provengono dal campo specificato.

Lo strumento Spartiacque supporta solo un layer raster di direzione di flusso di input D8.

Quando si specificano le posizioni del punto di versamento di input come dati feature, il campo predefinito sarà il primo campo valido disponibile. Se non esiste alcun campo valido, viene utilizzato il campo ObjectID (ad esempio, OID o FID).

Scegli layer di direzione flusso

Il raster di input che mostra la direzione del flusso fuori da ogni cella.

Il raster di direzione di flusso può essere creato utilizzando lo strumento Direzione flusso o l'attività FlowDirection nel servizio RasterAnalysisTools, eseguire utilizzando il tipo di direzione di flusso predefinito D8.

Specifica posizioni del punto di versamento di input

Le posizioni del punto di versamento di input.

Per un raster, ciò rappresenta le celle oltre le quali verranno determinate l'area o il bacino pluviale di contribuzione. Tutte le celle che non sono NoData verranno utilizzate come celle di origine. Per un input di punto, ciò rappresenta le posizioni oltre le quali verranno determinate l'area o il bacino pluviale di contribuzione.

Scegli campo punto versamento

Il campo usato per assegnare valori alle posizioni del punto di versamento.

Se il layer di punti di versamento di input è un raster, viene utilizzato Valore. Se è una feature, viene utilizzato un campo numerico. Se il campo contiene valori a virgola mobile, verranno troncati in numeri interi.

Nome del layer dei risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.

È possibile specificare il nome di una cartella in I miei contenuti in cui salvare il risultato usando la casella a discesa Salva risultato in.
