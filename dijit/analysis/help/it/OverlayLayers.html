Sovrapponi layer

La sovrapposizione combina due layer in un unico layer. Si può pensare all'overlay come a un peering attraverso una pila di mappe e alla creazione di una singola mappa contenente tutte le informazioni della pila. La sovrapposizione è più di una fusione di linee; tutti gli attributi delle feature nella sovrapposizione sono portati al risultato finale. La sovrapposizione costituisce la risposta ad una delle più elementari domande di geografia, ovvero "Cosa sta sopra e cosa sta sotto?" Di seguito sono riportati degli esempi:

  • Quali lotti sono compresi nella pianura alluvionale da 100 anni?
  • Quali strade sono comprese in determinate regioni?
  • Quale uso del suolo si fa di un determinato tipo di terreno?
  • Quali pozzi si trovano in basi militari abbandonate?

Se l'opzione Usa estensione mappa corrente è selezionata, verranno sovrapposte solo le feature nel layer di input e di sovrapposizione visibili nell'estensione mappa corrente. Se è deselezionata, verranno sovrapposte tutte le feature sia nel layer di input che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Scegli il layer di input

Il layer di punti, linea o area che verrà sovrascritto.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Scegli layer di analisi in fondo all'elenco a discesa. Con questa opzione si apre una galleria in cui è possibile individuare e selezionare dataset di varie fonti da utilizzare nell'analisi.

Scegliere il metodo di sovrapposizione

Specifica il metodo di sovrapposizione che sarà usato per definire come il layer di input e il layer di sovrapposizione saranno combinati.

  • Intersezione: il risultato conterrà solo quelle feature o porzioni di feature nel layer di input che si sovrappongono alle feature nel layer di sovrapposizione. Quando si intersecano feature di linea o di area, è possibile specificare il tipo di caratteristiche per l'output.

    Punti e linee

    Punti con linee

    Linee con linee

    Linee con aree

    Aree con aree

  • Unione: il risultato conterrà feature sia del layer di input che di quello di sovrapposizione. Questa opzione è disponibile solo se sia il layer di input che il layer di sovrapposizione contengono feature di area.
  • Cancella: il risultato conterrà le feature o le parti di feature del layer di input che non risultano sovrapposte ad aree del layer di sovrapposizione. Questa opzione è disponibile quando le feature di input sono le stesse o un tipo di feature di ordine inferiore del layer di sovrapposizione.

Se l'opzione Usa estensione mappa corrente è selezionata, verranno sovrapposte solo le feature nel layer di input e di sovrapposizione visibili nell'estensione mappa corrente. Se è deselezionata, verranno sovrapposte tutte le feature sia nel layer di input che in quello di sovrapposizione, anche se esterne all'estensione mappa corrente.

Nome layer risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il tipo di feature, ovvero area, linea o punto, è uguale al layer di sovrapposizione. Il nome predefinito è basato sul metodo di sovrapposizione e sul nome del layer di input. Se si tratta di un layer esistente, ti viene richiesto di fornire un altro nome.

Utilizzare la casella a discesa Salva risultato in per specificare il nome di una cartella di I miei contenuti in cui salvare il risultato.
