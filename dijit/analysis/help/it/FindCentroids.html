Trova Centroidi

Questo strumento funziona con un layer di feature multipuntuale, lineare o areale per determinare il centro rappresentativo di ogni feature.

Se l'opzione Usa estensione mappa corrente è selezionata, verranno analizzate solo le feature nel layer di input multipuntuale, lineare o areale visibile nell’estensione mappa corrente. Se questa opzione è deselezionata, verranno analizzate tutte le feature del layer di input, anche se esterne all'estensione mappa corrente.

Scegliere il layer le cui feature saranno utilizzate per generare i centroidi

Il layer multipuntuale, lineare o areale che verrà utilizzato per trovare i centroidi.

Oltre a scegliere un layer dalla mappa, è possibile selezionare Scegli layer di analisi in fondo all'elenco a discesa. Con questa opzione si apre una galleria in cui è possibile individuare e selezionare dataset di varie fonti da utilizzare nell'analisi.

Mostrami posizioni di output

Quando viene selezionato per default nel vero centroide l’output restituirà la posizione reale del centroide per ogni feature di input. I centroidi saranno posizionati nella seguente maniera:

  • Feature multipuntuali—Il centroide di output sarà posizionato sulle coordinate medie di x ed y di tutti i punti della feature multipuntuale.
  • Feature lineari—Il centroide di output sarà posizionato sulle coordinate medie ponderate x ed y dei mezzi punti di tutti segmenti di linea della feature lineare. Il peso è basato sulla lunghezza del segmento.
  • Feature areale - Il centroide di output sarà posizionato nel centro dell’area.

Se l’opzione contenuti da feature di input è selezionata, le feature puntuali di output saranno contenute entro o dentro ogni feature di input. Le feature puntuali risultanti appariranno come segue:

  • Feature multipuntuali - Il centroide di output coinciderà con uno dei punti del gruppo multipuntuale.
  • Feature lineari—Il centroide di output sarà sulla linea. Se la linea è curva, l’output sarà al centro della linea.
  • Feature areali—Il punto di output sarà dentro l’area della feature.

Nome del layer dei risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il nome del layer esiste già, verrà richiesto di fornire un altro nome.

Utilizzare il menù a discesa Salva risultato in per specificare il nome di una cartella in I miei contenuti in cui salvare il risultato.
