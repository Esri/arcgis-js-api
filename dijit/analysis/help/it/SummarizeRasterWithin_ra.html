Riepiloga raster entro

Questo strumento riassume i valori di un raster all'interno delle zone di un altro set di dati.

Di seguito sono riportate alcune applicazioni di esempio:

  • Dati un layer di confini spartiacque e un layer di confini di utilizzo del suolo per tipo di utilizzo, è possibile calcolare la superficie totale in acri del tipo di utilizzo del suolo per ciascuno spartiacque.
  • Dati un layer di lotti di una regione ed un layer di confini di città, è possibile riepilogare il valore medio di lotti non occupati entro ciascun confine di città.

Se l'opzione Usa estensione mappa corrente è selezionata, vengono analizzate solo quelle feature nel layer di input e nel layer da riepilogare visibili all'interno dell'estensione mappa corrente. Se è deselezionata, vengono analizzate tutte le feature presenti nel layer di input e nel layer da riepilogare, anche se esterne all'estensione mappa corrente.

Scegli layer di area per riepilogare un layer raster entro confini definiti

Il layer che definisce i confini delle aree, o zone, che verranno riepilogati.

Il layer può essere un raster o dati feature. Per i raster, le zone sono definite da tutte le posizioni nell'input aventi lo stesso valore di cella. Le aree non devono essere contigue.

Scegli campo per definire i confini

Questo è l'attributo del layer che verrà utilizzato per definire i confini delle aree.

Si supponga, ad esempio, che il primo layer di input definisca i confini dell'unità di gestione, con attributi che definiscono la regione, il distretto e l'ID particella di ciascuna unità. Si dispone anche di un layer raster che definisce un indice di biodiversità per ciascuna posizione. Con il campo selezionato, è possibile decidere di calcolare la biodiversità media a livello locale, di distretto o di regione.

Scegli layer raster per il riepilogo

Le celle raster in questo layer saranno riepilogate in base alle aree (zone) comprese.

Scegli statistiche da calcolare

Dopo aver scelto un layer da riepilogare, è possibile, facoltativamente, calcolare statistiche di qualsiasi attributo numerico dei punti, delle linee o delle aree che si trovano all'interno del layer di area di input.

I tipi di statistiche disponibili quando il campo selezionato è un intero sono Media, Maggioranza, Massimo, Mediana, Minimo, Minoranza, Percentile, Intervallo, Deviazione standard, Somma e Varietà.

Se il campo è a virgola mobile, le opzioni sono Media, Massimo, Minimo, Intervallo, Deviazione standard e Somma.

Valore percentile

Il percentile da calcolare. Il valore predefinito è 90 , che indica il 90esimo percentile.

I valori spaziano da 0 a 100. Il percentile numero 0 è essenzialmente l'equivalente della statistica minima e il percentile numero 100 è l'equivalente della statistica massima. Un valore pari a 50 produrrà essenzialmente lo stesso risultato di una statistica mediana.

Questa opzione è supportata soltanto se il parametro Scegli statistiche da calcolare è impostato su Percentile.

Ignora valori mancanti nei calcoli

Se si sceglie di ignorare i valori mancanti, solo le celle che hanno un valore nel layer da riepilogare verranno utilizzate per determinare il valore di output per tale area. In caso contrario, se mancano dei valori in un punto qualunque di un'area, le informazioni non saranno sufficienti per eseguire calcoli statistici per tutte le celle in quella zona e tale area riceverà un valore null (NoData) nell'output.

Elabora come multidimensionale

Specifica il modo in cui vengono eleborati i raster di input se sono multidimensionali. Se scegli di elaborarli come multidimensionali, si calcoleranno le statistiche per tutte le dimensioni del dataset multidimensionale di input. Altrimenti, si calcoleranno le statistiche della sezione corrente del dataset multidimensionale di input, come da impostazioni predefinite.

Nome layer risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il layer esiste già, verrà chiesto di fornire un altro nome.

È possibile specificare il nome di una cartella in I miei contenuti in cui salvare il risultato usando la casella a discesa Salva risultato in.
