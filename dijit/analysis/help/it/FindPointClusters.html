Trova cluster punti

Questo strumento trova i cluster di feature punto entro il disturbo circostante in base alla loro distribuzione spaziale.

Se l'estensione Usa mappa corrente è selezionata, solo le feature nel layer di punti di input all'interno dell'estensione mappa corrente verranno analizzate. Se questa opzione è deselezionata, verranno analizzate tutte le feature del layer di punti di input, anche se esterne all'estensione mappa corrente.

Scegliere il layer per il quale verranno trovati i cluster

Il layer puntuale in cui si trovano i cluster.

Numero minimo di punti da considerare cluster

Il numero minimo di feature da considerare cluster. Tutti i cluster con meno feature di quelli forniti verranno considerati disturbo.

Limita l'intervallo di ricerca a (opzionale)

La distanza minima per considerare le feature come parte dello stesso cluster. Se questo campo viene lasciato in bianco, lo strumento eseguirà HDBSCAN, che utilizza distanze variabili per separare cluster di varie densità da disturbi sparsi. Se viene fornito un valore per questo campo, lo strumento eseguirà DBSCAN, che utilizza una distanza specifica per separare cluster densi da disturbi sparsi.

Nome del layer dei risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il nome del layer esiste già, verrà richiesto di fornire un altro nome.

Utilizzare il menù a discesa Salva risultato in per specificare il nome di una cartella in I miei contenuti in cui salvare il risultato.
