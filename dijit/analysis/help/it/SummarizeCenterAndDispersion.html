Riepiloga centro e dispersione

Questo strumento trova feature centrali e distribuzioni direzionali. Può essere utilizzato per rispondere a domande quali:

  • Dove è il centro?
  • Quale feature è più accessibile da parte di tutte le altre feature?
  • Como sono disperse, compatte o integrate le feature?
  • Vi sono tendenze direzionali?

Se l’opzione Usa estensione mappa corrente è selezionata, verranno analizzati solo dati alle feature nel layer di input visibili all’interno dell’estensione mappa corrente. Se questa opzione è deselezionata, verranno analizzate tutte le feature del layer di input, anche se esterne all'estensione mappa corrente.

Scegli un layer da descrivere

Il layer puntuale, lineare o areale verrà utilizzato per trovare feature centrali o distribuzioni direzionali.

Tipi riepilogo

Il metodo per riepilogare centro e dispersione.

  • Feature centrale: identifica la feature localizzata più centralmente in un dataset.
  • Centro medio: identifica il centro geografico (o il centro della concentrazione) per un set di feature.
  • Centro mediano: identifica la posizione che riduce al minimo la distanza euclidea complessiva per le feature in un set di dati.
  • Ellisse: crea ellissi standard di deviazione per riepilogare le caratteristiche spaziali delle feature geografiche: tendenza centrale, dispersione e tendenze direzionali. Le ellissi possono assumere il formato di 1, 2 o 3 deviazioni standard.

Peso per (opzionale)

Il campo utilizzato per ponderare le posizioni in base alla loro importanza relativa.

Raggruppa per (opzionale)

Il campo utilizzato per raggruppare feature per calcoli separati di feature centrali o dispersione.

Nome del layer dei risultati

Il nome del layer che verrà creato in I miei contenuti e aggiunto alla mappa. Il nome predefinito è basato sul nome dello strumento e sul nome del layer di input. Se il nome del layer esiste già, verrà richiesto di fornire un altro nome.

Utilizzare il menù a discesa Salva risultato in per specificare il nome di una cartella in I miei contenuti in cui salvare il risultato.
