Vegetation überwachen

Dieses Werkzeug führt eine arithmetische Operation für die Bänder eines Multiband-Raster-Layers durch, um die Informationen zur Vegetationsabdeckung des Einzugsgebiets aufzudecken.

Die Eingabedaten auswählen

Wählen Sie den Multiband-Raster-Layer aus. Stellen Sie sicher, dass das Eingabe-Raster über die entsprechenden Bänder verfügt.

Methode zum Überwachen der Vegetation auswählen

Wählen Sie die Methode zum Berechnen des Vegetationsindex-Layers aus. Mit den unterschiedlichen Vegetationsindizes können bestimmte Features hervorgehoben oder verschiedene Arten von Rauschen reduziert werden.

  • Global Environmental Monitoring Index — GEMI ist ein nicht linearer Vegetationsindex für globale Umgebungsüberwachung aus Satellitenbildern. Er ähnelt dem NDVI, ist aber weniger empfindlich gegen atmosphärische Störungen. Er wird von nacktem Boden beeinflusst. Daher wird er nicht zur Verwendung in Bereichen mit wenig oder nicht allzu dichter Vegetation empfohlen.
  • Green Vegetation Index - Landsat TM — Der GVI wurde ursprünglich aus Landsat-MSS-Bilddaten entworfen, wurde aber für die Verwendung mit Landsat TM-Bilddaten geändert. Er wird auch als Landsat TM Tasseled Cap Green Vegetation Index bezeichnet. Dieser Überwachungsindex kann auch mit Bilddaten verwendet werden, deren Bänder die gleichen Spektraleigenschaften aufweisen.
  • Modified Soil Adjusted Vegetation Index — MSAVI2 ist ein Vegetationsindex, der versucht, Helligkeitseinflüsse des nackten Erdbodens der SAVI-Methode zu minimieren.
  • Normalized Difference Vegetation Index — NDVI ist ein standardisierter Index, der Ihnen das Erstellen eines Bildes mit Grünanteilen (relative Biomasse) ermöglicht. Dieser Index nutzt den Kontrast der Eigenschaften zweier Bänder aus einem multispektralen Raster-Dataset: die Absorption durch die Chlorophyllpigmente im roten Band und den Pflanzen-Reflexionsgrad im nahinfraroten Band (NIR).
  • Perpendicular Vegetation Index — Der PVI ist einem Difference Vegetation Index ähnlich; er ist jedoch gegen atmosphärische Variationen empfindlich. Wenn Sie diese Methode verwenden, um verschiedene Bilder miteinander zu vergleichen, sollten Sie sie nur für Bilder verwenden, die atmosphärisch korrigiert wurden. Diese Informationen können von Ihrem Datenanbieter bereitgestellt werden.
  • Soil-Adjusted Vegetation Index — SAVI ist ein Vegetationsindex, der versucht, Einflüsse durch die Bodenhelligkeit mit einem Bodenhelligkeits-Korrekturfaktor zu minimieren. Dies wird oft in trockenen Regionen verwendet, wo die vegetative Bodendeckung niedrig ist.
  • Sultan's Formula — Bei der Sultan-Formel wird ein bestimmter Algorithmus auf ein Sechsband-8-Bit-Bild angewendet, um ein Dreiband-8-Bit-Bild zu erzeugen. Das sich ergebende Bild hebt Felsformationen, so genannte Ophiolite, an Küstenlinien hervor. Diese Formel wurde auf Grundlage der TM- und ETM-Bänder einer Landsat 5- oder Landsat 7-Szene entworfen.
  • Transformed Soil-Adjusted Vegetation Index — Der Transformed-SAVI ist ein Vegetationsindex, der versucht, Bodenhelligkeitseinflüsse zu minimieren, indem angenommen wird, dass die Bodenlinie eine beliebige Neigung und einen beliebigen Schnittpunkt hat.
  • Visible Atmospherically Resistant Index — Der VARI ist ein Vegetationsindex, mit dem der Vegetationsanteil ausschließlich anhand von Informationen des sichtbaren Spektrums geschätzt wird. Für diese Methode sind Bänder im roten, grünen und blauen Bereich des Spektrums erforderlich.

Indizes für das NIR- und rote Band angeben

Geben Sie die Bandindizes für das NIR- und das rote Band an.

Multiband-Satelliten- und Luftbildsensoren erfassen Informationen, die in Wellenlängenbänder unterteilt werden, welche wiederum durch Bandindizes identifiziert werden. Für diese Methode der Vegetationsüberwachung müssen Sie die Bandindizes für das Nahinfrarot-Wellenlängenband (NIR) und das rote Wellenlängenband angeben.

Indizes für das rote, grüne und blaue Band angeben

Geben Sie die Bandindizes für das rote, grüne und blaue Band an.

Multiband-Satelliten- und Luftbildsensoren erfassen Informationen, die in Wellenlängenbänder unterteilt werden, welche wiederum durch Bandindizes identifiziert werden. Für den Visible Atmospherically Resistant Index müssen Sie die Bandindizes für das rote, grüne und blaue Wellenlängenband angeben.

Indizes für das blaue, rote, NIR-, SWIR-1- und Wärmeinfrarot-Band angeben

Geben Sie die Bandindizes für das blaue Band, das rote Band, das Nahinfrarot-Band (NIR), das Kurzwelleninfrarot-Band (SWIR) und das Wärmeinfrarot-Band an.

Multiband-Satelliten- und Luftbildsensoren erfassen Informationen, die in Wellenlängenbänder unterteilt werden, welche wiederum durch Bandindizes identifiziert werden. Für den Index nach der Sultan-Formel müssen Sie die Bandindizes für das blaue Band, das rote Band, das Nahinfrarot-Band (NIR), das Kurzwelleninfrarot-Band (SWIR) und das Wärmeinfrarot-Band angeben.

Indizes für das blaue, grüne, rote, NIR-, SWIR-1- und SWIR-2-Band angeben

Geben Sie die Bandindizes für das blaue, grüne, rote, NIR-, SWIR-1- und SWIR-2-Band an.

Multiband-Satelliten- und Luftbildsensoren erfassen Informationen, die in Wellenlängenbänder unterteilt werden, welche wiederum durch Bandindizes identifiziert werden. Für den Green Vegetation Index müssen Sie die Bandindizes für das blaue Band, das grüne Band, das rote Band, das Nahinfrarot-Band (NIR), das Kurzwelleninfrarot-1-Band (SWIR-1) und das Kurzwelleninfrarot-2-Band (SWIR-2) angeben.

Neigung der Bodenlinie

Die Neigung der Bodenlinie. Die Neigung ist die ungefähre lineare Beziehung zwischen NIR- und Rot-Bändern in einem Scatterplot.

Dieser Parameter ist nur für die Methode "Transformed Soil-Adjusted Vegetation Index" gültig.

Intercept

Dies ist der NIR-Wert, wenn der Reflexionswert des roten (Rot) Bandes für die speziellen Bodenlinien 0 ist.

Dieser Parameter ist nur für die Methode "Transformed Soil-Adjusted Vegetation Index" gültig.

Grüne vegetative Bodenbedeckung

Legt den Anteil an grüner vegetativer Bodenbedeckung fest, der für den Soil-Adjusted Vegetation Index erforderlich ist.

Zulässige Werte:

  • 1 = Flächen ohne grüne Vegetations-Bodendeckung
  • 0,5 = Flächen mit mäßiger grüner Vegetations-Bodendeckung
  • 0 = Flächen mit hoher grüner Vegetations-Bodendeckung

Anpassungsfaktor

Gibt den Anpassungsfaktor zur Minimierung der Auswirkungen des Bodens an, der für die Methode "Transformed Soil-Adjusted Vegetation Index" erforderlich ist. Der Standardwert ist 0,08.

Ein niedriger Wert bedeutet, dass die Auswirkungen des Bodens ignoriert werden. Ein hoher Wert bedeutet, dass das Ergebnis durch die Bodenbeschaffenheit beeinflusst wird.

Name des Ergebnis-Layers

Der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, werden Sie aufgefordert, einen anderen Namen einzugeben.

Sie können den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis über das Dropdown-Feld Ergebnis speichern in gespeichert wird. Wenn Sie über Berechtigungen zum Erstellen gekachelter und dynamischer Bilddaten-Layer verfügen, können Sie angeben, welcher Layer-Typ in der Ausgabe generiert werden soll, indem Sie das Dropdown-Feld "Ergebnis speichern als" verwenden.
