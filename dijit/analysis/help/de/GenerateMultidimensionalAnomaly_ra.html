Multidimensionale Abweichung generieren

Berechnet die Abweichung für jeden Ausschnitt in einem vorhandenen multidimensionalen Raster, um ein neues multidimensionales Raster zu generieren.

Als Abweichung wird die Differenz zwischen einer Beobachtung und dem zugehörigen Standard-, Mittel- oder Medianwert bezeichnet.

Mit diesem Werkzeug werden die Abweichungen im zeitlichen Verlauf für eine oder mehrere Variablen in einem multidimensionalen Raster berechnet. Wenn abgesehen von der Zeitdimension auch eine nicht zeitliche Dimension vorhanden ist, wird die Abweichung an jedem Schritt in der zusätzlichen Dimension berechnet.

Beispiel: Ihnen liegen monatliche Meerestemperaturdaten vor, wobei die Temperatur für jeden Meter Tiefe bis zu einer Meerestiefe von 100 Metern erfasst wurde. Sie möchten damit die Temperaturabweichungen als Differenz vom jährlichen Mittelwert berechnen. Mit diesem Werkzeug werden die Temperaturabweichungen basierend auf einem jährlichen Mittelwert berechnet, wenn Sie für das Zeitintervall zum Berechnen des Mittelwerts die Option Jährlich festlegen. Daraufhin werden die Abweichungswerte für jeden Meter bis zu einer Tiefe von 100 Metern zurückgegeben.

Dieses Werkzeug unterstützt nur multidimensionale Raster-Datasets mit Zeitdimension.

Wenn die Option Aktuelle Kartenausdehnung verwenden aktiviert ist, werden die Variablen des Layers analysiert, die in der aktuellen Kartenausdehnung sichtbar sind. Wenn die Option deaktiviert ist, werden die Variablen des gesamten Layers analysiert, auch diejenigen, die sich außerhalb der aktuellen Kartenausdehnung befinden.

Methode zum Ermitteln von Abweichungen ausw채hlen

Gibt die Methode an, die zum Berechnen der Abweichung verwendet wird.

  • Differenz vom Mittelwert: Die Differenz zwischen einem Pixelwert und dem Mittelwert des Wertes dieses Pixels in den vom Intervall definierten Abschnitten wird berechnet. Dies ist die Standardeinstellung.
  • Prozentuale Differenz vom Mittelwert: Die prozentuale Differenz zwischen einem Pixelwert und dem Mittelwert des Wertes dieses Pixels in den vom Intervall definierten Abschnitten wird berechnet.
  • Prozentsatz vom Mittelwert: Der Prozentsatz des Mittelwertes wird berechnet.
  • Z-Wert: Der Z-Wert jedes Pixels wird berechnet. Ein Z-Wert von 0 bedeutet, dass der Pixelwert mit dem Mittelwert identisch ist. Ein Z-Wert von 1 bedeutet, dass der Pixelwert um 1 Standardabweichung vom Mittelwert abweicht. Ein Z-Wert von 2 bedeutet, dass der Pixelwert um 2 Standardabweichungen vom Mittelwert abweicht usw.
  • Differenz vom Medianwert: Die Differenz zwischen einem Pixelwert und dem mathematischen Medianwert der Werte dieses Pixels in den vom Intervall definierten Abschnitten wird berechnet.
  • Prozentuale Differenz vom Medianwert: Die prozentuale Differenz zwischen einem Pixelwert und dem mathematischen Medianwert der Werte dieses Pixels in den vom Intervall definierten Abschnitten wird berechnet.
  • Prozentsatz vom Medianwert: Der Prozentsatz des mathematischen Medianwertes wird berechnet.

Für die Berechnung von Abweichungswerten sind die folgenden mathematischen Methoden verfügbar. Abweichungen können berechnet werden, indem entweder der Mittelwert oder der Medianwert als Durchschnittswert definiert wird. Ist die Datenverteilung verzerrt, wird der Mittelwert durch Ausreißer stark beeinflusst. Daher ist der Medianwert oder die Z-Wert-Methode für diese Art Daten möglicherweise besser geeignet.

  • Differenz vom Mittelwert = x - µ
    • x = Pixelwert in einem Abschnitt
    • µ = Mittelwert der Werte dieses Pixels über das gegebene Zeitintervall
  • Prozentuale Differenz vom Mittelwert = |x - µ| / [(x + µ)/2]
    • x = Pixelwert in einem Abschnitt
    • µ = Mittelwert der Werte dieses Pixels über das gegebene Zeitintervall
    • |x - µ| = absoluter Wert der Differenz zwischen dem Wert und dem Mittelwert
  • Prozentsatz vom Mittelwert = x / µ
    • x = Pixelwert in einem Abschnitt
    • µ = Mittelwert der Werte dieses Pixels über das gegebene Zeitintervall
  • Differenz vom Medianwert = x - ß
    • x = Pixelwert in einem Abschnitt
    • ß = Medianwert der Werte dieses Pixels über das gegebene Zeitintervall
  • Prozentuale Differenz vom Medianwert = |x - ß| / [(x + ß)/2]
    • x = Pixelwert in einem Abschnitt
    • ß = Medianwert der Werte dieses Pixels über das gegebene Zeitintervall
    • |x - ß| = absoluter Wert der Differenz zwischen dem Wert und dem Medianwert
  • Prozentsatz vom Medianwert = x / ß
    • x = Pixelwert in einem Abschnitt
    • ß = Medianwert der Werte dieses Pixels über das gegebene Zeitintervall
  • Z-Wert = (x - µ) / S
    • x = Pixelwert in einem Abschnitt
    • µ = Mittelwert der Werte dieses Pixels über das gegebene Zeitintervall
    • S = Standardabweichung der Werte dieses Pixels über das gegebene Zeitintervall

Zeitintervall zum Berechnen des Mittelwerts ausw채hlen

Gibt das Zeitintervall an, das zum Berechnen des Mittelwertes verwendet wird.

  • Alle: Berechnet den Mittelwert in allen Abschnitten für jedes Pixel.
  • Jährlich: Berechnet den jährlichen Mittelwert für jedes Pixel.
  • Monatlich wiederkehrend: Berechnet den monatlichen Mittelwert für jedes Pixel.
  • Wöchentlich wiederkehrend: Berechnet den wöchentlichen Mittelwert für jedes Pixel.
  • Täglich wiederkehrend: Berechnet den täglichen Mittelwert für jedes Pixel.
  • Stündlich: Berechnet den stündlichen Mittelwert für jedes Pixel.
  • Externes Raster: Ein vorhandenes Raster-Dataset wird referenziert, das den Mittel- oder Medianwert für jedes Pixel enthält.

Bilddaten-Layer des Mittelwertes als Referenz ausw채hlen

Gibt das Referenz-Raster-Dataset an, das einen zuvor berechneten Mittelwert für jedes Pixel enthält. Die Abweichungen werden durch den Vergleich mit diesem Mittelwert berechnet.

Fehlende Werte in Berechnung ignorieren

Gibt an, ob fehlende Werte bei der Analyse ignoriert werden.

  • Aktiviert: Bei der Analyse werden alle gültigen Pixel entlang einer angegebenen Dimension einbezogen und alle NoData-Pixel ignoriert. Dies ist die Standardeinstellung.
  • Deaktiviert: Wenn NoData-Werte für die Pixel entlang einer angegebenen Dimension vorhanden sind, wird bei der Analyse "NoData" zurückgegeben.
