Raster extrahieren

Mit diesem Werkzeug kann ein Raster auf eine Grenze zugeschnitten werden. Dies kann eine rechteckige Fläche sein oder eine Form, die Sie interaktiv auf dem Bildschirm definieren. Sie können das Raster auf die Ausdehnung der Fläche ausschneiden, die aktuell auf der Karte angezeigt wird, oder auf das Untersuchungsgebiet, das durch ein Polygon definiert wird.

Zu extrahierende Eingabedaten auswählen

Der auszuschneidende Eingabe-Layer.

Ausschneidegeometrie festlegen

Wählen Sie den Ausdehnungstyp aus, der beim Ausschneiden der Geometrie des Layers verwendet werden soll:

  • Aktuelle Ausdehnung verwenden: Die aktuelle Kartenausdehnung dient zum Ausschneiden des resultierenden Layers. Dies ist die Standardeinstellung.
  • Untersuchungsgebiet auswählen: Ermöglicht das Definieren einer Geometrie zum Ausschneiden mit dem Zeichenwerkzeug.
  • Layer auswählen: Wählen Sie einen Feature- oder Bilddaten-Layer aus, um die Ausdehnung zu definieren, oder wählen Sie einen Polygon-Feature-Layer aus, um die Geometrie zum Ausschneiden zu definieren. Wenn ein Polygon-Feature-Layer ausgewählt wird, wird ein Kontrollkästchen angezeigt, mit dem angegeben werden kann, ob Eingabe-Features als Geometrie zum Ausschneiden verwendet werden sollen. Diese Option ist standardmäßig deaktiviert. Zum Definieren der Ausdehnung wird in diesem Fall der Eingabe-Feature-Layer verwendet. Ist das Kontrollkästchen aktiviert, werden die Eingabe-Features als Geometrie zum Ausschneiden verwendet.

Ausschneidetyp festlegen

Geben Sie an, ob Sie die Fläche innerhalb oder außerhalb der Ausschneidegeometrie beibehalten möchten:

  • Daten innerhalb der Geometrie: Bilddaten innerhalb der Ausschneideausdehnung werden extrahiert. Dies ist die Standardeinstellung.
  • Daten außerhalb der Geometrie: Bilddaten außerhalb der Ausschneideausdehnung werden extrahiert.

Name des Ergebnis-Layers

Der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, werden Sie aufgefordert, einen anderen Namen einzugeben.

Sie können den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis über das Dropdown-Feld Ergebnis speichern in gespeichert wird. Wenn Sie über Berechtigungen zum Erstellen gekachelter und dynamischer Bilddaten-Layer verfügen, können Sie angeben, welcher Layer-Typ in der Ausgabe generiert werden soll, indem Sie das Dropdown-Feld "Ergebnis speichern als" verwenden.
