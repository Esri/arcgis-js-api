Abflussgebiet

Dieses Werkzeug ermittelt die relevante Fläche über einer Gruppe von Zellen in einem Raster.

Ein Eingabepunkt am Fluss Mississippi in Louisiana erstellt ein Abflussgebiet-Feature, das die Gebiete von New York bis Montana und bis nach Kanada einschließt.

Fließrichtungs-Layer auswählen

Das Eingabe-Raster, das die Fließrichtung aus jeder Zelle anzeigt.

Das Fließrichtungs-Raster kann mit der Raster-Funktion "Fließrichtungs-Raster" und dem standardmäßigen Fließrichtungstyp D8 erstellt werden.

Abflusspunktpositionen der Eingabe angeben

Die Abflusspunktpositionen der Eingabe.

Hierbei kann es sich um ein Raster oder ein Punkt-Feature handeln. Bei einem Raster stellt dies Zellen dar, über denen die relevante Fläche bzw. das Reservoir, ermittelt wird. Alle Zellen, die keine NoData-Zellen sind, werden als Quellenzellen verwendet. Bei einer Punkteingabe stellt dies Positionen dar, über denen die relevante Fläche, also das Einzugsgebiet, ermittelt wird.

Abflusspunkt-Feld auswählen

Das Feld für die Zuweisung von Werten zu den Abflusspunktpositionen.

Wenn der Eingabe-Layer für den Abflusspunkt ein Raster ist, wird Value verwendet. Handelt es sich um ein Feature, wird ein numerisches Feld verwendet. Wenn das Feld Gleitkommawerte enthält, werden sie in Ganzzahlwerte gekürzt.

Name des Ergebnis-Layers

Der Name des Layers, der in Eigene Inhalte erstellt und der Karte hinzugefügt wird. Der Standardname basiert auf dem Werkzeugnamen und dem Namen des Eingabe-Layers. Wenn der Layer bereits vorhanden ist, werden Sie aufgefordert, einen anderen Namen einzugeben.

Sie können den Namen eines Ordners in Eigene Inhalte angeben, in dem das Ergebnis über das Dropdown-Feld Ergebnis speichern in gespeichert wird. Wenn Sie über Berechtigungen zum Erstellen gekachelter und dynamischer Bilddaten-Layer verfügen, können Sie angeben, welcher Layer-Typ in der Ausgabe generiert werden soll, indem Sie das Dropdown-Feld "Ergebnis speichern als" verwenden.
