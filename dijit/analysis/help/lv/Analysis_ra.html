Deep Learning

Šie rīki tiek izmantoti, lai noteiktu vai klasificētu konkrētus attēla elementus vai klasificētu pikseļus rastra datu kopā. Deep Learning ir mākslīgā intelekta mašīnmācību metode, kas nosaka elementus attēlos, izmantojot vairākus slāņus neironu tīklos, kur katrs slānis spēj izgūt vienu vai vairākus unikālus attēla elementus. Šie rīki izmanto modeļus, kas ir trenēti noteikt īpašas iezīmes trešās puses Deep Learning struktūrās, piemēram, TensorFlow, CNTK un Keras—un izvades elementus vai klases kartes.

  • Klasificēt pikseļus, izmantojot Deep Learning, veido klasificētu rastru, izmantojot Deep Learning modeli.
  • Noteikt objektus, izmantojot Deep Learning izveido elementu klasi, kas satur objektus, kurus funkcija atrod ievades rastrā, izmantojot apmācītu Deep Learning modeli.
  • Klasificēt objektus, izmantojot Deep Learning izveido elementu klasi vai tabulu, kurā katram ievades rastra ievades objektam tiek piešķirts klases apzīmējums, izmantojot apmācītu Deep Learning modeli.

Daudzdimensiju analīze

Daudzdimensiju analīzes rīku komplektā pieejamie rīki jums ļauj analizēt zinātniskos datus dažādu mainīgo un dimensiju ietvaros.

Daudzdimensiju dati ir dažādos laika periodos, dziļumos un augstumos ievākti dati. Šī veida datus parasti izmanto atmosfēras, okeanogrāfijas un zemes zinātnē. Šajā rīku komplektā jūsu varat analizēt daudzdimensiju rastra datus dažādos formātos, tostarp netCDF, HDF, GRIB, daudzdimensiju mozaīkas datu kopā un Esri Cloud Raster Format (CRF).

Tālāk redzamajā tabulā ir norādīti daudzdimensiju analīzes rīki un īss katra rīka aprasts.

  • Rīks Apkopot daudzdimensiju rastru ģenerē daudzdimensiju rastra datu kopu, apkopojot esošā daudzdimensiju rastra mainīgos dimensijas ietvaros.
  • Rīks Ģenerēt daudzdimensiju anomāliju aprēķina katras sadaļas anomāliju daudzdimensiju rastrā, lai ģenerētu daudzdimensiju rastru.
  • Rīks Ģenerēt tendences rastru izvērtē katra pikseļa tendenci dimensijā vienam vai vairākiem multidimensiju rastra mainīgajiem.
  • Rīks Prognozēt, izmantojot tendenču rastru aprēķina prognozēto multidimensiju rastru, izmantojot rīka Ģenerēt tendenču rastru izvades tendenču rastru.
  • Rīks Atrast argumenta statistiku izgūst dimensijas vērtību jeb joslas indeksu, pie kura katram pikselim multidimensiju vai vairāku joslu rastrā tiek panākta konkrēta statistika.
