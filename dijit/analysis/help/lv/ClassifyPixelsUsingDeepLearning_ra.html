Funkcija Klasificēt pikseļus, izmantojot Deep Learning

Šis rīks izmanto apmācītu Deep Learning modeli uz ievades rastra, lai izgatavotu klasificētu rastru, un katram derīgajam pikselim ir piešķirta klases atzīme.

Ja tiek atzīmēta opcija Lietot pašreizējo kartes pārklājumu, tiks analizēts tikai rastra apgabals, kas ir redzams pašreizējā kartes pārklājumā. Ja tā netiek atzīmēta, viss rastrs tiks analizēts, pat ja tas atrodas ārpus pašreizējā kartes pārklājuma.

Izvēlieties pikseļu klasificēšanai izmantoto attēlu

Ievades attēls, kas jāklasificē.

Tas var būt attēla servisa URL, rastra slānis vai attēlu servisa slānis.

Izvēlieties Deep Learning modeli pikseļu klasificēšanai

Ievades Deep Learning pakotnes ( .dlpk) vienība.

Deep Learning pakotni veido Esri modeļa definīcijas JSON fails ( .emd), Deep Learning binārā modeļa fails, un pēc izvēles izmantojamā Python rastra funkcija.

Norādiet Deep Learning modeļa argumentus

Funkciju argumenti tiek definēti Python raster funkciju klasē, uz kuru atsaucas ievades modelis. Šeit jūs uzskaitāt papildu Deep Learning parametrus un argumentus eksperimentiem un pilnveidošanai, piemēram, ticamības slieksni jutīguma korekcijai.

Argumentu nosaukumus aizpilda Python moduļa lasīšanas rīks rastra analīzes serverī.

Apstrādes režīms

Norāda, kā tiks apstrādātas visas rastra vienības mozaīkas datu kopā vai attēlu servisā. Šis parametrs tiek lietots, ja ievades rastrs ir mozaīkas datu kopa vai attēlu serviss.

  • PROCESS_AS_MOSAICKED_IMAGE — Visi mozaīkas datu kopas vai attēlu servisa rastra vienības tiks savienotas un apstrādātas. Tā ir noklusējuma vērtība.
  • PROCESS_ITEMS_SEPARATELY — Visas rastra vienības mozaīkas datu kopā vai attēlu servisā tiks apstrādātas kā atsevišķi attēli.

Rezultātu slāņa nosaukums

Slāņa nosaukums, kas tiks izveidots sadaļā Mans saturs un pievienots kartei. Noklusējuma nosaukums ir balstīts uz rīka nosaukumu un ievades slāņa nosaukumu. Ja šāds slānis jau pastāv, jums tikts lūgts norādīt citu nosaukumu.

Mapes nosaukumu var norādīt mapē Mans saturs, kur rezultāts tiks saglabāts, izmantojot nolaižamo lodziņu Saglabāt rezultātu. Ja jums ir privilēģijas izveidot gan mozaīkas, gan dinamiskus attēlu slāņus, varat arī norādīt, kuru slāņa veidu ģenerēt izvadē, izmantojot nolaižamo lodziņu Saglabāt rezultātu kā.
