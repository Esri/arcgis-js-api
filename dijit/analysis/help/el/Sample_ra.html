Δείγμα

Το εργαλείο αυτό δημιουργεί έναν πίνακα ή ένα feature class σημείων όπου φαίνονται οι τιμές των κελιών από ένα raster ή ένα σύνολο raster για καθορισμένες τοποθεσίες. Οι τοποθεσίες καθορίζονται από κελιά raster, πολυγωνικά στοιχεία, πολυγραμμικά στοιχεία ή ένα σύνολο σημείων.

Οι τιμές των pixel θα εξαχθούν από όλα τα raster εισόδου σε κάθε τοποθεσία. Θα δημιουργηθεί ένας πίνακας ή ένα feature class σημείων με πεδία που θα περιέχουν τις τιμές των pixel για κάθε raster εισόδου. Στον πίνακα εξόδου δεν θα συμπεριληφθούν πρόσθετα γνωρίσματα από τον πίνακα του raster εισόδου.

Κάθε συνδυασμός raster, π.χ. μονής φασματικής ζώνης, πολλών φασματικών ζωνών, πολυδιάστατων δεδομένων ή πολλών μεταβλητών μπορεί να καθοριστεί ως raster εισόδου. Η δομή του πίνακα εξόδου αλλάζει όταν τα raster είναι πολυδιάστατων δεδομένων. Όποτε ένα raster πολλών φασματικών ζωνών καθορίζεται ως ένα από τα raster εισόδου, θα χρησιμοποιούνται όλες οι φασματικές ζώνες στο εν λόγω raster εισόδου. Για να επεξεργαστείτε ορισμένες επιλεγμένες φασματικές ζώνες από ένα εισαγόμενο raster πολλών φασματικών ζωνών, δημιουργήστε πρώτα ένα νέο raster dataset που θα απαρτίζεται από τις συγκεκριμένες φασματικές ζώνες. Χρησιμοποιήστε το αποτέλεσμα στη λίστα των raster εισόδου.

Η τοποθεσία εισόδου μπορεί να είναι raster ή σημειακό στοιχείο:

  • Raster—Όσα pixel περιέχουν έγκυρες τιμές (όχι NoData) χρησιμοποιούνται για την εξαγωγή των τιμών των pixel από όλα τα raster εισόδου και το κέντρο του pixel χρησιμοποιείται για τις τοποθεσίες σημείων.
  • Σημείο—Θα γίνει δειγματοληψία των τιμών στην τοποθεσία κάθε σημείου.
  • Πολυγραμμή ή πολύγωνο—Θα γίνει δειγματοληψία των τιμών στο κεντροειδές του πολυγώνου ή της πολυγραμμής. Αν το raster εισόδου είναι πολυδιάστατων δεδομένων, οι τιμές θα συναθροιστούν με βάση τον τύπο που έχετε καθορίσει για την επιλογή Στατιστικά στοιχεία.

Η δειγματοληψία στα raster εισόδου γίνεται με την εγγενή τους χωρική αναφορά και ανάλυση. Αν υπάρχουν πολλαπλά raster εισόδου με διαφορετικές χωρικές αναφορές, οι τοποθεσίες εισόδου προβάλλονται στη χωρική αναφορά κάθε μεμονωμένου raster από το οποίο εξάγονται οι τιμές. Το περιβάλλον ανάλυσης εφαρμόζεται στις τοποθεσίες εισόδου, αλλά τα raster εισόδου δεν θα τηρήσουν το περιβάλλον ανάλυσης.

Στις τοποθεσίες που εξάγουν τιμές από κελιά με τιμή NoData στο raster εισόδου θα δοθεί τιμή <null> στον πίνακα εξόδου.

Για να εκτελέσετε ανάλυση με στοιχεία πολλαπλών σημείων, πρέπει πρώτα να τα μετατρέψετε σε στοιχεία ενός σημείου προτού τα χρησιμοποιήσετε στο συγκεκριμένο εργαλείο.

Η χωρική αναφορά των συντεταγμένων x,y στο δείγμα πίνακα εξόδου είναι η ίδια όπως στο raster τοποθεσίας εισόδου ή στα σημειακά στοιχεία, εκτός αν υπάρχει καθορισμένη χωρική αναφορά στο περιβάλλον της παραμέτρου Σύστημα συντεταγμένων εξόδου. Αν η χωρική αναφορά των δεδομένων της τοποθεσίας εισόδου είναι άγνωστη, η χωρική αναφορά των συντεταγμένων x,y είναι επίσης άγνωστη.

Η χωρική αναφορά του feature class εξόδου είναι η ίδια όπως στο raster της τοποθεσίας εισόδου ή στα σημειακά στοιχεία, εκτός αν υπάρχει καθορισμένη χωρική αναφορά στο περιβάλλον της παραμέτρου Σύστημα συντεταγμένων εξόδου.

Το περιβάλλον της παραμέτρου Παράγοντας παράλληλης επεξεργασίας υποστηρίζεται μόνο όποτε το raster εισόδου είναι πολυδιάστατων δεδομένων.

Αν είναι επιλεγμένη η Χρήση τρέχουσας έκτασης χάρτη, θα αναλυθούν οι μεταβλητές ή οι φασματικές ζώνες του θεματικού επιπέδου που είναι ορατές στην τρέχουσα έκταση του χάρτη. Αν δεν είναι επιλεγμένη, θα αναλυθούν οι μεταβλητές ή οι φασματικές ζώνες όλου του θεματικού επιπέδου, ακόμη και αν βρίσκονται έξω από την τρέχουσα έκταση του χάρτη.

Επιλέξτε τα imagery layer προς δειγματοληψία.

Ένα μεμονωμένο imagery layer ή μια λίστα imagery layer στις τιμές των οποίων θα γίνει δειγματοληψία με βάση τα δεδομένα της τοποθεσίας εισόδου.

Για να γίνει δειγματοληψία όλων των φετών δεδομένων από ένα imagery layer πολυδιάστατων δεδομένων, πρέπει να είναι το μόνο raster εισόδου.

Επιλογή θεματικού επιπέδου τοποθεσίας

Δεδομένα που προσδιορίζουν τις θέσεις από τις οποίες θέλετε να γίνει δειγματοληψία.

Τα δεδομένα της τοποθεσίας εισόδου μπορούν να είναι raster ή κατηγορία σημειακών, πολυγραμμικών ή πολυγωνικών στοιχείων.

Καθορισμός πεδίου μοναδικού αναγνωριστικού

Πεδίο που περιέχει διαφορετική τιμή για κάθε τοποθεσία ή στοιχείο στο raster της τοποθεσίας εισόδου ή στο feature class.

Καθορισμός πληροφοριών για την απόκτηση των δεδομένων τοποθεσίας (προαιρετικά)

Καθορίστε την ημερομηνία και την ώρα, το βάθος ή άλλα δεδομένα απόκτησης που είναι συσχετισμένα με τα κελιά ή τα στοιχεία τοποθεσίας.

Για κάθε διάσταση υπάρχουν τρεις επιλογές καθορισμού των πληροφοριών απόκτησης που είναι συσχετισμένες με κελιά ή στοιχεία τοποθεσίας:

  • Μεμονωμένο πεδίο ή μεμονωμένη τιμή—Όποτε χρησιμοποιείτε την επιλογή αυτή, χρειάζεται να καθορίσετε τις εξής πληροφορίες:
    • Πεδίο ή τιμή διάστασης—Επιλέξτε ένα πεδίο ή τιμές διαστάσεων από τα δεδομένα τοποθεσίας ή καταχωρίστε μια συγκεκριμένη τιμή διάστασης.
  • Μεμονωμένο πεδίο ή μεμονωμένη τιμή με σχετικές τιμές—Αν χρησιμοποιήσετε την επιλογή αυτή, χρειάζεται να καθορίσετε μαζί τις παρακάτω πληροφορίες για να ορίσετε ένα εύρος τιμών εντός του οποίου θα υπολογιστούν τα στατιστικά στοιχεία για τις μεταβλητές:
    • Πεδίο ή τιμή διάστασης—Επιλέξτε ένα πεδίο ή τιμές διαστάσεων από τα δεδομένα τοποθεσίας.
    • Σχετική τιμή πριν—Καθορίστε έναν μη αρνητικό αριθμό που θα αφαιρεθεί από το/την Πεδίο ή τιμή διάστασης για να καθοριστεί η αρχή του εύρους τιμών μιας διάστασης. Η τιμή αυτή αναπαριστά τον αριθμό των ημερών αν η διάσταση είναι ο χρόνος.
    • Σχετική τιμή μετά—Καθορίστε έναν μη αρνητικό αριθμό που θα προστεθεί στο/στη Πεδίο ή τιμή διάστασης για να καθοριστεί το τέλος του εύρους τιμών μιας διάστασης. Η τιμή αυτή αναπαριστά τον αριθμό των ημερών αν η διάσταση είναι ο χρόνος.
  • Πεδία ή τιμές αρχής και τέλους—Αν χρησιμοποιήσετε την επιλογή αυτή, χρειάζεται να καθορίσετε μαζί τις παρακάτω πληροφορίες για να ορίσετε ένα εύρος τιμών εντός του οποίου θα υπολογιστούν τα στατιστικά στοιχεία για τις μεταβλητές:
    • Πεδίο ή τιμές αρχής—Επιλέξτε ένα πεδίο ή τιμές διαστάσεων από τα δεδομένα τοποθεσίας ή καταχωρίστε μια συγκεκριμένη τιμή διάστασης για να καθοριστεί η αρχή του εύρους τιμών μιας διάστασης.
    • Πεδίο ή τιμές τέλους—Επιλέξτε ένα πεδίο ή τιμές διαστάσεων από τα δεδομένα τοποθεσίας ή καταχωρίστε μια συγκεκριμένη τιμή διάστασης εύρος τιμών για να καθοριστεί το τέλος του εύρους τιμών μιας διάστασης.

Καθορισμός πεδίου ή τιμής απόστασης μεταξύ ζωνών

Η καθορισμένη απόσταση γύρω από τα στοιχεία δεδομένων τοποθεσίας. Η απόσταση μεταξύ ζωνών καθορίζεται με τις γραμμικές μονάδες της χωρικής αναφοράς του στοιχείου τοποθεσίας. Αν το στοιχείο χρησιμοποιεί γεωγραφική αναφορά, η μονάδα θα είναι μοίρες.

Τα στατιστικά στοιχεία θα υπολογιστούν εντός αυτής της περιοχής ζώνης.

Επιλογή τύπου στατιστικών στοιχείων

Ο τύπος του στατιστικού στοιχείου που θα υπολογιστεί.

  • Ελάχιστη τιμή—Βρίσκει την ελάχιστη τιμή εντός του καθορισμένου εύρους τιμών.
  • Μέγιστη τιμή—Βρίσκει τη μέγιστη τιμή εντός του καθορισμένου εύρους τιμών.
  • Διάμεση τιμή—Βρίσκει τη διάμεση τιμή εντός του καθορισμένου εύρους τιμών.
  • Μέσος όρος—Υπολογίζει τον μέσο όρο για το καθορισμένο εύρος τιμών.
  • Άθροισμα—Υπολογίζει το άθροισμα των μεταβλητών εντός του καθορισμένου εύρους τιμών.
  • Πλειονότητα—Βρίσκει τη συχνότερα εμφανιζόμενη τιμή.
  • Μειονότητα—Βρίσκει τη σπανιότερα εμφανιζόμενη τιμή.
  • Τυπική απόκλιση—Υπολογίζει την τυπική απόκλιση.
  • Ποσοστημόριο—Υπολογίζεται μια καθορισμένη ποσοστιαία τιμή εντός του προσδιορισμένου εύρους τιμών.

Ποσοστιαία τιμή

Η ποσοστιαία τιμή που θα υπολογίζεται όποτε για την παράμετρο Τύπος στατιστικών στοιχείων έχει καθοριστεί η ρύθμιση Ποσοστημόριο.

Η τιμή αυτή μπορεί να κυμαίνεται από το 0 ως το 100. Η προεπιλεγμένη τιμή είναι 90.

Καθορισμός τεχνικής επαναδειγματοληψίας

Ο αλγόριθμος επαναδειγματοληψίας που θα χρησιμοποιηθεί κατά τη δειγματοληψία ενός raster.

  • Πλησιέστερο—Πλησιέστερος γείτονας, χρησιμοποιεί την τιμή του πλησιέστερου κελιού για να δώσει τιμή στο κελί εξόδου όταν γίνεται νέα δειγματοληψία. Αυτή είναι η προεπιλογή.
  • Διγραμμικό—Πραγματοποιεί διγραμμική παρεμβολή και υπολογίζει τη νέα τιμή ενός κελιού χρησιμοποιώντας την τιμή σταθμισμένης απόστασης των τεσσάρων πλησιέστερων κέντρων εισόδου.
  • Κυβικό—Προσδιορίζει τη νέα τιμή ενός pixel με βάση την προσαρμογή μιας ομαλής καμπύλης μέσα από τα κέντρα των 16 πλησιέστερων κελιών εισόδου.

Καθορισμός διάταξης εξόδου

Καθορίζεται αν οι δειγματοληπτούμενες τιμές θα εμφανίζονται σε γραμμές ή στήλες στον πίνακα εξόδου. Η επιλογή αυτή είναι διαθέσιμη μόνο όποτε το εισαγόμενο raster πολυδιάστατων δεδομένων περιέχει μία μεταβλητή και μία διάσταση και κάθε φέτα δεδομένων είναι raster μονής φασματικής ζώνης.

  • Οι δειγματοληπτούμενες τιμές εμφανίζονται στη γραμμή—Οι δειγματοληπτούμενες τιμές εμφανίζονται σε χωριστές γραμμές στον πίνακα εξόδου. Αυτή είναι η προεπιλογή.
  • Οι δειγματοληπτούμενες τιμές εμφανίζονται στη στήλη—Οι δειγματοληπτούμενες τιμές εμφανίζονται σε χωριστές στήλες στον πίνακα εξόδου. Η επιλογή αυτή είναι έγκυρη μόνο όποτε το εισαγόμενο raster πολυδιάστατων δεδομένων περιέχει μία μεταβλητή και μία διάσταση και κάθε φέτα δεδομένων είναι raster μονής φασματικής ζώνης.

Καθορισμός τύπου εξόδου

Καθορίζεται αν θα δημιουργείται ένα feature class σημείων με δειγματοληπτούμενες τιμές στον πίνακά του ή μόνο ένας πίνακας με δειγματοληπτούμενες τιμές.

  • Πίνακας—Δημιουργείται ένας πίνακας με δειγματοληπτούμενες τιμές. Αυτή είναι η προεπιλογή.
  • Στοιχεία—Δημιουργείται ένα feature class σημείων με δειγματοληπτούμενες τιμές στον πίνακά του. Αν τα εισαγόμενα στοιχεία τοποθεσίας είναι πολυγραμμές ή πολύγωνα, κάθε σημείο στο εξαγόμενο αποτέλεσμα είναι το κεντροειδές κάθε γεωμετρίας των εισαγόμενων στοιχείων τοποθεσίας.

Όνομα θεματικού επιπέδου αποτελεσμάτων

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο ήδη υπάρχει, θα εμφανιστεί προτροπή για να δώσετε άλλο όνομα.

Μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα με χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε.
