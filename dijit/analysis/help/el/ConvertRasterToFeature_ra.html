Μετατροπή Raster σε Στοιχείο

Αυτό το εργαλείο μετατρέπει ένα raster σε feature dataset, όπως σημεία, γραμμές ή πολύγωνα.

Η παράμετρος Επιλέξτε πεδίο σάς επιτρέπει να επιλέξετε το πεδίο γνωρίσματος του συνόλου raster δεδομένων εισόδου που θα μετατραπεί σε γνώρισμα στην κλάση στοιχείων εξόδου. Αυτή η παράμετρος είναι διαθέσιμη μόνο για raster ακεραίου που έχουν πρόσθετα γνωρίσματα στο συσχετισμένο πίνακα.

Το raster εισόδου μπορεί να έχει οποιοδήποτε μέγεθος κελιού.

Τα raster ακεραίου μπορούν να μετατραπούν σε σημειακά, γραμμικά ή πολυγωνικά στοιχεία. Ωστόσο, τα raster κινητής υποδιαστολής μπορούν να μετατραπούν μόνο σε σημειακά στοιχεία.

Αν είναι επιλεγμένη η Χρήση τρέχουσας έκτασης χάρτη, θα μετατραπούν μόνο τα κελιά του raster που είναι ορατά στην τρέχουσα έκταση του χάρτη. Αν δεν είναι επιλεγμένη, θα μετατραπούν όλα τα κελιά στο θεματικό επίπεδο raster, ακόμη και αν βρίσκονται έξω από την τρέχουσα έκταση του χάρτη. Πρέπει να το λάβετε υπόψη εάν το θεματικό επίπεδο raster είναι μεγάλο ή έχει πολύ υψηλή ανάλυση.

Μερικές περιπτώσεις χρήσης αυτού του εργαλείου είναι οι εξής:

  • Δημιουργία πολυγωνικών στοιχείων από raster ταξινόμησης χρήσης γης.
  • Οι ροές που εντοπίζονται σε ένα ψηφιακό μοντέλο εδάφους (DEM) μπορούν να μετατραπούν σε γραμμικά στοιχεία.
  • Ένα ψηφιακό μοντέλο εδάφους (DEM) μπορεί να μετατραπεί σε σημειακά στοιχεία που μπορούν να χρησιμοποιηθούν σε μια ροή εργασιών που δεν βασίζεται σε raster.

Επιλέξτε θεματικό επίπεδο raster για μετατροπή

Το raster θεματικό επίπεδο εισόδου που θα μετατραπεί σε feature layer.

Επιλέξτε πεδίο

Πεδίο που καθορίζει την τιμή της μετατροπής.

Η τιμή μπορεί να είναι ακέραιος ή κείμενο.

Ένα πεδίο που περιέχει τιμές κινητής υποδιαστολής μπορεί να χρησιμοποιηθεί μόνο εάν η έξοδος παραπέμπει σε σημειακό σύνολο δεδομένων.

Το προεπιλεγμένο είναι το Value πεδίο, το οποίο περιέχει την τιμή σε κάθε κελί του raster.

Επιλέξτε τύπο εξόδου

Καθορίζει τον τύπο εξόδου.

  • Σημείο—Το raster θα μετατραπεί σε σύνολο σημειακών δεδομένων. Αυτή είναι η προεπιλογή.
  • Γραμμή—Το raster θα μετατραπεί σε feature dataset γραμμικών δεδομένων.
  • Πολύγωνο—Το raster θα μετατραπεί σε feature dataset πολυγωνικών δεδομένων.

Απλοποίηση γραμμών ή πολυγώνων

Καθορίζει αν οι γραμμές ή τα πολύγωνα θα απλοποιηθούν (θα εξομαλυνθούν). Η εξομάλυνση γίνεται με τέτοιο τρόπο ώστε η γραμμή να περιέχει τον ελάχιστον αριθμό τμημάτων ενώ ταυτόχρονα θα παραμείνει όσο το δυνατόν πιο κοντά στα αρχικά άκρα των κελιών του raster.

  • Επιλεγμένη ρύθμιση—Τα γραμμικά ή πολυγωνικά στοιχεία θα εξομαλυνθούν ώστε να εξαχθεί ένα πιο γενικευμένο αποτέλεσμα. Αυτή είναι η προεπιλογή.
  • Μη επιλεγμένη ρύθμιση—Τα γραμμικά ή πολυγωνικά στοιχεία θα ακολουθήσουν τα όρια κελιών του raster dataset.

Η παράμετρος αυτή υποστηρίζεται μόνο αν ο τύπος εξόδου είναι γραμμή ή πολύγωνο.

Δημιουργία πολυτμηματικών στοιχείων

Καθορίζει αν τα πολύγωνα εξόδου θα αποτελούνται από μονοτμηματικά ή πολυτμηματικά στοιχεία.

  • Επιλεγμένη ρύθμιση—Πολυτμηματικά στοιχεία θα δημιουργηθούν με βάση τα πολύγωνα που έχουν την ίδια τιμή.
  • Μη επιλεγμένη ρύθμιση—Μεμονωμένα (μονοτμηματικά) στοιχεία θα δημιουργηθούν για κάθε πολύγωνο. Αυτή είναι η προεπιλογή.

Η παράμετρος αυτή υποστηρίζεται μόνο αν ο τύπος εξόδου είναι πολύγωνο.

Ανώτατος αριθμός κορυφών ανά πολυγωνικό στοιχείο

Το όριο κορυφών χρησιμεύει για την υποδιαίρεση ενός πολυγώνου σε μικρότερα πολύγωνα.

Αν αφήσετε αυτό το πεδίο κενό, τα πολύγωνα εξόδου δεν θα διαιρεθούν. Αυτή είναι η προεπιλογή.

Η παράμετρος αυτή υποστηρίζεται μόνο αν ο τύπος εξόδου είναι πολύγωνο.

Όνομα θεματικού επιπέδου αποτελεσμάτων

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο ήδη υπάρχει, θα εμφανιστεί προτροπή για να δώσετε άλλο όνομα.

Μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα με χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε.
