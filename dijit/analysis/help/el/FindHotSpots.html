Εύρεση Hot Spots

Το εργαλείο "Εύρεση Hot Spot" προσδιορίζει αν υπάρχει κάποια στατιστικά σημαντική συστάδα στο χωρικό μοτίβο των δεδομένων σας.

  • Είναι πράγματι τα σημεία σας (εγκληματικές ενέργειες, δέντρα, τροχαία ατυχήματα) συγκεντρωμένα σε συστάδες; Πώς μπορείτε να είστε βέβαιοι;
  • Ανακαλύψατε ένα στατιστικά σημαντικό Hot Spot (για δαπάνες, βρεφική θνησιμότητα, συνέπεια σε υψηλή βαθμολογία σε εξετάσεις) ή μήπως ο χάρτης θα αποκάλυπτε κάτι άλλο, αν αλλάζατε τον τρόπο απεικόνισής του;
Με το εργαλείο "Εύρεση Hot Spot" θα μπορέσετε να απαντήσετε σε αυτές τις ερωτήσεις με βεβαιότητα.

Ακόμη και τα τυχαία χωρικά μοτίβα παρουσιάζουν κάποιο βαθμό συστάδας. Επιπλέον, τα μάτια μας και το μυαλό μας προσπαθούν φυσιολογικά να βρουν μοτίβα ακόμη και όταν δεν υπάρχουν. Συνεπώς, είναι δύσκολο να γνωρίζετε αν τα μοτίβα στα δεδομένα σας είναι το αποτέλεσμα πραγματικών χωρικών διεργασιών ή απλώς ένα τυχαίο αποτέλεσμα. Για το λόγο αυτό, οι ερευνητές και οι αναλυτές χρησιμοποιούν στατιστικές μεθόδους όπως η Εύρεση Hot Spots (Getis-Ord Gi*) για την ποσοτικοποίηση χωρικών μοτίβων. Όταν βρίσκετε σημαντικές συστάδες, από την άποψη της στατιστικής, στα δεδομένα σας, έχετε πολύτιμες πληροφορίες. Η γνώση του πού και πώς παρουσιάζονται συστάδες, μπορεί να παρέχει σημαντικά στοιχεία σχετικά με τις διεργασίες που προωθούν τα μοτίβα που βλέπετε. Η γνώση ότι οι διαρρήξεις σε σπίτια, για παράδειγμα, είναι σταθερά περισσότερες σε συγκεκριμένες συνοικίες, αποτελεί πολύ σημαντική πληροφορία όταν θέλετε να σχεδιάσετε αποτελεσματικές στρατηγικές αποτροπής, να κατανείμετε τους περιορισμένους πόρους αστυνόμευσης, να ξεκινήσετε προγράμματα παρακολούθησης συνοικιών, να εγκρίνετε λεπτομερείς έρευνες για εγκλήματα ή να προσδιορίσετε πιθανούς υπόπτους.

Επιλέξτε το θεματικό επίπεδο για το οποίο θα υπολογιστούν hot spots

Το σημειακό ή πολυγωνικό θεματικό επίπεδο για το οποίο θα βρεθούν hot και cold spots.

Εύρεση συστάδων υψηλής και χαμηλής συγκέντρωσης

Η ανάλυση αυτή απαντά στην ερώτηση: Πού συγκεντρώνονται χωρικά υψηλές και χαμηλές τιμές;

Αν τα δεδομένα σας είναι σημεία και επιλέξετε το Μετρήσεις σημείων, το εργαλείο αυτό θα υπολογίσει τη χωρική κατανομή των σημειακών στοιχείων για να απαντήσει στην ερώτηση: Πού συγκεντρώνονται ή διασκορπίζονται τα σημεία με μη αναμενόμενο τρόπο;

Αν επιλέξετε ένα πεδίο, το εργαλείο αυτό θα υπολογίσει τη χωρική κατανομή των τιμών που σχετίζονται με κάθε στοιχείο για να απαντήσει στην ερώτηση: Πού συγκεντρώνονται υψηλές και χαμηλές τιμές;

Μέτρηση σημείων εντός

Η προεπιλογή είναι να μετρούνται σημεία εντός ενός δικτυωτού πλέγματος, που δημιουργείται από το εργαλείο με βάση τα σημειακά δεδομένα. Εναλλακτικά, μπορείτε να επιλέξετε τη μέτρηση σημείων εντός ενός εξάγωνου πλέγματος ή την παροχή ενός θεματικού επιπέδου περιοχής (συνήθως, αυτά αντιπροσωπεύουν διοικητικές περιοχές, όπως περιοχές απογραφής, κοινοτικά όρια ή επαρχίες) για να απαντήσετε στην ερώτηση: Δεδομένου του αριθμού των σημείων που μετρήθηκαν μέσα σε κάθε στοιχείο περιοχής, υπάρχουν τοποθεσίες με στατιστικά σημαντικές χωρικές συστάδες υψηλών ή χαμηλών τιμών μέτρησης σημείων;

Καθορίστε πού μπορεί να υπάρχουν σημεία

Είτε σχεδιάστε είτε δώστε ένα θεματικό επίπεδο που ορίζει το χώρο όπου περιστατικά μπορούν να απαντήσουν στην ερώτηση: Μέσα στις περιοχές, υπάρχουν τοποθεσίες με μη αναμενόμενες υψηλές ή χαμηλές συγκεντρώσεις σημείων;

Τα στοιχεία της περιοχής που σχεδιάζετε ή τα στοιχεία στο θεματικό επίπεδο περιοχής που καθορίζετε θα πρέπει να προσδιορίζουν τη θέση όπου θα μπορούσαν να εμφανιστούν σημεία. Για να σχεδιάσετε αυτές τις περιοχές, κάντε κλικ στο κουμπί Σχεδίαση και κάντε κλικ σε μια τοποθεσία στο χάρτη για να δημιουργήσετε ένα σχήμα περιοχής. Για να σχεδιάσετε πρόσθετες περιοχές, κάντε κλικ στο κουμπί σχεδίασης ξανά και μετά κάντε κλικ σε μια τοποθεσία στο χάρτη για να συνεχίστε.

Διαίρεση με

Ορισμένες φορές μπορεί να θέλετε να αναλύσετε μοτίβα που λαμβάνουν υπόψη υποκείμενες κατανομές. Για παράδειγμα, αν τα σημεία σας αντιπροσωπεύουν εγκλήματα, η διαίρεση με το συνολικό πληθυσμό θα είχε ως αποτέλεσμα μια ανάλυση των εγκλημάτων κατά κεφαλή και όχι απλώς το πλήθος των εγκλημάτων. Η επιλογή ενός περιγραφικού γνωρίσματος για διαίρεση με αυτό αναφέρεται συχνά ως "κανονικοποίηση".

Η επιλογή Esri Population θα εμπλουτίσει κάθε στοιχείο περιοχής με τιμές πληθυσμού, οι οποίες στη συνέχεια θα χρησιμοποιηθούν ως το περιγραφικό γνώρισμα για διαίρεση. Αυτή η επιλογή θα χρησιμοποιήσει μονάδες.

Επιλογές

Το εργαλείο βρίσκει τις βέλτιστες ρυθμίσεις για τις προεπιλογές Μέγεθος κελιού και Ζώνη απόστασης με βάση τα χαρακτηριστικά των δεδομένων σας. Ωστόσο, αν έχετε συγκεκριμένες τιμές για Μέγεθος κελιού ή Ζώνη απόστασης που έχουν νόημα για την ανάλυσή σας, μπορείτε να χρησιμοποιήσετε το μενού Επιλογές για να ορίσετε αυτές τις τιμές.

Οι Επιλογές είναι επίσης χρήσιμες όταν πραγματοποιείτε ανάλυση σε διαφορετικά σύνολα δεδομένων, επειδή σας επιτρέπουν να διατηρείτε τις τιμές για Ζώνη απόστασης και Μέγεθος κελιού ίδιες σε πολλαπλά σύνολα δεδομένων. Στη συνέχεια, μπορείτε να συγκρίνετε κατάλληλα τα αποτελέσματα, για παράδειγμα, ποσοστά παχυσαρκίας και διαβήτη ή ακόμη και ποσοστά εγκληματικότητας για δύο διαφορετικά έτη.

Μέγεθος κελιού

Το μέγεθος των κελιών του πλέγματος εντός του οποίου γίνεται η μέτρηση σημείων.

Όταν χρησιμοποιείτε ένα εξάγωνο πλέγμα για μέτρηση σημείων, αυτή η απόσταση χρησιμοποιείται ως το ύψος των εξαγώνων.

Ζώνη απόστασης

Κάθε στοιχείο αναλύεται στο πλαίσιο των γειτονικών στοιχείων που βρίσκονται εντός της απόστασης που καθορίζετε. Το εργαλείο θα υπολογίσει μια προεπιλεγμένη απόσταση ή μπορείτε να χρησιμοποιήσετε αυτή την επιλογή για να ορίσετε μια συγκεκριμένη απόσταση που έχει νόημα για την ανάλυσή σας

Για παράδειγμα, αν μελετάτε μοτίβα μετακίνησης και γνωρίζετε ότι η μέση διάρκεια μετακίνησης προς τη δουλειά είναι για παράδειγμα 15 μίλια, ίσως θέλετε να χρησιμοποιήσετε μια ζώνη απόστασης 15 μιλίων για την ανάλυσή σας.

Όνομα θεματικού επιπέδου αποτελεσμάτων

Δώστε ένα όνομα για το θεματικό επίπεδο που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Αυτό το επίπεδο αποτελεσμάτων θα εμφανίσει στατιστικά σημαντικές συστάδες υψηλών και χαμηλών τιμών ή πλήθους σημείων. Αν το όνομα του εξαγόμενου θεματικού επιπέδου υπάρχει ήδη, θα σας ζητηθεί να το μετονομάσετε.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μου όπου θα αποθηκευτεί το αποτέλεσμα.
