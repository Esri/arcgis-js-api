Εξαγωγή raster

Το εργαλείο μπορεί να εκτελέσει αποκοπή raster σε ένα όριο, είτε σε μια ορθογώνια περιοχή είτε σε ένα σχήμα που καθορίζετε διαδραστικά στην οθόνη. Μπορείτε να εκτελέσετε αποκοπή στην έκταση της περιοχής που εμφανίζεται τη δεδομένη στιγμή στο χάρτη σας ή στην περιοχή μελέτης που ορίζεται από ένα πολύγωνο.

Επιλογή δεδομένων εισόδου για εξαγωγή

Το θεματικό επίπεδο εισόδου για αποκοπή.

Καθορίστε γεωμετρικό τύπο αποκοπής

Επιλέξτε τον τύπο έκτασης που θα χρησιμοποιείται κατά την αποκοπή του γεωμετρικού τύπου του θεματικού επιπέδου:

  • Χρήση τρέχουσας έκτασης—Η τρέχουσα έκταση χάρτη θα χρησιμοποιείται για την αποκοπή του εξαγόμενου θεματικού επιπέδου. Αυτή είναι η προεπιλογή.
  • Επιλέξτε περιοχή μελέτης—Σας επιτρέπει να ορίσετε ένα γεωμετρικό τύπο αποκοπής χρησιμοποιώντας το εργαλείο σχεδίασης.
  • Επιλέξτε θεματικό επίπεδο—Επιλέξτε feature layer ή imagery layer για να ορίσετε την έκταση, ή επιλέξτε πολύγωνο για να καθορίσετε την αποκοπή του γεωμετρικού τύπου. Όταν επιλεχθεί πολυγωνικό feature layer, προβάλλεται ένα κουτάκι ελέγχου για να προσδιορίσει αν θα χρησιμοποιηθούν στοιχεία εισόδου για γεωμετρία αποκοπής. Μη επιλεγμένο ως προεπιλογή. Σημαίνει πως το feature layer εισόδου χρησιμοποιείται για τον καθορισμό της έκτασης. Αν είναι επιλεγμένο σημαίνει τη χρήση στοιχείων εισόδου για τη γεωμετρία αποκοπής.

Καθορισμός τύπου αποκοπής

Καθορίστε αν θέλετε να διατηρηθεί η περιοχή εντός ή εκτός της γεωμετρίας της αποκοπής σας.

  • Δεδομένα εντός γεωμετρίας—Θα αφαιρεθούν οι δορυφορικές εικόνες που βρίσκονται εντός της περιοχής αποκοπής. Αυτή είναι η προεπιλογή.
  • Δεδομένα εντός γεωμετρίας—Θα αφαιρεθούν οι δορυφορικές εικόνες που βρίσκονται εκτός της περιοχής αποκοπής.

Όνομα θεματικού επιπέδου αποτελεσμάτων

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο ήδη υπάρχει, θα εμφανιστεί προτροπή για να δώσετε άλλο όνομα.

Μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα με χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε. Εάν έχετε τα δικαιώματα για τη δημιουργία θεματικών επιπέδων πλαισίων και δυναμικών προβολών 3D μπορείτε να καθορίσετε τον τύπο θεματικού επιπέδου που θα δημιουργηθεί στην έξοδο χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος ως.
