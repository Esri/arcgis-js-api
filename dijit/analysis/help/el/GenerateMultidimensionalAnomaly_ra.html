Δημιουργία πολυδιάστατου raster κατόπιν υπολογισμού επιμέρους ανωμαλιών

Υπολογίζει την ανωμαλία για κάθε τμήμα σε ένα προϋπάρχον raster πολυδιάστατων δεδομένων για να δημιουργηθεί ένα νέο raster πολυδιάστατων δεδομένων.

Ανωμαλία είναι απόκλιση μιας παρατήρησης από την τυπική, τη μέση ή τη διάμεση τιμή της.

Το εργαλείο αυτό υπολογίζει ανωμαλίες ενός κάποιου χρονικού διαστήματος για μία ή περισσότερες μεταβλητές σε ένα πολυδιάστατο raster. Αν έχετε μια μη χρονική διάσταση συν τη χρονική διάσταση, η ανωμαλία θα υπολογιστεί σε κάθε βήμα της πρόσθετης διάστασης.

Για παράδειγμα, έχετε δεδομένα θερμοκρασίας ωκεανών που έχουν συλλεχθεί σε μηνιαία βάση και κάθε 1 μέτρο σε βάθος έως και 100 μέτρα και θέλετε να υπολογίσετε τις ανωμαλίες της θερμοκρασίας ως αποκλίσεις από τον ετήσιο μέσο όρο. Το εργαλείο αυτό θα προσδιορίσει τις ανωμαλίες της θερμοκρασίας με βάση έναν ετήσιο μέσο όρο αν καθορίσετε τη ρύθμιση Ετησίως ως χρονικό διάστημα για τον υπολογισμό του μέσου όρου και θα επιστρέψει τις τιμές της ανωμαλίας για καθένα από 100 βάθη.

Το συγκεκριμένο εργαλείο υποστηρίζει μόνο raster dataset πολυδιάστατων δεδομένων που έχουν χρονική διάσταση.

Αν είναι επιλεγμένη η Χρήση τρέχουσας έκτασης χάρτη, θα αναλυθούν μεταβλητές του θεματικού επιπέδου που είναι ορατές στην τρέχουσα έκταση του χάρτη. Αν δεν είναι επιλεγμένη, θα αναλυθούν οι μεταβλητές όλου του θεματικού επιπέδου, ακόμη και αν βρίσκονται έξω από την τρέχουσα έκταση του χάρτη.

Επιλογή πολυδιάστατου imagery layer για τη δημιουργία ανωμαλίας

Το imagery layer εισόδου ενός raster dataset πολυδιάστατων δεδομένων.

Επιλογή μεταβλητών για τις οποίες θα δημιουργηθούν οι ανωμαλίες

Η μεταβλητή ή οι μεταβλητές για την οποία ή τις οποίες θα υπολογιστούν οι ανωμαλίες. Αν δεν καθορίσετε μεταβλητές, θα συναθροιστούν όλες οι μεταβλητές με χρονική διάσταση.

Επιλογή μεθόδου για τη δημιουργία της ανωμαλίας

Καθορίζει τη μέθοδο που θα χρησιμοποιηθεί για τον υπολογισμό της ανωμαλίας.

  • Διαφορά από τον μέσο όρο—Θα υπολογιστεί η διαφορά μεταξύ της τιμής ενός pixel και τις μέσης τιμής του στα τμήματα που καθορίζονται από το χρονικό διάστημα. Αυτή είναι η προεπιλογή.
  • Ποσοστιαία διαφορά από τον μέσο όρο—Θα υπολογιστεί η ποσοστιαία διαφορά μεταξύ της τιμής ενός pixel και της μέσης τιμής του στα τμήματα που καθορίζονται από το χρονικό διάστημα.
  • Ποσοστό του μέσου όρου—Θα υπολογιστεί το ποσοστό του μέσου όρου.
  • Βαθμολογία z—Θα υπολογιστεί η βαθμολογία z για κάθε pixel. Αν η βαθμολογία z είναι 0, υποδεικνύει ότι η τιμή του pixel είναι ίδια με τον μέσο όρο. Αν η βαθμολογία z είναι 1, υποδεικνύει ότι η τιμή του pixel είναι 1 τυπική απόκλιση από τον μέσο όρο. Αν η βαθμολογία z είναι 2, υποδεικνύει ότι η τιμή του pixel είναι 2 τυπικές αποκλείσεις από τον μέσο όρο και ούτω καθεξής.
  • Ποσοστιαία διαφορά από τον διάμεσο—Θα υπολογιστεί η διαφορά μεταξύ της τιμής ενός pixel και του μαθηματικού διάμεσου των τιμών του στα τμήματα που καθορίζονται από το χρονικό διάστημα.
  • Ποσοστιαία διαφορά από τον διάμεσο—Θα υπολογιστεί η ποσοστιαία διαφορά μεταξύ της τιμής ενός pixel και του μαθηματικού διάμεσου των τιμών του στα τμήματα που καθορίζονται από το χρονικό διάστημα.
  • Ποσοστό του διάμεσου—Θα υπολογιστεί το ποσοστό του μαθηματικού διάμεσου.

Οι μαθηματικές μέθοδοι για τον υπολογισμό των τιμών ανωμαλίας παρατίθενται παρακάτω. Οι ανωμαλίες μπορούν να υπολογιστούν με χρήση είτε της μέσης είτε της διάμεσης τιμής ως ορισμού του μέσου όρου. Αν η κατανομή των δεδομένων είναι ασύμμετρη, ο μέσος όρος ενδέχεται να επηρεαστεί σημαντικά από έκτοπα οπότε για δεδομένα αυτού του τύπου ίσως ενδείκνυται περισσότερο η μέθοδος διάμεσης τιμής ή βαθμολογίας z.

  • Διαφορά από τον μέσο όρο = x - µ
    • x = τιμή pixel σε ένα τμήμα
    • µ = μέσος όρος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
  • Ποσοστιαία διαφορά από τον μέσο όρο = |x - µ| / [(x + µ)/2]
    • x = τιμή pixel σε ένα τμήμα
    • µ = μέσος όρος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
    • |x - µ| = απόλυτη τιμή της διαφοράς μεταξύ της τιμής και του μέσου όρου
  • Ποσοστό του μέσου όρου = x / µ
    • x = τιμή pixel σε ένα τμήμα
    • µ = μέσος όρος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
  • Διαφορά από τον διάμεσο = x - ß
    • x = τιμή pixel σε ένα τμήμα
    • ß = διάμεσος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
  • Ποσοστιαία διαφορά από τον διάμεσο = |x - ß| / [(x + ß)/2]
    • x = τιμή pixel σε ένα τμήμα
    • ß = διάμεσος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
    • |x - ß| = απόλυτη τιμή της διαφοράς μεταξύ της τιμής και του διάμεσου
  • Ποσοστό του διάμεσου = x / ß
    • x = τιμή pixel σε ένα τμήμα
    • ß = διάμεσος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
  • Βαθμολογία z = (x - µ) / S
    • x = τιμή pixel σε ένα τμήμα
    • µ = μέσος όρος των τιμών αυτού του pixel κατά το δεδομένο χρονικό διάστημα
    • S = η τυπική απόκλιση όρος των τιμών του pixel κατά το δεδομένο χρονικό διάστημα

Επιλογή χρονικού διαστήματος για τον καθορισμό του μέσου όρου

Καθορίζει το χρονικό διάστημα που θα χρησιμοποιηθεί για τον υπολογισμό του μέσου όρου.

  • Όλα—Υπολογίζεται ο μέσος όρος από όλα τα τμήματα για κάθε pixel.
  • Ανά έτος—Υπολογίζεται ο ετήσιος μέσος όρος για κάθε pixel.
  • Επανάληψη κάθε μήνα—Υπολογίζεται ο μηνιαίος μέσος όρος για κάθε pixel.
  • Επανάληψη κάθε εβδομάδα—Υπολογίζεται ο εβδομαδιαίος μέσος όρος για κάθε pixel.
  • Επανάληψη κάθε ημέρα—Υπολογίζεται ο ημερήσιος μέσος όρος για κάθε pixel.
  • Ανά ώρα—Υπολογίζεται ο ωριαίος μέσος όρος για κάθε pixel.
  • Εξωτερικό raster—Γίνεται παραπομπή προς ένα προϋπάρχον raster dataset που περιέχει τη μέση ή τη διάμεση τιμή για κάθε pixel.

Επιλογή imagery layer μέσου όρου ως πηγής αναφοράς

Καθορίζει το raster dataset αναφοράς που έναν παλαιότερα υπολογισμένο μέσο όρο για κάθε pixel. Οι ανωμαλίες θα υπολογιστούν σε σύγκριση με τον εν λόγω μέσο όρο.

Παράβλεψη των τιμών που λείπουν κατά τον υπολογισμό

Καθορίζει αν οι τιμές που λείπουν παραβλέπονται κατά την ανάλυση.

  • Επιλεγμένη δυνατότητα— Στην ανάλυση θα συμπεριληφθούν όλα τα έγκυρα pixel κατά μήκος μιας δεδομένης διάστασης και θα παραβλεφθούν όσα pixel έχουν τιμή NoData. Αυτή είναι η προεπιλογή.
  • Μη επιλεγμένη ρύθμιση—Τα αποτελέσματα της ανάλυσης θα είναι NoData αν υπάρχουν τιμές NoData για το pixel κατά μήκος της δεδομένης διάστασης.

Όνομα θεματικού επιπέδου αποτελεσμάτων
