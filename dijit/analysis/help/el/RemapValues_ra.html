Νέα αποτύπωση τιμών

Το εργαλείο σάς επιτρέπει να αλλάξετε ή να αναταξινομήσετε τις τιμές των pixel για τα raster δεδομένα. Γίνεται νέα αποτύπωση των τιμών pixel ορίζοντας ένα εύρος τιμών pixel αντιστοιχισμένων σε μια τιμή pixel εξόδου. Η τιμή pixel εξόδου μπορεί να είναι μια έγκυρη τιμή ή μια τιμή NoData, δηλαδή pixel που δεν έχουν γνωστή αντιστοιχισμένη τιμή.

Αν είναι επιλεγμένη η Χρήση τρέχουσας έκτασης χάρτη, θα γίνει νέα αποτύπωση μόνο των κελιών που είναι ορατά εντός της τρέχουσας έκτασης χάρτη. Αν δεν είναι επιλεγμένη, θα γίνει νέα αποτύπωση όλων των κελιών στο raster, ακόμη και αν βρίσκονται έξω από την τρέχουσα έκταση του χάρτη.

Δεδομένα εισόδου

Τα δεδομένα εισόδου των οποίων οι τιμές θα αποτυπωθούν εκ νέου.

Νέα αποτύπωση τιμών pixel σε νέες τιμές

Εισαγάγετε ένα εύρος τιμών ορίζοντας μια ελάχιστη και μια μέγιστη τιμή, και κατόπιν αντιστοιχίστε μια τιμή εξόδου ή μια τιμή NoData στο καθορισμένο εύρος.

Για νέα αποτύπωση ενός καινούργιου εύρους τιμών, πατήστε Προσθήκη για να προστεθεί μια σειρά. Κάντε κλικ σε κάθε τιμή για να επεξεργαστείτε τον πίνακα στα εύρη και τη νέα αποτυπωμένη τιμή που θέλετε. Όταν προσδιορίζετε τα εύρη τιμών, ισχύουν δύο σημαντικοί κανόνες:

  • Η ελάχιστη τιμή του εύρους περιλαμβάνεται ενώ η μέγιστη τιμή εξαιρείται.
  • Η επεξεργασία των σειρών του πίνακα γίνεται από πάνω προς τα κάτω.

Ο παρακάτω πίνακας δείχνει ότι η ελάχιστη τιμή περιλαμβάνεται στην έξοδο ενώ η μέγιστη τιμή εξαιρείται από την έξοδο.
