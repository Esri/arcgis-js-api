Αντιγραφή στο Data Store

Το εργαλείο αντιγράφει ένα θεματικό επίπεδο εισόδου σε ένα data store. Τα δεδομένα θα αντιγραφούν στο ArcGIS Data Store και θα αποθηκευτούν στο σχεσιακό ή στο χωρο-χρονικό data store.

Για παράδειγμα, μπορείτε να αντιγράψετε στοιχεία που είναι αποθηκευμένα σε ένα κοινόχρηστο αρχείο μαζικών δεδομένων σε ένα σχεσιακό data store και να ορίσετε ότι θα αντιγράφονται μόνο τα στοιχεία εντός της τρέχουσας έκτασης χάρτη. Έτσι θα δημιουργούνταν ένα φιλοξενούμενο feature service μόνο με τα στοιχεία εντός της καθορισμένης έκτασης χάρτη.

Διαλέξτε θεματικό επίπεδο προς αντιγραφή

Το θεματικό επίπεδο ή ο πίνακας που θα αντιγραφεί.

Εκτός από την επιλογή ενός θεματικού επιπέδου από το χάρτη σας, μπορείτε επίσης να επιλέξετε Αναζήτηση θεματικών επιπέδων στο κάτω μέρος της αναπτυσσόμενης λίστας για αναζήτηση στα περιεχόμενά σας ενός κοινόχρηστου αρχείου συνόλου δεδομένων με big data ή ενός feature layer.

Μπορείτε να χρησιμοποιήσετε όλα τα στοιχεία ή να επιλέξετε (προαιρετικό)

Προαιρετικά, μπορείτε να χρησιμοποιήσετε το κουμπί επιλογής για να ορίσετε τις τοποθεσίες προς αντιγραφή.

Διαλέξτε το ArcGIS Data Store όπου θα γίνει η αντιγραφή

Το data store στο οποίο θα αντιγραφεί το θεματικό επίπεδο. Αυτή η παράμετρος θα παρακάμψει την επιλογή στα Περιβάλλοντα ανάλυσης Υπάρχουν οι εξής επιλογές:

  • Χωρο-χρονικό—Αυτή είναι η προεπιλογή.
  • Σχεσιακό.

Όνομα εξαγόμενου θεματικού επιπέδου

Αυτό είναι το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο υπάρχει ήδη, το εργαλείο θα αποτύχει.

Με τη χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε, μπορείτε να καθορίσετε το όνομα ενός φακέλου στην ενότητα Το περιεχόμενό μουόπου θα αποθηκευτεί το αποτέλεσμα.
