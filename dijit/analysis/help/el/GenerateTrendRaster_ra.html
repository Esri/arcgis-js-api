Δημιουργία Raster Τάσεων

Υπολογίζει την τάση για κάθε pixel κατά μήκος μιας διάστασης για μία ή περισσότερες μεταβλητές σε ένα πολυδιάστατο raster.

Επιλέξτε το imagery layer πολυδιάστατων δεδομένων για το οποίο θα αναλυθεί η τάση.

Το imagery layer εισόδου ενός raster πολυδιάστατων δεδομένων το οποίο θα αναλυθεί.

Επιλέξτε τη διάσταση κατά μήκος της οποίας θα αναλυθεί η τάση των μεταβλητών.

Η διάσταση κατά μήκος της οποίας θα εξαχθεί μια τάση για τη μεταβλητή ή τις μεταβλητές που έχουν επιλεχθεί στην ανάλυση.

Επιλέξτε τις μεταβλητές για τις οποίες θα αναλυθεί η τάση.

Η μεταβλητή ή οι μεταβλητές για την οποία ή τις οποίες θα υπολογιστούν οι τάσεις. Αν δεν έχει καθοριστεί καμία μεταβλητή, θα αναλυθεί η πρώτη μεταβλητή στο raster πολυδιάστατων δεδομένων.

Επιλογή του τύπου γραμμής για την προσαρμογή των τιμών των μεταβλητών κατά μήκος μιας διάστασης

Καθορίζει τον τύπο της γραμμής που θα χρησιμοποιηθεί για την προσαρμογή των τιμών pixel κατά μήκος μιας διάστασης.

  • Γραμμική—Οι τιμές των pixel των μεταβλητών θα προσαρμοστούν κατά μήκος μιας γραμμικής γραμμής τάσης. Αυτή είναι η προεπιλογή.
  • Πολυωνυμική—Οι τιμές των pixel των μεταβλητών θα προσαρμοστούν κατά μήκος μιας πολυωνυμικής γραμμής τάσης.
  • Αρμονική—Οι τιμές των pixel των μεταβλητών θα προσαρμοστούν κατά μήκος μιας αρμονικής γραμμής τάσης.
  • Mann-Kendall—Οι τιμές pixel θα αξιολογηθούν με τη δοκιμή τάσης Mann-Kendall.
  • Εποχικό-Kendall—Οι μεταβλητές τιμές pixel θα αξιολογηθούν με τη δοκιμή τάσης Εποχικού-Kendall.

Καθορίστε τη μονάδα χρόνου για τη διάρκεια μιας εποχικής περιόδου.

Προσδιορίζει τη μονάδα χρόνου που θα χρησιμοποιηθεί για το μήκος ενός εποχιακού διαστήματος κατά την εκτέλεση της Εποχιακής δοκιμής Kendall.

  • Ημέρες—Η μονάδα για τη διάρκεια της εποχικής περιόδου είναι ημέρες. Αυτή είναι η προεπιλογή.
  • Μήνες—Η μονάδα για τη διάρκεια της εποχικής περιόδου είναι μήνες.

Καθορισμός της διάρκειας του αρμονικού κύκλου

Η διάρκεια της περιοδικής μεταβολής στο μοντέλο. Η παράμετρος αυτή απαιτείται όποτε η καθορισμένη τιμή για τον τύπο γραμμής της τάσης είναι Αρμονική. Π.χ. το πράσινο των φύλλων συχνά έχει έναν ισχυρό κύκλο μεταβολής ανά έτος οπότε η διάρκεια του κύκλου είναι 1 έτος. Τα ωριαία δεδομένα θερμοκρασίας έχουν έναν ισχυρό κύκλο μεταβολής ανά ημέρα οπότε η διάρκεια του κύκλου είναι 1 ημέρα.

Η προεπιλεγμένη διάρκεια είναι 1 έτος για δεδομένα με ετήσιο κύκλο μεταβολής.

Η παράμετρος αυτή περιλαμβάνεται στην ανάλυση τάσεων μόνο όποτε η αναλυόμενη διάσταση είναι ο χρόνος και η καθορισμένη τιμή για τον τύπο γραμμής της τάσης είναι Αρμονική.

Επιλογή της μονάδας χρόνου για τη διάρκεια του αρμονικού κύκλου

Καθορίζει τη μονάδα χρόνου που θα χρησιμοποιηθεί για τη διάρκεια ενός αρμονικού κύκλου.

  • Ημέρες—Η μονάδα για τη διάρκεια του αρμονικού κύκλου είναι ημέρες.
  • Έτη—Η μονάδα για τη διάρκεια του αρμονικού κύκλου είναι έτη. Αυτή είναι η προεπιλογή.

Η παράμετρος αυτή περιλαμβάνεται στην ανάλυση τάσεων μόνο όποτε η αναλυόμενη διάσταση είναι ο χρόνος και η καθορισμένη τιμή για τον τύπο γραμμής της τάσης είναι Αρμονική.

Καθορισμός του αριθμού της συχνότητας για την προσαρμογή αρμονικής τάσης

Η συχνότητα που θα χρησιμοποιηθεί κατά την προσαρμογή αρμονικής τάσης. Η παράμετρος αυτή καθορίζει τη συχνότητα των κύκλων σε ένα έτος.

Η προεπιλεγμένη τιμή είναι 1 ή ένας αρμονικός κύκλος ανά έτος.

Η παράμετρος αυτή περιλαμβάνεται στην ανάλυση τάσεων μόνο όποτε η αναλυόμενη διάσταση είναι ο χρόνος.

Καθορισμός του αριθμού του βαθμού του πολυωνύμου για την προσαρμογή της τάσης

Ο αριθμός του βαθμού του πολυωνύμου που θα χρησιμοποιηθεί κατά την προσαρμογή πολυωνυμικής τάσης.

Η παράμετρος αυτή περιλαμβάνεται στην ανάλυση τάσεων μόνο όποτε η αναλυόμενη διάσταση είναι ο χρόνος.

Η προεπιλεγμένη τιμή είναι 2 ή πολυώνυμο δεύτερου βαθμού.

Επιλογή στατιστικών στοιχείων μοντέλου που θα συμπεριληφθούν στο raster τάσεων

Το εργαλείο αυτό παρέχει τρεις τύπους στατιστικών στοιχείων γραμμής προσαρμογής τάσης: σφάλμα μέσης τετραγωνικής ρίζας (RMSE), το στατιστικό στοιχείο ελέγχου καλής προσαρμογής της τιμής R στο τετράγωνο (R στο τετράγωνο) και το στατιστικό στοιχείο της τιμής p για τον συντελεστή κλίσης (τιμή P του συντελεστή κλίσης). Σε κάθε στατιστικό στοιχείο υπάρχει αντίστοιχο πλαίσιο ελέγχου για να καθορίσετε αν θα υπολογιστεί και θα συμπεριληφθεί ως μία από τις πολλές φασματικές ζώνες στο εξαγόμενο raster τάσεων.

  • RMSE

    • Επιλεγμένη ρύθμιση—Το RMSE θα υπολογιστεί και θα συμπεριληφθεί ως μία από τις πολλές φασματικές ζώνες στο εξαγόμενο raster τάσεων. Αυτή είναι η προεπιλογή.
    • Μη επιλεγμένη ρύθμιση—Το RMSE δεν θα υπολογιστεί.

  • R στο τετράγωνο

    • Επιλεγμένη ρύθμιση—Η τιμή R στο τετράγωνο θα υπολογιστεί και θα συμπεριληφθεί ως μία από τις πολλές φασματικές ζώνες στο εξαγόμενο raster τάσεων.
    • Μη επιλεγμένη ρύθμιση—Η τιμή R στο τετράγωνο δεν θα υπολογιστεί. Αυτή είναι η προεπιλογή.

  • Τιμή p του συντελεστή κλίσης

    • Επιλεγμένη ρύθμιση—Η τιμή p θα υπολογιστεί και θα συμπεριληφθεί ως μία από τις πολλές φασματικές ζώνες στο εξαγόμενο raster τάσεων.
    • Μη επιλεγμένη ρύθμιση—Η τιμή p δεν θα υπολογιστεί. Αυτή είναι η προεπιλογή.

Παράβλεψη των τιμών που λείπουν κατά τον υπολογισμό

Καθορίζει αν οι τιμές που λείπουν παραβλέπονται κατά την ανάλυση.

  • Επιλεγμένη δυνατότητα— Στην ανάλυση θα συμπεριληφθούν όλα τα έγκυρα pixel κατά μήκος μιας δεδομένης διάστασης και θα παραβλεφθούν όσα pixel έχουν τιμή NoData. Αυτή είναι η προεπιλογή.
  • Μη επιλεγμένη δυνατότητα— Τα αποτελέσματα της ανάλυσης θα είναι NoData αν υπάρχουν τιμές NoData για τα pixel κατά μήκος της δεδομένης διάστασης.

Όνομα θεματικού επιπέδου αποτελεσμάτων

Το όνομα του θεματικού επιπέδου που θα δημιουργηθεί στην ενότητα Το περιεχόμενό μου και θα προστεθεί στο χάρτη. Το προεπιλεγμένο όνομα βασίζεται στο όνομα του εργαλείου και στο όνομα του θεματικού επιπέδου εισόδου. Αν το θεματικό επίπεδο ήδη υπάρχει, θα εμφανιστεί προτροπή για να δώσετε άλλο όνομα.

Μπορείτε να καθορίσετε μέσα στον φάκελο Το περιεχόμενό μου το όνομα ενός άλλου φακέλου όπου θα αποθηκευτεί το αποτέλεσμα με χρήση του πτυσσόμενου πλαισίου Αποθήκευση αποτελέσματος σε. Εάν έχετε τα δικαιώματα για τη δημιουργία θεματικών επιπέδων πλαισίων και δυναμικών προβολών 3D μπορείτε να καθορίσετε τον τύπο θεματικού επιπέδου που θα δημιουργηθεί στην έξοδο χρησιμοποιώντας το πτυσσόμενο πλαίσιο Αποθήκευση αποτελέσματος ως.
