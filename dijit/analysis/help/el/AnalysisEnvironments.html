Ρυθμίσεις Περιβάλλοντος Ανάλυσης Πύλης

Ρυθμίσεις περιβάλλοντος για ανάλυση σε Portal for ArcGIS και ArcGIS Online.

Αντικατάσταση θεματικού επιπέδου αποτελεσμάτων

Επιλογή για αντικατάσταση υπαρχόντων θεματικών επιπέδων. Όταν είναι επιλεγμένη και υπάρχει ήδη στα περιεχόμενά σας θεματικό επίπεδο με το ίδιο όνομα, αυτό θα αντικατασταθεί.

  • Επιλεγμένη—Αν υπάρχει ήδη στα περιεχόμενά σας θεματικό επίπεδο με το ίδιο όνομα, αυτό θα αντικατασταθεί. Η έξοδος θα έχει το ίδιο σύστημα συντεταγμένων με την είσοδο.
  • Μη επιλεγμένη—Αν υπάρχει ήδη το όνομα που δίδεται στο θεματικό επίπεδο που προκύπτει, θα δημιουργηθεί σφάλμα και πρέπει να μετονομαστεί το θεματικό επίπεδο που προκύπτει, για να λειτουργήσει το εργαλείο. Αυτή είναι η προεπιλεγμένη ρύθμιση.

Κλείσιμο widget ανάλυσης μετά την εκτέλεση της ανάλυσης

Επιλογή για να παραμείνει ανοιχτό και ενεργό το παράθυρο διαλόγου της ανάλυσης ενώ γίνεται επεξεργασία από το εργαλείο. Αυτή η ρύθμιση θα σας επιτρέψει να κάνετε εύκολα τροποποιήσεις στις παραμέτρους ανάλυσης, ενώ εξακολουθεί να εκτελείται η προηγούμενη εργασία ανάλυσης. Η ανάλυση είναι μια επαναληπτική διαδικασία, και διατηρώντας το παράθυρο διαλόγου του εργαλείου ανοιχτό, έχετε τη δυνατότητα να κάνετε εύκολα αλλαγές στις παραμέτρους και να επαναλαμβάνετε την ανάλυση.

  • Επιλεγμένη—Το παράθυρο του εργαλείου ανάλυσης θα κλείσει και η εστίαση θα επιστρέψει στον πίνακα περιεχομένων. Αυτή είναι η προεπιλεγμένη ρύθμιση
  • Μη επιλεγμένη—Το παράθυρο του εργαλείου ανάλυσης παραμένει ανοιχτό και ενεργό.

Αποθήκευση αποτελεσμάτων ανάλυσης ως φιλοξενούμενων θεματικών επιπέδων στοιχείων

Επιλογή για να καθορίσετε αν το θεματικό επίπεδο των αποτελεσμάτων το οποίο δημιουργήθηκε από την ανάλυσή σας θα είναι συλλογή στοιχείων ή φιλοξενούμενο θεματικό επίπεδο στοιχείων.

  • Επιλεγμένη—Τα αποτελέσματα της ανάλυσής σας θα δημιουργήσουν ένα φιλοξενούμενο θεματικό επίπεδο στοιχείων και θα δημιουργηθεί ένα αντικείμενο στα περιεχόμενά σας. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Μη επιλεγμένη—Τα αποτελέσματα της ανάλυσής σας θα δημιουργήσουν μια συλλογή στοιχείων.

Σύστημα συντεταγμένων εξόδου

Ρύθμιση περιβάλλοντος για ανάλυση raster που χρησιμοποιείται για τον προσδιορισμό του συστήματος συντεταγμένων του θεματικού επιπέδου που προκύπτει.

  • Ίδιο με την είσοδο—Το αποτέλεσμα της ανάλυσής σας θα είναι στο ίδιο σύστημα συντεταγμένων με την είσοδο. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Όπως καθορίζεται—Επιλέξτε το σύστημα συντεταγμένων για τα αποτελέσματα της ανάλυσής σας. Όταν επιλέξετε τη δυνατότητα αυτή, κάντε κλικ στο εικονίδιο της υφηλίου για να επιλέξετε από μια λίστα γνωστών συστημάτων συντεταγμένων ή προσθέστε τη χωρική αναφορά WKID στο παρεχόμενο πλαίσιο.
  • Όνομα <θεματικού επιπέδου>—Επιλέξτε ένα υφιστάμενο θεματικό επίπεδο στο web χάρτη που θα χρησιμοποιηθεί για τον προσδιορισμό του συστήματος συντεταγμένων του αποτελέσματος της ανάλυσής σας.

Επεξεργασία συστήματος συντεταγμένων

Ρύθμιση περιβάλλοντος για το GeoAnalytics που χρησιμοποιείται για τον προσδιορισμό του συστήματος συντεταγμένων που θα χρησιμοποιηθεί στην ανάλυση κατά την πραγματοποίηση της ανάλυσης.

Για το GeoAnalytics, ορισμένες αναλύσεις απαιτούν την εκτέλεση της επεξεργασίας σε προβολικό σύστημα συντεταγμένων. Οποιοδήποτε εργαλείο έχει εφαρμόσει αποθήκευση δεδομένων ή γραμμικές μονάδες απαιτεί τη χρήση ενός προβολικού συστήματος συντεταγμένων.

  • Ίδιο με την είσοδο—Η επεξεργασία της ανάλυσής σας θα γίνει στο ίδιο σύστημα συντεταγμένων με την είσοδο. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Όπως καθορίζεται—Επιλέξτε το σύστημα συντεταγμένων για την επεξεργασία της ανάλυσής σας. Όταν επιλέξετε τη δυνατότητα αυτή, κάντε κλικ στο εικονίδιο της υφηλίου για να επιλέξετε από μια λίστα γνωστών προβολικών συστημάτων συντεταγμένων ή προσθέστε τη χωρική αναφορά WKID στο παρεχόμενο πλαίσιο.
  • Όνομα <θεματικού επιπέδου>—Επιλέξτε υπάρχον θεματικό επίπεδο από τον διαδικτυακό χάρτη σας. Το σύστημα συντεταγμένων του θεματικού επιπέδου που έχει επιλεγεί θα χρησιμοποιηθεί για τον προσδιορισμό του συστήματος συντεταγμένων που θα χρησιμοποιηθεί για την επεξεργασία της ανάλυσής σας.

Έκταση

Ρύθμιση περιβάλλοντος για τον προσδιορισμό της έκτασης ή των ορίων που θα χρησιμοποιηθούν κατά την πραγματοποίηση της ανάλυσης. Όλα τα στοιχεία εισόδου που βρίσκονται εντελώς μέσα στην καθορισμένη έκταση ή που τέμνουν την καθορισμένη έκταση θα χρησιμοποιηθούν στην ανάλυση.

  • Προεπιλεγμένη ρύθμιση—Χρησιμοποιήστε την έκταση που παρέχει το εργαλείο.
  • Όπως καθορίζεται—Εισαγάγετε χειροκίνητα τις συντεταγμένες για να προσδιορίσετε την έκταση.
  • Όνομα <θεματικού επιπέδου>—Επιλέξτε ένα υφιστάμενο θεματικό επίπεδο στο web χάρτη. Η χωρική έκταση του θεματικού επιπέδου που έχει επιλεγεί θα χρησιμοποιηθεί για τον προσδιορισμό της έκτασης που θα χρησιμοποιηθεί για την επεξεργασία της ανάλυσής σας.

Αγκίστρωση raster

Κατά την πραγματοποίηση ανάλυσης raster, η ρύθμιση του περιβάλλοντος Αγκίστρωση raster προσαρμόζει την έκταση του εξαγόμενου θεματικού επιπέδου raster, ώστε να αντιστοιχεί στην ευθυγράμμιση κελιών του καθορισμένου θεματικού επιπέδου της Αγκίστρωσης raster.

Μέγεθος κελιού

Κατά την πραγματοποίηση ανάλυσης raster, αυτό το περιβάλλον ορίζει το μέγεθος κελιού ή ανάλυση που θα χρησιμοποιηθεί για τη δημιουργία του εξαγόμενου θεματικού επιπέδου raster. Η προκαθορισμένη ανάλυση εξόδου προσδιορίζεται από το μεγαλύτερο μέγεθος κελιού του θεματικού επιπέδου raster εισόδου.

  • Ελάχιστο όριο δεδομένων εισόδου—Χρησιμοποιήστε το μικρότερο μέγεθος κελιού όλων των θεματικών επιπέδων εισόδου.
  • Μέγιστο όριο δεδομένων εισόδου—Χρησιμοποιήστε το μεγαλύτερο μέγεθος κελιού όλων των θεματικών επιπέδων εισόδου. Αυτή είναι η προεπιλεγμένη ρύθμιση.
  • Όπως καθορίζεται—Καθορίστε μια αριθμητική τιμή, για να ορίσετε το μέγεθος κελιού. Αν είναι επιλεγμένο, η προκαθορισμένη τιμή είναι 1.
  • Θεματικό επίπεδο <όνομα>—Καθορίστε το μέγεθος κελιού στο πλεγματικό θεματικό επίπεδο που έχετε επιλέξει.

Μάσκα

Κατά την πραγματοποίηση ανάλυσης raster, χρησιμοποιήστε το περιβάλλον Μάσκα, για να καθορίσετε ένα θεματικό επίπεδο που θα χρησιμοποιηθεί για τον προσδιορισμό της περιοχής ενδιαφέροντος. Μόνο τα κελιά που βρίσκονται εντός της μάσκας ανάλυσης θα ληφθούν υπόψη κατά τη λειτουργία της ανάλυσης.

  • Η μάσκα μπορεί να είναι raster ή θεματικό επίπεδο στοιχείων.
  • Αν η μάσκα ανάλυσης είναι raster, όλα τα κελιά με κάποια τιμή θα ληφθούν υπόψη για τον ορισμό της μάσκας. Τα κελιά ενός raster μάσκας με τιμή NoData θα θεωρηθούν ότι βρίσκονται εκτός της μάσκας και θα έχουν την τιμή NoData στο θεματικό επίπεδο αποτελεσμάτων της ανάλυσης.
  • Αν η μάσκα ανάλυσης είναι θεματικό επίπεδο στοιχείων, θα μετατραπεί εσωτερικά σε raster κατά την εκτέλεση. Για τον λόγο αυτό, βεβαιωθείτε ότι έχετε δώσει στις επιλογές και τις τιμές που ενδείκνυνται για την ανάλυσή σας.

ArcGIS Data Store

Τα αποτελέσματα GeoAnalytics αποθηκεύονται σε ένα ArcGIS Data Store και εκτίθενται ως ένα θεματικό επίπεδο στοιχείων στο Portal for ArcGIS. Στις περισσότερες περιπτώσεις, τα αποτελέσματα πρέπει να αποθηκεύονται σε ένα χωροχρονικό data store. Αυτή είναι η προεπιλεγμένη ρύθμιση. Ακολουθούν κάποιοι λόγοι για τους οποίους ίσως θελήσετε να αποθηκεύσετε αποτελέσματα σε σχεσιακό data store:

  • Για να χρησιμοποιήσετε τα αποτελέσματα σε συνεργασία μεταξύ portal
  • Για ενεργοποίηση δυνατοτήτων συγχρονισμού με τα αποτελέσματά σας

Δεν πρέπει να χρησιμοποιήσετε σχεσιακό data store αν περιμένετε αύξηση αποτελεσμάτων GeoAnalytics και επιθυμείτε να εκμεταλλευτείτε τις δυνατότητες του χωροχρονικού big data store για διαχείριση μεγάλων όγκων δεδομένων.
