Отслеживать растительность

Этот инструмент выполняет арифметическую операцию над каналами многоканального растрового слоя для извлечения данных о растительном покрове изучаемой области.

Выбрать входные данные

Выберите многоканальный растровый слой. Убедитесь, что во входном растре имеются соответствующие каналы.

Выбрать метод отслеживания состояния растительности

Выберите метод вычисления слоя индекса растительности. Различные индексы растительности помогают выявить определенные объекты или снизить шумы различной природы.

  • Global Environmental Monitoring Index — Индекс глобального мониторинга окружающей среды (GEMI) – нелинейный индекс растительности для глобального мониторинга окружающей среды по спутниковым снимкам. Он подобен NDVI, но менее чувствителен к влиянию атмосферы. На него влияет обнаженная почва; поэтому его не рекомендуется использовать в областях с редкой или умеренно густой растительностью.
  • Green Vegetation Index - Landsat TM — Индекс зеленой растительности (GVI) изначально был разработан на основе снимков Landsat MSS, затем модифицирован для снимков Landsat TM. Он также известен как индекс зеленой растительности Landsat TM Tasseled Cap. Этот индекс мониторинга можно применять к снимкам, каналы которых обладают такими же спектральными характеристиками.
  • Modified Soil Adjusted Vegetation Index — Модифицированный индекс растительности с коррекцией по почве (MSAVI2) пытается минимизировать влияние обнаженной почвы на метод SAVI.
  • Normalized Difference Vegetation Index — NDVI - это стандартизированный индекс, позволяющий создавать изображение, отображающее зелень (относительную биомассу). Этот индекс использует контраст характеристик двух каналов из набора мультиспектральных растровых данных: поглощение пигмента хлорофилла в красном канале и высокой отражательной способности растительности в инфракрасном канале (NIR).
  • Perpendicular Vegetation Index — Перпендикулярный индекс растительности (PVI) подобен разностному индексу растительности; однако он чувствителен к атмосферным отклонениям. При использовании этого метода для сравнения разных изображений его следует применять только к изображениям, скорректированным по атмосфере. Эта информация может быть предоставлена поставщиком данных.
  • Soil-Adjusted Vegetation Index — Индекс растительности с коррекцией по почве (SAVI) – это индекс растительности, который пытается минимизировать влияние яркости почвы с помощью коэффициента коррекции яркости почвы. Он часто используется в пустынных областях, где растительное покрытие незначительно.
  • Sultan's Formula — При методе Султана используется шестиканальное 8-битное изображение, к которому применяется определенный алгоритм для получения трехканального 8-битного изображения. В полученном изображении выделяются скальные образования на побережьях, называемые офиолитами. Эта формула была разработана на основе каналов TM и ETM сцены Landsat 5 или 7.
  • Transformed Soil-Adjusted Vegetation Index — Преобразованный индекс растительности с коррекцией по почве (Transformed-SAVI) – это индекс растительности, который пытается минимизировать влияние яркости почвы путем предположения, что линия почвы имеет произвольный уклон и пересечение.
  • Visible Atmospherically Resistant Index — VARI - это индекс растительности, который оценивает наличие растительности, используя только информацию из видимого спектра. Для этого метода необходимо наличие красного, синего и зеленого каналов видимого спектра.

Уклон линии почвы. Уклоном является примерное линейное отношение между NIR и red каналами диаграммы рассеивания.

Этот параметр действует только для метода Трансформированного корректированного почвенного индекса (ТКПИ).

Значение покрытия зеленой растительности

Определяет значение покрытия зеленой растительности, необходимое для расчета Модифицированного корректированного почвенного индекса (MSAVI).

Возможны следующие значения:

  • 1 = области без зеленого растительного покрова
  • 0.5 = области с умеренным растительным покровом
  • 0 = области с зеленым растительным покровом

Коэффициент настройки

Определяет фактор сопоставления, который помогает снизить эффект почвы, он также требуется для метода ТКПИ. Значение, предлагаемое по умолчанию, равно 0.08.

Более низкое значение означает, что эффект почвы игнорируется. Более высокое значение означает, что почва оказывает влияние на конечный результат.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
