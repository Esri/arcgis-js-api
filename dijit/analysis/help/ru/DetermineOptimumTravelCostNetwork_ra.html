Определить оптимальную стоимость перемещения по сети

Инструмент вычисляет оптимальную стоимость перемещения по сети на основании входных регионов.

Одним из примеров применения этого инструмента может быть поиск лучшей сети для машин экстренной помощи.

Если опция Использовать текущий экстент карты включена, будут анализироваться только объекты входного слоя, которые видны в момент анализа в текущем экстенте. Если опция отключена, анализ будет выполнен для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Выбор растра слоя расположения объектов для вычисления путей между объектами

Входные регионы, которые должны быть соединены в сеть наименьшей стоимости.

Регионы могут быть заданы сервисом изображений или сервисом объектов.

В сервисе изображений регионы определяются как группа непрерывных (смежных) ячеек с одинаковыми значениями. Регионы должны иметь уникальные номера. Ячейки, не входящие ни в один регион, должны иметь значение NoData. Растр должен быть целочисленным, а его значения могут быть как положительными, так и отрицательными.

Для векторного слоя тип входных данных может быть точка, линия или полигон.

Выбор растрового слоя стоимости расстояния

Растр, определяющий полное сопротивление или стоимость движения в плоскости через каждую ячейку.

Значение в каждом местоположении ячейки представляет собой стоимость за единицу расстояния для перемещения через ячейку. Для получения общей стоимости прохождения через ячейку, значение каждой ячейки умножается на разрешение ячейки, с учетом возможности движения по диагонали.

Значения в растре стоимости могут быть целыми числами или числами с плавающей точкой, но они не могут быть отрицательными или нулевыми (стоимость не может быть отрицательной или нулевой).

Имя линейного слоя итогового оптимального пути

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
