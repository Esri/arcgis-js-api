Поиск кластеров точек

Инструмент находит кластеры среди точечных объектов в окружающем шуме на основе их пространственного распределения.

Если отмечен экстент Использовать текущую карту, то изучаться будут только объекты входного слоя точек в пределах экстента карты. Если отметку снять, то будут проанализированы все объекты входного слоя точек, даже если они находятся вне текущего экстента карты.

Выберите слой, для которого будут найдены кластеры

Точечный слой, в котором нужно выявить кластеры.

Минимальное число точек, которые будут считаться кластером

Минимальное число объектов для включения в кластер. Любой кластер с меньшим числом объектом будет учитываться как шум.

Ограничить диапазон поиска до (дополнительно)

Максимальное расстояние между объектами, которые могут быть включены в один кластер. Если значение пустое, инструмент запустит HDBSCAN, который использует различные расстояния для отделения кластеров с различными плотностями от окружающего шумe. Если значение задано, инструмент запустит DBSCAN, который использует указанное расстояние для отделения плотных кластеров от окружающего шума.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
