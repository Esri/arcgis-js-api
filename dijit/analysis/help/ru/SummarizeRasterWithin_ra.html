Суммировать растр в пределах

Этот инструмент суммирует значения растра в пределах зон другого набора данных.

Примерами применения могут быть:

  • Используя слой границ водоразделов и слой с границами типов землепользования, можно определить общую площадь земель различных типов для каждого водораздела.
  • Используя слой земельных участков в округе и слой с границами городов, можно вычислить среднюю стоимость свободных участков в границах городов.

Если опция Использовать текущий экстент карты включена, будут анализироваться только те объекты входного слоя и слоя объединения, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты входного слоя и слоя объединения, даже если они находятся вне текущего экстента карты.

Выбрать площадной слой для суммирования растрового слоя в пределах указанных границ

Слой, задающий границы областей, или зон, которые будут суммироваться.

Слой должен содержать растровые или векторные данные. Для растров все зоны задаются всеми местоположениями с одинаковым значением ячейки во входных данных. Области не обязательно должны быть непрерывными.

Выберите поле для определения границ

Это атрибут слоя, который будет использоваться для задания границ областей.

Представьте, к примеру, что первый слой содержит границы административных подразделений с атрибутами, задающими регион, район и ID участка каждого подразделения. У вас также есть растровый слой, определяющий индекс биоразнообразия для каждого места. Выбрав поле, вы можете вычислить среднее биоразнообразие на локальном, районном или региональном уровне.

Выберите слой растра для суммирования

Ячейки растра данного слоя будут суммированы по областям (зонам), в которые они попадают.

Выберите статистику для вычисления

Выбрав слой для суммирования, дополнительно вы можете вычислить статистику любых числовых атрибутов точек, линий или полигонов, которые попадают в пределы границ входного полигонального слоя.

Когда поле является целочисленным, могут быть вычислены следующие типы статистики: Среднее, Максимум, Медиана, Минимум, Меньшинство, Процентиль, Диапазон, Среднеквадратическое отклонение, Сумма и Разнообразие.

Когда поле содержит значения с плавающей точкой, доступные типы – это Среднее, Максимум, Минимум, Диапазон, Среднеквадратическое отклонение и Сумма.

Значение процентиля

Вычисляемый процентиль. Значение по умолчанию – 90, определяющее 90-й процентиль.

Значения могут варьироваться от 0 до 100. 0-й процентиль по существу эквивалентен минимуму в статистике, а 100-й - максимуму. Значение 50 даст по существу тот же результат, что и медиана в статистике.

Эта опция поддерживается, только если параметр Выбрать статистику для вычисления задан как Процентиль.

Игнорировать пропущенные значения при вычислениях

Если вы решите игнорировать пропущенные значения для формирования выходного значения для области будут использованы только ячейки со значением а суммируемом слое. В противном случае при наличии в области пропущенных значений будет предполагаться, что для выполнения статистических расчетов для всех ячеек в этой зоне недостаточно информации, и поэтому соответствующая область получит пустое значение (NoData) на выходе.

Обработать как многомерный

Определяет, как будут обрабатываться входные растры, если они являются многомерными. Если вы выбираете обработку как многомерного, статистика будет вычисляться для всех измерений входного многомерного набора данных. В противном случае статистика будет вычисляться из текущего среза многомерного набора данных, это установка по умолчанию.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в.
