Суммировать растр в пределах

Этот инструмент суммирует значения растра в пределах зон другого набора данных.

Некоторые примеры применения этого инструмента включают следующее:

  • Используя слой границ водоразделов и слой с границами типов землепользования, можно определить общую площадь земель различных типов для каждого водораздела.
  • Используя слой земельных участков в округе и слой с границами городов, можно вычислить среднюю стоимость свободных участков в границах городов.

Если опция Использовать текущий экстент карты включена, будут анализироваться только те объекты входного слоя и слоя суммирования, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты входного слоя и слоя суммирования, даже если они находятся вне текущего экстента карты.

Выбрать площадной слой для суммирования растрового слоя в пределах указанных границ

Вход, задающий границы зон или областей, которые будут суммироваться.

Входные данные должны быть представлены целочисленным растром или пространственными данными. Для растров все зоны задаются всеми местоположениями с одинаковым значением ячейки во входных данных. Для объектов зоны могут перекрываться. Области не обязательно должны быть непрерывными.

Выберите поле для определения границ

Поле, которое содержит значения, определяющие каждую зону.

Это может быть целочисленное или строковое поле набора данных зоны.

Выберите слой растра для суммирования

Растр, содержащий значения, на основании которых будет вычислена статистика.

Может быть целочисленным или с плавающей точкой. Но если тип данных – числа с плавающей точкой, не будут вычислены такие зональные характеристики, как большинство, меньшинство и дисперсия.

Выберите статистику для вычисления

Задает вычисляемый тип статистики.

Вход может быть одним из следующих: Среднее, Большинство, Максимум, Медиана, Минимум, Меньшинство, Процентиль, Диапазон, Среднеквадратическое отклонение, Сумма или Разнообразие. По умолчанию это Среднее.

Значение процентиля

Вычисляемый процентиль. Значение по умолчанию – 90, определяющее 90-й процентиль.

Значение может варьироваться от 0 до 100. Результат указания 0-го процентиля, по существу, эквивалентен минимуму в статистике, а 100-го - максимуму. Значение 50 даст, по существу, тот же результат, что и медиана в статистике.

Этот параметр поддерживается, только если параметр Выбрать статистику для вычисления задан как Процентиль.

Игнорировать пропущенные значения при вычислениях

Определяет, будут ли игнорироваться пропущенные значения в анализе.

  • Отмечено – анализ будет включать ячейки, имеющие значение во входном растре в пределах любой определенной зоны, чтобы задавать выходное значение зоны. Ячейки NoData в растре значений будут игнорироваться в вычислении статистики. Используется по умолчанию.
  • Не отмечено – Результатом анализа будет значение NoData, если есть какие-либо значения NoData для ячеек растра значений в пределах определенной зоны.

Обработать как многомерный

Определяет, как будут обрабатываться входящие сервисы многомерных изображений, если они многомерные.

  • Отмечено – статистика будет вычисляться из текущего среза входного многомерного набора данных. Используется по умолчанию.
  • Не отмечено – статистика будет вычисляться для всех измерений входного многомерного набора данных.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
