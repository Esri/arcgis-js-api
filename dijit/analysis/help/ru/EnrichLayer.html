Обогатить слой

Этот инструмент обогащает ваши данные, собирая факты о людях, местах и деловой активности вокруг объектов данных, что дает возможность получать дополнительную информацию о местоположениях, которую вы не сможете получить, используя только карту. Например: что за люди живут здесь? Чем заняты люди, живущие в этом месте? Каков их образ жизни и окружающая среда? Какова деловая активность в этом месте?

В результате вы получите новый слой, содержащий всю демографическую и географическую информацию из полученных наборов данных. Эта новая информация добавляется в таблицу в виде полей.

Если опция Использовать текущий экстент карты включена, обогащение будет выполнено только для тех объектов входного слоя, которые видны в момент анализа в текущем экстенте. Если опция отключена, обогащение будет выполнено для всех объектов входного слоя, даже если они находятся вне текущего экстента карты.

Выберите слой для обогащения новыми данными.

Точечный, линейный или площадной слой, к которому будут добавлены новые атрибутивные данные.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Переменные обогащения

Щелкните Выбрать переменные, чтобы открыть Обзор данных. С помощью окна Обзор данных вы сможете находить и выбирать отдельные переменные обогащения или сразу несколько переменных.

Район выбирается автоматически на основании ваших данных. Если в ваших данных содержится информация о других странах, то вы можете выбрать другой район.

Для облегчения доступа переменные обогащения организованы в наборы данных. При выборе наборе данных вы увидите только несколько наиболее популярных переменных обогащения этого набора. Если щелкнуть Листать, то будут показаны категории в данном наборе. В каждой из категорий вы можете либо выбрать целую подкатегорию, либо перейти в одну из подкатегорий для выборки отдельных переменных обогащения. Вы можете также использовать окно поиска для поиска определенной переменной в категории.

По мере выборки переменных для обогащения, они будут добавляться в тележку для покупок, расположенную в углу браузера Обзор данных. Используйте тележку для покупок для наблюдения за количеством выбранных вами переменных.

Каждая выбранная вами переменная будет добавлена как поле в слое результатов.

Определите области для обогащения

Задайте область вокруг входных точек или линий, которые будут обогащаться, используя либо прямолинейное расстояние, либо режим движения. Для расстояния по прямой введите значение до 1000 миль (1609 км). Режимы передвижения могут использоваться, только если входной слой содержит точки. Доступные режимы передвижения задаются администратором вашей организации. При использовании одного из режимов движения, измерения проводятся вдоль дорог или пешеходных дорожек, также учитываются правила, такие как улицы с односторонним движением, запрет разворотов и т.д. Когда вы выберете способ передвижения и вид измерения, введите продолжительность или расстояние не более 5 часов (300 минут) или 300 миль (482.8 километров).

Если опция Возвращения результата в виде ограничивающих областей включена, результат получится в виде площадных объектов, определенных значениями времени или расстояния выше. Если опция не включена, результатом будут точечные объекты.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
