Водосборная область

Инструмент геообработки, который определяет область, расположенную выше набора ячеек растра.

Входная точка на реке Миссисипи в Луизиане создает пространственный объект водораздельной области, охватывающий районы от Нью-Йорка до Монтаны и Канады.

Выберите слой направления потока

Входной растр, показывающий направление стока для каждой ячейки.

Растр направления стока можно создать с помощью функции растра Направление стока, выбрав тип направления стока D8 по умолчанию.

Указание входных положений точек устьев

Входные положения точек устьев.

Это может быть растровый или точечный объект. Для растра этот параметр представляет ячейки, выше которых будет определяться водосборная область. В качестве входных ячеек будут использоваться все ячейки, значение которых отличается от значения NoData. Для набора данных точечных объектов этот параметр представляет местоположения, для которых будет определяться расположенная выше них водосборная область.

Выберите поле точек устьев

Поле, используемое для присвоения значений местоположениям точек устьев.

Если входной слой точек слоя является растром, то используется Value. Если же это объекты, то используется числовое поле. Если поле содержит значения с плавающей точкой, они будут округлены до целых чисел.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
