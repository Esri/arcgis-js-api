Создать многомерную аномалию

Вычисляет аномалию для каждого среза в существующем многомерном растре с целью создания нового многомерного растра.

Аномалия – это отклонение наблюдения от его стандартного или среднего значения.

Этот инструмент вычисляет аномалии во времени для одной или нескольких переменных в многомерном растре. Если в дополнение к измерению времени имеется измерение без времени, аномалия будет вычисляться на каждом шаге дополнительного измерения.

Например, у вас есть ежемесячные данные о температуре океана, собранные через 1 м до глубины 100 м, и вы хотите рассчитать такие температурные аномалии, как отклонения от среднегодового значения. Этот инструмент определит температурные аномалии на основе среднегодового значения, если вы установите Ежегодно для параметра временной интервал для вычисления среднего значения, и возвратит значения аномалий для каждой из 100 глубин.

Этот инструмент поддерживает только многомерные наборы растровых данных, которые имеют измерение времени.

Если отмечена опция Использовать текущий экстент карты, будут проанализированы только переменные слоя, видимые в текущем экстенте карты. Если опция отключена, будут анализироваться переменные всего слоя, даже если они находятся за пределами текущего экстента карты.

Выберите многомерный слой изображений для генерации аномалии

Слой входных изображений многомерного набора растровых данных.

Выберите переменные, для которых будут генерироваться аномалии.

Переменная или переменные, для которых будут рассчитываться аномалии. Если переменная не указана, будут проанализированы все переменные с измерением времени.

Выберите метод для генерации аномалий

Задает метод, который будет использоваться для вычисления аномалии.

  • Отличие от среднего – Будет рассчитана разница между значением пиксела и средним значением этого пиксела в срезах, которые определяются данным интервалом. Используется по умолчанию.
  • Процентное отличие от среднего – Будет рассчитана разница в процентах между значением пиксела и средним значением этого пиксела в срезах, которые определяются данным интервалом.
  • Процент от среднего – Будет рассчитан процент от среднего.
  • Z-оценка – Для каждого пиксела будет рассчитана Z-оценка. Z-оценка 0 указывает, что значение пиксела идентично среднему значению. Z-оценка 1 означает, что значение пиксела составляет 1 стандартное отклонение от среднего значения. Если z-оценка равна 2, значение пиксела равно 2 стандартным отклонениям от среднего значения и т. д.
  • Отличие от медианного – Будет рассчитана разница между значением пиксела и математическими медианными значениями этого пиксела в срезах, которые определяются данным интервалом.
  • Процентное отличие от медианного – Будет рассчитана разница в процентах между значением пиксела и математическими медианными значениями этого пиксела в срезах, которые определяются данным интервалом.
  • Процент от медианного – Будет рассчитан процент от математического медианного значения.

Далее перечислены математические методы расчета аномальных значений. Аномалии могут быть рассчитаны с использованием среднего или медианного значения в качестве определения среднего. Если распределение данных искажено, то на среднее значение могут сильно влиять выбросы, поэтому метод медианного значения или z-оценки может лучше подходить для этого типа данных.

  • Отличие от среднего = x - µ
    • x = значения пиксела в срезе
    • µ = среднее значение этого пиксела за заданный временной интервал
  • Процентное отличие от среднего = |x - µ| / [(x - µ)/2]
    • x = значения пиксела в срезе
    • µ = среднее значение этого пиксела за заданный временной интервал
    • |x - µ| = абсолютное значение разницы между текущим и средним значениями
  • Процент от среднего = x / µ
    • x = значения пиксела в срезе
    • µ = среднее значение этого пиксела за заданный временной интервал
  • Отличие от медианного = x - ß
    • x = значения пиксела в срезе
    • µ = медианное значение этого пиксела за заданный временной интервал
  • Процентное отличие от медианного = |x - ß| / [(x + ß)/2]
    • x = значения пиксела в срезе
    • µ = медианное значение этого пиксела за заданный временной интервал
    • |x - ß| = абсолютное значение разницы между текущим и медианным значениями
  • Процент от медианного = x / ß
    • x = значения пиксела в срезе
    • µ = медианное значение этого пиксела за заданный временной интервал
  • Z-оценка = (x - µ) / S
    • x = значения пиксела в срезе
    • µ = среднее значение этого пиксела за заданный временной интервал
    • S = стандартное отклонение значений пикселов за заданный интервал времени

Выберите временной интервал для расчета среднего

Задает временной интервал, который будет использоваться для вычисления среднего значения.

  • Все – Вычисляет среднее значение по всем срезам для каждого пиксела.
  • Ежегодно – Вычисляет среднегодовое значение для каждого пиксела.
  • Повтор ежемесячно – Вычисляет среднемесячное значение для каждого пиксела.
  • Повтор еженедельно – Вычисляет среднее значение за неделю для каждого пиксела.
  • Повтор ежедневно – Вычисляет среднее значение за сутки для каждого пиксела.
  • На часовой основе – Вычисляет среднечасовое значение для каждого пиксела.
  • Внешний растр – Приводится имеющийся набор растровых данных, который содержит среднее или медианное значение для каждого пиксела.

Выбор слоя изображений среднего значения в качестве базового

Задается базовый набор растровых данных, который содержит ранее вычисленное среднее значение для каждого пиксела. Аномалии будут вычислены в сравнении с этим средним.

Игнорировать пропущенные значения при вычислениях

Определяет, будут ли игнорироваться пропущенные значения в анализе.

  • Отмечено – анализ будет включать все действительные пикселы по данному измерению и игнорировать любые пикселы NoData. Используется по умолчанию.
  • Не отмечено – Результатом анализа будет значение NoData, если для этого пиксела по данному измерению есть хотя бы одно значение NoData.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
