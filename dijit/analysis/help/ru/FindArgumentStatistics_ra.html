Найти статистику аргументов

Извлекает значение измерения или индекс канала, при котором достигается заданная статистика для каждого пиксела в многомерном или многоканальном растре.

Используйте этот инструмент для извлечения значения измерения (например, даты, высоты или глубины), при котором достигается определенная статистика в стеке растров в многомерном наборе растровых данных. Или найдите число каналов, при котором эта статистика достигается в многоканальном растре.

Например, вам надо узнать месяц, в котором каждый пиксел достигал самой высокой температуры поверхности моря за 30 лет сбора данных. Другой пример – у вас восьмиканальное изображение данных дистанционного зондирования, и вы хотите узнать, в каком канале было достигнуто минимальное значение отражательной способности для каждого пиксела.

Если отмечена опция Использовать текущий экстент карты, анализироваться будут только переменные или каналы слоя, видимые в текущем экстенте карты. Если опция отключена, будут анализироваться переменные или каналы всего слоя, даже если они находятся за пределами текущего экстента карты.

Выбор многомерного или многоканального слоя изображений для анализа

Входной многомерный или многоканальный растр, который будет анализироваться.

Выбор измерения (переменная или канал) для извлечения статистики

Измерение, из которого будет извлекаться статистика. Этот параметр не требуется, если входной растр не является многомерным.

Выбор переменной (переменных) для анализа

Переменная или переменные, которые будут анализироваться. Если входной растр не является многомерным, значения пикселов многоканального растра считаются переменной. Если входной растр является многомерным, а переменная не задана, то будут анализироваться все переменные в пределах заданного измерения.

Например, чтобы найти годы, в которые значения температуры были самыми высокими, укажите температуру в качестве переменной для анализа. Если вы не укажете ни одной переменной, а у вас есть данные и по температуре, и по выпадению осадков, то анализироваться будут обе переменные, и выходной многомерный растр будет включать обе переменные.

Выбрать тип статистики

Задает статистику для извлечения из переменной или переменных по заданному измерению.

  • Аргумент минимума – будет получено значение величины, при котором достигается минимальное значение переменной. Это значение по умолчанию.
  • Аргумент максимума – будет получено значение величины, при котором достигается максимальное значение переменной.
  • Аргумент медианы – будет получено значение величины, при котором достигается медианное значение переменной.
  • Аргумент продолжительности – максимальная продолжительность, для которой значения переменной попадают между минимальным и максимальным значениями.

Множественное случайное значение

Значение пиксела, используемое для указания того, что данная статистика аргумента была достигнута более одного раза во входном наборе растровых данных. Если не указано, то значение пиксела будет значением величины, которое было достигнуто статистикой аргумента в первый раз.

Указать минимальное значение

Минимальное значение переменной, которое должно использоваться для получения продолжительности.

Этот параметр является обязательным, если в качестве типа статистики выбрана Продолжительность.

Указать максимальное значение

Максимальное значение переменной, которое должно использоваться для получения продолжительности.

Этот шаг является обязательным, если в качестве типа статистики выбрана Продолжительность.

Выбрать определение измерения

Определяет, как будет извлекаться статистика из заданного измерения.

  • Все – статистика будет извлекаться по всем измерениям. Это значение по умолчанию.
  • Ключевое слово интервала – статистика будет извлекаться из измерения времени в соответствии с ключевым словом интервала.

Выберите интервал ключевых слов

Единица времени, для которой будет извлекаться статистика.

Например, у вас есть пять лет ежедневных данных о температуре поверхности моря, и вы хотите узнать, в каком году наблюдалась максимальная температура. На этом шаге задайте тип статистики Аргумент максимума, задайте определение измерения Ключевое слово интервала, а интервал ключевого слова – Ежегодно.

Или же, если вы желаете узнать, в каком месяце постоянно наблюдалась самая высокая температура, задайте для этого шага тип статистики Аргумент максимума, задайте определение измерения Ключевое слово интервала, а интервал ключевого слова – Повтор ежемесячно. В результате будет создан растр, в котором каждый пиксел содержит месяц, в котором данная статистика наблюдалась за пятилетний период (например, 18.08.2008, 25.08.2016, 07.08.2013).

Этот шаг необходим, когда в качестве измерения, откуда будет извлекаться статистика, задан StdTime, а для определения способа извлечения этой статистики из измерения времени задано Ключевое слово интервала.

  • Повтор ежедневно – Статистика будет собираться по дням.
  • Повтор еженедельно – Статистика будет собираться по неделям.
  • Повтор ежемесячно – Статистика будет собираться по месяцам.
  • Повтор каждый квартал – Статистика будет собираться поквартально.
  • Ежечасно –Статистика будет извлечена за тот час, в течение которого она состоялась.
  • Ежедневно – Статистика будет извлечена за тот день, в течение которого она состоялась.
  • Еженедельно – Статистика будет извлечена за ту неделю, в течение которой она состоялась.
  • Ежемесячно – Статистика будет извлечена за тот месяц, в течение которого она состоялась.
  • Ежеквартально – Статистика будет извлечена за тот квартал, в течение которого она состоялась.
  • Ежегодно – Статистика будет извлечена за тот год, в течение которого она состоялась.

Игнорировать пропущенные значения при вычислениях

Определяет, будут ли игнорироваться пропущенные значения в анализе.

  • Если поле отмечено, анализ будет включать все действительные пикселы по данному измерению и игнорировать любые пикселы NoData. Это значение по умолчанию.
  • Если поле не будет отмечено, то результатом анализа будет значение NoData, если для этого пиксела по данному измерению есть хотя бы одно значение NoData.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
