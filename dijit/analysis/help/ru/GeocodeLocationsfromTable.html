Геокодирование адресов из таблицы

Геокодирование – это способ преобразования адресов в координаты. Этот инструмент геокодирует адреса из таблицы, файла CSV, или файла Excel в слой объектов или в выходной файл, где координаты записываются в дополнительных столбцах.

Выбор входной таблицы

Входная таблица с адресами для геокодирования. Входной файл должен быть таблицей, файлом CSV или файлом Excel.

Выбор локатора

Выберите сервис локатора для геокодирования. Если сервис локатора, который вы хотите использовать, не находится в ниспадающем меню, свяжитесь с администратором, чтобы сервис локатора настроили для пакетного геокодирования.

Страна

Если все ваши адреса находятся в одной стране, выберите эту страну, чтобы обеспечить более точные результаты. В противном случае, оставьте этот параметр пустым.

Выбор полей данных

Сервис локатора ищет определенные поля в данных, например, адрес, город, штат и почтовый индекс. Укажите, какие поля данных в вашем входном файле или таблице должны соответствовать каким полям для локатора, используя ниспадающие списки. Чем больше соответствий полей вы укажите, тем точнее будут ваши результаты. Если входная таблица содержит все данные в одном столбце, выберите Одно поле и выберите одно поле данных из ниспадающего меню. В другом случае выберите Несколько полей и укажите соответствие полей локатора и столбцов в вашей таблице.

Выбор выходного формата

Формат выходных результатов. Если вы выбираете создание сервиса объектов, адреса из входной таблицы будут добавлены на карту как слой точечных объектов. Опции CSV и XLS добавляют файлы с результатами задания геокодирования в Мои ресурсы, когда задание будет выполнено.

Имя выходного результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
