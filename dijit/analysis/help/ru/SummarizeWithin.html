Суммировать в пределах

Этот инструмент находит объекты (и части объектов), которые расположены в границах областей первого входного слоя. Например:

  • Используя слой границ водоразделов и слой с границами типов землепользования, можно определить общую площадь земель различных типов для каждого водораздела.
  • Используя слой земельных участков в округе и слой с границами городов, можно вычислить среднюю стоимость свободных участков в границах городов.

Если опция Использовать текущий экстент карты включена, будут анализироваться только те объекты входного слоя и слоя объединения, которые видны в момент анализа в текущем экстенте. Если опция отключена, будут анализироваться все объекты входного слоя и слоя объединения, даже если они находятся вне текущего экстента карты.

Выберите площадной слой для суммирования объектов в пределах границ областей

Площадной слой, который будет использоваться для суммирования объектов, попадающих в пределы границ областей.

Вы можете выбрать между заданным слоем областей или бинами, которые генерируются при работе инструмента. При генерации бинов, заданное для опции Квадрат число и единицы измерения определяют размеры квадрата. Для Шестиугольника число и единицы измерения определяют расстояние между параллельными сторонами.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Выберите слой для суммирования

Объекты этого слоя, которые расположены в границах объектов указанного выше входного слоя, будут суммированы.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Добавить статистику из слоя для суммирования

Дополнительно вы можете вычислить статистику числовых атрибутов точек, которые укладываются в пределы границ объектов.

При включении опции Число точек будет вычислено общее число точек в границах каждого объекта. Кроме того, вы можете вычислить статистику числовых атрибутов точек, которые укладываются в границы областей.

  • Сумма – суммарное значение указанного поля.
  • Среднее – вычисляет среднее арифметическое указанного поля.
  • Минимум – определяет минимальное значение указанного поля.
  • Максимум – определяет максимальное значение указанного поля.
  • Средне-квадратическое отклонение – средне-квадратическое отклонение, вычисленное для значений в указанном поле.

Если в слое суммирования отсутствуют числовые поля, дополнительная статистика будет недоступна.

Если в слое суммирования отсутствуют числовые поля, дополнительная статистика будет недоступна.

  • Средне-квадратическое отклонение – средне-квадратическое отклонение, вычисленное для значений в указанном поле.

Если в слое суммирования отсутствуют числовые поля, дополнительная статистика будет недоступна.

Добавить статистику из

Выбрав слой для суммирования, дополнительно вы можете вычислить статистику любых числовых атрибутов точек, линий или полигонов, которые попадают в пределы границ входного полигонального слоя.

  • Сумма – суммарное значение указанного поля
  • Среднее – вычисляет среднее арифметическое указанного поля
  • Минимум – определяет минимальное значение указанного поля
  • Максимум – определяет максимальное значение указанного поля
  • Средне-квадратическое отклонение – средне-квадратическое отклонение, вычисленное для значений в указанном поле.

В зависимости от типа объектов для суммирования, вы можете вычислить общее количество точек, общую длину линий или общую площадь.

Если в слое суммирования отсутствуют числовые поля, дополнительная статистика будет недоступна.

Выбрать поле для группировки

Это атрибут объектов Суммирования, который может использоваться для вычисления статистики отдельно для каждого уникального значения атрибута. Например, предположим первый входной слой содержит границы городов и объектами суммирования являются земельные участки. Один из атрибутов земельных участков – Status, который содержит два значения: VACANT и OCCUPIED (Свободен и Занят). Для вычисления общей площади свободных и занятых участков в границах городов, используйте Status для группировки по атрибуту. Статистика для каждой группы, наряду с количеством объектов в пределах границ каждой области, будет отображена во всплывающем окне результатов.

Добавить меньшинство, большинство

Эта опция активна, когда вы выбираете поле для группировки. Если вы хотите определить, какие атрибутивные значения в пределах каждой группы первого входного слоя анализа составляют меньшинство или большинство, выберите Добавить меньшинство, большинство. Если вы указали опцию Добавить меньшинство, большинство, к результирующему слою будут добавлены два новых поля. Если вы также выберите Добавить процент, к результирующему слою будут добавлены еще два дополнительных поля, содержащих величины в процентах для большинства и меньшинства в пределах каждой группы.

  • Если вы выполняете суммирование точек, меньшинство и большинство вычисляется на основе количества точек в пределах каждой группы во входном площадном слое.
  • Если вы выполняете суммирование линий, меньшинство и большинство вычисляется на основе длины линий в пределах каждой группы во входном площадном слое.
  • При суммировании площадей меньшинство и большинство вычисляется на основе суммы площадей в пределах каждой группы во входном площадном слое.

Добавить проценты

Эта опция активна, когда вы выбираете поле для группировки. Выберите Добавить проценты если вы хотите вычислить величину в процентах для каждого атрибутивного значения в пределах каждой группы. К результирующей таблице будет добавлено новое поле, содержащие величину в процентах для каждого атрибутивного значения в пределах каждой группы. Если вы также выберите Добавить меньшинство, большинство, к результирующему слою будут добавлены еще два дополнительных поля, содержащих величины в процентах для большинства и меньшинства в пределах каждой группы.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
