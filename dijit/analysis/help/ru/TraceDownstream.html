Трассировать вниз по течению

Этот инструмент определяет трассу, или путь потока, вниз по течению из точек входного слоя.

Например, предположим, у вас есть точечные объекты, представляющие источники загрязнений, и вы хотите определить, где в изучаемой области могут пройти загрязненные потоки. Вы можете использовать инструмент Трассировка вниз по течению для определения пути прохождения загрязнений. Этот путь также можно разделить на отдельные сегменты линий, задав расстояние и единицы измерения. Возвращаемая линия может иметь длину всего потока, заданную максимальную длину, или обрезаться по вырезающим объектам, таким как изучаемая область. Во многих случаях, если возвращается общая длина потока, она соответствует пути от источника до океана.

В результате вы получаете линию, отображающую путь потока вниз по течению. Если для параметра Разбивать путь на сегменты задано значение длины, линия будет разделена на отрезки указанной длины. Например, если входная точка обозначает место утечки загрязнений, а река, в которую они попадают, имеет скорость течения 0.6 мили в час, или 15 миль в день, может быть удобно разбить линию реки на участки по 15 миль. При этом вы будете видеть, где находятся загрязнения в каждый следующий день. Линия будет обозначена градуированными цветами на основе расстояния и содержать надписи, в соответствии с расстоянием от начала потока.

Полученный поток будет иметь все атрибуты входных точек, а также дополнительные атрибуты.

  • PointID – содержит ID из входной точки.
  • LengthKm – содержит общую длину нисходящего потока в километрах.
  • Description и DataResolution – содержат информацию, относящуюся к гидрологическим данным, используемым для расчета трассировки.
  • FromDistance и ToDistance – содержат расстояния от и до для каждого сегмента. Эти атрибуты сохраняются, если используется опция Разбивать путь на сегменты.

Если опция Использовать текущий экстент карты включена, будут анализироваться только то точечные объекты входного слоя, которые отображаются в текущем экстенте. Если опция отключена, будут использоваться все объекты входного слоя, даже если они находятся вне текущего экстента карты.

Точечные объекты, которые используются как начальные точки для трассировки

Точечные объекты, которые используются как начальные точки для трассировки. Выходная таблица будет содержать одну запись для каждой входной точки.

Выбирая слой для вашей карты, вы можете также щелкнуть Выбрать слои анализа в нижней части ниспадающего списка. Эта опция открывает галерею, где вы можете найти и выбрать наборы данных из разных источников для своего анализа.

Разбивать путь на сегменты

Линия потока будет разбита на сегменты определенной длины. Например, если входная точка обозначает место утечки загрязнений, а река, в которую они попадают, имеет скорость течения 0.6 мили в час, или 15 миль в день, может быть удобно разбить линию реки на участки по 15 миль. При этом вы будете видеть, где находятся загрязнения в каждый следующий день. Линия будет обозначена градуированными цветами на основе расстояния и содержать надписи, в соответствии с расстоянием от начала потока. Линия полученного потока будет состоять из нескольких сегментов, у каждого из которых будут поля FromDistance и ToDistance.

Максимальное расстояние трассировки

Определяет общую длину возвращаемой линии. Если вы укажите площадные объекты для вырезания потока, линия будет обрезана в соответствии с этим объектом, независимо от заданной здесь длины.

Вырезать результат по

Нарисуйте либо укажите слой с областью, по которой необходимо обрезать поток. Например, если вам нужны результаты только для изучаемой области, например, округа, укажите объект-округ, чтобы трассировка закончилась при достижении его границы.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.

Запуск этого инструмента не использует кредиты.
