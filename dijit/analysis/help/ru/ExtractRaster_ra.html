Извлечь растр

Этот инструмент вырезает растр по границе, заданной либо как прямоугольная область, либо в соответствии с фигурой, обведенной на экране. Вы можете вырезать растр по текущему экстенту или по области изучения, заданной полигоном.

Входной слой для вырезания.

Задать вырезающую геометрию

Выберите тип экстента, использующегося при вырезании геометрии слоя.

  • Использовать текущий экстент – для вырезания слоя используется текущий экстент карты. Это значение по умолчанию.
  • Выбрать область изучения – позволяет задать вырезающую геометрию с помощью инструмента рисования.
  • Выбрать слой – выберите векторный слой или слой изображений для определения экстента или выберите полигональный векторный слой для определения вырезающей геометрии. Если выбран полигональный векторный слой, отображается флажок, с помощью которого можно указать, следует ли использовать входные объекты для вырезающей геометрии. По умолчанию не отмечено. Это означает, что для определения экстента используется входной векторный слой. Если отмечено, входные объекты используются для вырезающей геометрии.

Указать тип вырезания

Укажите, хотите ли вы сохранить область внутри либо снаружи вырезающей геометрии.

  • Данные внутри геометрии — будут извлечены части изображения внутри экстентов вырезания. Это значение по умолчанию.
  • Данные вне геометрии — будут извлечены части изображения вне экстентов вырезания.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если имя слоя уже используется, появится запрос ввести новое имя.

Вы можете указать имя папки в Моих ресурсах, где будет сохранен результат, с помощью ниспадающего списка Сохранить результат в. Если у вас есть права на создание слоев листов изображений и слоев динамических изображений, вы можете выбрать типы генерируемых выходных слоев, используя ниспадающий список Сохранить результат как.
