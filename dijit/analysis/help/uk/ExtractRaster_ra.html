Виділити растр

Цей інструмент вирізає растр по кордону, заданому або як прямокутна область, або у відповідності з фігурою, обведеною на екрані. Ви можете вирізати растр за поточним екстентом області, який відображається на карті, або за областю вивчення, заданою полігоном.

Вказати геометрію вирізання

Виберіть тип екстенту для вирізання геометрії шару:

  • Використовувати поточний екстент—Поточний екстент карти буде використовуватися для вирізання результуючого шару. Це значення за замовчуванням.
  • Вибрати область, що вивчається — дозволяє визначати геометрію вирізання, використовуючи інструмент малювання.
  • Вибрати шар — вибрати векторний шар або шар зображень для визначення екстенту або вибрати векторний шар полігонів для визначення геометрія вирізання. Коли вибрано векторний шар полігонів, відображається прапорець для вказання того, чи треба використовувати вхідні об'єкти для геометрії вирізання. За замовчуванням, прапорець знято. Це означає, що вхідний векторний шар використовується для визначення екстенту. Якщо прапорець встановлено, це означає, що вхідні об'єкти використовуються для геометрії вирізання.

Вказати тип вирізання

Вкажіть, де слід зберегти область: всередині чи зовні геометрії вирізання:

  • Дані всередині геометрії — витягатимуться зображення всередині екстентів вирізання. Це значення за замовчуванням.
  • Дані поза геометрії — витягатимуться зображення поза екстентів вирізання.

Ім'я шару результатів

Назва шару, який буде створено в Мої ресурси і додано до карти. Ім'я за замовчуванням визначається на основі імені інструменту та імені вхідного шару. Якщо шар уже існує, вам буде запропоновано вказати іншу назву.
