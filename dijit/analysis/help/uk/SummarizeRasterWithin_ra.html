Підсумувати растр в межах

Цей інструмент підсумовує значення растру в межах зон іншого набору даних.

Деякі з прикладів застосування:

  • З урахуванням шару водозбірних кордонів та шару кордонів землекористування за типом землекористування, обчисліть загальну площу типу землекористування для кожної водозбірної області.
  • З урахуванням шару ділянок в окрузі та шару кордонів міста підсумуйте середнє значення вільних ділянок в межах кордону кожного міста.

Якщо вибрано опцію Використовувати поточний екстент карти, будуть аналізуватись тільки ті об’єкти у вхідному шарі та шарі для підсумування, які видимі в межах поточного екстенту карти. Якщо цю опцію не вибрано, будуть використовуватись всі об'єкти у вхідному шарі та шарі для підсумування, навіть якщо вони знаходяться за межами поточного екстенту карти.

Вибрати шар областей для підсумування растрового шару в визначених кордонах

Шар, який визначає кордони областей або зон для підсумування.

Шар може представляти собою растр або векторні дані. Для растрів зони визначаються всіма місцями розташування у вхідних даних, назви осередків яких співпадають. Області не обов'язково повинні бути неперервними.

Вибрати поле для визначення кордонів

Це атрибут шару, який буде використаний для визначення кордонів областей.

Наприклад, припустимо, перший вхідний шар визначає кордони одиниці управління і має атрибути, які визначають регіон, район і ідентифікатор ділянки кожної одиниці. Також є растровий шар, який визначає індекс біорізноманіття для кожного місця розташування. Обравши відповідне поле, можна обчислити середнє біорізноманіття на локальному, районному або регіональному рівні.

Вибрати растровий шар для підсумування

Осередки растру у цьому шарі будуть підсумовані за областями (зони), в які вони укладаються.

Вибрати статистику для обчислення

Після вибору шару для підсумування ви можете додаткового обчислити статистику для будь-яких числового атрибуту точок, ліній або областей в межах вхідного шару областей.

Якщо обране поле містить ціле значення, доступні такі типи статистики як Середнє значення, Більшість, Максимум, Мінімум, Меншість, Процентиль, Діапазон, Стандартне відхилення, Сума і Дисперсія.

Якщо поле містить дані з плаваючою точкою, доступні такі опції як Середнє значення, Максимум, Мінімум, Діапазон, Стандартне відхилення і Сума.

Процентильне значення

Процентиль, який треба розрахувати. За замовчуванням: 90, означає 90-й процентиль.

Значення можуть знаходитися в діапазоні від 0 до 100. 0-й процентиль по суті еквівалентний статистиці «Мінімум», а 100-й процентиль еквівалентний максимуму. Значення 50 по суті даватиме такий самий результат, що й статистика «Медіана».

Ця опція підтримується тільки якщо для параметру Обрати статистику для розрахунку задано Процентиль.

Ігнорувати відсутні значення в обчисленні

Якщо обрати ігнорування відсутніх значень, при визначенні вихідного значення для цієї області будуть підсумовуватися тільки осередки зі значенням у шарі. У протилежному випадку, якщо будь-де в області є відсутні значення, буде вважатися, що інформації для статистичного обчислення всіх осередків у цій зоні недостатньо, і результатом для цієї зони буде нульове значення (NoData).

Обробляти як багатовимірний

Указує, як будуть оброблятися вхідні растри, якщо вони є багатовимірними. Якщо обрана обробка багатовимірних даних, статистика розраховуватиметься для всіх вимірів вхідного набору багатовимірних даних. В іншому випадку статистика розраховуватиметься, виходячи з поточного зрізу вхідного набору багатовимірних даних. Даний спосіб обраний за замовчуванням.

Ім'я шару результатів

Назва шару, який буде створено в Мої ресурси і додано до карти. Ім'я за замовчуванням визначається на основі імені інструменту та імені вхідного шару. Якщо шар уже існує, вам буде запропоновано вказати іншу назву.

Використовуючи розкривний список Зберегти результат у, можна вказати ім'я папки в Мій вміст, де зберігатиметься результат.
