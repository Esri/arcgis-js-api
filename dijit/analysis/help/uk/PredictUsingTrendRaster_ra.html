Прогнозувати з використанням растру тренду

Обчислює прогнозований багатовимірний растр з використанням вихідного растру тренду, отриманого за допомогою інструменту Згенерувати растр тренду.

Якщо вибрано опцію Використовувати поточний екстент карти, будуть аналізуватися тільки змінні шару, видимі в межах поточного екстенту карти. Якщо цю опцію не вибрано, будуть аналізуватися всі змінні шару, навіть якщо вони знаходяться за межами поточного екстенту карти.

Вибрати багатовимірний шар зображень для прогнозування

Вхідний багатовимірний растр тренду з інструменту Згенерувати растр тренду.

Вибрати змінну(-нні), яка прогнозуватиметься.

Змінна або змінні, що прогнозуватимуться в аналізі. Якщо змінні не вказані, використовуватимуться всі дані.

Вибрати метод, використовуваний для надання прогнозних значень виміру

Вказує метод, який використовується для надання прогнозних значень виміру.

  • За значенням — прогноз розраховуватиметься для єдиного значення виміру або списку значень виміру, вказаного на наступному кроці. Це значення за замовчуванням. Наприклад, ви бажаєте розрахувати річну кількість опадів для 2050, 2100 та 2150 року.
  • За інтервалом — прогноз розраховуватиметься для інтервалу виміру, визначеному початковим та кінцевим значенням. Наприклад, ви бажаєте розрахувати річну кількість опадів для кожного року між 2050 та 2150.

Вказати значення виміру для прогнозування

Одне або декілька значень виміру, що треба включити в прогноз.

Значення часу, глибини та висоти мають відповідати формату значень виміру, що використовуються для генерування растру тренду. Якщо растр тренду згенеровано для виміру StdTime, форматом має бути РРРР-ММ-ДДЧГГ:ХХ:СС, наприклад, 2050-01-01Ч00:00:00. Декілька значень відокремлюються крапкою з комою.

Цей параметр потрібний, коли для визначення значень виміру для прогнозування вибрано метод За значенням.

Початок

Початкова дата, висота або глибина інтервалу виміру, що треба використовувати в прогнозі.

Кінець

Кінцева дата, висота або глибина інтервалу виміру, що треба використовувати в прогнозі.

Визначте інтервал виміру для прогнозування

Визначте початкову та кінцеву дату, висоту або глибину інтервалу виміру, що треба використовувати в прогнозі.

Вказати число кроків між початковим та кінцевим значеннями

Число кроків між двома значеннями виміру, що треба включити в прогноз. Значення за замовчуванням 1.

Наприклад, для прогнозування значень температури за кожні п’ять років використовуйте значення 5.

Вибрати одиницю, що використовуватиметься для інтервалу часових значень

Вказує одиницю, що використовуватиметься для інтервалу значень. Це застосовується, тільки коли виміром аналізу є вимір часу.

Ім'я шару результатів

Назва шару, який буде створено в Мої ресурси і додано до карти. Ім'я за замовчуванням визначається на основі імені інструменту та імені вхідного шару. Якщо шар уже існує, вам буде запропоновано вказати іншу назву.
