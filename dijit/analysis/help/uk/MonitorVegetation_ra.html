Контролювати рослинність

Цей інструмент виконує арифметичну операцію над каналами багатоканального растрового шару для пошуку інформації про рослинний покрив в області інтересу.

Вибрати вхідні дані

Вибрати багатоканальний растровий шар. Переконайтеся, що у вхідному растрі доступні належні канали.

Вибрати метод для контролю рослинності

Виберіть метод для обчислення індексного шару рослинності. Різноманітні індекси рослинності можуть допомогти виділити певні об’єкти або зменшити різноманітні шуми.

  • Global Environmental Monitoring Index — GEMI — це нелінійний індекс рослинності для глобального моніторингу навколишнього середовища за супутниковими знімками. Він подібний до NDVI, але менш чутливий до впливу атмосфери. На нього впливає оголений ґрунт, тому його не рекомендується використовувати в областях з рідкою або помірно густою рослинністю.
  • Green Vegetation Index - Landsat TM — GVI був первісно розроблений на основі знімків Landsat MSS і модифікований для знімків Landsat TM. Він також відомий як індекс зеленої рослинності Landsat TM Tasseled Cap. Його можна також застосовувати до знімків, канали яких мають такі самі спектральні характеристики.
  • Modified Soil Adjusted Vegetation Index — MSAVI2 — це індекс рослинності, який намагається мінімізувати вплив оголеного ґрунту на метод SAVI.
  • Normalized Difference Vegetation Index — NDVI — це стандартизований індекс, який дозволяє генерувати зображення, що показує зелений покрив або відносну біомасу. Цей індекс використовує контрастність характеристик двох каналів з набору мультиспектральних растрових даних — поглинання пігментом хлорофілу в червоному каналі та високу віддзеркалювальну здатність рослинних матеріалів у ближньому інфрачервоному каналі (NIR).
  • Perpendicular Vegetation Index — PVI подібний до різницевого індексу рослинності, однак він чутливий до атмосферних відхилень. При використанні цього методу для порівняння різних зображень його слід застосовувати тільки до зображень, скоригованих за атмосферою. Ця інформація може бути надана вашим постачальником даних.
  • Soil-Adjusted Vegetation Index — SAVI — це індекс рослинності, який намагається мінімізувати вплив яскравості ґрунту за допомогою коефіцієнта корекції яскравості ґрунту. Він часто використовується у пустельних областях, де рослинний покрив незначний.
  • Sultan's Formula — При методі Султана береться шестиканальне 8-бітне зображення і за допомогою спеціального алгоритму створюється трьохканальне 8-бітне зображення. В отриманому зображенні виділяються скельні утворення на узбережжях, які називаються офіоліти. Ця формула була розроблена на основі каналів TM і TM сцени Landsat 5 або 7.
  • Transformed Soil-Adjusted Vegetation Index — Перетворений SAVI — це індекс рослинності, який намагається мінімізувати вплив яскравості ґрунту шляхом припущення, що лінія ґрунту має довільний ухил і перетин.
  • Visible Atmospherically Resistant Index — VARI — це індекс рослинності, який оцінює наявність рослинності, використовуючи лише інформацію з видимого спектра. Для цього методу потрібні канали в червоній, зеленій та синій частинах спектра.

Указати індекси каналів для ближнього ІЧ-каналу та червоного каналу.

Багатоканальні супутникові або повітряні датчики фіксують інформацію, яка розбита на канали довжини хвилі, які ідентифікуються за індексами каналів. Для цього методу моніторингу рослинності потрібно, щоб були задані індекси каналів для каналів довжини хвилі близької ІЧ і червоної частини спектру.

Багатоканальні супутникові або повітряні датчики фіксують інформацію, яка розбита на канали довжини хвилі, які ідентифікуються за індексами каналів. Указати індекси каналів для синього, зеленого, червоного, NIR, КХІЧ-1 і КХІЧ -2 каналів.

Багатоканальні супутникові або повітряні датчики фіксують інформацію, яка розбита на канали довжини хвилі, які ідентифікуються за індексами каналів. Для індексу зеленої рослинності необхідно вказати індекси каналів для синього, зеленого, червоного, ближнього інфрачервоного (NIR), короткохвильового інфрачервоного (КХІЧ-1, КХІЧ-2) та термального каналів довжини хвилі.

Ухил лінії ґрунту

Ухил лінії ґрунту. Ухил — це наближене лінійне відношення між ближнім ІЧ і червоним каналом на діаграмі розсіяння.

Цей параметр дійсний тільки для методу «Перетворений індекс рослинності з корекцією за ґрунтом».

Перетин

Значення близького ІЧ-каналу, коли значення віддзеркалення червоного (Red) каналу становить 0 для певних ліній ґрунту.

Кількість зеленого рослинного покриву

Визначає кількість зеленого рослинного покриву, яка потрібна для методу індексу рослинності з поправкою на ґрунт.

Допустимі значення:

  • 1 = Області без зеленого рослинного покриву
  • 0,5 = Області з помірним зеленим рослинним покривом
  • 0 = Області з щільним зеленим рослинним покривом

Коефіцієнт поправки

Задає коефіцієнт поправки, який допомагає мінімізувати вплив ґрунту, що потрібно для методу «Перетворений індекс рослинності з корекцією за ґрунтом». Значення за замовчуванням 0,08.

Низьке значення означає, що вплив ґрунту ігнорується. Високе значення означає, що є вплив ґрунту на результат.
