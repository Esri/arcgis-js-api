Determine Optimum Travel Cost Network

Цей інструмент обчислює мережу з оптимальною вартістю переміщення на основі набору вхідних регіонів.

One example application of this tool is finding the best network for emergency vehicles.

If Use current map extent is checked, only those locations in the input layer that are visible within the current map extent will be analyzed. If unchecked, all locations in the input layer will be analyzed, even if they are outside the current map extent.

Choose raster or feature location layer to determine travel paths between

The input regions that are to be connected by the least-cost network.

Regions can be defined by either an image service or a feature service.

For image service, the regions are defined by groups of contiguous (adjacent) cells of the same value. Each region must be uniquely numbered. The cells that are not part of any region must be NoData. The raster type must be integer, and the values can be either positive or negative.

For feature service, the input can be point, line or polygon.
