Згенерувати багатовимірну аномалію

Обчислює аномалію для кожного зрізу в існуючому багатовимірному растрі для генерування нового багатовимірного растру.

Аномалія — ​​це відхилення спостереження від його стандартного, середнього або медіанного значення.

Цей інструмент обчислює аномалії в часі для однієї або більше змінних в багатовимірному растрі. Якщо на додаток до виміру часу є вимір без часу, аномалія буде обчислюватися на кожному кроці додаткового виміру.

Наприклад, у вас є щомісячні дані з температури океану, зібрані через кожний 1 м до глибини 100 м, і ви хочете розрахувати такі температурні аномалії, як відхилення від середньорічного значення. Цей інструмент визначить температурні аномалії на основі середньорічного значення, якщо ви встановите Щорічно для параметру Часовий інтервал розрахунку середнього значення, і поверне значення аномалій для кожної зі 100 глибин.

Цей інструмент підтримує тільки багатовимірні набори растрових даних, в яких є вимір часу.

Якщо вибрано опцію Використовувати поточний екстент карти, будуть аналізуватися тільки змінні шару, видимі в межах поточного екстенту карти. Якщо цю опцію не вибрано, будуть аналізуватися всі змінні шару, навіть якщо вони знаходяться за межами поточного екстенту карти.

Обрати багатовимірний шар зображень для генерування аномалії

Вхідний шар зображень багатовимірного набору растрових даних.

Обрати змінну(і), для яких будуть згенеровані аномалії

Змінна або змінні, для яких будуть обчислюватися аномалії. Якщо не вказано жодних змінних, будуть проаналізовані всі змінні з виміром часу.

Обрати метод для генерування аномалії

Задає метод, який буде використовуватися для обчислення аномалії.

  • Відмінність від середнього — розраховуватиметься різниця між значенням пікселя та середнім значенням цього пікселя по зрізах, визначених інтервалом. Це значення за замовчуванням.
  • Відсоткова відмінність від середнього — розраховуватиметься відсоткова різниця між значенням пікселя та середнім значенням цього пікселя по зрізах, визначених інтервалом.
  • Відсоток від середнього — розраховуватиметься відсоток від середнього.
  • Z-оцінка — розраховуватиметься z-оцінка для кожного пікселя. Z-оцінка 0 указує на те, що значення пікселя дорівнює середньому значенню. Z-оцінка 1 указує на те, що значення пікселя становить 1 середньоквадратичне відхилення від середнього значення. Z-оцінка 2 указує на те, що значення пікселя становить 2 середньоквадратичних відхилення від середнього значення, і так далі.
  • Відмінність від медіани — розраховуватиметься різниця між значенням пікселя та математичною медіаною значень цього пікселя по зрізах, визначених інтервалом.
  • Відсоткова відмінність від медіани — розраховуватиметься відсоткова різниця між значенням пікселя та математичною медіаною значень цього пікселя по зрізах, визначених інтервалом.
  • Відсоток від медіани — розраховуватиметься відсоток від математичної медіани.

Нижче наведені математичні методи для обчислення значень аномалії. Аномалії можуть розраховуватися з використанням середніх або медіанних значень як визначень середнього значення. Якщо розподіл даних несиметричний, середнє значення може сильно залежати від останців, тому для цього типу даних може краще підходити метод медіанного значення або z-оцінки.

  • Відмінність від середнього = x - µ
    • x = значення пікселя у зрізі
    • µ = середнє значення цього пікселя за заданий часовий інтервал
  • Відсоткова відмінність від середнього = |x - µ| / [(x + µ)/2]
    • x = значення пікселя у зрізі
    • µ = середнє значення цього пікселя за заданий часовий інтервал
    • |x - µ| = абсолютне значення різниці між значенням і середнім значенням
  • Відсоток від середнього = x / µ
    • x = значення пікселя у зрізі
    • µ = середнє значення цього пікселя за заданий часовий інтервал
  • Відмінність від медіани = x - ß
    • x = значення пікселя у зрізі
    • ß = медіана значень цього пікселя за заданий часовий інтервал
  • Відсоткова відмінність від медіани = |x - ß| / [(x + ß)/2]
    • x = значення пікселя у зрізі
    • ß = медіана значень цього пікселя за заданий часовий інтервал
    • |x - ß| = абсолютне значення різниці між значенням і медіаною
  • Відсоток від медіани = x / ß
    • x = значення пікселя у зрізі
    • ß = медіана значень цього пікселя за заданий часовий інтервал
  • Z-оцінка = (x - µ) / S
    • x = значення пікселя у зрізі
    • µ = середнє значення цього пікселя за заданий часовий інтервал
    • S = середньоквадратичне відхилення значень пікселя за заданий часовий інтервал

Обрати часовий інтервал для обчислення середнього значення

Задає часовий інтервал, який буде використовуватися для обчислення середнього значення.

  • Всі — розраховує середнє по всіх зрізах для кожного пікселя.
  • Щороку — розраховує середньорічне для кожного пікселя.
  • Повторюваний щомісячний — розраховує середньомісячне значення для кожного пікселя.
  • Повторюваний щотижневий — розраховує середньотижневе для кожного пікселя.
  • Повторюваний щоденний — розраховує середньоденне значення для кожного пікселя.
  • Щогодини — розраховує середньогодинне значення для кожного пікселя.
  • Зовнішній растр — дається посилання на існуючий набір растрових даних, який містить середнє або медіанне значення для кожного пікселя.

Вибрати шар зображень середнього в якості посилання

Вказує набір растрових даних, на який дається посилання, котрий містить раніше розраховане середнє для кожного пікселя. Аномалії розраховуватимуться в порівнянні до цього середнього.

Ігнорувати відсутні значення в обчисленні

Визначає, чи будуть ігноруватися відсутні значення в аналізі.

  • Вибрано—Аналіз включатиме всі дійсні пікселі за даним виміром та ігнорувати будь-які пікселі NoData. Це значення за замовчуванням.
  • Прапорець знято — результатом аналізу буде значення NoData, якщо для пікселів за даним виміром є хоча б одне значення NoData.
