Derive Aspect

Цей інструмент створює карту експозиції із джерела даних рельєфу. Експозиція встановлює напрямок ухилу максимальної швидкості зміни значень від кожного осередку до сусідніх з ним. Експозиція може розглядатися як напрямок ухилу. Значеннями вихідного растру будуть компасні напрямки експозиції.

Input Data

The elevation layer for which the aspect of each cell location will be calculated.

Result layer name
