Зразок

Цей інструмент створює таблицю або клас точкових об’єктів, який показує значення осередків з растру або набору растрів для визначених місць розташування. Місця розташування визначаються за осередками растру, полігональними об'єктами, полілінійними об'єктами або за набором точок.

Деякі з прикладів застосування цього інструменту:

  • Витягування атрибутів місця розташування вишок стільникового зв'язку з декількох растрів, таких як землекористування, тип грунту, висота та відстань до дороги.
  • Витягування температури ґрунту на різній глибині для заданого набору точок.
  • На основі 30-річних щомісячних даних про температуру поверхні моря, підведення підсумків в місцях перебування тварин або в межах ареалів за певний час, коли спостерігали види тварин.

Якщо прапорець Використовувати поточний екстент карти встановлено, аналізуватимуться змінні або канали шару, видимі в межах поточного екстенту карти. Якщо прапорець знято, аналізуватимуться змінні або канали всього шару, навіть якщо вони знаходяться за межами поточного екстенту карти.

Виберіть шар(и) зображень для відбору зразків

Єдинє зображення або список зображень, значення котрих відбиратимуться на основі вхідних даних місць розташування.

Для відбору всіх зрізів з багатовимірного шару зображень, він має бути єдиним вхідним растром.

Вибрати шар місць розташування

Дані, що ідентифікують положення, котрі ви бажаєте відібрати.

Вхідними даними місць розташування можуть бути растр або точка, полілінія або клас полігональних об’єктів.

Вказати поле унікальних ID

Поле, що містить різне значення для кожного місця розташування або об'єкта у вхідному растрі місць розташування або класі об'єктів.

Вказати інформацію про отримання даних місць розташування (опціонально)

Вкажіть час, глибину або інші дані отримання, пов’язані з осередками місць розташування або об’єктами.

Для кожного виміру є три опції вказування інформації про отримання, пов’язаної з осередками місць розташування або об’єктами:

  • Єдине поле або значення — при виборі цієї опції необхідно вказати наступну інформацію:
    • Поле або значення виміру — виберіть поле значень виміру з даних місць розташування або введіть конкретне значення виміру.
  • Єдине поле або значення з відносними значеннями — якщо вибрано цю опцію, інформацію нижче необхідно вказати разом для визначення діапазону, в межах якого розраховуватиметься статистика для змінних:
    • Поле або значення виміру — виберіть поле значень виміру з даних місць розташування.
    • Відносне значення до — вкажіть невід’ємне число, використовуване для віднімання від Поля або значення виміру для визначення початку діапазону значень виміру. Це значення представляє число днів, якщо виміром є час.
    • Відносне значення після — вкажіть невід’ємне число, використовуване для додавання до Поля або значення виміру для визначення кінця діапазону значень виміру. Це значення представляє число днів, якщо виміром є час.
  • Поля або значення початку та кінця — якщо вибрано цю опцію, інформацію нижче необхідно вказати разом для визначення діапазону, в межах якого розраховуватиметься статистика для змінних:
    • Поле або значення початку — виберіть поле значень виміру з даних місць розташування або введіть конкретне значення виміру для визначення початку діапазону значень виміру.
    • Поле або значення кінця — виберіть поле значень виміру з даних місць розташування або введіть конкретне значення виміру для визначення кінця діапазону значень виміру.

Цей параметр доступний лише тоді, коли параметр Вибрати шар(-и) зображень для вибірки є багатовимірним растром.

Вказати поле або значення буферної відстані

Вказане значення навколо об’єктів даних місць розташування. Буферна відстань вказується в лінійних одиницях вимірювання просторової прив’язки об’єкту місця розташування. Якщо об’єкт використовує географічну прив’язку, одиницями будуть градуси.

Статистика розраховуватиметься в межах цієї буферної області.

Цей параметр доступний лише тоді, коли Вибрати щар(-и) зображень для вибірки є багатовимірним растром.

Вибрати тип статистики

Тип статистики, що треба розрахувати.

  • Мінімум — знаходить мінімальне значення в межах вказаного діапазону.
  • Максимум — знаходить максимальне значення в межах вказаного діапазону.
  • Медіана — знаходить медіанне значення в межах вказаного діапазону.
  • Середнє — розраховує середнє значення для вказаного діапазону.
  • Сума — розраховує суму змінних в межах вказаного діапазону.
  • Більшість — знаходить значення, що зустрічається найчастіше.
  • Меншість — знаходить значення, що зустрічається найрідше.
  • Середньоквадратичне відхилення — розраховує середньоквадратичне відхилення.
  • Процентиль — розраховує визначений процентиль в межах вказаного діапазону.

Цей параметр доступний лише тоді, коли параметр Вибрати шар(-и) зображень для вибірки є багатовимірним растром, а параметрами Вибрати шар місця розташування є полілінії або полігони, або вибрано Вказати поле або значення буферної відстані (опціонально).

Процентильне значення

Це значення може знаходитися в діапазоні від 0 до 100. Значення за замовчуванням — 90.

Цей параметр доступний, тільки якщо для параметра Обрати тип статистики задано Процентиль.

Вказати метод перерахунку

Алгоритм перерахунку, використовуваний при перерахунку растру.

  • Найближчий — найближчий сусід, використовує значення найближчого осередку для призначення значення вихідному осередку при перерахунку. Це значення за замовчуванням.
  • Білінійний — білінійна інтерполяція, яка визначає нове значення осередку на основі зваженого середнього значення відстані центрів чотирьох найближчих вхідних осередків.
  • Кубічний — кубічна згортка, яка визначає нове значення осередку на основі проведення згладженої кривої через 16 центрів найближчих вхідних осередків.

Вказати вихідне компонування

Вказує, як відображатимуться відібрані значення у вихідній таблиці: в рядках або стовпцях.

  • Відібрані значення відображаються в рядку — відібрані значення відображаються в окремих рядках у вихідній таблиці. Це значення за замовчуванням.
  • Відібрані значення відображаються в стовпці — відібрані значення відображаються в окремих стовпцях у вихідній таблиці.

Цей параметр доступний , тільки коли параметр Вибрати шар(-и) зображень для вибірки є багатовимірним растром і містить одну змінну та один вимір, а кожен зріз є одноканальним растром.

Вказати вихідний тип

Вказує, що треба згенерувати: клас точкових об’єктів з відібраними значеннями в його таблиці чи тільки таблицю з відібраними значеннями.

  • Таблиця — згенерувати таблицю з відібраними значеннями. Це значення за замовчуванням.
  • Об'єкти — згенерувати клас точкових об’єктів з відібраними значеннями в його таблиці. Якщо параметрами Вибрати шар місця розташування є полілінії або полігони, кожна точка у вихідних даних є центроїдом кожної геометрії вхідних об’єктів місць розташування.

Ім'я шару результатів

Назва шару, який буде створено в Мої ресурси і додано до карти. Ім'я за замовчуванням визначається на основі імені інструменту та імені вхідного шару. Якщо шар уже існує, вам буде запропоновано вказати іншу назву.

Використовуючи розкривний список Зберегти результат у, можна вказати ім'я папки в Мій вміст, де зберігатиметься результат.
