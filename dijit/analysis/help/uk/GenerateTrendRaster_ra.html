Згенерувати растр тренду

Оцінює тренд для кожного пікселя за виміром для однієї або більше змінних у багатовимірному растрі.

Вибрати багатовимірний шар зображень для аналізу тренду

Вхідний шар зображень багатовимірного растру, що аналізуватиметься.

Виберіть вимір, за яким аналізуватиметься тренд змінних

Вимір, за яким витягатиметься тренд для змінної або змінних, вибраних в аналізі.

Вибрати змінну(-нні) для аналізу тренду

Змінна або змінні, для яких розраховуватимуться тренди. Якщо змінна не вказана, аналізуватиметься перша змінна в багатовимірному растрі.

Вибрати тип лінії для припасування змінних значень за виміром

Вказує тип лінії, що треба використовувати для припасування до значень пікселів за виміром.

  • Лінійна — змінні значення пікселів припасовуватимуться уздовж лінійної лінії тренду. Це значення за замовчуванням.
  • Поліноміальна — змінні значення пікселів припасовуватимуться уздовж лінії тренду поліному другого порядку.
  • Гармонічна — змінні значення пікселів припасовуватимуться уздовж гармонічної лінії тренду
  • Манн-Кендалл — змінні значення пікселів оцінюватимуться, використовуючи тест трендів Манна-Кендалла.
  • Сезонний-Кендалл — змінні значення пікселів оцінюватимуться, використовуючи тест трендів "Сезонний-Кендалл".

Вказати одиницю часу для тривалості сезонного періоду

Вказує одиницю часу, що використовуватиметься для тривалості сезонного періоду при виконанні тесту "Сезонний-Кендалл".

  • Дні — одиницею вимірювання тривалості сезонного періоду є дні. Це значення за замовчуванням.
  • Місяці — одиницею вимірювання тривалості сезонного періоду є місяці.

Вказати тривалість гармонічного циклу

Тривалість періодичного варіювання моделі. Цей параметр потрібний, коли для типу лінії тренду задано Гармонічний. Наприклад, зелений покрив часто має один стійкий цикл варіювання протягом одного року, тому тривалість циклу становить 1 рік. Годинні температурні дані мають один стійкий цикл варіювання протягом одного дня, тому тривалість циклу становить 1 день.

Тривалість за замовчування становить 1 рік для даних, що варіюються за річним циклом.

Цей параметр включається в аналіз тренду, тільки коли виміром, що аналізується, є час, а для типу тренду задано Гармонічний.

Виберіть одиницю часу тривалості гармонічного циклу

Вказує одиницю часу, яку необхідно використовувати для тривалості гармонічного циклу.

  • Дні — одиницею вимірювання для тривалості гармонічного циклу є дні.
  • Роки — одиницею вимірювання для тривалості гармонічного циклу є роки. Це значення за замовчуванням.

Цей параметр включається в аналіз тренду, тільки коли виміром, що аналізується, є час, а для типу тренду задано Гармонічний.

Вказати число частоти для припасування гармонічного тренду

Частоту, котру треба використовувати в припасуванні гармонічного тренду. Цей параметр вказує частоту циклів в році.

Значення за замовчуванням: 1 або один гармонічний цикл на рік.

Цей параметр включається в аналіз тренду, тільки коли виміром, що аналізується, є час.

Вказати число порядку полінома для припасування тренду.

Число порядку полінома, що треба використовувати в припасуванні поліноміального тренду.

Цей параметр включається в аналіз тренду, тільки коли виміром, що аналізується, є час.

Значення за замовчуванням: 2 або поліном другого порядку.

Вибрати статистику моделі, котру треба включити в растр тренду

Цей інструмент надає три типи статистики лінії припасування тренду: середньоквадратична помилка (RMSE), статистика за критерієм згоди за R-квадратом (R-квадрат) і статистика за p-значенням для коефіцієнту ухилу (P-значення коефіцієнту ухилу). Кожна статистика має відповідний прапорець для вказування того, чи розраховуватиметься та включатиметься вона в якості одного з декількох каналів в вихідному растрі тренду.

  • Середньоквадратична помилка (RMSE)

    • Прапорець встановлено — RMSE розраховуватиметься та включатиметься в якості одного з декількох каналів в вихідному растрі тренду. Це значення за замовчуванням.
    • Прапорець знято — RMSE не розраховуватиметься.

  • R-квадрат

    • Прапорець встановлено — значення R-квадрата розраховуватиметься та включатиметься в якості одного з декількох каналів в вихідному растрі тренду.
    • Прапорець знято — значення R-квадрата не розраховуватиметься. Це значення за замовчуванням.

  • P-значення коефіцієнту ухилу

    • Прапорець встановлено — p-значення розраховуватиметься та включатиметься в якості одного з декількох каналів в вихідному растрі тренду.
    • Прапорець знято — p-значення не розраховуватиметься. Це значення за замовчуванням.

Ігнорувати відсутні значення в обчисленні

Визначає, чи будуть ігноруватися відсутні значення в аналізі.

  • Вибрано—Аналіз включатиме всі дійсні пікселі за даним виміром та ігнорувати будь-які пікселі NoData. Це значення за замовчуванням.
  • Не вибрано—Результатом аналізу буде значення NoData, якщо для пікселів за даним виміром є хоча б одне значення NoData.
