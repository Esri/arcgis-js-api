Extraer ráster

Esta herramienta permite recortar un ráster por un límite, ya sea un área rectangular o una forma que defina interactivamente en la pantalla. Puede recortar por la extensión del área que se muestra en el mapa o por el área de estudio definida mediante un polígono.

Elija los datos de entada para extraer

La capa de entrada para recortar.

Especifique la geometría de recorte

Elija el tipo de extensión que se debe utilizar al recortar la geometría de la capa:

  • Usar extensión actual: se utilizará la extensión de mapa actual para recortar la capa resultante. Esta es la opción predeterminada.
  • Elegir área de estudio: le permite definir una geometría de recorte utilizando la herramienta de dibujar.
  • Elegir capa: elija una capa de entidades o una capa de imágenes para definir la extensión o una capa de entidades poligonales para definir la geometría de recorte. Cuando se selecciona una capa de entidades poligonales, se muestra una casilla de verificación para especificar si se quieren usar entidades de entrada para recortar la geometría. De forma predeterminada, está desactivado. Significa que se utiliza la capa de entidades de entrada para definir la extensión. Si está activado, significa que se utilizan las entidades de entrada para recortar la geometría.

Especificar tipo de recorte

Especifique si desea conservar el área de dentro o fuera de su geometría de recorte:

  • Datos dentro de la geometría: se extraerán las imágenes que queden dentro de las extensiones de recorte. Esta es la opción predeterminada.
  • Datos fuera de la geometría: se extraerán las imágenes que queden fuera de las extensiones de recorte.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en. Si tiene privilegios para crear capas de imágenes tanto en teselas como dinámicas, también puede especificar qué tipo de capa se va a generar para la salida utilizando el cuadro desplegable Guardar el resultado como.
