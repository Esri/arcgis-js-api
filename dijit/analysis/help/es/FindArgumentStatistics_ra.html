Hallar estadísticas de argumentos

Extrae el valor de dimensión o el índice de banda en el que se consigue una determinada estadística para cada píxel de un ráster multidimensional o multibanda.

Utilice esta herramienta para extraer el valor de dimensión (por ejemplo la fecha, altura o profundidad) a la que se alcanza una estadística específica en la pila de rásteres en un dataset ráster multidimensional. También puede encontrar el número de banda a la que se alcanza la estadística en un ráster multibanda.

Por ejemplo, puede que desee conocer el mes en el que cada píxel alcanzó la temperatura de la superficie del mar más alta durante 30 años de captura de datos. Otro ejemplo es cuando tiene una imagen de teledetección de ocho bandas y desea conocer la banda en la que se alcanzó el valor de reflectancia mínimo para cada píxel.

Si Usar la extensión de mapa actual está activado, se analizarán las variables o bandas de la capa que estén visibles en la extensión de mapa actual. Si no está activado, se analizarán las variables o bandas de la capa completa, incluso si se encuentran fuera de la extensión de mapa actual.

Elegir capa de imágenes multidimensional o multibanda que se va a analizar

El ráster multibanda o multidimensional de entrada que se va a analizar.

Elegir dimensión de la que se extraerá la estadística

La dimensión de la que se extraerá la estadística. Si el ráster de entrada no es un ráster multidimensional, este parámetro no se requiere.

Elegir variables que se van a analizar

Las variables que se van a analizar. Si el ráster de entrada no es multidimensional, los valores de píxel del ráster multibanda se consideran la variable. Si el ráster de entrada es multidimensional y no se especifica ninguna variable, se analizarán todas las variables con la dimensión seleccionada.

Por ejemplo, para encontrar los años en los que los valores de temperatura fueron los más altos, especifique la temperatura como la variable que se va a analizar. Si no especifica ninguna variable y cuenta con variables de temperatura y precipitaciones, se analizarán ambas variables y el ráster multidimensional de salida incluirá ambas variables.

Elegir tipo de estadísticas

Especifica la estadística que se extraerá de las variables a lo largo de la dimensión proporcionada.

  • Argumento del mínimo: se extraerá el valor de dimensión en el que se alcanza el valor mínimo de la variable. Esta es la opción predeterminada.
  • Argumento del máximo: se extraerá el valor de dimensión en el que se alcanza el valor máximo de la variable.
  • Argumento del mediano: se extraerá el valor de dimensión en el que se alcanza el valor mediano de la variable.
  • Argumento de duración: la duración de dimensión más amplia para la que los valores de la variable se encuentran entre los valores mínimo y máximo.

Especificar valor de ocurrencia múltiple

El valor de píxel que se utilizará para indicar que se alcanzó una estadística de argumento determinada más de una vez en el dataset ráster de entrada. Si no se especifica, el valor de píxel será el valor de la dimensión la primera vez que se alcance la estadística de argumento.

Especificar valor mínimo

El valor mínimo de la variable que se utilizará para extraer la duración.

Este parámetro se requiere cuando se selecciona Duración como el tipo de estadísticas.

Especificar valor máximo

El valor máximo de la variable que se utilizará para extraer la duración.

Este paso se requiere cuando se selecciona Duración como el tipo de estadísticas.

Elegir definición de dimensión

Especifica cómo se extraerá la estadística de la dimensión.

  • Todo: la estadística se extraerá a través de todas las dimensiones. Esta es la opción predeterminada.
  • Palabra clave de intervalo: la estadística se extraerá de la dimensión de tiempo de acuerdo con la palabra clave de intervalo.

Elegir intervalo de palabra clave

La unidad de tiempo para la que se extraerá la estadística.

Por ejemplo, dispone de cinco años de datos de la temperatura diaria de la superficie del mar y desea saber el año en el que se observó la temperatura máxima. Establezca Argumento del máximo como el tipo de estadísticas, establezca Palabra clave interna como la definición de dimensión y establezca Cada año como el intervalo de palabra clave en este paso.

De forma alternativa, si desea conocer el mes en el que se observó de manera constante la temperatura máxima, establezca Argumento del máximo como el tipo de estadísticas, establezca la Palabra clave de intervalo como la definición de dimensión y establezca Recurrente mensualmente como el intervalo de palabra clave de este paso. Esto generará un ráster en el que cada píxel contiene el mes en el que se alcanzó la estadística a través del registro de cinco años (08/18/2018, 08/25/2016, 08/07/2013, por ejemplo).

Este paso se requiere cuando StdTime se establece en la dimensión desde la que extraerá la estadística y Palabra clave de intervalo se selecciona para definir cómo se extraerá la estadística de la dimensión de tiempo.

  • Recurrente diariamente: la estadística se extraerá a través de días.
  • Recurrente semanalmente: la estadística se extraerá a través de semanas.
  • Recurrente mensualmente: la estadística se extraerá a través de meses.
  • Recurrente trimestralmente: la estadística se extraerá a través de trimestres.
  • Cada hora: la estadística se extraerá para la hora en la que se alcanzó la estadística.
  • Cada día: la estadística se extraerá para el día en el que se alcanzó la estadística.
  • Cada semana: la estadística se extraerá para la semana en la que se alcanzó la estadística.
  • Cada mes: la estadística se extraerá para el mes en el que se alcanzó la estadística.
  • Cada trimestre: la estadística se extraerá para el trimestre el que se alcanzó la estadística.
  • Cada año: la estadística se extraerá para el año en el que se alcanzó la estadística.

Ignorar valores que faltan en cálculo

Especifica si los valores que faltan se ignorarán en el análisis.

  • Si se activa la casilla, el análisis incluirá todos los píxeles válidos a lo largo de una dimensión determinada y se ignorarán los píxeles NoData. Esta es la opción predeterminada.
  • Si se desactiva la casilla, el análisis dará como resultado NoData si hay valores NoData para el píxel a lo largo de la dimensión determinada.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en. Si tiene privilegios para crear capas de imágenes tanto en teselas como dinámicas, también puede especificar qué tipo de capa se va a generar para la salida utilizando el cuadro desplegable Guardar el resultado como.
