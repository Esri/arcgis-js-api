Cuenca hidrográfica

Esta herramienta determina el área de cuenca contribuyente sobre un conjunto de celdas de un ráster.

Por ejemplo, un punto de entrada en el río Misisipi en Luisiana creará una entidad de cuenca hidrográfica que se extiende para cubrir áreas desde Nueva York hasta Montana y Canadá.

Elegir capa de dirección del flujo

Ráster de entrada que muestra la dirección del flujo de cada celda.

El ráster de dirección de flujo se puede crear con la función ráster Dirección de flujo, con el tipo de dirección de flujo predeterminado, D8.

Especificar ubicaciones de punto de fluidez de entrada

Las ubicaciones de punto de fluidez de entrada.

Puede ser un ráster o una entidad de punto. Para un ráster, esto representa las celdas por sobre las que se determinará el área de contribución o la cuenca. Todas las celdas que no son NoData se utilizarán como celdas de origen. Para una entrada de punto, esto representa las ubicaciones sobre las que se determinará el área de contribución o la cuenca.

Elegir campo de punto de fluidez

El campo que se utiliza para asignar los valores a las ubicaciones de punto de fluidez.

Si la capa de puntos de fluidez de entrada es un ráster, se utiliza Value. Si es una entidad, se utiliza un campo numérico. Si el campo contiene valores de punto flotante, se truncarán en enteros.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en. Si tiene privilegios para crear capas de imágenes tanto en teselas como dinámicas, también puede especificar qué tipo de capa se va a generar para la salida utilizando el cuadro desplegable Guardar el resultado como.
