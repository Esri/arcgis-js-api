Cuenca hidrográfica

Esta herramienta determina el área de cuenca contribuyente sobre un conjunto de celdas de un ráster.

Una cuenca hidrográfica es el área de pendiente que contribuye con el flujo, generalmente agua, a una salida común como un drenaje concentrado. Puede ser parte de una cuenca hidrográfica más grande y también puede contener cuencas hidrográficas más pequeñas, denominadas subcuencas.

El valor de cada cuenca hidrográfica se tomará del valor del origen en la capa de puntos de fluidez de ráster o entidad de entrada. Cuando el punto de fluidez es un dataset ráster, se utilizan los valores de celda de forma predeterminada. Cuando el punto de fluidez es un dataset de entidad de puntos, los valores provendrán del campo especificado.

La herramienta Cuenca hidrográfica solo admite una capa ráster de dirección del flujo de entrada D8.

Al especificar las ubicaciones de punto de fluidez de entrada como datos de entidad, el campo predeterminado será el primer campo válido disponible. Si no existen campos válidos, el campo ObjectID (por ejemplo, OID o FID) será el valor predeterminado.

Elegir capa de dirección del flujo

Ráster de entrada que muestra la dirección del flujo de cada celda.

El ráster de dirección del flujo se puede crear con la herramienta Dirección de flujo o la tarea FlowDirection en el servicio RasterAnalysisTools, ejecutadas con el tipo de dirección de flujo predeterminado, D8.

Especificar ubicaciones de punto de fluidez de entrada

Las ubicaciones de punto de fluidez de entrada.

Para un ráster, esto representa las celdas por sobre las que se determinará el área de contribución o la cuenca. Todas las celdas que no son NoData se utilizarán como celdas de origen. Para una entrada de punto, esto representa las ubicaciones sobre las que se determinará el área de contribución o la cuenca.

Elegir campo de punto de fluidez

El campo que se utiliza para asignar los valores a las ubicaciones de punto de fluidez.

Si la capa de puntos de fluidez de entrada es un ráster, se utiliza un valor. Si es una entidad, se utiliza un campo numérico. Si el campo contiene valores de punto flotante, se truncarán en enteros.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en.
