Calcular distancia

Esta herramienta calcula la distancia euclidiana, dirección y asignación a partir de un solo origen o un conjunto de orígenes.

Las aplicaciones de ejemplo dan respuesta a preguntas como las siguientes:

  • ¿Cuál es la distancia a la población más cercana?
  • ¿Cuál es la distancia hasta la carretera más cercana?
  • ¿Cuál es la distancia hasta la fuente de agua más cercana?

Si se ha activado Usar la extensión del mapa actual, solo se analizarán aquellas entidades de la capa de entrada que estén visibles en la extensión de mapa actual. Si la opción está desactivada, se analizarán todas las entidades de la capa de entrada, aunque estén fuera de la extensión de visualización de mapa actual.

Elegir la capa de ubicación de entidad o ráster respecto a la cual se calculará la distancia

La capa que define los orígenes respecto de los cuales se calculará la distancia. La capa puede ser un servicio de imágenes o un servicio de entidades.

En el caso de un servicio de imágenes, el tipo de entrada puede ser de tipo entero o de punto flotante.

En el caso de un servicio de entidades, la entrada puede ser un punto, una línea o un polígono.

Distancia máxima a calcular

La distancia máxima a calcular. Por ejemplo, si indica un valor de 10.000 metros, la distancia euclidiana se calculará a partir de las ubicaciones de origen y hasta ese valor (distancia).

Si no se proporciona ningún valor, la distancia se calculará hasta la extensión definida.

Las unidades disponibles son: kilómetros, metros, millas y pies.

Tamaño de celda de salida

El tamaño de celda y las unidades para los rásteres de salida.

Las unidades pueden estar en kilómetros, metros, millas o pies.

Nombre de la capa de resultados de distancia

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en. Si tiene privilegios para crear capas de imágenes tanto en teselas como dinámicas, también puede especificar qué tipo de capa se va a generar para la salida utilizando el cuadro desplegable Guardar el resultado como.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en. El nombre de la capa que se creará en Mi contenido. Debe indicar un nombre que no exista aún en Mi contenido.

Elegir campo de asignación

Un campo de la capa de origen de entrada que contiene los valores que definen cada origen. Puede ser un campo de enteros o de cadenas de caracteres del dataset de origen.

El valor predeterminado es Value. Si se utiliza un servicio de entidades y no está disponible ningún campo Value, se utiliza el primer campo de entero o cadena de caracteres disponible. Si no existen campos válidos, se utilizará el campo ObjectID (por ejemplo, OID o FID, dependiendo del tipo de entrada de entidad).

Elegir la capa de entidades o ráster para representar barreras

El dataset que define las barreras.

En el caso de un servicio de imágenes, el tipo de entrada puede ser de tipo entero o de punto flotante. Las celdas que tengan algún valor (incluido cero) se tratarán como barreras, y las celdas con NoData no se tratarán como barreras.

En el caso de un servicio de entidades, la entrada puede ser un punto, una línea o un polígono.

Método de distancia

Determina si la distancia se calcula basándose en un método planar (Tierra plana) o geodésico (elipsoide).

  • Planar: el cálculo se realizará sobre un plano bidimensional proyectado usando un sistema de coordenadas cartesianas 2D. Este es el método predeterminado.
  • Geodésico: las distancias se calculan sobre el elipsoide. Por lo tanto, independientemente de la proyección de entrada o salida, los resultados no cambian.
