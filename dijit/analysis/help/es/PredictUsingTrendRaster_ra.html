Predecir con ráster de tendencia

Calcula un ráster multidimensional previsto utilizando el ráster de tendencia de salida de la herramienta Generar ráster de tendencia.

Si Usar la extensión de mapa actual está activado, se analizarán las variables de la capa que estén visibles en la extensión de mapa actual. Si no está activado, se analizarán las variables de la capa completa, incluso si se encuentran fuera de la extensión de mapa actual.

Elegir capa de imágenes multidimensional que se va a predecir

El ráster de tendencia multidimensional de entrada de la herramienta Generar ráster de tendencia.

Elegir variables que se van a predecir

Las variables que se van a predecir en el análisis. Si no se especifica ninguna variable, se utilizarán todas las variables.

Elegir el método utilizado para ofrecer valores de dimensión de predicción

Especifica el método utilizado para ofrecer valores de dimensión de predicción.

  • Por valor: la predicción se calculará para un único valor de dimensión o una lista de valores de dimensión especificados en el siguiente paso. Esta es la opción predeterminada. Por ejemplo, desea predecir la precipitación anual de los años 2050, 2100 y 2150.
  • Por intervalo: la predicción se calculará para un intervalo de la dimensión definida mediante un valor inicial y final. Por ejemplo, desea predecir la precipitación anual de cada año entre el 2050 y el 2150.

Especificar valores de dimensión para predicción

El valor o los valores de dimensión que se utilizarán en la predicción.

El formato de los valores de tiempo, profundidad y altura debe coincidir con el formato de los valores de dimensión utilizados para generar el ráster de tendencia. Si el ráster de tendencia se generó para la dimensión StdTime, el formato debería ser AAAA-MM-DDTHH:MM:SS, por ejemplo 2050-01-01T00:00:00. Varios valores se separan mediante punto y coma.

Este parámetro se requiere cuando se selecciona el método Por valor para definir los valores de dimensión para predicción.

Comenzar

La fecha, la altura o la profundidad de inicio del intervalo de dimensiones que se utilizarán en la predicción.

Finalizar

La fecha, la altura o la profundidad finales del intervalo de dimensiones que se utilizarán en la predicción.

Definir el intervalo de dimensión para predicción

Defina la fecha de inicio y fin, la altura o la profundidad de inicio del intervalo de dimensión que se utilizarán en la predicción.

Especificar el número de pasos entre los valores de inicio y fin

El número de pasos entre dos valores de dimensión que se incluirán en la predicción. El valor predeterminado es 1.

Por ejemplo, para predecir valores de temperatura cada cinco años, utilice un valor de 5.

Elija la unidad que se usará para el intervalo de valores de tiempo

Especifica la unidad que se usará para el intervalo de valores. Esto solo se aplica cuando la dimensión del análisis es una dimensión de tiempo.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en. Si tiene privilegios para crear capas de imágenes tanto en teselas como dinámicas, también puede especificar qué tipo de capa se va a generar para la salida utilizando el cuadro desplegable Guardar el resultado como.
