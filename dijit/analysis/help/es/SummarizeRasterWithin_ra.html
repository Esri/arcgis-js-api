Resumir ráster dentro de

Esta herramienta resume el valor de un ráster dentro de las zonas de otro dataset.

A continuación se indican algunas aplicaciones de ejemplo:

  • Dada una capa de límites de cuencas hidrográficas y una capa de límites de uso del suelo, calcule la superficie total del tipo de uso del suelo para cada cuenca hidrográfica.
  • Dada una capa de parcelas de un condado y una capa de límites de ciudades, resuma el valor medio de las parcelas vacías en cada límite de ciudad.

Si se ha activado Usar la extensión del mapa actual, solo se analizarán las entidades de la capa de entrada y la capa que se va a resumir que estén visibles en la extensión de mapa actual. Si la opción está desactivada, se analizarán todas las entidades de la capa de entrada y de la capa que se va a resumir, aunque estén fuera de la extensión de mapa actual.

Elija una capa de área para resumir una capa ráster en los límites definidos

La capa que define los límites de las áreas, o zonas, que se van a resumir.

La capa puede ser un ráster o datos de entidad. Para los rásteres, las zonas se definen mediante todas las ubicaciones de la entrada que tienen el mismo valor de celda. No es necesario que las áreas sean contiguas.

Elija un campo para definir los límites

Este es el atributo de la capa que se utilizará para definir los límites de las áreas.

Por ejemplo, supongamos que la primera capa de entrada define los límites de la unidad de gestión, con atributos que definen la región, el distrito y el ID de parcela de cada unidad. También tiene una capa ráster que define el índice de biodiversidad para cada ubicación. Con el campo seleccionado, puede decidir calcular la biodiversidad media en el nivel local, de distrito o regional.

Elija una capa ráster para resumir

Las celdas ráster de esta capa se resumirán por las áreas (zonas) en las que queden dentro.

Elija una estadística para calcular

Una vez que haya elegido una capa para resumir, podrá calcular estadísticas sobre cualquier atributo numérico de los puntos, líneas o áreas dentro de la capa de área de entrada.

Los tipos de estadísticas que están disponibles cuando el campo seleccionado es un entero son: Media, Mayoría, Máximo, Mediana, Mínimo, Minoría, Percentil, Rango, Desviación estándar, Suma y Variedad.

Si el campo es de punto flotante, las opciones son: Valor medio, Máximo, Mínimo, Rango, Desviación estándar y Suma.

Valor de percentil

El percentil que se va a calcular. El valor predeterminado es 90, que indica el 90.º percentil.

Los valores pueden variar de 0 a 100. El 0.º percentil equivale en esencia a la estadística mínimo, mientras que el 100.º percentil equivale a la estadística máximo. Un valor de 50 generará en esencia el mismo resultado que la estadística mediana.

Esta opción solo se admite si el valor del parámetro Elija una estadística para calcular es Percentil.

Ignorar valores que faltan en cálculo

Si opta por ignorar los valores que faltan, solo se utilizarán para determinar el valor de salida de dicha área las celdas que tengan un valor en la capa que se va a resumir. En caso contrario, si faltan valores en algún lugar del área, se considerará que no hay suficiente información para realizar cálculos estadísticos para todas las celdas de esa zona y esa área recibirá un valor nulo (NoData) en la salida.

Procesar como multidimensional

Especifica cómo se procesan los rásteres de entrada si son multidimensionales. Si se opta por procesar como multidimensional, las estadísticas se calculan a partir de todas las dimensiones del dataset multidimensional de entrada. De lo contrario, las estadísticas se calculan a partir del fragmento actual del dataset multidimensional de entrada, que es el predeterminado.

Nombre de capa de resultados

El nombre de la capa que se creará en Mi contenido y que se agregará al mapa. El nombre predeterminado se basa en el nombre de la herramienta y en el nombre de la capa de entrada. Si la capa ya existe, se le pedirá que indique otro nombre.

Puede especificar el nombre de una carpeta de Mi contenido en la que se guardará el resultado con el cuadro desplegable Guardar el resultado en.
