GeoAnalytics는 ArcGIS GeoAnalytics Server에서 분석 작업을 배포합니다. 이를 통해 여러 머신을 활용하여 더 많은 데이터를 보다 빠르게 분석할 수 있습니다. 현재 사용 가능한 도구는 다음과 같습니다.

  • 포인트 집약
  • 조인 피처
  • 추적 재생성
  • 속성 요약
  • 다음 범위 내 요약
  • 사건 감지(Detect Incidents)
  • 유사한 위치 찾기
  • 밀도 계산
  • 핫 스팟 찾기
  • 버퍼 생성
  • 데이터 저장소로 복사
  • 계산 필드(Calculate Field)
  • 테이블의 지오코드 위치(Geocode Locations from Table)

GeoAnalytics는 공간 및 시간 패턴 모두에서 강조 요소가 있는 대용량 데이터를 분석하는 데 중점을 둡니다.

데이터 요약

데이터 요약 도구모음에는 영역 내에 있거나 다른 피처 주변에 있는 피처와 해당 속성에 대한 총 개수, 길이, 면적, 설명이 포함된 기본 통계를 계산하는 도구가 포함되어 있습니다.

  • 포인트 집약 - 지정된 영역 또는 그룹 내에 속한 포인트에 대한 통계를 계산합니다.
  • 조인 피처 - 다른 피처와 공간, 시계열 또는 속성 대응비를 공유하는 피처에 대한 통계를 계산합니다.
  • 추적 재생성 - 동일한 추적에 속하는 포인트 또는 폴리곤에 대한 통계를 계산하고 입력을 추적으로 재생성합니다.
  • 속성 요약 - 속성을 공유하는 피처 또는 테이블 형식 데이터에 대한 통계를 계산합니다.
  • 범위 내 요약 - 서로 겹치는 영역 피처 및 속성에 대한 통계를 계산합니다.

포인트 집약(Aggregate Points)

이 도구는 포인트 피처 레이어와 영역 피처 레이어 또는 지정된 거리로 정의된 그룹을 사용하여 각 영역 또는 그룹 내에 속한 포인트를 확인하고 각 영역 또는 그룹 내의 모든 포인트에 대한 통계를 계산합니다. 필요한 경우 이 도구에 시간 슬라이싱을 적용할 수 있습니다.

예를 들면 다음과 같습니다.

  • 범죄 사건의 포인트 위치가 주어진 경우 도시별 또는 다른 행정 구역별 범죄 건수를 계산합니다.
  • 100km 그룹을 사용하여 프랜차이즈 위치의 최고 및 최저 월간 매출을 확인합니다.


조인 피처 (Join Features)

피처 레이어 또는 테이블 형식 데이터를 사용하여 입력 레이어 또는 테이블 간의 특정 대응비에 따라 피처와 레코드를 조인할 수 있습니다. 조인은 공간, 시계열 및 속성 대응비에 따라 결정되며 필요한 경우 요약 통계를 계산할 수 있습니다.

예를 들면 다음과 같습니다.

  • 시간과 함께 범죄 사건에 대한 포인트 위치가 주어진 경우, 서로 1시간 이내 발생한 범죄 사건에 대해 1킬로미터 이내의 공간 대응비를 지정하여 범죄 데이터를 해당 범죄 사건에 조인함으로써 시공간에서 일련의 범죄가 서로 근접하게 발생하는지를 확인할 수 있습니다.
  • 인구 통계 정보가 포함된 우편 번호 테이블과 주거 건물을 나타내는 영역 피처가 주어진 경우, 인구 통계 정보를 거주지에 조인하여 각 거주지에 인구 통계 정보가 포함되도록 할 수 있습니다.


추적 재생성(Reconstruct Tracks)

인스턴트 시간을 나타내는 포인트 또는 폴리곤 피처의 레이어에 시간이 활성화되어 있는 경우 이 도구를 사용하여 추적에 속하게 될 입력 피처를 정하고 입력의 순서를 시간 순으로 정합니다. 필요한 경우 각 추적 내의 입력 피처에 대한 통계를 계산할 수 있습니다.

예를 들면 다음과 같습니다.

  • 허리케인 측정치 시간과 포인트 위치가 주어진 경우 허리케인의 평균 풍속과 최대 풍압을 계산합니다.
  • 선박의 GPS 측정치를 사용하여 라인으로 항로를 재생성하세요.


속성 요약 (Summarize Attributes)

이 도구는 피처 또는 테이블 형식 데이터를 사용하여 특정 필드에 대한 통계를 요약합니다.

예를 들면 다음과 같습니다.

  • COMPANY_NAME 필드가 있는 식료품점 위치가 주어진 경우, 회사 이름별로 매장을 요약하여 회사별 통계를 확인합니다.
  • COMPANY_NAME 필드와 COUNTY 필드가 있는 식료품점 테이블이 주어진 경우, 회사 이름 및 구/군별로 매장을 요약하여 각 구/군 내 각 회사에 대한 통계를 확인합니다.


다음 범위 내 요약(Summarize Within)

두 레이어 간에 겹치는 영역(및 영역의 일부)을 찾아서 해당 영역에 대한 통계를 계산합니다.

예를 들면 다음과 같습니다.

  • 하천 유역 레이어와 토지 이용 유형별 토지 이용 영역 레이어가 주어진 경우 각 하천 유역에 대한 토지 이용 유형의 총 면적을 계산합니다.
  • 도시 내 필지 레이어와 도시 경계 레이어가 주어진 경우 각 도시 내의 공지에 대한 평균 지가를 요약합니다.


위치 찾기 (Find Locations)

이러한 도구는 지정한 여러 조건을 충족하는 영역을 식별하는 데 사용됩니다.

  • 사건 감지는 시계열 레이어와 함께 작동하여 어떤 피처가 지정한 조건을 충족하는지 확인합니다.
  • 테이블의 지오코드 위치는 주소를 좌표로 변환합니다.
  • 유사한 위치 찾기 - 지정한 조건에 따라 하나 이상의 참조 위치와 가장 유사한 위치를 찾습니다.

사건 감지

이 도구는 인스턴트 시간을 나타내는 포인트, 라인, 영역 또는 테이블의 시계열 레이어와 함께 작동됩니다. 이 도구는 순서가 순차적으로 지정된 피처, 즉 추적을 사용하여 어떤 피처가 관심 사건인지 파악합니다. 사건은 지정한 조건에 따라 결정됩니다.

예를 들면 다음과 같습니다.

  • 풍속 속성을 기반으로 범주 4에 속하는 허리케인의 측정치를 감지했습니다.
  • 선박 속도를 나타내는 필드로 선박 이동에 따른 GPS 측정치를 사용하여 5개의 이전 측정치 평균보다 1.5배 더 큰 측정치를 찾습니다.


테이블의 지오코드 위치(Geocode Locations from Table)

주소를 좌표로 변환합니다. 빅데이터 파일 공유 테이블에 이 도구를 사용합니다.


유사한 위치 찾기

지정한 조건에 따라 하나 이상인 참조 위치에 대한 후보 검색 레이어의 위치 유사성을 측정하여 유사한 위치를 찾습니다.

예를 들면 다음과 같습니다.

  • 직원 수와 연간 매출을 조사하여 가장 유사한 매장을 찾습니다.
  • 인구, 연간 성장률, 세제 수입 간의 관계를 조사하여 가장 유사한 100개 도시를 찾습니다.


패턴 분석

이러한 도구는 데이터에 있는 공간 패턴을 식별, 수량화 및 시각화합니다.

  • 밀도 계산은 일부 현상의 알려진 수량을 이용하여 이러한 수량을 맵에서 분산시킵니다.
  • 핫 스팟 찾기 - 데이터의 공간 패턴에서 통계적으로 중요한 군집을 식별합니다.

밀도 계산(Calculate Density)

밀도 계산 도구는 맵에서 일부 현상(포인트의 속성으로 표시됨)의 알려진 수량을 분산시켜 포인트 피처에서 밀도 맵을 생성합니다. 결과는 밀도가 나타난 영역의 레이어입니다.

예를 들면 다음과 같습니다.

  • 구/군 내의 병원 밀도를 계산합니다. 결과 레이어가 병원에 대한 높고 낮은 접근성을 지역에 표시하며 신규 병원 건설 장소를 결정하는 데 이 정보를 사용할 수 있습니다.
  • 과거 산불이 발생한 위치를 기반으로 산불 발생 위험도가 높은 지역을 식별합니다.
  • 새 도로 건설 장소를 계획하기 위해 주요 고속도로에서 멀리 있는 커뮤니티를 찾습니다.


핫 스팟 찾기(Find Hot Spots)

핫 스팟 찾기 도구는 데이터의 공간 패턴에 통계적으로 유의한 군집이 있는지 확인합니다.

  • 포인트(범죄 사건, 나무, 교통 사고)가 실제로 군집되어 있습니까? 어떻게 확신할 수 있습니까?
  • 통계적으로 유의한 핫 스팟(지출, 영아 사망률, 지속적으로 높은 시험 점수)을 발견했습니까? 아니면 심볼화 방식을 변경한 경우 맵에 다른 정보가 나타납니까?
핫 스팟 찾기 도구를 사용하면 확신을 가지고 이러한 질문에 답변할 수 있습니다.


인접도 사용

공간 분석에서 가장 많이 제기되는 일반 질문 중 하나인 "무엇과 무엇이 인접합니까?"라는 질문에 대한 답을 찾는데 사용하는 도구입니다.

  • 버퍼 생성 - 피처에서 지정된 거리에 있는 영역을 생성합니다.

버퍼 생성 (Create Buffers)

버퍼는 포인트, 라인 또는 폴리곤 피처에서 지정된 거리를 포함하는 영역입니다.

예를 들면 다음과 같습니다.

  • 선형 하천 피처를 사용하는 경우 각 하천에 대해 하천 너비의 50배로 버퍼를 생성하여 제안된 하천 경계를 확인합니다.
  • 국가를 나타내는 영역이 주어진 경우 각 국가에 대해 200해리의 버퍼를 생성하여 해상 경계를 확인합니다.


데이터 관리 (Manage Data)

지리 데이터 및 테이블 형식 데이터의 일상적인 관리에 사용되는 도구입니다.

  • 필드 계산 - 새 필드를 생성하거나 기존 필드를 수정하고 콘텐츠에서 새 레이어를 생성합니다.
  • 데이터 저장소에 복사 - 데이터를 ArcGIS Data Store에 복사하고 콘텐츠에서 레이어를 생성합니다.

데이터 저장소로 복사(Copy to Data Store)

입력 피처 레이어 또는 테이블을 ArcGIS Data Store에 복사하고 ArcGIS Enterprise의 콘텐츠에서 레이어를 생성합니다.

예를 들면 다음과 같습니다.

  • 시각화를 위해 빅데이터 파일 공유의 .csv 파일 컬렉션을 시공간 데이터 저장소에 복사합니다.
  • 시공간형 데이터 저장소에 저장되어 있는 현재 맵 범위의 피처를 관계형 데이터 저장소로 복사합니다.


계산 필드(Calculate Field)

새 필드 또는 기존 필드의 값을 계산하고 ArcGIS Enterprise의 콘텐츠에서 레이어를 생성합니다.

예를 들면 다음과 같습니다.

  • 기존 필드인 total을 필드 total_2016, total_2017 및 total_2018 수익의 합으로 수정합니다.
  • windspeed 및 pollutant 같은 필드 값을 기반으로 위험 수준을 분류하는 새 필드를 생성합니다.
