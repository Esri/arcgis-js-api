다음 범위 내 래스터 요약

이 도구는 다른 데이터셋의 구역 내에서 래스터 값을 요약합니다.

다음과 같은 몇 가지 적용 예가 있습니다.

  • 유역 경계 레이어와 토지 이용 유형별 토지 이용 경계 레이어가 주어진 경우 각 유역에 대한 토지 이용 유형의 총 면적을 계산합니다.
  • 구/군 내 필지 레이어와 도시 경계 레이어가 주어진 경우 각 도시 경계 내의 공지에 대한 평균 값을 요약합니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 입력 레이어 및 요약할 레이어의 피처만 분석되고, 이 옵션을 선택하지 않는 경우 현재 맵 범위를 벗어나는 피처까지 포함하여 입력 레이어와 요약할 레이어에 있는 피처가 전부 분석됩니다.

정의된 경계 내의 래스터 레이어를 요약할 영역 레이어 선택

영역이나 구역의 경계를 정의하는 레이어로, 요약될 레이어입니다.

래스터나 피처 데이터가 레이어로 사용될 수 있습니다. 래스터의 경우 구역은 셀 값이 같은 입력의 모든 위치에 의해 정의됩니다. 영역은 인접하지 않아도 됩니다.

경계를 정의할 필드 선택

이 필드는 영역의 경계를 정의하는 데 사용될 레이어의 속성입니다.

예를 들어 첫 번째 입력 레이어가 지역, 구역 및 각 단위의 필지 ID를 정의하는 속성을 사용하여 관리 단위 경계를 정의한다고 가정해 보겠습니다. 각 위치의 생물 다양성 색인을 정의하는 래스터 레이어도 있습니다. 선택한 필드를 사용하여 지방, 구역 또는 지역 수준에서의 평균 생물 다양성을 계산해볼 수 있습니다.

요약할 래스터 레이어 선택

이 레이어의 래스터 셀은 포함되어 있는 영역(구역)별로 요약됩니다.

계산할 통계 선택

요약할 레이어를 선택하면, 필요한 경우 입력 영역 레이어 내에 있는 포인트, 라인 또는 영역의 수치 속성에 대한 통계를 계산할 수 있습니다.

선택한 필드가 정수인 경우 사용 가능한 통계 유형은 평균(Average), 다수(Majority), 최대(Maximum), 중앙값(Median), 최소(Minimum), 소수(Minority), 백분위수(Percentile), 범위(Range), 표준편차(Standard Deviation), 합계(Sum), 다양(Variety)입니다.

부동 소수점 필드인 경우에는 평균(Average), 최대(Maximum), 최소(Minimum), 범위(Range), 표준편차(Standard Deviation), 합계(Sum)을 선택할 수 있습니다.

백분위수 값

계산할 백분위수입니다. 기본값은 90이며 90번째 백분위수를 나타냅니다.

값의 범위는 0~100일 수 있습니다. 0번째 백분위수는 기본적으로 최소 통계와 같고 100번째 백분위수는 최대 통계와 같습니다. 50의 값은 기본적으로 중앙값 통계와 동일한 결과를 생성합니다.

이 옵션은 계산할 통계 선택 매개변수가 백분위수로 설정된 경우에만 지원됩니다.

계산 내 누락 값 무시

누락 값을 무시하도록 선택하면 요약될 레이어에 값이 있는 셀만 해당 영역의 결과 값을 확인하는 데 사용됩니다. 이 옵션을 선택하지 않으면 영역 내에 누락 값이 있을 경우 해당 구역의 모든 셀에 대해 통계 계산을 수행하기 위한 정보가 부족하여 해당 영역에 대한 결과에는 null (NoData) 값이 포함되는 것으로 간주됩니다.

다차원으로 처리

입력 래스터가 다차원일 경우 처리 방법을 지정합니다. 다차원으로 처리하도록 선택하는 경우 입력 다차원 데이터셋의 모든 차원에서 통계가 계산됩니다. 그렇지 않으면 입력 다차원 데이터셋의 현재 슬라이스에서 통계가 계산됩니다(기본값).

결과 레이어 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 결과가 저장될 내 콘텐츠에서 폴더 이름을 지정할 수 있습니다.
