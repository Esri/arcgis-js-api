래스터 추출

이 도구는 래스터를 경계(직사각형 영역 또는 화면에서 대화형으로 정의한 쉐이프)에 클립할 수 있습니다. 맵에 현재 표시된 영역의 범위 또는 폴리곤에 의해 정의된 처리 범위로 클립할 수 있습니다.

추출할 입력 데이터 선택

클립할 입력 레이어입니다.

클리핑 지오메트리 지정

레이어의 지오메트리를 클립할 때 사용할 범위 유형을 선택합니다.

  • 현재 범위 사용 - 결과 레이어를 클립하는 데 현재 맵 범위가 사용됩니다. 이 옵션이 기본 설정입니다.
  • 처리 범위 선택 - 그리기 도구를 사용하여 클리핑 지오메트리를 정의할 수 있습니다.
  • 레이어 선택 - 피처 레이어나 영상 레이어를 선택하여 범위를 정의하거나 폴리곤 피처 레이어를 선택하여 클리핑 지오메트리를 정의합니다. 폴리곤 피처 레이어가 선택된 경우 클리핑 지오메트리에 입력 피처 사용 여부를 지정하는 체크 박스가 표시됩니다. 기본적으로 선택 취소되어 있습니다. 즉, 범위를 정의하는 데 입력 피처 레이어가 사용됩니다. 선택된 경우, 클리핑 지오메트리에 입력 피처가 사용됩니다.

클리핑 유형 지정

클립 지오메트리 내부 또는 외부 영역을 유지할지 여부를 지정합니다.

  • 지오메트리 내부 데이터 - 클립 범위 내부의 영상이 추출됩니다. 이 옵션이 기본 설정입니다.
  • 지오메트리 외부 데이터 - 클립 범위 외부의 영상이 추출됩니다.

결과 레이어 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 결과가 저장될 내 콘텐츠에서 폴더 이름을 지정할 수 있습니다. 타일 형태 및 동적 영상 레이어를 모두 생성할 수 있는 권한이 있는 경우 다른 이름으로 결과 저장 드롭다운 상자를 사용하여 출력에서 생성할 레이어 유형을 지정할 수 있습니다.
