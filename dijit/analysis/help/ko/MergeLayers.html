레이어 병합

이 도구는 두 레이어의 피처를 새 레이어에 복사합니다. 병합할 모든 레이어에 같은 피처 유형(포인트, 라인 또는 영역)이 있어야 합니다. 입력 레이어의 필드를 결합하고 복사할 방법을 제어할 수 있습니다. 예를 들면 다음과 같습니다.

  • 공립학교 레이어와 사립학교 레이어가 있으며 모든 학교를 표시하는 단일 레이어가 필요합니다.
  • 레이어가 2개 있고, 각 레이어에는 인접 마을에 대한 필지 정보가 포함되어 있습니다. 이러한 레이어를 단일 레이어로 결합하여 2개의 입력 레이어에 이름과 유형이 같은 필드만 유지할 수 있습니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 피처만 병합됩니다. 선택하지 않으면 현재 맵 범위 외부에 있는 위치를 포함하여 두 레이어에 있는 모든 피처가 병합됩니다.

병합할 레이어 선택

첫 번째 레이어와 병합할 레이어를 선택합니다.

병합 필드 수정

기본적으로 두 입력의 모든 필드가 결과에 포함됩니다. 필드가 결과에 포함되는 방식을 다음과 같이 변경할 수 있습니다.

  • 이름 바꾸기 - 결과에서는 필드의 이름이 바뀝니다.
  • 제거 - 결과에서는 필드가 제거됩니다.
  • 일치 - 필드 이름이 동일하게 설정되고, 병합할 레이어의 필드 값이 일치하는 결과 필드에 복사됩니다.

두 레이어에 이름 및 데이터 유형이 동일한 필드가 포함되어 있는 경우 두 필드의 콘텐츠가 해당 이름의 한 필드로 병합됩니다. 이는 병합 필드 수정 매개변수에 일치가 사용된 필드에는 적용되지 않습니다.

결과 레이어 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어 이름이 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 메뉴를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
