데이터 저장소로 복사(Copy to Data Store)

이 도구는 입력 레이어를 받아 데이터 저장소에 복사합니다. 데이터는 ArcGIS Data Store에 복사되며 관계형 또는 시공간형 데이터 저장소에 저장됩니다.

예를 들어 빅 데이터 파일 공유에 저장되어 있는 피처를 관계형 데이터 저장소에 복사하되 현재 맵 범위 내의 피처만 복사되도록 지정할 수 있습니다. 이 경우 지정된 맵 범위 내에 있는 피처만 포함된 호스팅 피처 서비스가 생성됩니다.

복사할 레이어 선택

복사될 레이어나 테이블입니다.

맵에서 레이어를 선택하는 것 이외에, 드롭다운 목록의 하단에 있는 레이어 찾아보기를 선택하여 빅 데이터 파일 공유 데이터셋 또는 피처 레이어의 콘텐츠를 찾아볼 수 있습니다.

모든 피처를 사용하거나 원하는 위치를 선택할 수 있음(선택)

필요에 따라 선택 버튼을 사용하여 복사될 위치를 식별할 수 있습니다.

복사될 ArcGIS 데이터 저장소 선택

레이어가 복사될 데이터 저장소입니다. 이 매개변수는 분석 환경의 선택 항목을 덮어씁니다. 옵션은 다음과 같습니다.

  • 시공간형 - 이 옵션이 기본 설정입니다.
  • 관계형.

결과 레이어 이름

내 콘텐츠에서 생성되어 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있으면 도구를 사용할 수 없습니다.

결과 저장: 드롭다운 상자를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
