포인트 집계(Aggregate Points)

이 도구는 포인트 피처 레이어 및 영역 피처 레이어에서 작동합니다. 먼저 각 영역에 속한 포인트를 확인합니다. 이 영역 내 포인트의 공간 관계를 확인한 후에는 영역 내 모든 포인트에 대한 통계가 계산되어 영역에 할당됩니다. 가장 기본적인 통계는 영역 내에 있는 포인트의 수이지만 다른 통계도 확인할 수 있습니다.

예를 들어 카페 위치에 대응하는 포인트 피처와 구/군에 대응하는 영역 피처가 있다고 가정하고 구/군별 커피 판매량을 요약해 보겠습니다. 카페에 TOTAL_SALES 속성이 있는 것으로 가정할 때 각 시/군/구 내의 모든 TOTAL_SALES 합계, 각 시/군/구 내의 최소 또는 최대 TOTAL_SALES, 각 시/군/구 내의 모든 판매량에 대한 표준편차를 구할 수 있습니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 입력 포인트 레이어와 영역 레이어의 피처만 분석되고, 이 옵션을 선택하지 않는 경우에는 현재 맵 범위를 벗어나는 피처까지 포함하여 입력 포인트 레이어와 영역 레이어에 있는 피처가 전부 분석됩니다.

면적으로 집약할 포인트가 포함된 레이어 선택

영역으로 집약되는 포인트 레이어입니다.

맵에서 레이어를 선택하는 것 외에도 드롭다운 목록 하단에서 분석 레이어를 선택할 수 있습니다. 해당 옵션을 사용하면 다양한 원본에서 분석에 사용할 데이터셋을 검색하고 선택할 수 있는 갤러리가 열립니다.

집약 면적이 포함된 레이어 선택

포인트가 집약되는 영역 레이어입니다. 이 레이어의 영역 경계 내에 있는 모든 포인트 수가 계산되고 사용자가 선택한 통계를 사용하여 선택적으로 요약됩니다.

제공한 영역 레이어 또는 도구 실행 시 생성되는 그룹 중에 선택할 수 있습니다. 빈을 생성할 때 사각형의 경우 지정된 수 및 단위가 사각형의 높이 및 길이를 결정합니다. 육각형의 경우 지정된 수 및 단위가 평행 측면 간 거리를 결정합니다.

맵에서 레이어를 선택하는 것 외에도 드롭다운 목록 하단에서 분석 레이어를 선택할 수 있습니다. 해당 옵션을 사용하면 다양한 원본에서 분석에 사용할 데이터셋을 검색하고 선택할 수 있는 갤러리가 열립니다.

포인트 없는 영역 유지

내부에 포인트가 없는 영역을 결과 레이어에 표시할지 여부를 결정합니다. 이 옵션을 선택하지 않으면 포인트가 없는 영역이 결과에 표시되지 않고, 이 옵션을 선택하면 포인트가 없는 영역이 결과에 표시됩니다.

통계 추가

입력 포인트 레이어의 속성에 대한 통계(예: 합계, 평균, 최소값, 최대값, 표준편차)를 계산할 수 있습니다. 결과 레이어에는 계산된 각 통계에 대한 새 속성이 포함됩니다. 속성과 통계를 선택(새 드롭다운 상자가 표시됨)하여 원하는 개수의 통계를 추가할 수 있습니다.

요약할 레이어에 숫자 필드가 없는 경우 추가 통계를 사용할 수 없습니다.

그룹화할 필드 선택

입력 포인트 레이어의 속성을 사용하여 통계 그룹을 생성할 수 있습니다. 예를 들어 네이버후드 경계 지역의 범죄를 집계하는 경우 서로 다른 5가지 범죄 유형으로 구성된 Crime_type 속성을 사용할 수 있습니다. 각각의 고유한 범죄 유형이 그룹을 구성하고, 각각의 고유한 Crime_type 값에 대해 선택한 통계가 계산됩니다. 각 그룹에 대한 통계와 포인트 수가 결과 레이어 팝업에 표시됩니다. 속성을 그룹화하도록 선택할 때 2개의 결과물 즉, 결과 레이어와 결과 테이블이 만들어집니다. 이들에는 각 고유 그룹에 대한 통계가 포함됩니다.

소수(Minority), 다수(Majority) 추가

그룹화할 필드를 선택하면 체크 박스가 활성화됩니다. 어떤 속성 값이 각 그룹 내의 소수(열세)인지, 또는 각 경계 내의 다수(우세)인지 알아 보려면 소수/다수 추가를 선택하세요. 소수(Minority), 다수(Majority) 추가를 선택하면 새로운 필드 두 개가 결과 레이어에 추가됩니다. 백분율 추가를 선택하는 경우에도 새로운 필드 두 개가 각 그룹 내의 소수(Minority), 다수(Majority) 속성 값의 백분율을 포함한 결과 레이어에 추가됩니다.

예를 들어 당신이 도시 계획자이고 모든 도시 공원 내에서 어느 수목의 종류가 소수에 속하고 다수에 속하는지 알아 보려면 공원 영역으로 수목 데이터를 집약하고 이 옵션을 선택하세요.

백분율 추가

그룹화할 필드를 선택하면 체크박스가 활성화됩니다. 각 그룹 내에 있는 각 속성 값의 백분율을 찾으려는 경우 백분율 추가를 선택합니다. 새 필드가 각 그룹 내에 있는 각 속성 값의 백분율을 포함하는 결과 테이블에 추가됩니다. 소수(Minority)/다수(Majority) 추가도 체크된 경우 추가 필드 두 개가 각 그룹 내에 있는 소수(Minority)/다수(Majority) 속성 값의 백분율을 포함한 결과 레이어에 추가됩니다.

결과 레이어 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어 이름이 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 메뉴를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
