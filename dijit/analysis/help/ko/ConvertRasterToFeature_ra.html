래스터를 피처로 변환(Convert Raster To Feature)

이 도구는 래스터를 포인트, 라인 또는 폴리곤 등의 피처 데이터셋으로 변환합니다.

필드 선택 매개변수를 통해 입력 래스터 데이터셋의 속성 필드 중 결과 피처 클래스의 속성으로 사용될 필드를 선택할 수 있습니다. 이 매개변수는 연관된 테이블에 추가 속성이 있는 정수 래스터에 대해서만 사용할 수 있습니다.

입력 래스터의 셀 크기는 다양할 수 있습니다.

정수 래스터는 포인트, 라인 또는 폴리곤 피처로 변환할 수 있지만 부동 소수점 래스터는 포인트 피처로만 변환됩니다.

현재 맵 범위 사용을 선택한 경우 현재 맵 범위 내에 표시된 래스터 셀만 변환되고, 선택하지 않으면 현재 맵 범위 외부에 있더라도 래스터 레이어의 모든 셀이 변환됩니다. 래스터 레이어가 대형이거나 고해상도인 경우 이 점에 유의하세요.

이 도구의 용도는 다음과 같습니다.

  • 토지 이용 분류 래스터에서 폴리곤 피처를 생성합니다.
  • DEM에서 식별된 스트림을 라인 피처로 변환할 수 있습니다.
  • DEM을 비래스터 기반의 워크플로에서 사용할 수 있는 포인트 피처로 변환할 수 있습니다.

변환할 래스터 레이어 선택

피처 레이어로 변환될 입력 래스터 레이어입니다.

필드 선택

변환 값을 지정하는 필드입니다.

임의의 정수 또는 텍스트 값이 사용될 수 있습니다.

부동 소수점 값이 포함된 필드는 결과가 포인트 데이터셋으로 변환되는 결과일 경우에만 사용할 수 있습니다.

각 래스터 셀의 값을 포함하는 Value 필드가 기본값입니다.

결과 유형 선택

결과 유형을 지정합니다.

  • 포인트 - 래스터가 포인트 데이터셋으로 변환됩니다. 이 옵션이 기본 설정입니다.
  • 라인 - 래스터가 라인 피처 데이터셋으로 변환됩니다.
  • 폴리곤 - 래스터가 폴리곤인 피처 데이터셋으로 변환됩니다.

라인 또는 폴리곤 단순화

라인 또는 폴리곤의 간소화(스무스) 여부를 지정합니다. 기존 래스터 셀 가장자리와 최대한 유사한 모양을 유지하면서 라인에 세그먼트의 최소값을 포함하는 방법으로 부드럽게 처리합니다.

  • 선택 - 라인 또는 폴리곤 피처가 단순한 모양으로 부드럽게 처리되어 더욱 간략화된 결과가 만들어집니다. 이 옵션이 기본 설정입니다.
  • 선택 취소 - 라인 또는 폴리곤 피처가 래스터 데이터셋의 셀 경계를 따르게 됩니다.

이 매개변수는 결과 유형이 라인이나 폴리곤인 경우에만 지원됩니다.

멀티파트 피처 생성

결과 폴리곤이 싱글파트로 구성될지 또는 멀티파트로 구성될지를 지정합니다.

  • 선택 - 동일한 값을 가진 폴리곤을 기반으로 멀티파트 피처가 생성됩니다.
  • 선택 취소 - 각 폴리곤에 대해 개별(싱글파트) 피처가 생성됩니다. 이 옵션이 기본 설정입니다.

이 매개변수는 결과 유형이 폴리곤인 경우에만 지원됩니다.

폴리곤 피처당 최대 버텍스 수

폴리곤을 더 작은 폴리곤으로 세분화하는 데 사용되는 버텍스 한도입니다.

이 매개변수를 비워두면 결과 폴리곤이 분할되지 않습니다. 이 옵션이 기본 설정입니다.

이 매개변수는 결과 유형이 폴리곤인 경우에만 지원됩니다.

결과 레이어 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 결과가 저장될 내 콘텐츠에서 폴더 이름을 지정할 수 있습니다.
