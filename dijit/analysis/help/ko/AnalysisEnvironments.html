포털 분석 환경 설정

Portal for ArcGIS 및 ArcGIS Online에서의 분석을 위한 환경 설정입니다.

결과 레이어 덮어쓰기

기존 레이어를 덮어쓰는 옵션입니다. 이 옵션을 선택한 경우 같은 이름이 있는 레이어가 콘텐츠 내에 이미 있으면 덮어씁니다.

  • 선택함 - 같은 이름이 있는 레이어가 콘텐츠 내에 이미 있으면 덮어씁니다. 결과에 입력과 같은 좌표체계가 있게 됩니다.
  • 선택하지 않음 - 제공된 결과 레이어 이름이 이미 있는 경우 오류가 발생하며 도구를 실행하려면 결과 레이어의 이름을 다시 지정해야 합니다. 이 방법이 기본값입니다.

분석을 실행한 후 분석 위젯 종료하기

도구를 실행하는 동안 분석 대화 창을 계속 열어두고 활성화하는 옵션입니다. 이 설정을 통해 이전 분석 작업이 계속 실행되는 동안에도 분석 매개변수를 쉽게 수정할 수 있습니다. 분석은 반복적인 과정이며 도구 대화 창을 계속 열어두면 매개변수를 쉽게 변경하고 분석을 다시 실행할 수 있습니다.

  • 선택함 - 분석 도구 창이 종료되며 포커스가 목차로 돌아갑니다. 이 옵션이 기본 설정입니다.
  • 선택하지 않음 - 분석 도구 창이 열려 있고 활성화됩니다.

분석 결과를 호스팅 피처 레이어로 저장

분석에서 만든 결과 레이어를 피처 컬렉션 또는 호스팅 피처 레이어로 저장할지 지정하는 옵션입니다.

  • 선택함 - 분석 결과가 호스팅 피처 레이어로 생성되며 항목이 콘텐츠 내에 생성됩니다. 이 방법이 기본값입니다.
  • 선택하지 않음 - 분석 결과가 피처 컬렉션으로 생성됩니다.

결과 좌표체계

결과 레이어의 좌표체계를 정의하는 데 사용하는 래스터 분석의 환경 설정입니다.

  • 입력과 동일—분석 결과가 입력과 동일한 좌표체계로 표시됩니다. 이 방법이 기본값입니다.
  • 지정한 대로—분석 결과에 대한 좌표체계를 선택합니다. 이 옵션을 선택하는 경우 지구 아이콘을 클릭하여 알고 있는 좌표체계의 목록에서 선택하거나, 제공된 상자에 공간 참조 WKID를 추가합니다.
  • 레이어 <이름>—분석 결과의 좌표체계를 결정하는 데 사용할 웹 맵에서 기존 레이어를 선택합니다.

좌표체계 처리

분석을 실행하는 동안 사용할 좌표체계를 결정하는 데 사용하는 GeoAnalytics의 환경 설정입니다.

GeoAnalytics의 경우 일부 분석에는 투영된 좌표체계에서 실행되는 처리가 필요합니다. 데이터 또는 선형 단위의 비닝을 구현한 모든 도구에는 투영된 좌표체계가 필요합니다.

  • 입력과 동일—분석이 입력과 동일한 좌표체계에서 처리됩니다. 이 방법이 기본값입니다.
  • 지정한 대로—분석을 처리할 좌표체계를 선택합니다. 이 옵션을 선택하는 경우 지구 아이콘을 클릭하여 알고 있는 투영된 좌표체계의 목록에서 선택하거나, 제공된 상자에 공간 참조 WKID를 추가합니다.
  • 레이어 <이름>—웹 맵에서 기존 레이어를 선택합니다. 선택한 레이어의 좌표체계는 분석을 처리하는 데 사용할 좌표체계를 결정하는 데 사용됩니다.

범위(Extent)

분석을 실행하는 동안 사용할 범위 또는 경계를 결정하는 데 사용하는 환경 설정입니다. 완전히 범위 내에 있거나 지정된 범위를 교차하는 모든 입력 피처가 분석에서 사용됩니다.

  • 기본 - 도구에서 제공된 범위를 사용합니다.
  • 지정한 대로—범위를 지정할 좌표체계를 수동으로 입력합니다.
  • 레이어 <이름>—웹 맵에서 기존 레이어를 선택합니다. 선택한 레이어의 공간 범위는 분석을 처리하는 데 사용되는 범위를 결정하는 데 사용됩니다.

래스터 스냅

래스터 분석을 실행할 때 래스터 스냅 환경을 설정하면 지정한 래스터 스냅 레이어의 셀 정렬과 일치하도록 결과 래스터 레이어의 범위가 조정됩니다.

셀 크기

래스터 분석을 실행할 때 이 환경은 결과 래스터 레이어를 생성하는 데 사용할 셀 크기 또는 해상도를 설정합니다. 기본 결과 해상도는 입력 래스터 레이어의 가장 큰 셀 크기로 결정됩니다.

  • 입력 최소—모든 입력 레이어의 가장 작은 셀 크기를 사용합니다.
  • 입력 최대—모든 입력 레이어의 가장 큰 셀 크기를 사용합니다. 이 방법이 기본값입니다.
  • 지정한 대로—셀 크기를 설정하도록 숫자 값을 지정합니다. 이 옵션을 선택하는 경우 기본값은 1입니다.
  • 레이어 <이름> - 셀 크기를 선택한 래스터 레이어로 설정합니다.

마스크(Mask)

래스터 분석을 실행할 때 마스크 환경을 사용하여 관심 영역을 정의하는 데 사용할 레이어를 지정합니다. 분석 마스크 내에 해당하는 이러한 셀만 분석 작업에서 고려됩니다.

  • 마스크는 래스터 또는 피처 레이어가 될 수 있습니다.
  • 분석 마스크가 래스터인 경우 값을 가진 모든 셀이 마스크를 정의하는 데 고려됩니다. NoData인 마스크 래스터의 셀은 마스크 외부에 있는 것으로 간주되며 분석 결과 레이어에서 NoData가 됩니다.
  • 분석 마스크가 피처 레이어인 경우 실행 시 래스터로 내부 전환됩니다. 이러한 이유로 및 이 분석에 맞게 적절히 설정되도록 주의를 기울여야 합니다.

ArcGIS 데이터 스토어

GeoAnalytic 결과는 ArcGIS Data Store에 저장되며 Portal for ArcGIS에서 피처 레이어로 나타납니다. 대부분의 경우 결과는 시공간적 데이터 저장소에 저장되어야 합니다. 이 설정이 기본값입니다. 하지만 다음과 같은 이유로 결과를 관계형 데이터 저장소에 저장할 수도 있습니다.

  • 결과를 포털 간 컬래버레이션에 사용하기 위해
  • 기능을 결과와 동기화하기 위해

GeoAnalytics 결과가 늘어날 것으로 예상되며 시공간형 빅데이터 저장소의 기능을 활용하여 대용량 데이터를 처리하려는 경우에는 관계형 데이터 저장소를 사용하지 마세요.
