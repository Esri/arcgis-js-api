테이블의 지오코드 위치(Geocode Locations from Table)

지오코딩은 주소에서 좌표로 변환하는 방법입니다. 이 도구는 테이블, CSV 파일 또는 Excel 파일의 주소를 추가 열로 좌표가 있는 결과 파일 또는 피처 레이어로 변환합니다.

입력 테이블 선택

지오코딩할 주소가 있는 입력 테이블입니다. 테이블, CSV 파일 또는 Excel 파일에 입력해야 합니다.

로케이터 선택

지오코딩할 로케이터 서비스를 선택합니다. 사용할 로케이터 서비스가 드롭다운 메뉴에 없으며, 관리자에게 연락하여 로케이터 서비스를 일괄 지오코딩에 대해 구성하도록 합니다.

국가

모든 주소가 한 국가 내에 있으면 가장 정확한 결과를 위해 해당 국가를 선택합니다. 그렇지 않으면 이 매개변수를 비워 둡니다.

데이터 필드 선택

로케이터 서비스에서는 데이터의 특정 필드(예: 주소, 시/군/구, 시/도 및 우편번호)를 예상합니다. 드롭다운 상자를 사용하여 입력 파일 또는 테이블의 데이터 필드를 로케이터에서 예상하는 필드와 일치하도록 합니다. 매핑하는 필드가 많을수록 결과가 더 정확합니다. 입력 테이블에서 모든 데이터가 열 한 개에 있으면 단일 필드를 선택하고 드롭다운 메뉴에서 데이터 필드를 선택합니다. 그렇지 않으면 여러 필드를 선택하고 적합한 로케이터 필드를 테이블의 여러 필드 열에 매핑합니다.

결과 형식 선택

결과 형식입니다. 피처 서비스를 생성하도록 선택했으면 입력 테이블의 주소가 맵에 포인트 피처 레이어로 추가됩니다. CSV 및 XLS 옵션에서는 작업이 완료되면 지오코딩 작업 결과가 포함된 파일을 내 콘텐츠에 추가합니다.

결과 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어 이름이 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 메뉴를 사용하여 내 콘텐츠에서 결과를 저장할 폴더의 이름을 지정할 수 있습니다.
