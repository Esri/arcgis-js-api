샘플

이 도구는 정의된 위치에 대해 한 래스터 또는 래스터 집합의 셀 값을 보여주는 테이블 또는 포인트 피처 클래스를 생성합니다. 위치는 래스터 셀, 폴리곤 피처, 폴리라인 피처 또는 포인트 집합으로 정의됩니다.

각 위치의 모든 입력 래스터에서 픽셀값이 추출됩니다. 테이블 또는 포인트 피처 클래스는 각 입력 래스터의 픽셀값이 포함된 필드로 생성됩니다. 입력 래스터 테이블의 추가 속성은 결과 테이블에 포함되지 않습니다.

단일밴드, 다중밴드, 다차원 또는 다변량과 같은 래스터의 조합을 입력 래스터로 지정할 수 있습니다. 입력 래스터가 다차원이면 결과 테이블의 구조가 변경됩니다. 다중밴드 래스터가 입력 래스터 중 하나로 지정되면 해당 입력의 모든 밴드가 사용됩니다. 따라서 입력 다중밴드 래스터의 일부 밴드를 선택해 처리하려면 먼저 해당 특정 밴드로 구성된 새 래스터 데이터셋을 생성해야 합니다. 그 결과를 입력 래스터 목록에 사용합니다.

입력 위치는 래스터 또는 포인트 피처일 수 있습니다.

  • 래스터 - 유효한 값(NoData가 아님)이 포함된 픽셀을 사용하여 모든 입력 래스터에서 픽셀값을 추출하며, 픽셀의 중심이 포인트 위치에 사용됩니다.
  • 포인트 - 각 포인트 위치에 값이 샘플링됩니다.
  • 폴리라인 또는 폴리곤 - 폴리라인 또는 폴리곤 중심에 값이 샘플링됩니다. 입력 래스터가 다차원인 경우, 값은 통계 유형에 따라 집계됩니다.

입력 래스터는 네이티브 공간 기준 체계와 해상도로 샘플링됩니다. 공간 기준 체계 서로 다른 여러 입력 래스터가 있는 경우 입력 위치는 값이 추출되는 각 개별 래스터의 공간 기준 체계에 투영됩니다. 분석 환경은 입력 위치에만 적용되며 입력 래스터에는 적용되지 않습니다.

입력 래스터의 NoData 셀에서 값을 추출하는 위치의 경우 결과 테이블에 <null> 값이 제공됩니다.

멀티포인트 피처 분석을 수행하려면 이러한 피처를 이 도구에 사용하기 전에 먼저 단일 포인트 피처로 변환해야 합니다.

결과 샘플 테이블 내 x,y 좌표의 공간 기준 체계는 결과 좌표계 환경에 지정되어 있지 않는 한 입력 위치 래스터 또는 포인트 피처와 동일합니다. 입력 위치 데이터의 공간 기준 체계를 알 수 없다면 x,y 좌표의 공간 기준 체계도 알 수 없습니다.

결과 피처 클래스의 공간 기준 체계는 결과 좌표계 환경에 지정되어 있지 않는 한 입력 위치 래스터 또는 포인트 피처와 동일합니다.

병렬 처리 계수 환경은 입력이 다차원 래스터인 경우에만 지원됩니다.

현재 맵 범위 사용을 선택하면 현재 맵 범위 내에 보이는 레이어의 변수 또는 밴드가 분석됩니다. 선택하지 않으면 현재 맵 범위 외부에 있는 변수를 포함하여 전체 레이어의 변수 또는 밴드가 분석됩니다.

샘플링할 영상 레이어 선택

입력 위치 데이터를 기반으로 값이 샘플링되는 단일 영상 레이어 또는 영상 레이어 목록입니다.

다차원 영상 레이어의 모든 슬라이스를 샘플링하려면 다차원 영상 레이어가 유일한 입력 래스터여야 합니다.

위치 레이어 선택

샘플링할 위치를 식별하는 데이터입니다.

래스터, 포인트, 폴리라인, 폴리곤 피처 클래스를 입력 위치 데이터로 사용할 수 있습니다.

고유 ID(Unique ID) 필드 지정

입력 위치 래스터 또는 피처 클래스의 모든 위치나 피처에 대해 서로 다른 값이 포함되는 필드입니다.

위치 데이터 취득 정보 지정(선택 사항)

위치 셀 또는 피처와 연관된 시간, 깊이 또는 기타 취득 데이터를 지정합니다.

각 디멘전에 대해, 다음과 같은 세 가지 옵션을 사용하여 위치 셀 또는 피처와 연관된 취득 정보를 지정할 수 있습니다.

  • 단일 필드 또는 값 - 이 옵션을 선택하면 다음 정보를 지정해야 합니다.
    • 디멘전 필드 또는 값 - 위치 데이터에서 디멘전 값의 필드를 선택하거나 특정 디멘전 값을 입력합니다.
  • 상대 값이 포함된 단일 필드 또는 값 -이 옵션을 선택하면 변수의 통계 계산 범위를 정의할 때 다음의 정보를 함께 지정해야 합니다.
    • 디멘전 필드 또는 값 - 위치 데이터에서 디멘전 값의 필드를 선택합니다.
    • 이전 상대 값 - 디멘전 필드 또는 값에서 뺄 음수가 아닌 숫자를 지정하여 디멘전 값 범위의 시작을 정의합니다. 이 값은 디멘전이 시간인 경우 일수를 나타냅니다.
    • 이후 상대 값 - 디멘전 필드 또는 값에 더할 음수가 아닌 숫자를 지정하여 디멘전 값 범위의 끝을 정의합니다. 이 값은 디멘전이 시간인 경우 일수를 나타냅니다.
  • 시작 및 종료 필드 또는 값 -이 옵션을 선택하면 변수의 통계 계산 범위를 정의할 때 아래의 정보를 함께 지정해야 합니다.
    • 시작 필드 또는 값 - 위치 데이터에서 디멘전 값의 필드를 선택하거나 특정 디멘전 값을 입력하여 디멘전 값 범위의 시작을 정의합니다.
    • 종료 필드 또는 값 - 위치 데이터에서 디멘전 값의 필드를 선택하거나 특정 디멘전 값을 입력하여 디멘전 값 범위의 종료를 정의합니다.

버퍼 거리 필드 또는 값 지정

위치 데이터 피처 주변의 지정 거리입니다. 버퍼 거리는 위치 피처 공간 기준 체계의 선형 단위로 지정됩니다. 피처가 지리 기준 체계를 사용할 경우 단위는 도입니다.

통계는 이 버퍼 영역 내에서 계산됩니다.

통계 유형 선택

계산될 통계 유형입니다.

  • 최소값 - 지정된 범위 내에서 최소값을 찾습니다.
  • 최대값 - 지정된 범위 내에서 최대값을 찾습니다.
  • 중앙값 - 지정된 범위 내에서 중앙값을 찾습니다.
  • 평균 - 지정된 범위의 평균을 계산합니다.
  • 합계 - 지정된 범위 내에 있는 변수의 합계를 계산합니다.
  • 다수 - 가장 많이 발생한 값을 찾습니다.
  • 소수 - 가장 적게 발생한 값을 찾습니다.
  • 표준편차 - 표준편차를 계산합니다.
  • 백분위수 - 지정된 범위 내에서 정의된 백분위수를 계산합니다.

백분위수 값

통계 유형 매개변수가 백분위수로 설정된 경우 계산할 백분위수입니다.

이 값의 범위는 0~100일 수 있습니다. 기본값은 90입니다.

리샘플링 기법 지정

래스터를 샘플링할 때 사용되는 리샘플링 알고리즘입니다.

  • 최근린 - 최근린입니다. 이 방법이 기본값입니다.
  • 이중 선형 - 이중 선형 보간입니다.
  • 3차 - 3차 회선 보간법입니다.

결과 레이아웃 지정

샘플링된 값이 결과 테이블의 행에 표시되는지 또는 열에 표시되는지를 지정합니다. 이 옵션은 입력 다차원 래스터에 하나의 변수와 하나의 디멘전이 포함되어 있고 각 슬라이스가 단일밴드 래스터인 경우에만 사용할 수 있습니다.

  • 샘플링된 값이 행에 표시됨 - 샘플링된 값이 결과 테이블의 개별 행에 나타납니다. 이 방법이 기본값입니다.
  • 샘플링된 값이 열에 표시됨 - 샘플링된 값이 결과 테이블의 개별 열에 나타납니다. 이 옵션은 입력 다차원 래스터에 하나의 변수와 하나의 디멘전이 포함되어 있고 각 슬라이스가 단일밴드 래스터인 경우에만 유효합니다.

결과 유형 지정

샘플링된 값이 테이블에 포함되어 있는 포인트 피처 클래스를 생성할지 또는 샘플링된 값이 포함되어 있는 테이블만 생성할지를 지정합니다.

  • 테이블 - 샘플링된 값이 포함되어 있는 테이블을 생성합니다. 이 방법이 기본값입니다.
  • 피처 - 샘플링된 값이 테이블에 포함되어 있는 포인트 피처 클래스를 생성합니다. 입력 위치 피처가 폴리라인 또는 폴리곤인 경우 결과의 각 포인트는 입력 위치 피처의 각 지오메트리의 중심입니다.

결과 레이어 이름

내 콘텐츠에 생성되고 맵에 추가되는 레이어의 이름입니다. 기본 이름은 도구 이름과 입력 레이어 이름을 기반으로 합니다. 레이어가 이미 있는 경우 다른 이름을 입력하라는 메시지가 나타납니다.

결과 저장: 드롭다운 상자를 사용하여 결과가 저장될 내 콘텐츠에서 폴더 이름을 지정할 수 있습니다.
