Interpolar Pontos

Esta ferramenta permite a você prever os valores em novos locais baseado em medidas a partir de uma coleção de pontos. A ferramenta obtém os dados de ponto com valores em cada ponto e retorna um raster de valores previstos:

Outros casos de uso desta ferramenta incluem o seguinte:

  • Uma área de gerenciamento da qualidade do ar tem sensores que medem níveis de poluição. Interpolar Pontos pode ser utilizada para predizer níveis de poluição em locais que não têm sensores, como locais com populações de risco—escolas ou hospitais, por exemplo.
  • Prever concentrações de metal pesado em colheitas baseado em amostras tiradas de plantas individuais.
  • Prever níveis de nutriente do solo (nitrogênio, fósforo, potássio, e assim por diante) e outros indicadores (como condutividade elétrica) para estudar suas relações com rendimento da cultura e prescrever quantidades precisas de fertilizante para cada local no campo.
  • Os aplicativos meteorológicos incluem predição de temperaturas, chuva e variáveis associadas (como chuva ácida).

Otimizar para

Escolha sua preferência para velocidade versus precisão.

Predições mais precisas tomam mais tempo para calcular. Este parâmetro altera os valores padrão de vários outros parâmetros de Interpolar Pontos para otimizar a velocidade de cálculo, precisão de resultados ou um equilíbrio dos dois. Por padrão, a ferramenta otimizará para equilibrar.

Transformar dados para distribuição normal

Escolha se transformar seus dados para a distribuição normal.

A interpolação é mais precisa para dados que seguem uma distribuição normal (em forma de sino). Se os seus dados parecerem não estarem normalmente distribuídos, você deverá executar uma transformação.

Tamanho dos modelos locais

A ferramenta Interpolar Pontos funciona construindo modelos de interpolação locais que são misturados para criar o mapa de predição final. Este parâmetro controla quantos pontos estarão contidos em cada modelo local. Os valores menores tornarão os resultados mais locais e poderão revelar efeitos em pequena escala, mas podem introduzir alguma instabilidade nos cálculos. Os valores maiores serão mais estáveis, mas alguns efeitos locais podem estar ausentes.

O valor pode variar de 30 até 500, mas valores típicos estão entre 50 e 200.

Número de Vizinhos

As predições são calculadas baseadas em pontos vizinhos. Este parâmetro controla quantos pontos serão utilizados no cálculo. A utilização de um número maior de vizinhos geralmente produzirá resultados mais precisos, mas os resultados demoram mais para calcular.

Este valor pode variar de 1 até 64, mas valores típicos estão entre 5 e 15.

Tamanho da Célula de Saída

O tamanho da célula e a unidade para os rasters de saída.

As unidades disponíveis são Pés, Milhas, Metros e Quilômetros.

Erros de predição de saída

Escolha se você deseja criar um raster de erros padrão para os valores previstos.

Erros padrão são úteis, pois fornecem informações sobre a confiabilidade dos valores preditos. Uma regra geral simples é que o valor verdadeiro cairá dentro de dois erros padrão de 95 por cento de tempo do valor predito. Por exemplo, suponha que um novo local tem um valor previsto de 50 com um erro padrão de 5. Isto significa que a melhor suposição desta ferramenta é que o valor verdadeiro neste local é 50, mas ele razoavelmente pode ser tão baixo quanto 40 ou tão alto quanto 60. Para calcular este intervalo de valores razoável, multiplique o erro padrão por 2, adicione este valor ao valor previsto para obter o maior valor final do intervalo e subtraia do valor previsto para obter o menor valor final do intervalo.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em . Se você tiver os privilégios para criar camadas de imagens de mosaico e dinâmicas, será possível especificar qual tipo de camada gerar na saída usando a caixa suspensa Salvar resultado como.
