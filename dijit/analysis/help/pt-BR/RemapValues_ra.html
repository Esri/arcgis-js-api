Remapear Valores

Esta ferramenta permite que você altere ou reclassifique os valores de pixels dos dados raster. Os valores de pixels são remapeados especificando uma faixa de valores de pixel para mapear um valor de pixel de saída. O valor de pixel de saída pode ser um valor válido ou NoData, que são pixels que não têm um valor conhecido associado a eles.

Se Utilizar extensão de mapa atual estiver selecionada, somente as células visíveis dentro da extensão de mapa atual serão remapeadas. Se desmarcado, todas as células no raster serão processadas, ainda que eles estejam fora da extensão do mapa atual.

Dados de Entrada

Os dados de entrada que terão seus valores remapeados.

Remapear valores de pixel para novos valores

Insera uma faixa de valores especificando um valor mínimo e máximo e então atribua um valor de saída ou um valor de NoData para a faixa especificada.

Para remapear uma nova faixa de valores, clique em Adicionar, que adiciona uma linha. Clique em cada valor para editar a tabela nas faixas e o valor remapeado que você precisa. Quando você especificar as faixas de valor, há duas regras importantes que são empregadas:

  • O valor mínimo da faixa é inclusivo e o valor máximo é exclusivo.
  • As linhas na tabela estão processadas de cima para baixo.

Ambas as regras são empregadas para reordenar os valores. Já que cada linha é processada de cima para baixo, estes valores tomam precedência ao determinar a saída, mesmo quando as faixas são utilizadas.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em . Se você tiver os privilégios para criar camadas de imagens de mosaico e dinâmicas, será possível especificar qual tipo de camada gerar na saída usando a caixa suspensa Salvar resultado como.
