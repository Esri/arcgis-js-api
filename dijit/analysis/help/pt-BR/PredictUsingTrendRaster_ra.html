Prever Utilizando Raster de Tendência

Computes a forecasted multidimensional raster using the output trend raster from the Generate Trend Raster tool.

Se Utilizar extensão de mapa atual estiver selecionada, as variáveis da camada que são visíveis dentro da extensão de mapa atual serão analisadas. Se desmarcada, as variáveis da camada total serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Escolher camada de imagens multidimensionais para prever

O raster de tendência multidimensional de entrada da ferramenta Gerar Raster de Tendência .

Escolher as variáveis que serão previstas

A variável ou variáveis que serão previstas na análise. Se nenhuma variável for especificada, todas as variáveis serão utilizadas.

Escolha o método utilizado para fornecer valores de dimensão de previsão

Especifica o método utilizado para fornecer valores de dimensão de previsão.

  • Por Valor—A previsão será calculada para um valor de dimensão único ou uma lista de valores de dimensão especificados na próxima etapa. Este é o padrão. Por exemplo, você deseja prever a precipitação anual pelos anos 2050, 2100 e 2150.
  • Por Intervalo—A previsão será calculada para um intervalo de dimensão definido por um valor inicial e um valor final. Por exemplo, você deseja prever a precipitação anual para cada ano entre 2050 e 2150.

Especificar valores de dimensão para previsão

O valor de dimensão ou valores a serem utilizados na previsão.

O formato do tempo, profundidade, e valores de altura devem corresponder ao formato dos valores de dimensão utilizados para gerar o raster de tendência. Se o raster de tendência foi gerado para a dimensão StdTime, o formato deverá ser YYYY-MM-DDTHH:MM:SS, por exemplo 2050-01-01T00:00:00. Múltiplos valores são separados com um ponto-e-vírgula.

Este parâmetro é exigido quando o método Por Valor é selecionado para definir valores de dimensão para previsão.

Iniciar

A data inicial, altura ou profundidade do intervalo de dimensão a ser utilizado na previsão.

Final

A data final, altura ou profundidade do intervalo de dimensão a ser utilizado na previsão.

Definir o intervalo de dimensão para previsão

Define a data inicial e data final, altura ou profundidade do intervalo de dimensão a ser utilizado na previsão.

Especificar o número de etapas entre os valores inicial e final

O número de etapas entre dois valores de dimensão a ser incluído na previsão. O valor padrão é 1.

Por exemplo, para prever valores de temperatura a cada cinco anos, utilize um valor de 5.

Escolha a unidade que será utilizada para o intervalo de valor do tempo.

Especifica a unidade que será utilizada para o intervalo de valor. Isto se aplica somente quando a dimensão de análise é uma dimensão de tempo.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em . Se você tiver os privilégios para criar camadas de imagens de mosaico e dinâmicas, será possível especificar qual tipo de camada gerar na saída usando a caixa suspensa Salvar resultado como.
