Classificar Objetos Utilizando Aprendizagem Detalhada

Esta ferramenta executa um modelo de aprendizagem detalhada testado em um raster de entrada para produzir uma tabela ou classe de feição na qual cada objeto de entrada tem um rótulo de classe atribuído.

Se Utilizar extensão de mapa atual estiver selecionada, somente as áreas do raster que são visíveis na extensão de mapa atual serão analisadas. Se não selecionada, o raster inteiro será analisado, mesmo que estiver fora da extensão de mapa atual.

Escolher imagem utilizada para classificar objetos

A imagem de entrada utilizada para detectar objetos.

Escolher a camada de feição para objetos (Opcional)

A camada de feição de entrada de ponto, linha ou polígono que identifica a localização de cada objeto para ser classificado e rotulado. Cada linha na camada de feição de entrada representa um objeto único.

Se nenhuma camada de feição de entrada for especificada, a ferramenta assume que cada imagem de entrada contenha um objeto único a ser classificado. Se a imagem de entrada ou imagens utilizam uma referência espacial, a saída a partir da ferramenta é uma camada de feição, em que a extensão de cada imagem é utilizada como o geometria de limite de cada feição rotulada. Se a imagem de entrada ou imagens não forem referenciadas espacialmente, a saída da ferramenta é uma tabela contendo o valor do ID da imagem e os rótulos de classe de cada imagem.

Escolher modelo de aprendizado detalhado utilizado para detectar classificar objetos

O item do pacote de aprendizado detalhado de entrada ( .dlpk).

O pacote de aprendizado detalhado é composto do arquivo JSON da definição de modelo da Esri ( .emd), o arquivo de modelo binário de aprendizado detalhado e, opcionalmente, a função de raster do Python a ser utilizada.

Modo de Processamento

Especifica como todos os itens do raster em um conjunto de dados de mosaico ou um serviço de imagem serão processados. Este parâmetro é aplicado quando o raster de entrada é um conjunto de dados de mosaico ou um serviço de imagem.

  • PROCESS_AS_MOSAICKED_IMAGE — Todos os itens do raster no conjunto de dados de mosaico ou serviço de imagem serão colocados em mosaico e processados. Este é o padrão.
  • PROCESS_ITEMS_SEPARATELY — Todos os itens do raster no conjunto de dados de mosaico ou serviço de imagem serão processados como imagens separadas.

Especificar parâmetros do modelo de aprendizagem detalhada

Os parâmetros da função são definidos na classe de função de raster do Python referenciada pelo modelo de entrada. É aqui que você lista os parâmetros adicionais de aprendizado detalhado e os parâmetros para experimentos e refinamentos, como um limite de confiança para ajustar a sensibilidade.

Os nomes dos parâmetros são preenchidos pela ferramenta da leitura do módulo Python no servidor de análise do raster.

Define o nome do campo de rótulo de classe.

O nome do campo que conterá o rótulo de classificação na camada de feição de saída.

Se nenhum nome de campo for especificado, um novo campo denominado ClassLabel será gerado na classe de feição de saída.

Modo do processamento

Especifica como todos os itens de raster em um de serviço de imagem serão processados.

  • Processar como imagem mosaicada—Todos os itens de raster no serviço de imagem serão mosaicados e processados. Este é o padrão.
  • Processar itens separadamente—Todos os itens do raster no serviço de imagem serão processados como imagens separadas.
