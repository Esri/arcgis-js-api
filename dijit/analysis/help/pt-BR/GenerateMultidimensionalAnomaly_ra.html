Gerar Anomalia Multidimensional

Computes the anomaly for each slice in an existing multidimensional raster to generate a new multidimensional raster.

Uma anomalia é a divergência de uma observação de seu valor padrão, média ou mediana.

Esta ferramenta calcula anomalias com o passar do tempo para uma ou mais variáveis em um raster multidimensional. Se você tiver uma dimensão de não tempo além da dimensão de tempo, a anomalia será calculada em cada etapa na dimensão adicional.

Por exemplo, você tem dados mensais de temperatura do oceano, coletado a cada 1 metro de profundidade até 100 metros e você deseja calcular as anomalias de temperatura como divergências a partir da média anual. Esta ferramenta determinará as anomalias de temperatura baseado em uma média anual se você configurar Anual como o intervalo temporal para calcular a média, e retornará valores de anomalia para cada uma das 100 profundidades.

Isto ferramenta somente suporta conjuntos de dados raster multidimensional que tem uma dimensão de tempo.

Se Utilizar extensão de mapa atual estiver selecionada, as variáveis da camada que são visíveis dentro da extensão de mapa atual serão analisadas. Se desmarcada, as variáveis da camada total serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Escolha a camada de imagens multidimensional para gerar anomalia

A camada de imagens de entrada de um conjunto de dados raster multidimensional.

Escolha variáveis em que as anomalias serão geradas

A variável ou variáveis em que as anomalias serão calculadas. Se nenhuma variável for especificada, todas as variáveis com uma dimensão de tempo serão analisadas.

Escolha o método para gerar anomalia

Especifica o texto que será utilizado para calcular a anomalia.

  • Diferença da Média—A diferença entre um valor de pixel e a média deste valor do pixel através de partes definidas pelo intervalo será calculada. Este é o padrão.
  • Diferença de Porcentagem da Média—A diferença de porcentagem entre um valor de pixel e a média deste valor do pixel através das partes definidas pelo intervalo será calculada.
  • Porcentagem da Média—A porcentagem da média será calculada.
  • Pontuação Z—A pontuação z de cada pixel será calculada. Uma pontuação z de 0 indica que o valor do pixel é à média. Uma pontuação z de 1 indica que o valor do pixel é 1 desvio padrão da média. Uma pontuação z for 2, o valor do pixel é 2 desvio padrão da média e assim por diante.
  • Diferença da Média—A diferença entre um valor de pixel e a média matemática destes valores do pixel através de partes definidas pelo intervalo será calculada.
  • Diferença de Porcentagem da Média—A diferença de porcentagem entre um valor de pixel e a média matemática destes valores do pixel através de partes definidas pelo intervalo será calculada.
  • Porcentagem de Mediano—A porcentagem da média matemática será calculada.

Os métodos matemáticos para calcular valores de anomalia são listados abaixo. As anomalias podem ser calculadas utilizando a média ou os valores medianos como a definição da média. Se a distribuição de dados for inclinada, a médio pode estar altamente influenciada por valores atípicos, então o valor mediano ou método de pontuação z pode ser melhor utilizado para este tipo de dados.

  • Diferença da média = x - µ
    • x = valor de pixel em uma parte
    • µ = média destes valores de pixel durante o intervalo de tempo fornecido
  • Diferença de porcentagem da média = |x - µ| / [(x + µ)/2]
    • x = valor de pixel em uma parte
    • µ = média destes valores de pixel durante o intervalo de tempo fornecido
    • |x - µ| = valor absoluto da diferença entre o valor e a média
  • Porcentagem da média = x - µ
    • x = valor de pixel em uma parte
    • µ = média destes valores de pixel durante o intervalo de tempo fornecido
  • Diferença de mediano = x - ß
    • x = valor de pixel em uma parte
    • ß = mediano destes valores do pixel durante o intervalo de tempo fornecido
  • Diferença de porcentagem do mediano = |x - ß| / [(x + ß)/2]
    • x = valor de pixel em uma parte
    • ß = mediano destes valores do pixel durante o intervalo de tempo fornecido
    • |x - ß| = valor absoluto da diferença entre o valor e o mediano
  • Porcentagem do mediano = x / ß
    • x = valor de pixel em uma parte
    • ß = mediano destes valores do pixel durante o intervalo de tempo fornecido
  • Pontuação Z = (x - µ) / S
    • x = valor de pixel em uma parte
    • µ = média destes valores de pixel durante o intervalo de tempo fornecido
    • S = o desvio padrão dos valores do pixel durante o intervalo de tempo fornecido

Escolha intervalo temporal para calcular a média

Especifica o intervalo temporal que será utilizado para calcular a média.

  • Todos—Calcula a média através de todas as partes de cada pixel.
  • Anual—Calcula a média anual de cada pixel.
  • Mensalmente recorrente—Calcula a média mensal de cada pixel.
  • Semanalmente Recorrente—Calcula a média semanal de cada pixel.
  • Diariamente Recorrente—Calcula a média diária de cada pixel.
  • A Cada Hora—Calcula a média de hora de cada pixel.
  • Raster Externo—Um conjunto de dados raster existente que contém o valor médio ou mediano de cada pixel que é referenciado.

Escolher camada de imagens da média como referência

Especifica o conjunto de dados raster de referência que contém uma média anteriormente calculada para cada pixel. As anomalias serão calculadas em comparação com esta média.

Ignorar valores ausentes no cálculo

Especifica se os valores ausentes são ignorados na análise.

  • Marcado—A análise incluirá todos os pixels válidos ao longo de uma dimensão fornecida e ignorará quaisquer pixels de NoData. Este é o padrão.
  • Desmarcado—A análise resultará em NoData se houver quaisquer valores de NoData para os pixels ao longo da dimensão fornecida.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em . Se você tiver os privilégios para criar camadas de imagens de mosaico e dinâmicas, será possível especificar qual tipo de camada gerar na saída usando a caixa suspensa Salvar resultado como.
