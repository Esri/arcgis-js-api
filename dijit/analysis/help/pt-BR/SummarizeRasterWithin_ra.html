Resumir Raster Dentro

Esta ferramenta resume os valores de um raster dentro das zonas de outro conjunto de dados.

Alguns exemplos de aplicativos desta ferramenta incluem o seguinte:

  • Fornecida uma camada de limites da bacia e uma camada de limites de uso do solo por tipo de uso do solo, calcule a área total em acres do tipo de uso do solo para cada bacia.
  • Fornecida uma camada de lotes em um município e uma camada de limites da cidade, resuma o valor médio de lotes desocupados dentro de cada limite da cidade.

Se Utilizar extensão de mapa atual estiver selecionada, somente aquelas feições na camada de entrada e camada a serem resumidas que são visíveis dentro da extensão de mapa atual serão analisadas. Se não selecionada, todas as feições em ambas a camada de entrada e a camada a ser resumida serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Escolher camada de área para resumir uma camada raster dentro dos limites definidos

A entrada que define os limites das zonas, ou áreas, que serão resumidas.

A entrada pode ser um raster inteiro ou dados de feições. Para rasters, as zonas são definidas por todos os locais na entrada que tenham o mesmo valor de célula. Para feições, as zonas podem ser sobrepostas. As áreas não têm que ser contíguas.

Escolher campo para definir os limites

O campo que contém os valores que definem cada zona.

Pode ser um número inteiro ou um campo de string do conjunto de dados da zona.

Escolher camada raster para resumir

O raster que contém os valores sobre os quais calcular uma estatística.

Pode ser inteiro ou ponto flutuante. No entanto, quando é do tipo ponto flutuante, as opções para calcular maioria, minoria e variedade não estarão disponíveis.

Escolher estatística para calcular

Especifica o tipo de estatística a ser calculado.

A entrada pode ser uma das seguintes: Average, Maioria, Máximo, Mediano, Mínimo, Minoria, Percentil, Intervalo, Desvio padrão, Soma ou Variedade. O padrão é Média.

Valor de percentil

O percentil para calcular. O padrão é 90, indicando o 90º percentil.

O valor podem variar de 0 até 100. O resultado da especificação do 0º percentil é essencialmente equivalente ao da estatística mínima e o 100º percentil é equivalente ao máximo. Um valor de 50 produzirá essencialmente o mesmo resultado que a estatística mediana.

Este parâmetro é suportada somente se o parâmetro Escolher estatística para calcular estiver configurado para Percentil.

Ignorar valores ausentes no cálculo

Especifica se os valores ausentes são ignorados na análise.

  • Marcado—A análise incluirá as células que possuem um valor no raster do valor de entrada dentro de qualquer zona específica para determinar o valor de saída para essa zona. As células NoData no raster de valor serão ignoradas no cálculo da estatística. Este é o padrão.
  • Desmarcado—A análise resultará em NoData se houver algum valor NoData para as células do raster de valor dentro de uma zona específica.

Processar como multidimensional

Especifica como os serviços de imagem multidimensional de entrada serão processados ​​se forem multidimensionais.

  • Marcado—As estatísticas serão calculadas a partir da parte atual do conjunto de dados multidimensional de entrada. Este é o padrão.
  • Desmarcado—As estatísticas serão calculadas para todas as dimensões do conjunto de dados multidimensional de entrada.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em . Se você tiver os privilégios para criar camadas de imagens de mosaico e dinâmicas, será possível especificar qual tipo de camada gerar na saída usando a caixa suspensa Salvar resultado como.
