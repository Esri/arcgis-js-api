Resumir Raster Dentro

Esta ferramenta resume os valores de um raster dentro das zonas de outro conjunto de dados.

Alguns aplicativos de exemplo incluem o seguinte:

  • Fornecida uma camada de limites da bacia e uma camada de limites de uso do solo por tipo de uso do solo, calcule a área total em acres do tipo de uso do solo para cada bacia.
  • Fornecida uma camada de lotes em um município e uma camada de limites da cidade, resuma o valor médio de lotes desocupados dentro de cada limite da cidade.

Se Utilizar extensão de mapa atual estiver selecionada, somente aquelas feições na camada de entrada e camada a serem resumidas que são visíveis dentro da extensão de mapa atual serão analisadas. Se não selecionada, todas as feições em ambas a camada de entrada e a camada a ser resumida serão analisadas, mesmo que estiverem fora da extensão de mapa atual.

Escolher camada de área para resumir uma camada raster dentro dos limites definidos

A camada que define os limites das áreas ou zonas, que serão resumidas.

A camada pode ser um raster ou dados de feição. Para rasters, as zonas são definidas por todos os locais na entrada que tenham o mesmo valor de célula. As áreas não têm que ser contíguas.

Escolher campo para definir os limites

Este é o atributo da camada que será utilizada para definir os limites das áreas.

Por exemplo, suponha que a primeira camada de entrada define os limites da unidade de gerenciamento, com atributos que definem a região, o distrito e o parcel ID de cada unidade. Você também tem uma camada raster definindo um índice de biodiversidade de cada local. Com o campo que você selecionar, você pode decidir calcular a biodiversidade média no local, distrito ou níveis regionais.

Escolher camada raster para resumir

As células raster desta camada serão resumidas pelas áreas (zonas) que elas caem dentro.

Escolher estatística para calcular

Após você escolher uma camada para resumir, você pode opcionalmente calcular estatísticas de qualquer atributo numérico dos pontos, linhas ou áreas dentro da camada de área de entrada.

Os tipos de estatísticas disponíveis quando o campo selecionado for inteiro serão Média, Maioria, Máximo, Mediana, Mínimo, Minoria, Percentil, Intervalo, Desvio padrão, Soma e Variedade.

Se o campo for ponto flutuanrte, as opções são Média, Máximo, Mediano, Mínimo, Minoria, Faixa, Desvio padrão e Soma.

Valor de percentil

O percentil para calcular. O padrão é 90, indicando o 90º percentil.

Os valores podem variar de 0 até 100. O 0º percentil é essencialmente equivalente à estatística mínima e o 100º percentil é equivalente à máxima. Um valor de 50 produzirá essencialmente o mesmo resultado que a estatística mediana.

Esta opção é suportada somente se o parâmetro Escolher estatística para calcular estiver configurado para Percentil.

Ignorar valores ausentes no cálculo

Se você optar por ignorar valores ausentes, somente as células que têm um valor na camada para ser resumida serão utilizadas para determinar o valor de saída desta área. Caso contrário, se houver valores ausentes em qualquer lugar em uma área, é julgado que não há informações suficientes para executar os cálculos estatísticos de todas as células nesta zona e que a área receberá um valor nulo (NoData) na saída.

Processar como multidimensional

Especifica como os rasters de entrada serão processados se forem multidimensionais. Se você escolher processar como multidimensional, as estatísticas serão calculadas a partir de todas as dimensões do conjunto de dados multidimensional de entrada. Caso contrário, as estatísticas serão calculadas a partir da divisão atual do conjunto de dados multidimensional de entrada, este é o padrão.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em .
