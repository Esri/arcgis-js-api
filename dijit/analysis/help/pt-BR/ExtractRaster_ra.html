Extrair Raster

Esta ferramenta pode recortar um raster para um limite, ou para uma área retangular ou para uma forma que você defina interativamente na tela. Você pode recortar na extensão da área que você tem atualmente exibida no seu mapa ou pela área de estudo definida por um polígono.

A camada de entrada para recortar.

Especificar geometria de recorte

Escolha o tipo de extensão para utilizar ao recortar a geometria da camada:

  • Utilizar extensão atual—A extensão de mapa atual será utilizada para recortar a camada resultante. Este é o padrão.
  • Escolher área de estudo—Permite a você definir uma geometria de recorte usando a ferramenta de desenhar.
  • Escolher camada—Escolha uma camada de feição ou camada de imagem para definir a extensão ou escolha uma camada de feição de polígono para definir a geometria de recorte. Quando uma camada de feição de polígono é selecionada, uma caixa de seleção é exibida para especificar se usar feições de entrada para recortar a geometria. Por padrão, está desmarcado. Significa que a camada de feição de entrada é usada para definir a extensão. Se estiver marcado, significa que as feições de entrada são usadas para recortar a geometria.

Especificar tipo de recorte

Especifique se deseja reter a área de dentro ou de fora de sua geometria de recorte:

  • Dados dentro da geometria—Imagens dentro das extensões de recorte serão removidas. Este é o padrão.
  • Dados fora da geometria—Imagens fora das extensões de recorte serão removidas.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Você pode especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo utilizando a caixa suspensa Salvar resultados em . Se você tiver os privilégios para criar camadas de imagens de mosaico e dinâmicas, será possível especificar qual tipo de camada gerar na saída usando a caixa suspensa Salvar resultado como.
