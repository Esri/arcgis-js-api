Skapa avrinningsområden

Det här verktyget fastställer avrinningsområdet, eller flödesackumuleringsområdet uppströms, för varje punkt i indatalagret.

Du kan till exempel ha punktgeoobjekt som representerar platser för vattenburna föroreningar och vill hitta de troliga källorna till föroreningen. Eftersom källan till föroreningen måste ligga någonstans i avrinningsområdet uppströms från punkten, kan du använda det här verktyget för att avgöra vilket avrinningsområde som innehåller källorna till föroreningen.

Platsen för indatapunkterna är viktig. Om de inte ligger exakt på en dräneringslinje kan de resulterande avrinningsområdena bli mycket små och inte särskilt användbara för din analys. Sökavstånd till närmaste dränering kan användas för att justera platsen för dina indatapunkter så att de hamnar precis på en dräneringslinje. Därmed kan du hitta det avrinningsområde som rinner till punkten på dräneringslinjen. De justerade platserna som används för att hitta avrinningsområdena skrivs till utdatalagret – dina indatapunkter ändras inte.

Resultatet blir ett grupplager som innehåller två lager: avrinningsområdena och de justerade punkter som används för att beräkna avrinningsområdena.

De resulterande avrinningsområdena kommer att ha alla attribut från indatapunkterna tillsammans med följande ytterligare attribut:

  • PourPtID – innehåller ID-värdet från indatapunkten
  • Area Square Kilometers– innehåller området i kvadratkilometer från avrinningsområdet uppströms
  • Description och DataResolution– innehåller information som rör de hydrologiska data som används för att beräkna spårningen

Om Använd aktuell kartutbredning är markerat används endast de geoobjekt som finns i den aktuella kartutbredningen vid beräkningen av avrinningsområdena. Om alternativet inte markeras analyseras alla geoobjekt i indatalagret, även om de ligger utanför den aktuella kartutbredningen.

Punktgeoobjekt som används för beräkning av avrinningsområden

Ett avrinningsområde (flödesackumuleringsområdet uppströms) beräknas för varje punkt.

Förutom att välja ett lager på din karta kan du välja Välj analyslager, som du hittar längst ned i listrutan. Det alternativet öppnar ett galleri där du kan bläddra till och välja datauppsättningar från diverse källor som du kan använda i din analys.

Sökavstånd till närmaste dränering

Om indatapunkterna ligger långt från en dräneringslinje, blir de resulterande avrinningsområdena troligtvis mycket små och är inte till mycket nytta i analysen, till exempel för att avgöra källan till föroreningen uppströms. I de flesta fall vill du att indatapunkterna ska snappa till närmaste dräneringslinje för att hitta de avrinningsområden som rinner till en punkt som finns längs dräneringslinjen. Ange ett sökavstånd om du vill hitta den närmaste dräneringslinjen. Om du inte anger ett sökavstånd kommer verktyget att beräkna och använda ett konservativt sökavstånd.

Om du vill använda den exakta platsen för din indatapunkt anger du sökavståndet noll.

Dräneringslinjerna har i analyssyfte beräknats i förväg av Esri med hjälp av hydrologiska standardmodeller. Om det inte finns någon dräneringslinje inom sökavståndet används den plats som innehåller den största flödesackumuleringen inom sökavståndet.

Resultatlagrets namn

Namnet på lagret som skapas i Mitt innehåll och läggs till i kartan. Standardnamnet baseras på verktygets namn och indatalagrets namn. Om lagernamnet redan finns uppmanas du ange ett annat namn.

Om du använder listrutan Spara resultat i kan du ange namnet på en mapp i Mitt innehåll där resultatet ska sparas.

Det här verktyget förbrukar inte krediter.
