Skapa siktområde

Verktyget Skapa siktområde används för att skapa områden där observatörer kan se objekt på marken. Indataobservatörspunkterna kan representera antingen observatörer (till exempel personer på marken eller spanare i ett brandtorn) eller objekt som observeras (till exempel vindkraftverk, vattentorn, fordon eller andra människor). Resultatområdena är de områden där observatörerna kan se de observerade objekten och tvärt om – de observerade objekten kan se observatörerna.

Både observatörerna och de observerade objekten kan ha en höjd över marken, och den här höjden används för att fastställa synligheten. Ett siktområde som beräknas för 140 meter höga vindkraftverk och två meter långa människor är till exempel normalt sett större än ett siktområde beräknat för 70 meter höga vindkraftverk och två meter långa människor.

Observera att beräkningen av siktområdet inte tar hänsyn till hinder som träd eller byggnader. Du kan tänka dig att beräkningen utförs på bar mark där det inte finns något annat än observatören och det observerade objektet.

Resultatlagret innehåller områdesgeoobjekt som representerar synliga områden. Fältet Frequency innehåller antalet analyspunkter som kan se varje område. Om du till exempel har fem indatapunkter som var och en representerar ett brandtorn, innehåller fältet Frequency värden mellan ett och fem. Områden med värdet ett går bara att se från ett torn, medan områden med värdet fem går att se från alla torn. Resultatlagret innehåller också fältet DEMResolution, som sparar upplösningen för den digitala höjdmodell (DEM) som används för att skapa siktområdena.

Om Använd aktuell kartutbredning är markerat analyseras endast de observatörspunkter som är synliga i den aktuella kartutbredningen. Om alternativet inte markeras analyseras alla observatörspunkter i indatalagret, även om de ligger utanför den aktuella kartutbredningen.

Punktgeoobjekt som representerar observatörsplatser

De punktgeoobjekt som ska användas som observatörsplatser.

Förutom att välja ett lager på din karta kan du välja Välj analyslager, som du hittar längst ned i listrutan. Det alternativet öppnar ett galleri där du kan bläddra till och välja datauppsättningar från diverse källor som du kan använda i din analys.

Höjd för observatörsplatser

Det här är höjden ovan mark för dina observatörsplatser. Standard är den ungefärliga längden för en person.

Höjd för andra objekt på marken

Det här är höjden för byggnader eller personer på marken som används för att fastställa synligheten. Det resulterande siktområdet är de områden där en observatörspunkt kan se dessa andra objekt. Motsatsen stämmer också, de övriga objekten kan se en observatörspunkt.

  • Om din observatörspunkt representerar vindkraftverk och du vill avgöra var personer som står på marken kan se dem, anger du den ungefärliga längden på en person (cirka 2 meter). Resultatet är de områden där en person som står på marken kan se vindkraftverken.
  • Om observatörspunkten representerar brandtorn och du vill avgöra vilka torn som kan se ett rökmoln som stiger 6 meter eller högre, anger du 6 meter som höjd. Resultatet är de områden där ett brandtorn kan se ett rökmoln som är minst 6 meter högt.
  • Om din observatörspunkt representerar utsiktsplatser längs vägar och stigar och du vill fastställa var vindkraftverk som är minst 150 meter höga kan ses, anger du 150 meter som höjd. Resultatet är de områden där en person som står på en utsiktsplats kan se ett vindkraftverk som är minst 150 meter högt.
  • Om observatörspunkten representerar utsiktsplatser och du vill avgöra hur stort område personer som står på marken kan se anger du noll. Resultatet är de platser som kan ses från den utsiktsplatsen.

Maximalt siktavstånd

Det här är ett brytavstånd där beräkningen av synliga områden stoppas. Bortom det här avståndet är det okänt om observatörspunkterna och de andra objekten kan se varandra. Stora värden ökar beräkningstiden. Det maximala tillåtna värdet är 50 kilometer.

Resultatlagrets namn

Namnet på lagret som skapas i Mitt innehåll och läggs till i kartan. Standardnamnet baseras på verktygets namn och indatalagrets namn. Om lagernamnet redan finns uppmanas du ange ett annat namn.

Om du använder listrutan Spara resultat i kan du ange namnet på en mapp i Mitt innehåll där resultatet ska sparas.

Det här verktyget förbrukar inte krediter.
