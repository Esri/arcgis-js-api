Combiner les couches

Cet outil copie les entités de deux couches dans une nouvelle couche. Les couches à combiner doivent toutes contenir les mêmes types d'entités (points, lignes ou surfaces). Vous pouvez déterminer la manière dont les champs des couches en entrée sont rassemblés et copiés, par exemple :

  • Vous disposez d’une couche d’écoles publiques et d’une couche d’écoles privées et souhaitez une couche unique affichant toutes les écoles.
  • Vous disposez de deux couches, chacune contenant des informations de parcelle pour les municipalités contiguës. Vous souhaitez les combiner en une même couche, en conservant uniquement les champs qui portent le même nom et possèdent le même type de données pour les deux couches en entrée.

Si l'option Utiliser l'étendue courante de la carte est sélectionnée, seules les entités visibles au sein de l'étendue courante de la carte seront combinées. Si l'option n'est pas sélectionnée, toutes les entités des deux couches seront combinées, même si elles se trouvent en dehors de l'étendue courante de la carte.

Modifier les champs en fusion

Par défaut, tous les champs des deux entrées sont reportés dans la sortie. Vous pouvez modifier la manière dont les champs sont maintenus dans la sortie de la manière suivante :

  • Renommer : le champ est renommé dans la sortie.
  • Supprimer : le champ est supprimé de la sortie.
  • Apparier : les noms de champs sont rendus équivalents. Les valeurs du champ de la couche à combiner sont alors copiées dans le champ en sortie apparié.

Si les deux couches contiennent un champ de nom et de type de données identiques, leur contenu est fusionné en un même champ de ce nom. Cela ne s’applique pas aux champs où Apparier est utilisé dans le paramètre Modifier les champs en fusion.

Nom de la couche résultat

Le nom de la couche qui sera créée dans My Content (Mon contenu) et ajoutée à la carte. Le nom par défaut repose sur le nom de l'outil et sur le nom de la couche en entrée. Si le nom de la couche existe déjà, vous êtes invité à fournir un autre nom.

Le menu déroulant Save result in (Enregistrer le résultat dans) permet de spécifier le nom d’un dossier dans My Content (Mon contenu) dans lequel sera enregistré le résultat.
