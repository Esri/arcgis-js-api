Générer un raster de tendance

Estime la tendance pour chaque pixel le long d’une dimension pour une ou plusieurs variables dans un raster multidimensionnel.

Choisir une couche d’imagerie multidimensionnelle pour analyser la tendance

Couche d’imagerie en entrée d’un raster multidimensionnel à analyser.

Choisir la dimension le long de laquelle analyser la tendance des variables

Dimension le long de laquelle une tendance sera extraite pour la ou les variables sélectionnées dans l’analyse.

Choisir la ou les variables pour analyser la tendance

La variable ou les variables pour lesquelles les tendances sont calculées. Si aucune variable n’est spécifiée, la première variable dans le raster multidimensionnel est analysée.

Choisir le type de ligne pour ajuster les valeurs de variable le long d’une dimension

Spécifie le type de ligne à utiliser pour ajuster les valeurs de pixel le long d’une dimension.

  • Linear (Linéaire) : les valeurs de pixel des variables sont ajustées le long d’une ligne de tendance linéaire. Il s’agit de l’option par défaut.
  • Polynomial (Polynomial) : les valeurs de pixel des variables sont ajustées le long d’une ligne de tendance polynomiale de second ordre.
  • Harmonic (Harmonique) : les valeurs de pixel des variables sont ajustées le long d’une ligne de tendance harmonique.
  • Mann-Kendall : les valeurs de pixel variables sont évaluées à l’aide du test de tendance de Mann-Kendall.
  • Seasonal Kendall (Kendall saisonnier) : les valeurs de pixel variables sont évaluées à l’aide du test de tendance de Kendall saisonnier.

Indiquez l’unité de temps de la durée d’une saison.

Spécifie l’unité de temps à utiliser pour mesurer la longueur de la période saisonnière lors du test de Kendall saisonnier

  • Days (Jours) : l’unité de durée d’une saison se définit en jours. Il s’agit de l’option par défaut.
  • Months (Mois) : l’unité de durée d’une saison se définit en mois.

Spécifier la longueur d’un cycle harmonique

Durée de la variation périodique à modéliser. Ce paramètre est requis lorsque le type de la ligne de tendance est défini sur Harmonic (Harmonique). Par exemple, la verdoyance des feuilles présente souvent un cycle de variation fort sur une seule année, de sorte que la longueur du cycle est de 1 an. Les données horaires de température présentent un cycle de variation fort tout au long d’une seule journée, de sorte que la longueur du cycle est de 1 jour.

La longueur par défaut est égale à 1 an pour les données qui varient sur un cycle annuel.

Ce paramètre est inclus uniquement dans l’analyse des tendances lorsque la dimension analysée est la dimension temporelle et que le type de la ligne de tendance est défini sur Harmonic (Harmonique).

Choisir l’unité temporelle de la longueur du cycle harmonique

Indique l’unité temporelle à utiliser pour la longueur d’un cycle harmonique.

  • Days (Jours) : l’unité de la longueur du cycle harmonique sont les jours.
  • Years (Années) : l’unité de la longueur du cycle harmonique sont les années. Il s’agit de l’option par défaut.

Ce paramètre est inclus uniquement dans l’analyse des tendances lorsque la dimension analysée est la dimension temporelle et que le type de la ligne de tendance est défini sur Harmonic (Harmonique).

Spécifier le numéro de fréquence pour l’ajustement de tendance harmonique

La fréquence à utiliser dans l’ajustement de tendance harmonique. Ce paramètre spécifie la fréquence des cycles au cours d’une année.

La valeur par défaut est de 1, soit un cycle harmonique par an.

Ce paramètre est inclus uniquement dans l’analyse des tendances lorsque la dimension analysée est la dimension temporelle.

Spécifier le numéro d’ordre polynomial pour l’ajustement de tendance

Numéro d’ordre polynomial à utiliser dans l’ajustement de tendance polynomial.

Ce paramètre est inclus uniquement dans l’analyse des tendances lorsque la dimension analysée est la dimension temporelle.

La valeur par défaut est 2 ou un polynôme du second degré.

Choisir les statistiques de modèle à inclure dans le raster de tendance

Cet outil fournit trois types de statistiques de la ligne d’ajustement de tendance : la racine carrée de l’erreur quadratique moyenne (EQM), la statistique de qualité de l’ajustement R-carré (R-carré) et la statistique de valeur p pour le coefficient de pente (Valeur p du coefficient de pente). Chaque statistique a une case à cocher correspondante pour indiquer si elle sera calculée et incluse comme l’un des nombreux canaux dans le raster de tendance en sortie.

  • EQM

    • Activé : l’EQM sera calculé et inclus comme l’un des nombreux canaux dans le raster de tendance en sortie. Il s’agit de l’option par défaut.
    • Désactivé : l’EQM ne sera pas calculée.

  • R-carré

    • Activé : la valeur R-carré sera calculée et incluse comme l’un des nombreux canaux dans le raster de tendance en sortie.
    • Désactivé : la valeur R-carré ne sera pas calculée. Il s’agit de l’option par défaut.

  • Valeur p du coefficient de pente

    • Activé : la valeur p sera calculée et incluse comme l’un des nombreux canaux dans le raster de tendance en sortie.
    • Désactivé : la valeur P ne sera pas calculée. Il s’agit de l’option par défaut.

Ignorer les valeurs manquantes dans le calcul

Spécifie si les valeurs manquantes sont ignorées lors de l’analyse.

  • Activé : l’analyse inclut tous les pixels valides selon une dimension donnée et ignore les pixels NoData. Il s’agit de l’option par défaut.
  • Désactivé : le résultat de l’analyse est NoData s’il existe des valeurs NoData pour les pixels selon la dimension donnée.

Nom de la couche de résultat

Le nom de la couche qui sera créée dans la page My Content (Mon contenu) et ajoutée à la carte. Le nom par défaut repose sur le nom de l'outil et sur le nom de la couche en entrée. Si la couche existe déjà, vous êtes invité à fournir un autre nom.

Vous pouvez spécifier le nom d’un dossier dans My Content (Mon contenu), dans lequel le résultat sera enregistré via la zone de liste déroulante Save result in (Enregistrer le résultat dans). Si vous avez les privilèges permettant de créer des couches tuilées et d’images dynamiques, vous pouvez spécifier le type de couche à générer en sortie à l’aide de la zone déroulante Save result as (Enregistrer le résultat sous).
