汇总范围内的栅格

该工具用于汇总另一个数据集区域内的栅格数据值。

其中的一些应用示例如下：

  • 按土地使用类型给定分水岭边界图层和土地使用边界图层，计算每个分水岭的土地使用类型的总面积。
  • 已知某县内宗地的图层和城市边界图层，汇总各城市边界内闲置宗地的平均值。

如果选中 使用当前地图范围，则仅分析当前地图范围内可见的输入图层和待汇总图层中的要素。 如果未选中，则会对输入图层和待汇总图层内的所有要素进行分析，包括超出当前地图范围的要素。

选择要在已定义边界内汇总栅格图层的区域图层

定义要汇总的区域或区边界的图层。

图层可以是栅格或要素数据。 对于栅格，区域将由输入中具有相同像元值的所有位置定义。 各区域无需相连。

选择要定义边界的字段

这是将用于定义区域边界的图层的属性。

例如，假设第一个输入图层使用定义各单元的地区、区域和宗地 ID 的属性定义管理单元边界。 您还具有用于定义每个位置的生物多样性指数的栅格图层。 使用所选字段可计算本地、区域或地区级别的平均生物多样性。

选择要汇总的栅格图层

此图层中的栅格像元将按其所在的区域（区）进行汇总。

选择要计算的统计数据

选择要汇总的图层后，您可以选择计算输入区域图层中点、线或区域的数值属性的统计数据。

如果选定字段为整型，则可用的统计类型包括平均值、众数、最大值、中位数、最小值、少数、百分比数、范围、标准差、总和以及变异度。

如果字段为浮点型，则选项包括平均值、最大值、最小值、范围、标准差以及总和。

百分数值

要计算的百分比数。 默认值为 90，表示 90%。

取值范围为 0 到 100。 0% 基本上等同于最小值统计数据，而 100% 则等同于最大值。 值 50 所生成的结果基本等同于中值统计数据的结果。

仅当 选择要计算的统计数据参数设置为 百分比数时，才支持此选项。

忽略计算中的缺失值

如果选择忽略缺失值，则仅使用在要汇总的图层中拥有值的像元来确定该区域的输出值。 否则，如果在区域内的任意位置存在缺失值，则会视作对该区域中所有像元执行统计计算的信息不足，该区域将在输出中收到空 (NoData) 值。

以多维方式处理

指定输入栅格的处理方式（如果它们是多维的）。 如果选择以多维方式处理，则将计算输入多维数据集的所有维度的统计数据。 否则，将计算输入多维数据集的当前剖切片中的统计数据，这是默认设置。

结果图层名称

将在 我的内容中创建并添加到地图中的图层的名称。 默认名称基于工具名称以及输入图层名称。 如果该名称的图层已存在，则系统将提示您提供其他名称。

您可以使用 将结果保存在下拉框指定 我的内容中的文件夹名称，结果将保存到该文件夹中。
