Temukan Statistik Argumen

Mengekstrak nilai dimensi atau indeks band di mana statistik yang diberikan dicapai untuk setiap piksel dalam raster multidimensional atau multipita.

Gunakan alat ini untuk mengekstraksi nilai dimensi (misalnya tanggal, tinggi, atau kedalaman) dengan statistik tertentu dicapai dalam tumpukan raster dalam set data raster multidimensional. Atau temukan nomor band dengan statistik dicapai dalam raster multiband.

Misalnya, Anda mungkin ingin tahu bulan di mana setiap piksel mencapai suhu permukaan laut tertinggi selama 30 tahun pengumpulan data. Contoh lain adalah Anda memiliki gambar penginderaan jarak jauh delapan-band dan Anda ingin tahu band dengan nilai reflektansi minimum tercapai untuk setiap piksel.

Jika Gunakan jangkauan peta saat ini dicentang, variabel atau band layer yang terlihat dalam jangkauan peta saat ini akan dianalisis. Jika tidak dicentang, variabel atau band layer penuh akan dianalisis, bahkan jika mereka berada di luar jangkauan peta saat ini.

Pilih layer citra multidimensional atau multi-band yang akan dianalisis

Raster multidimensional atau multiband input yang akan dianalisis.

Pilih dimensi dari mana statistik akan diekstrak

Dimensi asal statistik yang akan diekstrak. Jika raster input bukan raster multidimensional, parameter ini tidak diperlukan.

Pilih variabel yang akan dianalisis

Variabel yang akan dianalisis. Jika raster input bukan multidimensional, nilai-nilai piksel raster multiband akan dianggap sebagai variabel. Jika raster input multidimensional dan tidak ada variabel yang ditentukan, semua variabel dengan dimensi yang dipilih akan dianalisis.

Misalnya, untuk menemukan tahun dengan nilai suhu tertinggi, tentukan suhu sebagai variabel yang akan dianalisis. Jika Anda tidak menentukan variabel apa pun dan Anda memiliki variabel suhu dan curah hujan, kedua variabel tersebut akan dianalisis, dan raster multidimensional output akan menyertakan kedua variabel tersebut.

Pilih jenis statistik

Menentukan statistik untuk mengekstrak dari variabel atau variabel di sepanjang dimensi yang ditentukan.

  • Argumen minimum—Nilai dimensi saat nilai variabel minimum tercapai akan diekstrak. Ini adalah default.
  • Argumen maksimum—Nilai dimensi saat nilai variabel maksimum tercapai akan diekstrak.
  • Argumen median—Nilai dimensi saat nilai variabel median tercapai akan diekstrak.
  • Argumen durasi—Durasi dimensi terpanjang yang nilai variabelnya berada di antara nilai minimum dan maksimum.

Tentukan beberapa nilai kemunculan

Nilai piksel yang digunakan untuk menunjukkan bahwa statistik argumen yang diberikan tercapai lebih dari sekali dalam set data raster input. Jika tidak ditentukan, nilai piksel akan menjadi nilai dimensi saat statistik argumen pertama kali tercapai.

Tentukan nilai minimum

Nilai variabel minimum yang akan digunakan untuk mengekstrak durasi.

Parameter ini diperlukan ketika Durasi dipilih sebagai jenis statistik.

Tentukan nilai maksimum

Nilai variabel maksimum yang akan digunakan untuk mengekstrak durasi.

Langkah ini diperlukan ketika Durasi dipilih sebagai jenis statistik.

Pilih penentuan dimensi

Menentukan cara statistik akan diekstrak dari dimensi.

  • Semua—Statistik akan diekstrak di semua dimensi. Ini adalah default.
  • Kata Kunci Interval—Statistik akan diekstrak dari dimensi waktu sesuai dengan kata kunci interval.

Pilih interval kata kunci

Unit waktu saat statistik akan diekstrak.

Misalnya, Anda memiliki data suhu permukaan laut harian sepanjang lima tahun dan Anda ingin tahu kapan tahun saat suhu maksimum bisa diamati. Atur Argumen maksimum sebagai jenis statistik, atur Kata Kunci Interval sebagai penentuan dimensi, dan atur Tahunan sebagai interval kata kunci dalam langkah ini.

Atau, jika Anda ingin mengetahui bulan saat suhu maksimum diamati secara konsisten, atur Argumen maksimum sebagai jenis statistik, atur Kata Kunci Interval sebagai penentuan dimensi, dan atur Berulang Setiap Bulan sebagai interval kata kunci dalam langkah ini. Ini akan menghasilkan raster dengan setiap piksel berisi bulan tempat statistik dicapai di seluruh catatan lima tahun (08/18/2018, 08/25/2016, 08/07/2013, misalnya).

Langkah ini diperlukan ketika WaktuStandar diatur sebagai dimensi dari mana statistik akan diekstrak, dan Kata Kunci Interval dipilih untuk menentukan cara statistik akan diekstrak dari dimensi waktu.

  • Berulang setiap hari—Statistik akan diekstrak setiap hari.
  • Berulang setiap minggu—Statistik akan diekstrak setiap minggu.
  • Berulang setiap bulan—Statistik akan diekstrak setiap bulan.
  • Berulang setiap triwulan—Statistik akan diekstrak setiap triwulan.
  • Setiap jam—Statistik akan diekstrak pada jam saat statistik tersebut tercapai.
  • Harian—Statistik akan diekstrak pada hari saat statistik itu tercapai.
  • Mingguan—Statistik akan diekstrak pada minggu saat statistik tercapai.
  • Bulanan—Statistik akan diekstrak pada bulan saat statistik tersebut tercapai.
  • Setiap triwulan—Statistik akan diekstrak pada triwulan saat statistik tercapai.
  • Tahunan—Statistik akan diekstrak pada tahun saat statistik tersebut tercapai.

Abaikan nilai yang hilang dalam penghitungan

Menentukan apakah nilai yang hilang diabaikan dalam analisis.

  • Jika kotak dicentang, analisis akan mencakup semua piksel yang valid di sepanjang dimensi tertentu dan mengabaikan piksel NoData. Ini adalah default.
  • Jika kotak tidak dicentang, analisis akan menghasilkan NoData jika ada nilai NoData untuk piksel di sepanjang dimensi yang ditentukan.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
