Batas Air

Alat ini menentukan area kontribusi di atas rangkaian sel dalam raster.

Batas air adalah daerah dataran tinggi yang menyumbang aliran air — umumnya air — ke saluran umum sebagai drainase terkonsentrasi. Ini dapat menjadi bagian dari batas air yang lebih besar dan juga dapat berisi batas air yang lebih kecil, yang disebut subbasin.

Nilai setiap batas air akan diambil dari nilai sumber di raster input atau layer titik tuang fitur. Ketika titik tuang adalah kumpulan raster, nilai sel digunakan secara default. Ketika titik tuang adalah set data fitur titik, nilainya akan berasal dari kolom yang ditentukan.

Alat Watershed hanya mendukung layer raster arah aliran input D8.

Saat menentukan lokasi input titik tuang sebagai data fitur, kolom default akan menjadi kolom valid pertama yang tersedia. Jika tidak ada kolom yang valid, kolom ObjectID (misalnya, OID atau FID) akan menjadi default.

Pilih layer arah aliran

Input raster yang menunjukkan arah aliran keluar dari setiap sel.

Raster arah aliran dapat dibuat menggunakan alat Flow Direction atau tugas FlowDirection dalam layanan RasterAnalysisTools, jalankan menggunakan tipe arah aliran default D8.

Tentukan lokasi input titik tuang

Input lokasi titik tuang.

Untuk raster, ini menunjukkan sel-sel di atas mana area kontribusi, atau tangkapan, akan ditentukan. Semua sel yang bukan NoData akan digunakan sebagai sel sumber. Untuk input titik, ini mewakili lokasi di atas mana area kontribusi, atau tangkapan, akan ditentukan.

Pilih kolom titik tuang

Kolom yang digunakan untuk menetapkan nilai ke lokasi titik tuang.

Jika layer input pour point adalah raster, Value digunakan. Jika fitur, kolom numerik digunakan. Jika kolom berisi nilai-nilai floating-point, mereka akan dipotong menjadi bilangan bulat.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di.
