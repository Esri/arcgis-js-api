Daerah Aliran Sungai

Alat ini menentukan area kontribusi di atas rangkaian sel dalam raster.

Titik input di Sungai Mississippi di Louisiana akan membuat fitur batas air yang mencakup area dari New York ke Montana dan ke Kanada.

Pilih layer arah aliran

Input raster yang menunjukkan arah aliran keluar dari setiap sel.

Raster arah aliran dapat dibuat menggunakan fungsi raster Arah Aliran, dengan jenis arah aliran default D8.

Tentukan lokasi input titik tuang

Input lokasi titik tuang.

Ini dapat berupa raster atau fitur titik. Untuk raster, ini menunjukkan sel-sel di atas mana area kontribusi, atau tangkapan, akan ditentukan. Semua sel yang bukan NoData akan digunakan sebagai sel sumber. Untuk input titik, ini mewakili lokasi di atas mana area kontribusi, atau tangkapan, akan ditentukan.

Pilih kolom titik tuang

Kolom yang digunakan untuk menetapkan nilai ke lokasi titik tuang.

Jika layer titik tuang input adalah raster, Value akan digunakan. Jika layer input titik tuang adalah fitur, kolom numerik akan digunakan. Jika kolom berisi nilai-nilai floating-point, mereka akan dipotong menjadi bilangan bulat.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
