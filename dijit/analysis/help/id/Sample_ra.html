Sampel

Alat ini membuat feature class titik atau tabel yang menunjukkan nilai sel dari raster, atau rangkaian raster, untuk lokasi yang ditentukan. Lokasi ditentukan oleh sel raster, fitur poligon, fitur polyline, atau oleh serangkaian titik.

Beberapa contoh aplikasi alat ini meliputi hal berikut:

  • Ekstrak atribut dari lokasi menara seluler dari beberapa raster, seperti penggunaan lahan, jenis tanah, elevasi, dan jarak ke jalan.
  • Ekstrak suhu tanah di kedalaman yang berbeda untuk set titik tertentu.
  • Dari data suhu permukaan laut bulanan selama 30 tahun, rangkum di lokasi hewan atau dalam wilayah jelajah untuk waktu tertentu saat spesies hewan diamati.

Jika Gunakan jangkauan peta saat ini dicentang, variabel atau band layer yang terlihat dalam jangkauan peta saat ini akan dianalisis. Jika tidak dicentang, variabel atau band layer penuh akan dianalisis, bahkan jika mereka berada di luar jangkauan peta saat ini.

Pilih layer citra untuk sampel

Satu gambar atau daftar gambar yang nilainya akan diambil sampel berdasarkan data lokasi input.

Untuk mengambil sampel semua irisan dari layer citra multidimensional, irisan tersebut harus menjadi satu-satunya raster input.

Pilih layer lokasi

Posisi pengidentifikasi data yang ingin Anda ambil sampelnya.

Data lokasi input dapat berupa raster, atau titik, polyline, atau feature class poligon.

Tentukan kolom ID Unik

Kolom yang berisi nilai berbeda untuk setiap lokasi atau fitur dalam raster lokasi input atau feature class.

Tentukan informasi akuisisi data lokasi (Opsional)

Tentukan waktu, kedalaman, atau data akuisisi lainnya yang terkait dengan sel atau fitur lokasi.

Untuk setiap dimensi, ada tiga opsi untuk menentukan informasi akuisisi yang terkait dengan sel atau fitur lokasi:

  • Kolom atau nilai tunggal—Jika opsi ini dipilih, informasi berikut ini harus ditentukan:
    • Kolom atau nilai dimensi—Pilih kolom nilai dimensi dari data lokasi, atau masukkan nilai dimensi tertentu.
  • Nilai atau kolom tunggal dengan nilai relatif—Jika opsi ini dipilih, informasi berikut ini harus ditentukan bersama untuk menentukan rentang di mana statistik akan dihitung untuk variabel:
    • Kolom atau nilai dimensi—Pilih kolom nilai dimensi dari data lokasi.
    • Nilai relatif sebelumnya—Tentukan angka non-negatif yang digunakan untuk dikurangi dari Kolom atau nilai dimensi untuk menentukan awal rentang nilai dimensi. Nilai ini mewakili jumlah hari jika dimensinya adalah waktu.
    • Nilai relatif setelahnya—Tentukan angka non-negatif yang digunakan untuk ditambahkan ke Kolom atau nilai dimensi untuk menentukan akhir rentang nilai dimensi. Nilai ini mewakili jumlah hari jika dimensinya adalah waktu.
  • Kolom atau nilai awal dan akhir—Jika opsi ini dipilih, informasi di bawah ini perlu ditentukan bersama untuk menentukan rentang di mana statistik akan dihitung untuk variabel:
    • Kolom atau nilai awal—Pilih kolom nilai dimensi dari data lokasi, atau masukkan nilai dimensi tertentu, untuk menentukan awal rentang nilai dimensi.
    • Nilai kolom akhir—Pilih kolom nilai dimensi dari data lokasi, atau masukkan nilai dimensi tertentu, untuk menentukan akhir rentang nilai dimensi.

Parameter ini hanya tersedia jika parameter Pilih layer citra untuk diambil sampel merupakan raster multidimensional.

Tentukan kolom atau nilai jarak buffer

Jarak yang ditentukan di sekitar fitur data lokasi. Jarak buffer ditentukan dalam unit linear dari referensi spasial fitur lokasi. Jika fitur menggunakan referensi geografis, unit akan berupa derajat.

Statistik akan dihitung dalam area buffer ini.

Parameter ini hanya tersedia saat Pilih layer citra untuk diambil sampel merupakan raster multidimensional.

Pilih jenis statistik

Jenis statistik yang akan dihitung.

  • Minimum—Menemukan nilai minimum dalam rentang yang ditentukan.
  • Maksimum—Menemukan nilai maksimum dalam rentang yang ditentukan.
  • Median—Menemukan nilai median dalam rentang yang ditentukan.
  • Rata-rata—Menghitung rata-rata untuk rentang yang ditentukan.
  • Jumlah—Menghitung jumlah variabel dalam rentang yang ditentukan.
  • Mayoritas—Menemukan nilai yang paling sering muncul.
  • Minoritas—Menemukan nilai yang paling jarang muncul.
  • Simpangan Baku—Menghitung simpangan baku.
  • Persentil—Menghitung persentil yang ditentukan dalam rentang yang ditentukan.

Parameter ini hanya tersedia saat parameter Pilih layer citra untuk diambil sampel merupakan raster multidimensional, dan parameter Pilih layer lokasi merupakan polyline atau poligon, atau Tentukan kolom atau nilai jarak buffer (Opsional) ditentukan.

Nilai Persentil

Nilai ini dapat memiliki rentang dari 0 hingga 100. Defaultnya adalah 90.

Parameter ini hanya tersedia jika parameter Pilih Jenis statistik diatur ke Persentil.

Tentukan teknik pengambilan ulang sampel

Algoritme pengambilan ulang sampel digunakan saat mengambil sampel raster.

  • Terdekat—Lingkungan terdekat, yang menggunakan sel terdekat untuk menetapkan nilai ke sel output saat pengambilan sampel ulang. Ini adalah default.
  • Bilinear—Interpolasi bilinear, yang menentukan nilai baru sel berdasarkan rata-rata jarak tertimbang dari empat pusat sel input terdekat.
  • Kubik—Konvolusi kubik, yang menentukan nilai baru sel berdasarkan pemasangan kurva halus melalui 16 pusat sel input terdekat.

Tentukan tata letak output

Menentukan apakah nilai sampel muncul dalam baris atau kolom di tabel output.

  • Nilai sampel muncul dalam baris—Nilai sampel muncul dalam baris terpisah di tabel output. Ini adalah default.
  • Nilai sampel muncul dalam kolom—Nilai sampel muncul dalam kolom terpisah di tabel output.

Parameter ini hanya tersedia jika parameter Pilih layer citra untuk diambil sampel merupakan raster multidimensional dan berisi satu variabel dan satu dimensi, dan setiap irisan merupakan raster band tunggal.

Tentukan jenis output

Menentukan apakah akan membuat feature class titik dengan nilai sampel dalam tabelnya, atau hanya tabel dengan nilai sampel.

  • Tabel—Buat tabel dengan nilai sampel. Ini adalah default.
  • Fitur—Buat feature class titik dengan nilai sampel dalam tabelnya. Jika Pilih layer lokasi merupakan polyline atau poligon, setiap poin di output adalah centroid setiap geometri dari fitur lokasi input.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di.
