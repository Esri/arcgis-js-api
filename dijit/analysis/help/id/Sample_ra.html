Sampel

Alat ini membuat feature class titik atau tabel yang menunjukkan nilai sel dari raster, atau rangkaian raster, untuk lokasi yang ditentukan. Lokasi ditentukan oleh sel raster, fitur poligon, fitur polyline, atau oleh serangkaian titik.

Nilai piksel akan diekstrak dari semua raster input di setiap lokasi. Feature class titik atau tabel akan dibuat dengan kolom yang berisi nilai piksel untuk setiap raster input. Atribut tambahan dari tabel raster input tidak akan dimasukkan dalam tabel output.

Kombinasi raster apa pun, seperti band tunggal, multiband, multidimensional, atau multivarian, dapat ditentukan sebagai raster input. Struktur tabel output berubah saat raster input bersifat multidimensional. Ketika raster multiband ditentukan sebagai salah satu raster input, semua band dalam input itu akan digunakan. Untuk memproses pilihan band dari raster multiband input, pertama buat set data raster baru yang terdiri dari band-band tertentu. Gunakan hasilnya dalam daftar raster input.

Lokasi input dapat berupa fitur raster atau titik:

  • Raster—Piksel yang berisi nilai yang valid (bukan NoData) digunakan untuk mengekstraksi nilai piksel dari semua raster input, dan pusat piksel digunakan untuk lokasi titik.
  • Poin—Nilai akan diambil sampelnya di setiap lokasi titik.
  • Polyline atau poligon—Nilai akan diambil sampelnya pada poligon atau centroid polyline. Jika raster input multidimensional, nilainya akan diagregasi berdasarkan jenis Statistik.

Raster input diambil sampelnya dalam referensi dan resolusi spasial asli mereka. Jika ada beberapa raster input dengan referensi spasial yang berbeda, lokasi input diproyeksikan ke masing-masing referensi spasial raster individu dari mana nilai diekstrak. Lingkungan analisis diterapkan ke lokasi input, tetapi raster input tidak akan menaati lingkungan analisis.

Lokasi yang mengekstrak nilai dari sel NoData di raster input akan diberi nilai <null> di tabel output.

Untuk melakukan analisis dengan fitur multititik, pertama-tama ubahlah ke fitur titik tunggal sebelum menggunakannya dalam alat ini.

Referensi spasial dari koordinat x, y dalam tabel sampel output sama dengan raster lokasi input atau fitur titik, kecuali jika ditentukan dalam lingkungan Sistem Koordinat Output. Jika referensi spasial dari data lokasi input tidak diketahui, referensi spasial dari koordinat x, y juga tidak diketahui.

Referensi spasial dari feature class output sama dengan raster lokasi input atau fitur titik, kecuali jika ditentukan dalam lingkungan Sistem Koordinat Output.

Lingkungan Faktor Pemrosesan Paralel hanya didukung bila inputnya adalah raster multidimensional.

Jika Gunakan jangkauan peta saat ini dicentang, variabel atau band layer yang terlihat dalam jangkauan peta saat ini akan dianalisis. Jika tidak dicentang, variabel atau band layer penuh akan dianalisis, bahkan jika mereka berada di luar jangkauan peta saat ini.

Pilih layer citra untuk sampel

Layer citra tunggal atau daftar layer citra yang nilainya akan diambil sampel berdasarkan data lokasi input.

Untuk mengambil sampel semua irisan dari layer citra multidimensional, irisan tersebut harus menjadi satu-satunya raster input.

Pilih layer lokasi

Posisi pengidentifikasi data yang ingin Anda ambil sampelnya.

Data lokasi input dapat berupa raster, atau titik, polyline, atau feature class poligon.

Tentukan kolom ID Unik

Kolom yang berisi nilai berbeda untuk setiap lokasi atau fitur dalam raster lokasi input atau feature class.

Tentukan informasi akuisisi data lokasi (Opsional)

Tentukan waktu, kedalaman, atau data akuisisi lainnya yang terkait dengan sel atau fitur lokasi.

Untuk setiap dimensi, ada tiga opsi untuk menentukan informasi akuisisi yang terkait dengan sel atau fitur lokasi:

  • Kolom atau nilai tunggal—Bila opsi ini dipilih, informasi berikut perlu ditentukan:
    • Nilai atau kolom dimensi—Pilih kolom nilai dimensi dari data lokasi, atau masukkan nilai dimensi tertentu.
  • Nilai atau kolom tunggal dengan nilai relatif—Jika opsi ini dipilih, informasi di bawah ini perlu ditentukan bersama untuk menentukan rentang di mana statistik akan dihitung untuk variabel:
    • Nilai atau kolom dimensi—Pilih kolom nilai dimensi dari data lokasi.
    • Nilai relatif sebelumnya—tentukan angka non-negatif yang digunakan untuk dikurangi dari Nilai atau kolom dimensi untuk menentukan awal rentang nilai dimensi. Nilai ini mewakili jumlah hari jika dimensinya adalah waktu.
    • Nilai relatif setelahnya—tentukan angka non-negatif yang digunakan untuk ditambahkan ke Nilai atau kolom tunggal untuk menentukan akhir rentang nilai dimensi. Nilai ini mewakili jumlah hari jika dimensinya adalah waktu.
  • Nilai atau kolom awal dan akhir—Jika opsi ini dipilih, informasi di bawah ini perlu ditentukan bersama untuk menentukan rentang di mana statistik akan dihitung untuk variabel:
    • Nilai atau kolom awal—pilih kolom nilai dimensi dari data lokasi, atau masukkan nilai dimensi tertentu, untuk menentukan awal rentang nilai dimensi.
    • Nilai atau kolom akhir—pilih kolom nilai dimensi dari data lokasi, atau masukkan nilai dimensi tertentu, untuk menentukan akhir rentang nilai dimensi.

Tentukan kolom atau nilai jarak buffer

Jarak yang ditentukan di sekitar fitur data lokasi. Jarak buffer ditentukan dalam unit linear dari referensi spasial fitur lokasi. Jika fitur menggunakan referensi geografis, unit akan berupa derajat.

Statistik akan dihitung dalam area buffer ini.

Pilih jenis statistik

Jenis statistik yang akan dihitung.

  • Minimum—Menemukan nilai minimum dalam rentang yang ditentukan.
  • Maksimum—Menemukan nilai maksimum dalam rentang yang ditentukan.
  • Median—Menemukan nilai median dalam rentang yang ditentukan.
  • Rata-rata—Menghitung rata-rata untuk rentang yang ditentukan.
  • Jumlah—Menghitung jumlah variabel dalam rentang yang ditentukan.
  • Mayoritas—Menemukan nilai yang paling sering muncul.
  • Minoritas—Menemukan nilai yang paling jarang muncul.
  • Simpangan Baku—Menghitung simpangan baku.
  • Persentil—Menghitung persentil yang ditentukan dalam rentang yang ditentukan.

Nilai Persentil

Persentil yang akan dihitung saat parameter Jenis Statistik diatur ke Persentil.

Nilai ini dapat memiliki rentang dari 0 hingga 100. Defaultnya adalah 90.

Tentukan teknik pengambilan ulang sampel

Algoritme pengambilan ulang sampel digunakan saat mengambil sampel raster.

  • Terdekat—Lingkungan terdekat. Ini adalah default.
  • Bilinear—Interpolasi bilinear
  • Kubik—Konvolusi kubik

Tentukan tata letak output

Menentukan apakah nilai sampel muncul dalam baris atau kolom di tabel output. Opsi ini hanya tersedia saat raster multidimensional input berisi satu variabel dan satu dimensi, dan setiap irisan adalah raster band tunggal.

  • Nilai sampel muncul dalam baris—Nilai sampel muncul dalam baris terpisah di tabel output. Ini adalah default.
  • Nilai sampel muncul dalam kolom—Nilai sampel muncul dalam kolom terpisah di tabel output. Opsi ini hanya valid bila raster multidimensional input berisi satu variabel dan satu dimensi, dan setiap irisan adalah raster band tunggal.

Tentukan jenis output

Menentukan apakah akan membuat feature class titik dengan nilai sampel dalam tabelnya, atau hanya tabel dengan nilai sampel.

  • Tabel—Buat tabel dengan nilai sampel. Ini adalah default.
  • Fitur—Buat feature class titik dengan nilai sampel dalam tabelnya. Jika fitur lokasi input adalah polyline atau poligon, setiap titik dalam output adalah centroid setiap geometri dari fitur lokasi input.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak tarik-turun Simpan hasil di.
