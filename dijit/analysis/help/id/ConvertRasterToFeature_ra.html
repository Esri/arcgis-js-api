Konversi Raster ke Fitur

Alat ini mengubah raster ke set data fitur, seperti titik, garis, atau poligon.

Parameter Pilih kolom memungkinkan Anda untuk memilih kolom atribut manakah dari rangkaian data raster input yang akan menjadi sebuah atribut di kelas fitur output. Parameter ini hanya tersedia untuk raster bilangan bulat yang memiliki atribut tambahan di tabel terkait.

Raster input dapat memiliki ukuran sel berapa pun.

Raster bilangan bulat dapat dikonversi ke fitur titik, garis, atau poligon; tetapi, raster bilangan titik mengambang hanya dapat dikonversi ke fitur titik.

Jika Gunakan jangkauan peta saat ini dicentang, hanya sel raster yang terlihat di dalam jangkauan peta saat ini yang akan dikonversikan. Jika tidak dicentang, semua sel pada layer raster akan dikonversikan, meskipun berada di luar jangkauan peta saat ini. Perhatikan hal ini jika layer raster memiliki ukuran yang besar atau memiliki resolusi yang sangat tinggi.

Contoh penggunaan alat ini mencakup hal berikut:

  • Membuat fitur poligon dari raster klasifikasi penggunaan tanah.
  • Aliran yang teridentifikasi di DEM dapat dikonversikan ke fitur garis.
  • DEM dapat dikonversikan ke fitur titik yang dapat digunakan di alur kerja berbasis nonraster.

Pilih layer raster untuk dikonversi

Layer raster input yang akan dikonversikan ke feature layer.

Pilih kolom

Kolom yang menentukan nilai konversi.

Bisa saja berupa bilangan bulat atau nilai teks.

Kolom yang berisi nilai bilangan titik mengambang hanya dapat digunakan jika output adalah ke set data titik.

Default adalah kolom Value, yang berisi nilai di setiap sel raster.

Pilih tipe output

Menentukan jenis output.

  • Titik—Raster akan dikonversi ke set data titik. Ini adalah default.
  • Garis—Raster akan dikonversi ke set data fitur garis.
  • Poligon—Raster akan dikonversi ke set data fitur poligon.

Sederhanakan garis atau poligon

Menentukan apakah garis atau poligon akan disederhanakan (diperhalus). Perataan dilakukan sedemikian rupa sehingga garis berisi jumlah segmen minimum dan tetap sedekat mungkin dengan tepi sel raster asli.

  • Dicentang—Fitur garis atau poligon akan diperhalus untuk menghasilkan hasil yang lebih umum. Ini adalah default.
  • Tidak dicentang—Fitur garis atau poligon akan mengikuti batas sel set data raster.

Parameter ini hanya didukung jika jenis output adalah garis atau poligon.

Buat fitur multi-bagian

Menentukan apakah poligon output akan terdiri dari fitur satu-bagian atau multi-bagian.

  • Dicentang—Fitur multi-bagian akan dibuat berdasarkan poligon yang memiliki nilai yang sama.
  • Tidak Dicentang—Fitur individu (satu-bagian) akan dibuat untuk setiap poligon. Ini adalah default.

Parameter ini hanya didukung jika jenis outputnya adalah poligon.

Puncak maksimum per fitur poligon

Batas verteks digunakan untuk membagi poligon menjadi poligon yang lebih kecil.

Jika dibiarkan kosong, poligon output tidak akan terpecah. Ini adalah default.

Parameter ini hanya didukung jika jenis outputnya adalah poligon.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di.
