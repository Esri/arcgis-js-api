Ringkas Raster di Dalam

Alat ini merangkum nilai raster di dalam zona set data lain.

Beberapa aplikasi contoh termasuk hal berikut:

  • Dengan layer batas Daerah Aliran Sungai dan batas penggunaan lahan berdasarkan jenis penggunaannya, dapat dihitung luas total penggunaan lahan untuk setiap Daerah Aliran Sungai.
  • Dengan mengetahui layer bagian persil di suatu daerah dan layer batas kota, ringkaslah nilai rata-rata dari bagian persil yang kosong di dalam setiap batas kota.

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur-fitur pada layer input dan layer yang akan diringkas yang terlihat di jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, semua fitur pada layer input dan layer yang akan diringkas akan dianalisis meskipun berada di luar jangkauan peta saat ini.

Pilih area layer untuk meringkas layer raster di dalam batas yang ditentukan

Layer yang menentukan perbatasan area atau zona yang akan diringkas.

Layer dapat berupa raster atau data fitur. Untuk raster, zona ditentukan oleh semua lokasi di input yang memiliki nilai sel yang sama. Area tersebut tidak perlu berdampingan.

Pilih kolom untuk menentukan batas

Ini adalah artibut layer yang akan digunakan untuk menentukan perbatasan area.

Contoh, misalnya layer input pertama menentukan batas unit manajemen, dengan atribut yang menentukan wilayah, distrik, dan ID persil setiap unit. Anda pun memiliki layer raster yang menentukan indeks hayati untuk setiap lokasi. Dengan kolom yang Anda pilih, Anda dapat memutuskan untuk menghitung rata-rata hayati di tingkat lokal, distrik, atau wilayah.

Pilih layer raster untuk diringkas

Sel raster di layer ini akan diringkas berdasarkan area (zona) tempat posisi sel tersebut.

Pilih statistik yang akan dihitung

Setelah Anda memilih layer untuk diringkas, Anda dapat memilih untuk menghitung statistik dari setiap atribut numerik dari titik, garis, atau area di dalam layer area input.

Tipe statistik yang tersedia ketika kolom yang dipilih adalah bilangan bulat adalah Rata-rata, Mayoritas, Maksimal, Median, Minoritas, Persentil, Rentang, Standar deviasi, Jumlah, dan Keragaman.

Jika kolom adalah bilangan titik mengambang, opsinya adalah Rata-rata, Maksimal, Minimal, Rentang, Standar deviasi, dan Jumlah.

Nilai Persentil

Persentil yang dihitung. Defaultnya adalah 90, yang menunjukkan persentil ke-90.

Nilai dapat berkisar antara 0 hingga 100. Persentil ke-0 pada dasarnya setara dengan statistik minimal, dan persentil ke-100 setara dengan statistik maksimal. Nilai 50 pada dasarnya akan menghasilkan hasil yang sama dengan statistik median.

Opsi ini hanya didukung jika parameter Pilih statistik yang akan dihitung diatur ke Persentil.

Abaikan nilai yang hilang dalam penghitungan

Jika Anda memilih untuk mengabaikan nilai yang hilang, hanya sel yang memiliki nilai di dalam layer yang akan diringkas yang akan digunakan untuk menentukan nilai output untuk area tersebut. Jika tidak, jika ada nilai yang hilang di mana pun di dalam area, maka akan diasumsikan bahwa informasi tidak memadai untuk melakukan perhitungan statistik untuk semua sel di zona itu, dan area itu akan menerima nilai null (NoData) di output.

Proses sebagai multidimensional

Menentukan cara pemrosesan raster input jika raster input tersebut bersifat multidimensional. Jika Anda memilih untuk memproses sebagai multidimensional, statistik akan dihitung dari semua dimensi set data multidimensional input. Jika tidak, maka statistik akan dihitung dari irisan set data multidimensional input saat ini, ini defaultnya.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di.
