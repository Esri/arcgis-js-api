Hitung Densitas

Alat ini membuat peta kepadatan dari fitur titik atau garis dengan menyebarkan jumlah beberapa fenomena yang diketahui (disajikan sebagai atribut titik atau garis) pada peta. Hasilnya adalah layer area yang diklasifikasikan dari yang paling tidak padat hingga paling padat.

Beberapa contoh aplikasi alat ini meliputi hal berikut:

  • Membuat peta kepadatan kejahatan untuk membantu kepolisian mengalokasikan sumber daya secara tepat ke area dengan tingkat kejahatan tinggi.
  • Menghitung kepadatan rumah sakit dalam sebuah daerah. Layer hasil akan menampilkan area dengan aksesibilitas tinggi dan rendah terhadap rumah sakit, dan informasi ini dapat digunakan untuk memutuskan tempat rumah sakit baru seharusnya dibangun.
  • Mengenali area yang berisiko tinggi akan terjadinya kebakaran hutan berdasarkan lokasi historis kebakaran hutan.
  • Menemukan masyarakat yang jauh dari jalan raya utama untuk merencanakan lokasi pembangunan jalan baru.

Pilih titik input atau layer garis untuk menghitung kepadatan

Titik input atau garis yang akan digunakan untuk menghitung kepadatan.

Gunakan kolom hitung

Sediakan kolom yang menentukan jumlah insiden di setiap lokasi. Misalnya, jika Anda memiliki titik yang menggambarkan kota, Anda dapat menggunakan kolom yang berisi populasi kota sebagai kolom hitung, dan layer kepadatan populasi yang dihasilkan akan menunjukkan kepadatan populasi lebih besar dekat kota-kota dengan populasi yang lebih besar.

Jika pilihan default Tidak Ada digunakan, setiap lokasi akan diasumsikan mewakili jumlah tunggal.

Jarak pencarian

Masukkan sebuah jarak yang menentukan seberapa jauh untuk mencari guna menemukan fitur titik atau garis ketika menghitung nilai kepadatan. Misalnya, jika Anda memberikan jarak pencarian sebesar 10.000 meter, kepadatan di lokasi mana saja di dalam layer output dihitung berdasarkan fitur yang berada di dalam 10.000 meter dari lokasi. Lokasi mana saja yang tidak memiliki insiden apa pun dalam 10.000 meter akan menerima nilai kepadatan nol.

Jika tidak disediakan jarak, default akan dihitung berdasarkan lokasi fitur input dan nilai di kolom hitung (jika kolom hitung disediakan).

Satuan luas output

Tentukan satuan luas output Kepadatan adalah jumlah dibagi luas, dan parameter ini menentukan satuan luas dalam perhitungan kepadatan.

Satuan luas yang tersedia adalah Mil Persegi, Kaki Persegi, Meter Persegi, dan Kilometer Persegi.

Ukuran sel output

Ukuran dan unit sel untuk raster output.

Unit bisa jadi kilometer, meter, mil, atau kaki.

Pilih fitur penghalang (opsional)

Set data yang menentukan penghalang.

Penghalang dapat menjadi feature layer polyline atau fitur poligon.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
