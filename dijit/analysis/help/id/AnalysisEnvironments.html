Pengaturan Lingkungan Analisis Portal

Pengaturan lingkungan untuk analisis pada Portal for ArcGIS dan ArcGIS Online.

Timpa layer hasil

Opsi untuk menimpa layer yang ada. Saat dicentang, jika suatu layer dengan nama sama telah ada pada konten Anda, layer itu akan ditimpa.

  • Dicentang—jika suatu layer dengan nama sama telah ada pada konten Anda, layer itu akan ditimpa. Output akan memiliki sistem koordinat yang sama dengan inputnya.
  • Tidak dicentang—Jika nama layer hasil yang diberikan telah ada, kesalahan akan terjadi dan layer hasil harus diberi nama kembali sebelum alat dapat dieksekusi. Ini adalah default.

Tutup widget analisis setelah menjalankan analisis

Opsi untuk menjaga dialog analisis tetap terbuka dan aktif selama proses berlangsung. Pengaturan ini akan memungkinkan Anda secara mudah melakukan modifikasi pada parameter analisis Anda saat tugas analisis sebelumnya masih berjalan. Analisis adalah proses yang berulang dan dengan menjaga dialog alat tetap terbuka, akan memungkinkan Anda untuk dengan mudah melakukan pengubahan parameter dan menjalankan kembali analisis Anda.

  • Dicentang—Jendela alat analisis akan menutup dan fokus akan kembali ke daftar isi. Ini adalah default
  • Tidak dicentang—Jendela alat analisis akan tetap terbuka dan aktif.

Simpan hasil analisis sebagai feature layer yang di-host

Opsi untuk menentukan apakah layer hasil yang dibuat dari analisis Anda akan menjadi kumpulan fitur atau feature layer yang di-host.

  • Dicentang—Hasil analisis Anda akan dibuat sebagai feature layer yang di-host dan item akan dibuat pada konten Anda. Ini adalah default.
  • Tidak Dicentang—Hasil analisis Anda akan dibuat sebagai kumpulan fitur.

Sistem koordinat output

Pengaturan lingkungan untuk analisis raster digunakan untuk menentukan sistem koordinat layer hasil.

  • Sama seperti Input]—Hasil analisis Anda akan memiliki sistem koordinat yang sama dengan input. Ini adalah default.
  • Seperti yang ditentukan—Pilih sistem koordinat untuk hasil analisis Anda. Ketika opsi ini dipilih, klik ikon bola dunia untuk memilih dari daftar sistem koordinat yang diketahui, atau menambahkan referensi spasial WKID pada kotak yang disediakan.
  • Layer <nama>—Pilih layer yang sudah ada pada web map Anda yang akan digunakan untuk menentukan sistem koordinat hasil analisis Anda.

Memproses sistem koordinat

Pengaturan lingkungan untuk GeoAnalytics digunakan untuk menentukan sistem koordinat yang akan analisis Anda gunakan saat proses eksekusi berlangsung.

Untuk GeoAnalytics, beberapa analisis mengharuskan bahwa proses dieksekusi pada sistem koordinat terproyeksi. Setiap alat yang menerapkan penempatan data atau unit linear mengharuskan sistem koordinat terproyeksi.

  • Sama seperti input—Analisis Anda akan diproses pada sistem koordinat yang sama seperti input. Ini adalah default.
  • Seperti yang ditentukan—Pilih sistem koordinat untuk memproses analisis Anda. Ketika opsi ini dipilih, klik ikon bola dunia untuk memilih dari daftar sistem koordinat terproyeksi yang diketahui, atau menambahkan referensi spasial WKID pada kotak yang disediakan.
  • Layer <nama>—Pilih layer yang ada dari web map Anda. Sistem koordinat layer yang dipilih akan digunakan untuk menentukan sistem koordinat yang akan digunakan untuk memproses analisis Anda.

Jangkauan

Pengaturan lingkungan digunakan untuk menentukan jangkauan atau batasan yang akan digunakan saat eksekusi analisis. Seluruh fitur input yang sepenuhnya berada di dalam atau yang berpotongan dengan jangkauan tertentu akan digunakan pada analisis.

  • Default—Gunakan jangkauan yang disediakan dengan alat.
  • Seperti yang ditentukan—Secara manual masukkan koordinat untuk menentukan jangkauan.
  • Layer <nama>—Pilih layer yang ada dari web map Anda. Jangkauan spasial layer yang dipilih akan digunakan untuk menentukan jangkauan yang digunakan untuk memproses analisis Anda.

Snap Raster

Saat melakukan analisis raster, mengatur lingkungan Snap Raster akan menyesuaikan jangkauan dari output raster layer sehingga sesuai dengan pengaturan sel layer Snap Raster yang ditentukan.

Ukuran Sel

Saat melakukan analisis raster, lingkungan ini mengatur ukuran sel atau resolusi yang akan digunakan untuk membuat layer raster output. Resolusi output default ditentukan oleh ukuran sel terbesar dari layer raster input.

  • Input Minimum—Gunakan ukuran sel terkecil dari seluruh layer input.
  • Input Maksimum—Gunakan ukuran sel terbesar dari seluruh layer input. Ini adalah default.
  • Seperti yang ditentukan—Tentukan nilai angka untuk menetapkan ukuran sel. Jika ini dipilih, nilai default adalah 1.
  • Layer <name>—Set the cell size to the raster layer chosen.

Mask

Saat melakukan analisis raster, gunakan lingkungan Mask untuk menentukan layer yang akan digunakan untuk menentukan area penting Anda. Hanya sel yang berada dalam mask analisis yang akan diperhitungkan dalam operasi analisis.

  • Mask dapat berupa raster atau feature layer.
  • Jika mask analisis adalah raster, seluruh sel yang memiliki nilai akan diperhitungkan untuk menentukan mask. Sel pada raster mask yang merupakan NoData akan dianggap berada di luar masker dan akan menjadi NoData pada layer hasil analisis.
  • Jika masker analisis adalah feature layer, maka akan diubah secara internal menjadi raster saat eksekusi. Oleh karena itu, pastikan dan sudah diatur sesuai untuk analisis Anda.

Penyimpanan Data ArcGIS

Hasil GeoAnalytic disimpan di ArcGIS Data Store dan ditampilkan sebagai feature layer di Portal for ArcGIS. Pada banyak kasus, hasil harus disimpan di penyimpanan data spatiotemporal. Ini adalah default. Berikut adalah alasan mengapa Anda mungkin ingin menyimpan hasil di sebuah penyimpanan data relasional:

  • Untuk menggunakan hasil di kolaborasi portal-ke-portal
  • Untuk mengaktifkan kemampuan sinkronisasi dengan hasil Anda

Anda tidak boleh menggunakan penyimpanan data relasional jika Anda berharap hasil GeoAnalytics untuk meningkatkan dan ingin memanfaatkan kemampuan penyimpanan data besar spatiotemporal untuk menangani jumlah data yang besar.
