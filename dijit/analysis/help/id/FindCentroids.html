Temukan Titik Berat

Alat ini bekerja dengan layer fitur multititik, garis, atau area untuk menentukan pusat setiap fitur yang representatif.

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur di layer input multititik, garis atau area yang terlihat dalam jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, seluruh fitur pada layer input akan dianalisis, meskipun berada di luar jangkauan peta saat ini.

Pilih layer yang fiturnya akan digunakan untuk menghasilkan titik berat

Layer multititik, garis, atau area yang akan digunakan untuk menemukan titik berat.

Selain memilih sebuah layer dari peta, Anda dapat memilih Pilih Layer Analisis yang terdapat di bagian paling bawah daftar drop-down. Opsi ini membuka galeri di mana Anda dapat menelusuri dan memilih set data dari berbagai sumber untuk digunakan dalam analisis Anda.

Perlihatkan saya lokasi output

Jika Pada titik berat sebenarnya dicentang (default), output akan menghasilkan lokasi titik berat untuk setiap fitur input. Titik berat akan ditemukan sebagai berikut:

  • Fitur multititik—titik berat output akan dicari di koordinat rata-rata x dan y dari semua titik dalam fitur multititik.
  • Fitur garis—titik berat output akan dicari di koordinat rata-rata x dan y tertimbang titik tengah dari semau segmen garis dalam fitur garis. Berat didasarkan pada panjang segmen.
  • Fitur area—Titik berat output akan dicari di tengah area.

Jika dimuat oleh fitur input dicentang, fitur titik output akan dimuat oleh atau di dalam setiap fitur input. Fitur titik yang dihasilkan akan muncul sebagai berikut:

  • Fitur multititik—Titik berat output akan bertepatan dengan satu titik dalam kelompok multititik.
  • Fitur garis—Titik berat output akan berada di garis. Jika garis melengkung, output akan berada di titik tengah garis tersebut.
  • Fitur area—Titik output akan berada di dalam fitur area.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama layer input. Jika nama layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Dengan menggunakan Simpan hasil di menu drop-down, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.
