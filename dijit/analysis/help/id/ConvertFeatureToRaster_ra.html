Konversi Fitur ke Raster

Ala ini mengubah fitur ke set data raster.

Setiap kelas fitur yang berisi fitur titik, garis, atau poligon dapat diubah menjadi set data raster.

Pusat sel digunakan untuk memutuskan nilai oiksel raster output.

Tipe kolom input menentukan tipe raster output. Jika kolom berupa bilangan bulat, output raster akan berupa bilangan bulat; jika berupa bilangan titik mengambang, output akan berupa bilangan titik mengambang.

Pilih feature layer untuk dikonversi

Input feature layer untuk dikonversi ke rangkaian data raster.

Pilih kolom

Pilih kolom yang akan digunakan untuk mengisikan nilai pada output raster.

Ukuran sel output

Ukuran dan unit sel untuk raster output.

Unit yang tersedia adalah Mil, Kaki, Kilometer, dan Meter. Unit defaultnya adalah Kaki.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
