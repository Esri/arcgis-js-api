Jalur Optimal Sebagai Raster

Alat ini menghitung jalur optimal dari tujuan ke sumber sebagai raster.

Contoh aplikasi termasuk yang berikut:

  • Temukan rute yang optimal untuk jalan baru.
  • Temukan pelacakan alur hilir dari suatu lokasi.

Pilih raster atau fitur tujuan

Raster atau feature layer yang mengidentifikasi sel dari mana jalur optimal ditentukan ke sumber yang paling optimal.

Jika inputnya adalah raster, maka harus terdiri dari sel yang memiliki nilai valid (nol adalah nilai valid) untuk tujuan, dan sel yang tersisa harus diberi NoData. Raster input harus berjenis bilangan bulat.

Kolom tujuan (opsional)

Kolom digunakan guna mendapatkan nilai untuk lokasi tujuan.

Kolom ini harus berupa jenis bilangan bulat.

Pilih raster akumulasi jarak

Raster akumulasi jarak digunakan untuk menentukan jalur optimal dari tujuan ke sumber.

Raster akumulasi jarak biasanya dibuat dengan alat Akumulasi Jarak atau Alokasi Jarak. Setiap sel dalam raster akumulasi jarak mewakili jarak biaya akumulatif minimal atas permukaan dari setiap sel ke suatu set sel sumber.

Pilih arah balik atau raster arah aliran

Raster arah balik berisi arah yang dihitung dalam derajat. Arah mengidentifikasikan sel berikutnya di sepanjang jalur optimal kembali ke sumber biaya paling akumulatif terkecil dengan menghindari pembatasan.

Raster arah balik biasanya dibuat dengan alat Akumulasi Jarak atau Alokasi Jarak. Jika demikian, rentang nilainya adalah antara 0 derajat hingga 360 derajat. Nilai 0 disimpan untuk sel sumber. Arah timur (kanan) adalah 90 derajat, dan nilai bertambah searah jarum jam (180 adalah arah selatan, 270 adalah barat, dan 360 adalah utara).

Jalur optimal yang dibuat dapat berupa jalur aliran berdasarkan arah aliran D8. Untuk membuat jalur optimal dengan cara ini, gunakan raster arah aliran D8 sebagai input untuk parameter ini.

Jenis jalur (opsional)

Menentukan kata kunci yang menetapkan cara nilai dan zona di data tujuan input diinterpretasikan dalam perhitungan jalur.

  • Untuk Setiap Zona—Untuk setiap zona di data tujuan input, jalur berbiaya terkecil akan ditentukan. Dengan opsi ini, jalur dengan biaya terkecil untuk setiap zona dimulai di sel dengan pembobotan jarak biaya terendah di zona tersebut. Ini adalah default.
  • Tunggal Terbaik—Untuk semua sel di data tujuan input, jalur berbiaya terkecil akan dibuat dari sel dengan jalur berbiaya terkecil minimal ke sel sumber.
  • Untuk Setiap Sel—Untuk setiap sel dengan nilai valid di data tujuan input, jalur berbiaya terkecil akan ditentukan. Dengan opsi ini, setiap sel dari data tujuan input diperlakukan secara terpisah, dan jalur dengan biaya terkecil ditentukan untuk masing-masing dari sel.

Nama layer raster hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
