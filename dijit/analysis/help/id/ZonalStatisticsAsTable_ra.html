Statistik Zona sebagai Tabel

Alat ini merangkum nilai-nilai raster di dalam zona set data lain dan melaporkan hasilnya di dalam tabel.

Beberapa aplikasi contoh termasuk hal berikut:

  • Dengan adanya layer batas negara bagian dan persebaran curah hujan rata-rata di sepanjang wilayah Amerika Serikat, hitung persebaran peristiwa hujan ekstrem di negara bagian menggunakan daftar nilai persentil.
  • Dengan adanya layer zona ekologi yang tumpang-tindih dan layer raster kelembapan tanah, hitung semua statistik pendukung di dalam setiap zona untuk menghubungkan zona dengan ketersediaan kelembapan tanah.

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur-fitur pada layer input dan layer yang akan diringkas yang terlihat di jangkauan peta saat ini yang akan dianalisis. Jika tidak dicentang, semua fitur pada layer input dan layer yang akan diringkas akan dianalisis meskipun berada di luar jangkauan peta saat ini.

Pilih raster atau fitur zona

Layer yang menentukan perbatasan zona, atau area, yang nantinya akan diringkas.

Layer dapat berupa raster bilangan bulat atau data fitur. Untuk raster, zona ditentukan oleh semua lokasi di input yang memiliki nilai sel yang sama. Untuk fitur, zona dapat tumpang-tindih. Area tersebut tidak perlu berdampingan.

Kolom zona

Ini adalah atribut layer yang akan digunakan untuk menentukan perbatasan zona (area).

Contoh, misalnya layer input pertama menentukan batas unit manajemen, dengan atribut yang menentukan wilayah, distrik, dan ID persil setiap unit. Anda pun memiliki layer raster yang menentukan indeks hayati untuk setiap lokasi. Dengan kolom yang Anda pilih, Anda dapat memutuskan untuk menghitung rata-rata hayati di tingkat lokal, distrik, atau wilayah.

Pilih raster nilai

Sel raster di layer ini akan diringkas berdasarkan area (zona) tempat posisi sel tersebut.

Abaikan NoData dalam perhitungan (opsional)

Menentukan apakah NoData yang hilang diabaikan dalam analisis.

  • Dicentang—Analisis akan menyertakan sel yang memiliki nilai di raster nilai input dalam zona tertentu untuk menentukan nilai output zona tersebut. Sel NoData di raster nilai akan diabaikan di dalam penghitungan statistik. Ini adalah default.
  • Tidak dicentang—Analisis akan menghasilkan NoData, jika ada nilai NoData untuk sel raster nilai di dalam zona tertentu.

Jenis statistik (opsional)

Menentukan jenis statistik yang akan dihitung.

Inputnya bisa berupa salah satu dari pilihan berikut: Semua, Rata-rata, Mayoritas, Maksimal, Median, Minimal, Minoritas, Persentil, Rentang, Simpangan baku, Jumlah, Keragaman, Minimal dan Maksimal, Rata-rata dan Simpangan baku, atau Minimal, Maksimal, dan Rata-rata. Defaultnya adalah Semua.

Jika raster nilai input berupa bilangan bulat, semua statistik akan dihitung secara default. Jika jenis raster nilai adalah titik mengambang, maka semua statistik selain Median dan Persentil akan dihitung.

Jika jenis raster nilai input adalah titik mengambang, maka penghitungan zona untuk Mayoritas, Minoritas, dan Keragaman tidak akan dapat dihitung.

Nilai persentil (opsional)

Persentil yang akan dihitung. Sebuah nilai tunggal atau daftar yang memuat beberapa nilai dapat ditentukan. Defaultnya adalah 90, yang menunjukkan persentil ke-90.

Nilai dapat berkisar antara 0 hingga 100. Hasil dari menentukan persentil ke-0 pada intinya setara dengan hasil statistik minimal, dan persentil ke-100 setara dengan hasil statistik maksimal. Nilai 50 pada dasarnya akan menghasilkan hasil yang sama dengan statistik median.

Opsi ini hanya didukung jika parameter Jenis statistik ditetapkan ke Persentil atau Semua.

Jenis interpolasi persentil (opsional)

Menentukan metode interpolasi persentil yang akan digunakan ketika nilai dari raster input yang akan dihitung berjumlah genap.

Jika raster nilai input berjenis piksel integer, metode Terdekat yang akan digunakan. Jika raster nilai input merupakan jenis piksel titik mengambang, maka metode Linear yang akan digunakan. Ini adalah default.

Untuk metode Terdekat, nilai yang paling mendekati persentil yang diinginkan yang akan digunakan. Untuk metode Linear, rata-rata dari dua nilai terdekat dari persentil yang diinginkan yang akan digunakan.

Proses sebagai multidimensional (opsional)

Menentukan cara pemrosesan raster input jika raster input tersebut bersifat multidimensional.

  • Dicentang—Statistik akan dihitung dari irisan set data multidimensional input saat ini. Ini adalah default.
  • Tidak Dicentang—Statistik akan dihitung untuk semua dimensi set data multidimensional input.

Nama tabel hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.
