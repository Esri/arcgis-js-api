Buat Daerah Aliran Air

Alat ini menentukan batas air, atau area penghasil hulu, untuk setiap titik di layer input Anda.

Contohnya, layer ini mungkin merupakan layer yang berisi toko terbaik Anda atau desa yang paling parah terserang penyakit. Karena sumber kontaminasinya pasti di suatu tempat pada hulu batas air titik tersebut, Anda akan menggunakan alat ini untuk menentukan watershed yang mengandung sumber pengkontaminasi.

Lokasi titik masukan Anda adalah penting. Jika itu tidak ditentukan dengan tepat pada garis drainase, hasil batas air-nya mungkin sangat kecil dan tidak bermanfaat untuk analisis Anda. Cari jarak ke drainase terdekat dapat digunakan untuk menyesuaikan lokasi titik input Anda sehingga itu tepat berada di garis batas air untuk menemukan batas air yang mengalir ke titik yang berlokasi di titik drainase. Lokasi yang disesuaikan digunakan untuk mencari aliran sungai yang ditulis ke output lapisan—Anda poin masukan yang tidak dimodifikasi.

Hasilnya adalah sekelompok layer yang mengandung dua layer: batas air dan titik yang disesuaikan digunakan untuk menghitung batas air.

Batas air hasilnya akan memiliki seluruh atribut dari titik input Anda juga atribut tambahan ini:

  • PourPtID—Berisi nilai ID dari titik input
  • Area Square Kilometers—Berisi area dalam kilometer persegi pada batas air hulu
  • Description dan DataResolution—Berisi informasi terkait dengan data hidrologi yang digunakan untuk menghitung pelacakan

Jika Gunakan jangkauan peta saat ini dicentang, hanya fitur pada cakupan peta saat ini yang akan digunakan untuk menghitung batas air. Jika tidak dicentang, seluruh fitur pada layer input akan dihilangkan, meskipun berada di luar cakupan peta saat ini.

Fitur titik yang digunakan untuk menghitung batas air

Batas air (area yang berkontribusi hulu) akan dihitung untuk setiap titik.

Selain memilih sebuah layer dari peta, Anda dapat memilih Pilih Layer Analisis yang terdapat di bagian paling bawah daftar drop-down. Opsi ini membuka galeri di mana Anda dapat menelusuri dan memilih set data dari berbagai sumber untuk digunakan dalam analisis Anda.

Cari jarak ke drainase terdekat

Jika titik input Anda berlokasi jauh dari garis drainase, batas air hasilnya kemungkinan akan sangat kecil dan tidak begitu bermanfaat untuk analisis, misalnya menentukan sumber hulu dari kontaminasi. Pada sebagian besar kasus, Anda ingin titik input langsung ke garis drainase setempat untuk menemukan batas air yang mengalir ke titik yang berlokasi pada garis drainase. Untuk menemukan garis drainase terdekat, tentukan jarak pencarian. Jika Anda tidak menentukan jarak pencarian, alat akan menghitung dan menggunakan jarak pencarian konservatif.

Untuk menggunakan lokasi pasti titik input Anda, tentukan jarak pencarian yaitu nol.

Untuk tujuan analisis, garis drainase telah dihitung sebelumnya oleh Esri menggunakan model hidrologi standar. Jika tidak ada garis drainase pada jarak pencarian, lokasi yang mengandung akumulasi aliran tertinggi dalam jarak pencarian digunakan.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama layer input. Jika nama layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Dengan menggunakan Simpan hasil di menu drop-down, Anda dapat menentukan nama folder di Konten Saya sebagai tempat penyimpanan hasil.

Menjalankan alat ini tidak memakan kredit.
