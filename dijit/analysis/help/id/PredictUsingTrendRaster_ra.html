Prediksi Menggunakan Raster Tren

Menghitung perkiraan raster multidimensional menggunakan raster tren output dari alat Hasilkan Raster Tren.

Jika Gunakan jangkauan peta saat ini dicentang, variabel layer yang terlihat dalam jangkauan peta saat ini akan dianalisis. Jika tidak dicentang, variabel seluruh layer akan dianalisis, bahkan jika berada di luar jangkauan peta saat ini.

Pilih layer citra multidimensional yang akan diprediksi

Raster tren multidimensional input dari alat untuk Buat Raster Tren.

Pilih variabel yang akan diprediksi

Variabel yang akan diprediksi dalam analisis. Jika tidak ada variabel yang ditentukan, semua variabel akan digunakan.

Pilih metode yang digunakan untuk memberikan nilai dimensi prediksi

Menentukan metode yang digunakan untuk memberikan nilai dimensi prediksi.

  • Berdasarkan Nilai—Prediksi akan dihitung untuk nilai dimensi tunggal atau daftar nilai dimensi yang ditentukan pada langkah berikutnya. Ini adalah default. Misalnya, Anda ingin memperkirakan curah hujan tahunan untuk tahun 2050, 2100, dan 2150.
  • Berdasarkan Interval—Prediksi akan dihitung untuk interval dimensi yang ditentukan oleh nilai awal dan akhir. Misalnya, Anda ingin memperkirakan curah hujan tahunan untuk tahun 2050 dan 2150.

Tentukan nilai dimensi untuk prediksi

Nilai dimensi yang akan digunakan dalam prediksi.

Format nilai waktu, kedalaman, dan ketinggian harus cocok dengan format nilai dimensi yang digunakan untuk menghasilkan raster tren. Jika tren raster dihasilkan untuk dimensi WaktuStandar, formatnya harus TTTT-BB-HHTJJ:MM:DD, misalnya, 2050-01-01T00:00:00. Beberapa nilai dipisahkan dengan tanda titik koma.

Parameter ini diperlukan bila metode Berdasarkan nilai dipilih untuk menentukan nilai dimensi untuk prediksi.

Mulai

Tanggal mulai, ketinggian, atau kedalaman interval dimensi yang akan digunakan dalam prediksi.

Akhir

Tanggal akhir, ketinggian, atau kedalaman interval dimensi yang akan digunakan dalam prediksi.

Tentukan interval dimensi untuk prediksi

Tentukan tanggal awal dan akhir, tinggi, atau kedalaman interval dimensi yang akan digunakan dalam prediksi.

Tentukan jumlah langkah antara nilai awal dan akhir

Jumlah langkah antara nilai dua dimensi yang akan dimasukkan dalam prediksi. Nilai default adalah 1.

Misalnya, untuk memprediksi nilai suhu setiap lima tahun, gunakan nilai 5.

Pilih unit yang akan digunakan untuk interval nilai waktu

Menentukan unit yang akan digunakan untuk interval nilai. Ini hanya berlaku saat dimensi analisis adalah dimensi waktu.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
