Ekstrak Raster

Alat ini dapat menempelkan raster ke perbatasan, baik pada area persegi panjang atau bentuk yang Anda definisikan secara interaktif pada layar. Anda dapat menempelkan cakupan area yang saat ini Anda tampilkan di peta Anda atau dekat area studi yang ditentukan oleh sebuah poligon.

Pilih data input untuk diekstrak

Layer input untuk dipotong.

Tentukan geometri pemotongan

Pilih tipe jangkauan yang akan digunakan ketika memotong geometri di layer:

  • Gunakan jangkauan saat ini—Jangkauan peta saat ini akan digunakan untuk menempelkan layer yang dihasilkan. Ini adalah default.
  • Pilih area studi—Memungkinkan Anda menentukan geometri pemotongan menggunakan alat gambar.
  • Pilih lapisan—Pilih feature layer atau layer citra untuk menentukan luasnya, atau pilih layer fitur poligon untuk menentukan geometri pemotongan. Saat feature layer poligon dipilih, kotak centang akan ditampilkan untuk menentukan penggunaan fitur input untuk geometri pemotongan. Secara default, ini tidak dicentang. Ini berarti feature layer input digunakan untuk menentukan jangkauan. Jika dicentang, artinya fitur input akan digunakan untuk geometri pemotongan.

Tentukan jenis pemotongan

Tentukan apakah Anda ingin mempertahankan area di dalam atau di luar geometri potong Anda:

  • Data di dalam geometri—Citra di dalam jangkauan potong akan diekstrak. Ini adalah default.
  • Data di luar geometri—Citra di luar jangkauan potong akan diekstrak.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
