Petakan Ulang Nilai

Alat ini memungkinkan Anda untuk mengubah atau mereklasifikasi nilai piksel data raster. Nilai piksel dipetakan ulang dengan menentukan rentang nilai piksel pada peta ke nilai piksel output. Nilai piksel output dapat berupa nilai yang valid atau nilai NoData, yaitu piksel yang tidak memiliki hubungan dengan nilai yang telah diketahui.

Jika Gunakan jangkauan peta saat ini dicentang, hanya sel yang terlihat di dalam jangkauan peta saat ini yang akan dipetakan ulang. Jika tidak dicentang, semua sel pada raster akan diproses, meskipun berada di luar jangkauan peta saat ini.

Masukkan Data

Data input yang nilainya harus dipetakan ulang.

Petakan ulang nilai piksel ke nilai baru

Masukkan rentang nilai dengan menentukan nilai minimal dan maksimal, kemudian tetapkan nilai output atau nilai NoData ke rentang yang telah ditentukan.

Untuk memetakan ulang kisaran nilai yang baru, klik Tambah, yang akan menambahkan barisan. Klik pada setiap nilai untuk mengedit tabel ke rentang dan memetakan ulang nilai yang Anda butuhkan. Ketika Anda menentukan rentang nilai, ada dua aturan penting yang harus diterapkan:

  • Nilai minimum di rentang adalah inklusif dan nilai maksimum adalah eksklusif.
  • Baris di tabel diproses dari atas ke bawah.

Tabel di bawah menunjukkan bahwa nilai minimal disertakan dalam output, tetapi nilai maksimal tidak disertakan ke dalam output.

Nama layer hasil

Nama layer yang akan dibuat di Konten Saya dan ditambahkan ke peta. Nama default ditetapkan berdasarkan nama alat dan nama layer input. Jika layer tersebut sudah ada, Anda akan diminta untuk memberi nama lain.

Anda dapat menentukan nama folder di Konten Saya tempat hasil akan disimpan menggunakan kotak drop-down Simpan hasil di. Jika Anda memiliki hak istimewa untuk membuat layer citra dinamis dan tile, Anda dapat menentukan jenis layer yang akan dihasilkan dalam output menggunakan kotak drop-down Simpan hasil sebagai.
