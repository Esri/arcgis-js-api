Gerar Raster de Tendência

Estima a tendência para cada píxel ao longo de uma dimensão para uma ou mais variáveis num raster multidimensional.

Escolher uma camada de imagem multidimensional para analisar tendência

A camada de imagem de entrada de um raster multidimensional será analisada.

Escolher a dimensão ao longo da qual a tendência será analisada

A dimensão ao longo da qual uma tendência será extraída para a variável ou variáveis selecionadas na análise.

Escolher variável(is) para analisar tendência

A variável ou variáveis para as quais as tendências serão calculadas. Se não for especificada qualquer variável, será utilizada a primeira variável no raster multidimensional.

Escolher o tipo de linha para se ajustar aos valores variáveis ao longo de uma dimensão

Especifica o tipo de linha a utilizar para se ajustar aos valores de píxel ao longo de uma dimensão.

  • Linear — Valores de píxeis variáveis serão ajustados ao longo da linha de tendência linear. Isto é por omissão.
  • Polinomial — Valores de píxeis variáveis serão ajustados ao longo de uma linha de tendência polinomial de segunda ordem.
  • Harmónico — Valores de píxeis variáveis serão ajustados ao longo de uma linha de tendência harmónica.
  • Mann-Kendall — Os valores de píxeis variáveis serão avaliados usando o teste de tendência Mann-Kendall.
  • Kendall Sazonal — Os valores de píxeis variáveis serão avaliados usando o teste de tendência Kendall Sazonal.

Especificar a unidade de tempo para a duração do período sazonal

Especifica a unidade de tempo a ser usada para a duração de um período sazonal ao realizar o teste Kendall Sazonal.

  • Dias — A unidade para a duração do período sazonal é dias. Isto é por omissão.
  • Meses — A unidade para a duração do período sazonal é meses.

Especificar o comprimento do ciclo harmónico

O comprimento da variação periódica para o modelo. Este parâmetro é necessário quando o tipo de linha de tendência é definido como Harmónico. Por exemplo, o verde da folha tem frequentemente um ciclo forte de variação num único ano, pelo que a duração do ciclo é de 1 ano. Os dados de temperatura horária têm um ciclo forte de variação ao longo de um único dia, pelo que a duração do ciclo é 1 dia.

A duração predefinida é de 1 ano para os dados que variam num ciclo anual.

Este parâmetro só é incluído na análise de tendência quando a dimensão em análise é o tempo e o tipo de linha de tendência está definido para Harmónica.

Escolher a unidade de tempo da duração do ciclo harmónico

Especifica a unidade de tempo a utilizar para a duração de um ciclo harmónico.

  • Dias — A unidade para a duração do ciclo harmónico é dias.
  • Anos — A unidade para a duração do ciclo harmónico é anos. Isto é por omissão.

Este parâmetro só é incluído na análise de tendência quando a dimensão em análise é o tempo e o tipo de linha de tendência está definido para Harmónica.

Especificar o número de frequência para o ajuste da tendência harmónica

A frequência a utilizar no ajuste à tendência harmónica. Este parâmetro especifica a frequência de ciclos num ano.

O valor padrão é 1 ou um ciclo harmónico por ano.

Este parâmetro só é incluído na análise de tendência quando a dimensão em análise é o tempo.

Especificar o número de ordem polinomial a utilizar no ajuste à tendência

O número de ordem polinomial a utilizar no ajuste à tendência.

Este parâmetro só é incluído na análise de tendência quando a dimensão em análise é o tempo.

O valor padrão é 2 ou um polinómio de segunda ordem.

Escolher estatísticas modelo a incluir no raster de tendência

Esta ferramenta oferece três tipos de estatísticas de ajuste de tendência: o erro médio quadrático (RMSE), a estatística de boa qualidade de ajuste de Rquadrado (R quadrado) e valor P para o coeficiente de inclinação (valor p do coeficiente de inclinação). Cada estatística tem uma caixa de seleção correspondente para especificar se será calculada e incluída como uma das múltiplas bandas do raster de tendência de saída.

  • RMSE

    • Marcado — O RMSE será calculado e incluído como uma das múltiplas bandas do raster da tendência de saída. Isto é por omissão.
    • Desmarcado — O RMSE não será calculado.

  • R-quadrado

    • Marcado — O valor Rquadrado será calculado e incluído como uma das múltiplas bandas do raster de tendência de saída.
    • Desmarcado — O valor Rquadrado não será calculado. Isto é por omissão.

  • Valor p do coeficiente de inclinação

    • Marcado — O valor p será calculado e incluído como uma das múltiplas bandas do raster da tendência de saída.
    • Desmarcado — O valor p não será calculado. Isto é por omissão.

Ignorar valores em falta no cálculo

Especifica se os valores em falta são ou não ignorados na análise.

  • Assinalado — A análise irá incluir todos os píxeis válidos ao longo de uma determinada dimensão e ignorar quaisquer píxeis NoData. Isto é por omissão.
  • Desmarcado — A análise resultará em NoData se existirem valores NoData para os píxeis ao longo de uma determinada dimensão.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
