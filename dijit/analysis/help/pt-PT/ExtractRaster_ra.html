Extrair Raster

Esta ferramenta pode recortar um raster, transformando-o num limite, quer numa área retangular ou num formato definido interativamente por si no ecrã. Pode recortar a extensão da área atualmente em exibição no mapa, ou uma área de estudo definida por um polígono.

Selecionar dados de entrada para extrair

A camada de entrada a recortar.

Especificar geometria de recorte

Selecione o tipo de extensão a utilizar ao recortar a geometria da camada.

  • Utilizar extensão atual—A atual extensão do mapa será utilizada para recortar a camada resultante. Isto é por omissão.
  • Selecionar a área de estudo—Permite-lhe definir uma geometria de recorte, utilizando a ferramenta de desenho.
  • Selecionar camada—Selecione uma camada de elementos ou camada de imagens para definir a extensão ou selecione uma camada de elementos polígono para definir a geometria de recorte. Quando uma camada de elementos polígono é selecionada, é exibida uma caixa de validação para especificar se irá usar elementos de entrada para a geometria de recorte. Por predefinição, não está selecionada. Isto significa que a camada de elementos de entrada é usada para definir a extensão. Se estiver selecionada, significa que os elementos de entrada são usados para a geometria de recorte.

Especificar tipo de recorte

Especifique se pretende manter a área no interior ou fora da geometria de recorte:

  • Dados dentro da geometria—As imagens dentro das extensões de recorte serão extraídas. Isto é por omissão.
  • Dados fora da geometria—As imagens fora das extensões de recorte serão extraídas.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa. Se tiver os privilégios para criar camadas de imagens em mosaico e dinâmicas, pode especificar o tipo de camada que será gerado na saída utilizando a caixa pendente Guardar resultado como.
