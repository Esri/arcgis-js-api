Calcular Densidade

A ferramenta Calcular Densidade cria um mapa de densidade de ponto ou linha de elementos espalhando quantidades conhecidas de algum fenómeno (representados como atributos dos pontos ou linhas) através do mapa. O resultado é uma camada de áreas classificadas da menos densa para a mais densa.

Para a entrada de ponto, cada ponto deve representar a localização de algum evento ou incidente, e a camada resultado representa uma contagem do incidente por unidade de área. Um valor de densidade mais elevado numa nova localização significa que existem mais pontos perto desse local. Em muitos casos, a camada de resultado pode ser interpretada como uma superfície de risco para futuros eventos. Por exemplo, se os pontos de entrada representam locais de queda de relâmpagos, a camada resultado pode ser interpretada como uma superfície de risco para quedas de raios futuras.

Para a entrada de linha, a superfície da linha de densidade representa o valor total da linha que está perto de cada local. As unidades dos valores de densidade calculados são o comprimento de linha por unidade de área. Por exemplo, se as linhas representam rios, a camada resultado representará o comprimento total dos rios que se encontram dentro do raio de busca. Este resultado pode ser usado para identificar as áreas que são hospitaleiras para animais de pasto.

Outros casos de uso desta ferramenta incluem o seguinte:

  • Criação de mapas de densidade de crime para ajudar os departamentos de polícia a alocar adequadamente os recursos para áreas de alta criminalidade.
  • Calculando densidades de hospitais dentro de um município. A camada resultado mostrará as áreas com acessibilidades alta e baixa para os hospitais, e esta informação pode ser usada para decidir onde o novo deve ser construído.
  • Identificar áreas que estão em risco elevado de incêndios florestais com base em locais históricos de incêndios florestais.
  • Localizar as comunidades que estão longe de auto-estradas, a fim de planejar onde as novas estradas devem ser construídas.

Escolher ponto ou camada de linha da qual calcular densidade.

O ponto ou camada de linha da qual a densidade será calculada.

Além de escolher uma camada do seu mapa, pode selecionar Escolher Camada de Análise que se encontra na parte inferior da lista pendente. Esta opção abre uma galeria, na qual pode navegar e selecionar conjuntos de dados de várias fontes para utilizar na sua análise.

Use um campo de contagem

Fornecer um campo especificando o número de incidentes em cada local. Por exemplo, se tem pontos que representam as cidades, pode usar um campo que representa a população da cidade, como o campo de contagem, e a camada de densidade populacional resultante irá calcular as densidades populacionais maiores perto de cidades com populações maiores.

Se a opção por omissão Nenhum é usada, cada local assumirá a representação de uma única contagem.

Distância de Pesquisa

Introduza uma distância especificando o quão longe deve pesquisar para encontrar o ponto ou linha de elementos ao calcular os valores de densidade. Por exemplo, se proporcionar uma distância de busca de 10.000 metros, a densidade de qualquer local na camada de saída é calculada com base nas características que se encontram dentro dos 10.000 metros em redor do local. Qualquer local que não tenha nenhum incidente dentro de 10.000 metros receberá um valor de densidade de zero.

Se a distância não for fornecida, por omissão será calculado com base nos locais dos elementos de entrada e os valores do campo de contagem (se um campo de contagem é fornecido).

Recortar saída para

Ou desenhar ou fornecer uma camada especificando a área onde quer que as densidades sejam calculadas. Por exemplo, se está a calcular densidade de peixes num lago, pode usar o limite do lago neste parâmetro e o ponto de saída só vai desenhar dentro do limite do lago.

Classificar Por

Escolha como os valores de densidade serão classificados em áreas.

  • Intervalos iguais—Áreas são criadas de tal modo que a gama de valores de densidade é igual para cada área.
  • Intervalos geométricos—Áreas são baseadas em intervalos de classe que têm uma série geométrica. Este método assegura que cada intervalo de classe tem aproximadamente o mesmo número de valores dentro de cada classe, e que a variação entre os intervalos é consistente.
  • Intervalos naturais—Intervalos de classe para áreas são baseadas em agrupamentos naturais de dados. Os valores de quebra de classe são identificados de forma a agrupar o melhor possível valores semelhantes e a maximizar as diferenças entre classes
  • Área igual - as áreas são criadas de forma a terem todas a mesma área. Por exemplo, se a camada de resultado tiver mais valores de alta densidade do que de baixa densidade, serão criadas mais áreas para altas densidades.
  • Desvio por omissão—Áreas por omissão são criadas com base no desvio-padrão dos valores de densidade previsto.

Número de Classes

Especifica o número de classes (intervalo de valores previstos) na camada de resultado. Cada classe define o limite das áreas. Os valores destas classes serão determinados pelo parâmetro Classifique acima.

Unidades de área de saída

Especifique a unidade de área de saída. A densidade é dividida por contagem de área, e este parâmetro especifica a unidade de área no cálculo da densidade.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se o nome da camada já existir, ser-lhe-á pedido que introduza outro nome.

Ao utilizar o menu suspenso Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
