Converter Elemento para Raster

Esta ferramenta converte elementos para um conjunto de dados raster.

Qualquer classe de elementos que contenha elementos de ponto, linha ou polígono pode ser convertida para um conjunto de dados raster.

O centro da célula é utilizado para determinar o valor do pixel de raster de saída.

O tipo de campo de entrada determina o tipo de raster de saída. Caso o campo seja inteiro, o raster de saída será inteiro; caso seja um ponto flutuante, a saída será um ponto flutuante.

Selecionar camada de elementos para converter

A camada de elementos de entrada a converter para um conjunto de dado raster.

Escolher campo

Selecione o campo que será utilizado para atribuir valores ao raster de saída.

Tamanho de célula de saída

O tamanho da célula e unidade para os rasters de saída.

As unidades disponíveis são Milhas, Pés, Quilómetros e Metros. A predefinição é Pés.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa. Se tiver os privilégios para criar camadas de imagens em mosaico e dinâmicas, pode especificar o tipo de camada que será gerado na saída utilizando a caixa pendente Guardar resultado como.
