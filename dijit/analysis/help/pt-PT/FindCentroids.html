Encontrar Centróides

Esta ferramenta funciona com uma camada de elementos de múltiplos pontos, linhas ou área para determinar o centro representativo de cada elemento.

Se Utilizar atual extensão de mapa se encontrar selecionado, apenas os elementos na camada de múltiplos pontos, linhas, ou área visível na atual extensão de mapa serão analizados. Se não marcar a opção, serão analisados todos os elementos na camada de entrada, mesmo que estejam fora da extensão atual do mapa.

Selecionar camada cujos elementos serão utilizados para gerar centróides

A camada de múltiplos pontos. Linhas ou área que será utilizada para encontrar centróides.

Além de escolher uma camada do seu mapa, pode selecionar Escolher Camada de Análise que se encontra na parte inferior da lista pendente. Esta opção abre uma galeria, na qual pode navegar e selecionar conjuntos de dados de várias fontes para utilizar na sua análise.

Mostrem-me localizações de saída

Quando No verdadeiro centróide está selecionado (predefinição), a saída irá devolver a verdadeira localização do centróide para cada elemento de entrada. Os centróides estarão localizados como se segue:

  • Elementos de múltiplos pontos—O centróide de saída estará localizado na média das coordenadas x e y de todos os pontos no elemento de múltiplos pontos.
  • Elementos de linhas—O centróide de saída estará localizado na média ponderada das coordenadas x e y dos pontos intermédios no elemento de linhas. O peso baseia-se no comprimento do segmento.
  • Elementos de área—O centróide de saída estará localizado no centro da área.

Quando contido em elementos de entrada se encontra selecionado, os elementos de pontos de saída estarão contidos por, ou no interior de, cada elemento de entrada. Os elementos de pontos resultantes surgirão como se segue:

  • Elementos de múltiplos pontos—O centróide de saída será coincidente com um dos pontos no grupo de pontos múltiplos.
  • Elementos de linhas—O centróide de saída encontrar-se-á na linha. Caso a linha seja curvada, a saída encontrar-se-á no ponto intermédio da linha.
  • Elementos de área—O ponto de saída encontrar-se-á no interior do elementos de área.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se o nome da camada já existir, ser-lhe-á pedido que introduza outro nome.

Ao utilizar o menu suspenso Guardar resultado em pode definir o nome da pasta em O Meu Conteúdo onde o resultado será guardado.
