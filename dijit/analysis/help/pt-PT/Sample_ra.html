Amostra

Esta ferramenta cria uma tabela ou uma classe de elementos de pontos que mostra os valores das células a partir de um raster, ou conjunto de rasters, para localizações definidas. As localizações são definidas por células raster, elementos de polígono, elementos polilinha, ou por um conjunto de pontos.

Os valores de píxel serão extraídos a partir de todos os rasters de entrada em cada localização. Será criada uma tabela ou classe de elementos de pontos com campos que contêm os valores dos píxeis para cada raster de entrada. Os atributos adicionais da tabela de rasters de entrada não serão incluídos na tabela de saída.

Qualquer combinação de rasters, tais como banda única, multibanda, multidimensional, ou multivariada, pode ser especificada como rasters de entrada. A estrutura da tabela de saída muda quando os rasters de entrada são multidimensionais. Quando um raster multibanda é especificado como um dos rasters de entrada, todas as bandas dessa entrada serão utilizadas. Para processar uma seleção de bandas de um raster de entrada multibanda, é necessário criar primeiro um novo conjunto de dados raster composto por essas bandas específicas. Usar o resultado na lista de rasters de entrada.

A localização da entrada pode ser um raster ou um elemento de pontos.

  • Raster — Píxeis que contêm valores válidos (não NoData) são usados para extrair os valores de píxeis de todos os rasters de entrada, e o centro do pixel é usado para localizações de ponto.
  • Ponto — Os valores serão em cada localização do ponto.
  • Polilinha ou um polígono — Os valores serão no centroide do polígono ou da polilinha. Se o raster de entrada for multidimensional, os valores serão agregados com base no tipo de Estatística.

Os rasters de entrada são na sua referência espacial e resolução nativas. Se tivermos vários rasters de entrada com diferentes referências espaciais, as localizações de entrada são projetadas para cada referência espacial de cada raster individual a partir do qual os valores são extraídos. O ambiente de análise é aplicado às localizações de entrada, mas os raster de entrada não irão respeitar o ambiente de análise.

As localizações que extraem valores de células NoData no raster de entrada receberão um valor <null> na tabela de saída.

Para realizar a análise com elementos multiponto é necessário convertê-los para elementos de único ponto antes de usá-los nesta ferramenta.

A referência espacial das coordenadas x,y da tabela da amostra de saída são as mesmas que a localização do raster de entrada ou dos elementos de pontos, a menos que seja especificado no ambiente Sistema de Coordenadas de Saída. Se a referência espacial dos dados das localizações de entrada for desconhecida, a referência espacial das coordenadas x,y também é desconhecida.

A referência espacial da classe de elemento de saída é a mesma que a localização do raster de entrada ou dos elementos de pontos, a menos que seja especificado no ambiente Sistema de Coordenadas de Saída.

O ambiente Fator de rocessamento aralelo apenas é suportado quando a entrada é um raster multidimensional.

Se a opção Utilizar a extensão do mapa atual estiver marcada, serão analisadas as variáveis ou bandas da camada que se encontrem visíveis na extensão do mapa atual. Se estiver desmarcada, serão analisadas as variáveis ou bandas de toda a camada, mesmo que estejam fora da extensão do mapa atual.

Escolher as camadas

Uma camada de imagens simples ou uma lista de camadas de imagens, cujos valores serão com base nos dados de localização de entrada.

Para todas as secções de uma camada de imagens multidimensionais, esta deve ser o único raster de entrada.

Escolher camada de localização

Dados de identificação das posições que pretende

Os dados da localização de entrada podem ser um raster ou um ponto, uma polilinha ou uma classe de elementos de polígono.

Especificar campo de ID nico

Um campo que contém um valor diferente para cada localização ou no raster de localização de entrada ou classe de elementos.

Especificar as informações de aquisição de dados de localização (Opcional)

Especificar o tempo, profundidade, ou outros dados de aquisição associados com as células de localização ou elementos.

Para cada dimensão, existem três opções para especificar as informações de aquisição associado com células de localização ou elementos:

  • Campo único ou valor — Quando esa opção for selecionada, as seguintes informações devem ser especificadas:
    • Campo de dimensão ou valor — Selecione um campo de valores de dimensão dos dados de localização, ou introduza um valor de dimensão específico.
  • Campo único ou valor com os respetivos valores — Se esta opção for selecionada, as informações abaixo devem ser especificadas em conjunto para definir o intervalo dentro do qual serão calculadas as estatísticas para as variáveis:
    • Campo de dimensão ou valor — Selecione um campo de valores de dimensão dos dados de localização.
    • Valor relativo antes — Especifique um número não negativo utilizado para subtrair do Campo de dimensão ou valor para definir o início de um intervalo de valores de dimensão. Este valor representa o número de dias se a dimensão for tempo.
    • Valor relativo depois — Especifique um número não negativo usado para adicionar a Campo de dimensão ou valor para definir o fim de um intervalo de valores de dimensão. Este valor representa o número de dias se a dimensão for tempo.
  • Campos de início e fim — Se esta opção for selecionada, as informações abaixo devem ser especificadas em conjunto para definir o intervalo dentro do qual serão calculadas as estatísticas para as variáveis:
    • Campo de início ou valores — Selecione um campo de valores de dimensão dos dados de localização, ou introduza um valor de dimensão específico, para definir o início de um intervalo de valores de dimensão.
    • Campo de fim ou valores — Selecione um campo de valores de dimensão dos dados de localização, ou introduza um valor de dimensão específico, para definir o fim de um intervalo de valores de dimensão.

Especificar distância de campo ou valor do buffer

A distância especificada em torno dos elementos dos dados de localização. A distância do buffer é especificada em unidades lineares de referência espacial dos elementos de localização. Se o elemento usar uma referência geográfica, a unidade será graus.

As estatísticas serão calculadas dentro desta área de buffer.

Escolher tipo de estatística

O tipo de estatística a calcular.

  • Mínimo — Encontra o valor mínimo dentro do intervalo especificado.
  • Máximo — Encontra o valor máximo dentro do intervalo especificado.
  • Mediana — Encontra o valor mediano dentro do intervalo especificado.
  • Média — Calcula a média para o intervalo especificado.
  • Soma — Calcula a soma das variáveis dentro do intervalo especificado.
  • Maioria — Encontra o valor que ocorre com mais frequência.
  • Minoria — Encontra o valor que ocorre com menos frequência.
  • Desvio Padrão — Calcula o desvio padrão.
  • Percentil — Calcula o percentil definido dentro do intervalo especificado.

Valor de percentil

O percentil a calcular quando o parâmetro Tipo de Estatística está definido para Percentil.

Este de 0 a 100. O padrão é 90.

Especificar a técnica de reamostragem

O algoritmo de reamostragem usado na amostragem de um raster.

  • Mais próximo — Vizinho mais próximo. Isto é por omissão.
  • Bilinear — Interpolação bilinear
  • Cúbico — Convolução cúbica

Especificar o de saída

Especifica se os valores aparecem em linhas ou colunas na tabela de saída. Esta opção apenas está disponível quando o raster multidimensional de entrada contém uma variável e uma dimensão, e cada secção é um raster de banda única.

  • Os valores aparecem em linha — Os valores aparecem em linhas separadas na tabela de saída. Isto é por omissão.
  • Os valores aparecem em coluna — Os valores aparecem em colunas na tabela de saída. Esta opção apenas é válida quando o raster multidimensional de entrada contém uma variável e uma dimensão, e cada secção é um raster de banda única.

Especificar tipo de saída

Especifica se deve gerar uma classe de elementos de pontos com valores na sua tabela, ou apenas uma tabela com valores .

  • Tabela — Gerar uma tabela com valores . Isto é por omissão.
  • Elementos — Gerar uma classe de elementos de pontos com valores na sua tabela. Se os elementos de entrada de localização forem polilinhas ou polígonos, cada ponto na saída é o centroide de cada geometria dos elementos de entrada de localização.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
