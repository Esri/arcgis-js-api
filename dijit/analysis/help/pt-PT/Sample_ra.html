Amostra

Esta ferramenta cria uma tabela ou uma classe de elementos de pontos que mostra os valores das células a partir de um raster, ou conjunto de rasters, para localizações definidas. As localizações são definidas por células raster, elementos de polígono, elementos polilinha, ou por um conjunto de pontos.

Alguns exemplos de aplicações desta ferramenta incluem:

  • Extraia atributos para localização de torre de transmissão a partir de vários rasters, como uso do solo, tipo de solo, elevação e distância até à estrada.
  • Extraia a temperatura do solo a diferentes profundidades para um determinado conjunto de pontos.
  • A partir de 30 anos de dados mensais de temperatura da superfície do mar, resuma em localizações de animais ou dentro de territórios para um determinado momento quando uma espécie animal foi observada.

Se a opção Utilizar a extensão do mapa atual estiver marcada, serão analisadas as variáveis ou bandas da camada que se encontrem visíveis na extensão do mapa atual. Se estiver desmarcada, serão analisadas as variáveis ou bandas de toda a camada, mesmo que estejam fora da extensão do mapa atual.

Escolher as camadas

Uma imagem única ou uma lista de imagens, cujos valores serão amostrados com base nos dados de localização de entrada.

Para amostrar todas as secções de uma camada de imagens multidimensionais, esta deve ser o único raster de entrada.

Escolher camada de localização

Dados de identificação das posições que pretende amostrar

Os dados da localização de entrada podem ser um raster ou um ponto, uma polilinha ou uma classe de elementos de polígono.

Especificar campo de ID nico

Um campo que contém um valor diferente para cada localização ou no raster de localização de entrada ou classe de elementos.

Especificar as informações de aquisição de dados de localização (Opcional)

Especificar o tempo, profundidade, ou outros dados de aquisição associados com as células de localização ou elementos.

Para cada dimensão, existem três opções para especificar as informações de aquisição associado com células de localização ou elementos:

  • Campo único ou valor — Quando esta opção for selecionada, as seguintes informações têm de ser especificadas:
    • Campo ou valor de dimensão — Selecione um campo de valores de dimensão a partir dos dados de localização ou introduza um valor de dimensão específico.
  • Campo ou valor único com valores relativos — Se esta opção for selecionada, as informações abaixo têm de ser especificadas em conjunto para definir o intervalo dentro do qual serão calculadas as estatísticas para as variáveis:
    • Campo ou valor de dimensão — Selecione um campo de valores de dimensão a partir dos dados de localização.
    • Valor relativo antes — Especifique um número não negativo utilizado para subtrair do Campo ou valor de dimensão para definir o início de um intervalo de valores de dimensão. Este valor representa o número de dias se a dimensão for tempo.
    • Valor relativo depois — Especifique um número não negativo utilizado para subtrair do Campo ou valor de dimensão para definir o fim de um intervalo de valores de dimensão. Este valor representa o número de dias se a dimensão for tempo.
  • Campos ou valores de início e fim — Se esta opção for selecionada, as informações abaixo têm de ser especificadas em conjunto para definir o intervalo dentro do qual serão calculadas as estatísticas para as variáveis:
    • Campo ou valores de início — Selecione um campo de valores de dimensão a partir dos dados de localização ou introduza um valor de dimensão específico para definir o início de um intervalo de valores de dimensão.
    • Campo ou valores de fim — Selecione um campo de valores de dimensão a partir dos dados de localização ou introduza um valor de dimensão específico para definir o fim de um intervalo de valores de dimensão.

Este parâmetro só estará disponível quando o parâmetro Escolher camada(s) de imagem a amostrar for um raster multidimensional.

Especificar distância de campo ou valor do buffer

A distância especificada em torno dos elementos dos dados de localização. A distância do buffer é especificada em unidades lineares de referência espacial dos elementos de localização. Se o elemento usar uma referência geográfica, a unidade será graus.

As estatísticas serão calculadas dentro desta área de buffer.

Este parâmetro só estará disponível quando Escolher camada(s) de imagem a amostrar for um raster multidimensional.

Escolher tipo de estatística

O tipo de estatística a calcular.

  • Mínimo — Encontra o valor mínimo dentro do intervalo especificado.
  • Máximo — Encontra o valor máximo dentro do intervalo especificado.
  • Mediana — Encontra o valor mediano dentro do intervalo especificado.
  • Média — Calcula a média para o intervalo especificado.
  • Soma — Calcula a soma das variáveis dentro do intervalo especificado.
  • Maioria — Encontra o valor que ocorre com mais frequência.
  • Minoria — Encontra o valor que ocorre com menos frequência.
  • Desvio Padrão — Calcula o desvio padrão.
  • Percentil — Calcula o percentil definido dentro do intervalo especificado.

Este parâmetro só estará disponível quando o parâmetro Escolher camada(s) de imagem a amostrar for um raster multidimensional e os parâmetros Escolher camada de localização forem polilinhas ou polígonos ou Especificar campo ou valor de distância do buffer (Opcional) for especificado.

Valor de percentil

Este valor pode variar entre 0 e 100. O padrão é 90.

Este parâmetro só estará disponível quando o parâmetro Escolher tipo de estatística estiver definido como Percentil.

Especificar a técnica de reamostragem

O algoritmo de reamostragem usado na amostragem de um raster.

  • Mais próximo — Vizinho mais próximo que utiliza o valor da célula mais próxima para atribuir um valor à célula de saída durante a reamostragem. Isto é por omissão.
  • Bilinear — Interpolação bilinear que determina o novo valor de uma célula com base numa média de distância ponderada dos quatro centros de células de entrada mais próximos.
  • Cúbica — Convolução cúbica que determina o novo valor de uma célula com base na instalação de uma curva suave através dos 16 centros de células de entrada mais próximos.

Especificar o layout de saída

Especifica se os valores amostrados aparecem em linhas ou colunas na tabela de saída.

  • Os valores amostrados aparecem em linha — Os valores amostrados aparecem em linhas separadas na tabela de saída. Isto é por omissão.
  • Os valores amostrados aparecem em coluna — Os valores amostrados aparecem em colunas na tabela de saída.

Este parâmetro só estará disponível quando o parâmetro Escolher camada(s) de imagem a amostrar for um raster multidimensional e tiver uma variável e uma dimensão e cada secção for um raster de banda única.

Especificar tipo de saída

Especifica se deve gerar uma classe de elementos de pontos com valores amostrados na sua tabela, ou apenas uma tabela com valores amostrados.

  • Tabela — Gerar uma tabela com valores amostrados. Isto é por omissão.
  • Elementos — Gerar uma classe de elementos de pontos com valores amostrados na sua tabela. Se os parâmetros Escolher camada de localização forem polilinhas ou polígonos, cada ponto na saída é o centróide de cada geometria dos elementos de localização de entrada.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
