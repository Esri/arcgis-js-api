Monitorizar Vegetação

Esta ferramenta efetua uma operação aritmética nas bandas de uma camada raster multibanda para revelar informações de cobertura de vegetação da área de estudo.

Selecionar os dados de entrada

Selecione a camada raster multibanda. Garanta que o raster de entrada tem as bandas apropriadas disponíveis.

Selecionar método para monitorizar vegetação

Selecione o método a utilizar para calcular a camada de índice de vegetação. Os diferentes índices de vegetação podem ajudar a destacar certos elementos ou a reduzir vários ruídos.

  • Global Environmental Monitoring Index — GEMI é um índice de vegetação não-linear para monitorização ambiental global de imagens de satélite. É semelhante a NDVI, mas menos sensível a efeitos atmosféricos. É afetado por solo descoberto; portanto, não se recomenda a respetiva utilização em áreas de vegetação esparsa ou moderadamente densa.
  • Green Vegetation Index - Landsat TM — GVI foi originalmente concebido a partir de imagens Landsat MSS, mas foi modificado para poder ser utilizado com imagens Landsat TM. É também conhecido como índice de vegetação verde Landsat TM Tasseled Cap. Este índice de monitorização também pode ser utilizado com imagens cujas bandas partilham as mesmas características espectrais.
  • Modified Soil Adjusted Vegetation Index — MSAVI2 é um índice de vegetação que procura minimizar as influências do solo descoberto no método SAVI.
  • Normalized Difference Vegetation Index — NDVI é um índice estandardizado que lhe permite gerar uma imagem que apresenta verdor, ou biomassa relativa. Este índice tira partido do contraste entre as características de duas bandas de um conjunto de dados raster multiespectral; as absorções de pigmento de clorofila na banda Vermelho e a elevada refletividade dos materiais vegetais na banda de infravermelho-próximo (NIR).
  • Perpendicular Vegetation Index — PVI é semelhante a um índice de vegetação por diferença; no entanto, é sensível a alterações atmosféricas. Este método deve ser utilizado para comparar imagens apenas quando se trata de imagens que foram corrigidas ao nível atmosférico. Estas informações podem ser-lhe fornecidas pelo seu fornecedor de dados.
  • Soil-Adjusted Vegetation Index — SAVI é um índice de vegetação que procura minimizar as influências da luminosidade do solo, recorrendo a um fator de correção da luminosidade do solo. É muitas vezes utilizado em regiões áridas, onde a cobertura de vegetação é escassa.
  • Sultan's Formula — O processo Sultan's Formula recorre a uma imagem de 8 bits de seis bandas e aplica-lhe um algoritmo específico por forma a produzir uma imagem de 8 bits de três bandas. A imagem resultante destaca formações rochosas denominadas ofiolitos, em faixas costeiras. Esta fórmula foi criada com base nas bandas TM e ETM de uma cena Landsat 5 ou 7.
  • Transformed Soil-Adjusted Vegetation Index — Transformed-SAVI é um índice de vegetação que procura minimizar as influências da luminosidade do solo, assumindo a linha do solo como um declive e intersecção arbitrários.
  • Visible Atmospherically Resistant Index — O VARI é um índice de vegetação que estimou a presença de vegetação apenas com informações do espetro visível. Este método requer bandas nas partes vermelha, verde e azul do espetro.

Especificar índices para a banda NIR e banda Vermelho

Especifique os índices de banda para as bandas NIR e Vermelho.

Os sensores de satélite ou aéreos multibanda capturam informações que são divididas em bandas de comprimento de onda, que são identificadas com índices de banda. Este método de monitorização de vegetação requer que especifique os índices de banda para o infravermelho próximo (NIR) e bandas de comprimento de onda vermelho.

Especificar índices para as bandas Vermelho, Verde e Azul

Especifique os índices de banda para as bandas Vermelho, Verde e Azul.

Os sensores de satélite ou aéreos multibanda capturam informações que são divididas em bandas de comprimento de onda, que são identificadas com índices de banda. O Índice de Resistência Atmosférica Visível requer que especifique os índices de banda para as bandas de comprimento de onda Vermelho, Verde e Azul.

Especificar índices para as bandas Azul, Vermelho, NIR, SWIR-1 e Térmico

Especifique os índices de banda para as bandas Azul, Vermelho, Infravermelho próximo (NIR), Infravermelho de ondas curtas (SWIR) e Térmico.

Os sensores de satélite ou aéreos multibanda capturam informações que são divididas em bandas de comprimento de onda, que são identificadas com índices de banda. O índice Fórmula de Sultan requer que especifique os índices de banda para as bandas de comprimento de onda azul, vermelho, infravermelho próximo (NIR), infravermelho de ondas curtas (SWIR) e térmico.

Especificar índices para as bandas Azul, Verde, Vermelho, NIR, SWIR-1 e SWIR-2

Especifique os índices de banda para as bandas Azul, Verde, Vermelho, NIR, SWIR-1 e SWIR-2.

Os sensores de satélite ou aéreos multibanda capturam informações que são divididas em bandas de comprimento de onda, que são identificadas com índices de banda. O Índice de Vegetação Verde requer que especifique os índices de banda para as bandas de comprimento de onda azul, verde, vermelho, infravermelho próximo (NIR), infravermelho de ondas curtas 1 (SWIR-1) e infravermelho de ondas curtas 2 (SWIR-2).

Inclinação da linha do solo

A inclinação da linha do solo. A inclinação é a relação linear aproximada entre as bandas NIR e Red num gráfico de dispersão.

Este parâmetro á apenas válido para o método de Índice de Vegetação Ajustado ao Solo Transformado (Transformed Soil-Adjusted Vegetation Index)

Intersecção

O valor da NIR quando o valor de reflexão da banda vermelho (Red) é 0 para as linhas de solo específicas.

(a = NIR - sRed), quando Red é 0.

Este parâmetro á apenas válido para o método de Índice de Vegetação Ajustado ao Solo Transformado (Transformed Soil-Adjusted Vegetation Index)

Quantidade de cobertura de vegetação verde

Especifica a quantidade de cobertura de vegetação verde exigida pelo método do Índice de Vegetação Ajustado ao Solo.

Os valores válidos são como se seguem:

  • 1 = Áreas desprovidas de cobertura de vegetação
  • 0.5 = Áreas com cobertura de vegetação moderada
  • 0 = Áreas com elevada cobertura de vegetação

Fator de ajustamento

Especifica o fator de ajustamento para ajudar a minimizar o efeito de solo, que é exigido pelo método de Índice de Vegetação Ajustado ao Solo Transformado (Transformed Soil-Adjusted Vegetation Index). O valor padrão é 0.08.

Um valor baixo significa que está a ignorar o efeito do solo. Um valor elevado significa que o solo afetará o resultado.

Nome da camada resultante

O nome da camada que será criada em O Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e do nome da camada de entrada. Se a camada já existir, ser-lhe-á pedido que introduza outro nome.

É possível especificar o nome de uma pasta em O Meu Conteúdo, onde o resultado será guardado utilizando a opção Guardar Resultado em da caixa suspensa.
