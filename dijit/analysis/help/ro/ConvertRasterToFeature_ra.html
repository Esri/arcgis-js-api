Convertire raster în obiect spaţial

Acest instrument converteşte un raster într-un set de date de obiecte spaţiale, precum puncte, linii sau poligoane.

Parametrul Alegere câmp vă permite să alegeţi ce câmp de atribute din setul de date raster de intrare va deveni un atribut în clasa de obiecte spaţiale de ieşire. Acest parametru este disponibil doar pentru rastere cu numere întregi care au atribute suplimentare în tabelul asociat.

Rasterul de intrare poate avea orice dimensiune pentru celulă.

Rasterele cu numere întregi pot fi convertite în obiecte spaţiale puncte, linii sau poligoane; totuşi, rasterele cu puncte mobile pot fi convertite doar în obiecte spaţiale puncte.

Dacă se bifează Utilizare extindere curentă a hărţii, vor fi convertite numai celulele raster vizibile în extinderea curentă a hărţii. Dacă nu se bifează, toate celulele din stratul tematic raster vor fi convertite, chiar dacă se află în afara extinderii curente a hărţii. Reţineţi dacă stratul tematic raster este mare sau are o rezoluţie foarte înaltă.

Printre situaţiile de utilizare ale acestui instrument se numără:

  • Creați obiecte spațiale de tip poligon dintr-un raster de clasificare a utilizării terenului.
  • Fluxurile identificate într-un DEM pot fi convertite în obiecte spaţiale linii.
  • Un DEM poate fi convertit în obiecte spaţiale puncte care pot fi utilizate într-un flux de lucru nonraster.

Alegeţi stratul tematic raster pentru convertire

Stratul tematic raster de intrare care va fi convertit în straturi tematice de obiecte spaţiale.

Alegere câmp

Un câmp care specifică valoarea de conversie.

Poate să fie un număr întreg sau o valoare text.

Un câmp care conţine valori de puncte mobile poate fi utilizat doar dacă rezultatul este un set de date punctuale.

Valoarea implicită este câmpul Value, care conține valoarea din fiecare celulă raster.

Alegeţi tipul rezultatului

Specifică tipul rezultatului.

  • Punct—Rasterul va fi convertit într-un set de date tip punct. Aceasta este acţiunea implicită.
  • Linie—va fi convertit într-un set de date de obiect spațial tip linie.
  • Poligon—Rasterul va fi convertit într-un set de date de obiect spațial tip poligon.

Simplificare linii sau poligoane

Specifică dacă liniile sau poligoanele vor fi simplificate (aplatizate). Aplatizarea este realizată astfel încât linia să conțină un număr minim de segmente în timp ce rămâne cât mai aproape posibil de marginile celulei raster originale.

  • Bifat—Obiectele spațiale liniare sau poligonale vor fi aplatizate pentru a produce un rezultat mai generalizat. Aceasta este acţiunea implicită.
  • Nebifat—Obiectele spațiale liniare sau poligonale vor respecta granițele celulei setului de date raster.

Acest parametru este acceptat doar dacă tipul de rezultat este o linie sau un poligon.

Creare obiecte spaţiale cu mai multe părţi

Specifică dacă poligoanele rezultante vor consta în obiecte spațiale cu una sau mai multe părți.

  • Bifat—Obiectele spațiale cu părți multiple vor fi create pe baza poligoanelor care au aceeași valoare.
  • Nebifat—Obiectele spațiale individuale (cu o singură parte) vor fi create pentru fiecare poligon. Aceasta este acţiunea implicită.

Acest parametru este acceptat doar dacă tipul rezultatului este un poligon.

Vârfurile maxime pe obiect spațial poligonal

Limita vârfurilor folosită pentru a subîmpărți un poligon în poligoane mai mici.

Dacă este lăsat loc gol, poligoanele rezultante nu vor fi împărțite. Aceasta este acţiunea implicită.

Acest parametru este acceptat doar dacă tipul rezultatului este un poligon.

Nume strat tematic rezultat

Acesta este numele stratului tematic ce va fi creat în Conținutul Meu și va fi adăugat pe hartă. Numele implicit se bazează pe numele instrumentului şi numele stratului tematic de intrare. Dacă stratul tematic rezultat există deja, vi se va solicita să furnizaţi un alt nume.

Puteți specifica numele folderului în Resursele mele unde se va salva rezultatul folosind caseta drop-down Salvați rezultatele în.
