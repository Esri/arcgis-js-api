Legg lag oppå

Overlegg slår sammen to eller flere lag til ett lag. Et overlegg blir som om du ser ned gjennom en stabel med kart som ligger oppå hverandre, og oppretter et enkelt kart som inneholder all den informasjonen som finnes i hele kartstabelen. Overlegg er mye mer enn en sammenslåing av streker. Alle attributtene for geoobjektene som er en del av overlegget, videreføres til det endelige produktet. Overlegg brukes til å besvare et av det mest grunnleggende geografiske spørsmålene: "Hva ligger oppå hva?"For example:

  • Hvilke tomter ligger innenfor flomsonen for hundreårsflom?(Innenfor betyr det samme som oppå.)
  • Hvilke veier er innenfor hvilke fylker?
  • Hvilken arealbruk er det oppå hvilken jordart?
  • Hvilke brønner befinner seg på nedlagte militærbaser?

Hvis det er merket av for Bruk gjeldende kartutstrekning, er det bare geoobjektene i inndata- og overleggslaget som er synlige innenfor gjeldende kartutstrekning, som legges over. Hvis det ikke er merket av for dette alternativet, legges alle geoobjektene i både inndatalaget og overleggslaget over, selv om de befinner seg utenfor gjeldende kartutstrekning.

Choose input layer

Velg overleggsmetode

The overlay method defines how the input layer and the overlay layer are combined.

  • Intersect—The result will contain only those features or portions of features in the overlay layer that overlap with features in the input layer. Ved kryssende linje- eller områdegeoobjekter har du muligheten til å angi hvilken type geoobjekter du ønsker som resultat.

  • Union—The result will contain features from both the input layer and the overlay layer. The Union option is only available if both the input layer and the overlay layer contain area features.
  • Erase—The result will contain features or portions of features in the overlay layer that do not overlap with the areas in the input layer. Overleggslaget kan inneholde område-, linje- eller punktgeoobjekter.

If Use current map extent is checked, only the features in the input and overlay layer that are visible within the current map extent will be overlaid. If unchecked, all features in both the input layer and the overlay layer will be overlaid, even if they are outside the current map extent.

Navn på resultatlag

This is the name of the layer that will be created in My Content and added to the map. Det har samme geoobjektstype (områder, linjer eller punkt) som overleggslaget. Standardnavnet er basert på overleggsmetoden og navnet på inndatalaget. Hvis laget allerede finnes, blir du bedt om å oppgi et annet navn.

Using the Save result in drop-down box, you can specify the name of a folder in My Content where the result will be saved.
