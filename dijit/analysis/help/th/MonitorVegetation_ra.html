การตรวจสอบข้อมูลพืช

เครื่องมือนี้ใช้การคำนวณทางคณิตศาสตร์ให้กับแบนต่างๆ ของชั้นข้อมูลเรสเตอร์มมัลติแบน ในการแสดงข้อมูลพื้นที่ครอบคลุมของพืชในพื้นที่ศึกษา

เลือกข้อมูลอินพุท

เลือกชั้นข้อมูลแรสเตอร์มัลติแบน ตรวจสอบให้แน่ใจว่าแรสเตอร์ที่นำเข้ามีแบนที่เหมาะสมสามารถใช้ในการวิเคราะห์ได้

เลือกวิธีการในการตรวจสอบพืช

เลือกวิธีการที่จะใช้ในการคำนวณชั้นข้อมูลดัชนีพืชพรรณ ดัชนีพืชที่แตกต่างกันสามารถช่วยเน้นคุณสมบัติบางอย่าง หรือลดค่าสิ่งรบกวนต่าง ๆ

  • Global Environmental Monitoring Index — GEMI เป็นดัชนีพืชแบบไม่เชิงเส้น สำหรับการตรวจสอบด้านสิ่งแวดล้อมระดับโลกจากภาพถ่ายดาวเทียม มันคล้ายกับ NDVI แต่ไม่ไวต่อผลกระทบในชั้นบรรยากาศ โดยจะได้รับผลกระทบจากพื้นที่ดินเปล่า ดังนั้นจึงไม่แนะนำให้ใช้ในพื้นที่ที่มีพืชเบาบาง หรือ มีความหนาแน่นปานกลาง
  • Green Vegetation Index - Landsat TM — GVI ถูกออกแบบมาสำหรับภาพจาก Landsat MSS แต่ได้รับการแก้ไขให้ใช้ได้กับภาพ Landsat TM เป็นที่รู้จักกันในชื่อดัชนีพืช Tasseled Cap green Landsat TM ดัชนีการตรวจสอบนี้ยังสามารถนำมาใช้กับภาพที่มีแบนที่แสดงลักษณะสเปกตรัมเดียวกัน
  • Modified Soil Adjusted Vegetation Index — MSAVI2 เป็นดัชนีพืชพรรณที่พยายามที่จะลดอิทธิพลของพื้นที่ดินเปล่า ของวิธี SAVI
  • Normalized Difference Vegetation Index — NDVI เป็นดัชนีมาตรฐานช่วยให้คุณสามารถสร้างภาพที่แสดงความอ่อน หรือความเข้มของชีวมวล ดัชนีนี้จะใช้ประโยชน์ของความคมชัดในลักษณะของแบนสองแบน จากชุดข้อมูลราสเตอร์หลายช่วงความถึ่ ว่ามีการดูดกลืนเม็ดสีคลอโรฟิลในแถบสีแดง และสีสะท้อนแสงสูงของวัสดุพืชในช่วงแบนใกล้อินฟราเรด (NIR)
  • Perpendicular Vegetation Index — PVI คล้ายกับดัชนีความแตกต่างพืช แต่จะไวต่อการเปลี่ยนแปลงของชั้นบรรยากาศมากกว่า เมื่อใช้วิธีนี้ในการเปรียบเทียบภาพที่แตกต่างกันก็ควรจะใช้เฉพาะในภาพที่ได้รับการแก้ไขค่าแบบแอตโมสเฟียลิคอล ข้อมูลนี้สามารถดูได้จากผู้ที่ขายข้อมูลให้คุณ
  • Soil-Adjusted Vegetation Index — SAVI เป็นดัชนีพืชพรรณที่พยายามที่จะลดอิทธิพลของความสว่างของดินโดยใช้ปัจจัยการแก้ไขดินบริเวณที่มีความสว่าง วิธีนี้มักจะใช้ในดินแดนแห้งแล้งที่มีปกพืชคลุมอยู่ในระดับต่ำ
  • Sultan's Formula — กระบวนการประมาลผลด้วย สูตรของ Sultan จะใช้ภาพ หกแบน ขนาด 8 บิต ผ่านอัลกอริทึมที่เฉพาะเจาะจงเพื่อผลิตเป็นภาพ สามแบน ขนาด 8 บิต ส่งผลให้ภาพแสดงไฮไลท์หินที่เรียกว่าโอฟิโอไลต์บนชายฝั่ง สูตรนี้ได้รับการออกแบบบนพื้นฐานของ TM และ ETM แบนภาพจากของดาวเทียม LANDSAT 5 หรือ 7
  • Transformed Soil-Adjusted Vegetation Index — Transformed-SAVI เป็นดัชนีพืชพรรณที่พยายามที่จะลดอิทธิพลของความสว่างดิน ด้วยการสมมติว่าเส้นดินที่มีความลาดชันสูงจะโดนตัดออก
  • Visible Atmospherically Resistant Index — VARI คือดัชนีพืชพรรณที่ประมาณการการมีอยู่ของพืชพรรณ โดยใช้ข้อมูลจากสเปกตรัมที่มองเห็นได้เพียงอย่างเดียว วิธีการนี้ จะต้องการแบนด์ในส่วนสีแดง สีเขียว และสีน้ำเงินของสเปกตรัม

ระบุดัชนีของแบนด์ NIR และแบนด์สีแดง

ระบุดัชนีแบนด์สำหรับ NIR และแบนด์สีแดง

ดาวเทียมหลายแบนด์ หรือเซนเซอร์ทางอากาศ จะบันทึกข้อมูลที่ถูกแบ่งเป็นแบนด์ความยาวคลื่น ซึ่งจะถูกระบุด้วยดัชนีแบนด์ วิธีการตรวจสอบพืชพรรณนี้ จะกำหนดให้คุณต้องระบุดัชนีแบนด์สำหรับอินฟราเรดใกล้ (NIR) และแบนด์ความยาวคลื่นสีแดง

ปริมาณของพืชคลุมสีเขียว

ระบุปริมาณพืชพรรณสีเขียวที่ปกคลุมดิน ด้วยวิธีการดัชนีพืชพรรณที่ปรับแก้ดิน

ค่าที่ถูกต้องเป็นดังนี้:

  • 1 = พื้นที่ที่ไม่มีพืชสีเขียวครอบคลุม
  • 0.5 = พื้นที่ที่มีพืชสีเขียวครอบคลุม หรือ มีพืชสีเขียวในระดับปานกลาง
  • 0 = พื้นที่ที่มีพืชสีเขียวครอบคลุมสูง

ปัจจัยการปรับ

ระบุปัจจัยการปรรับเพื่อช่วยลดผลกระทบจากดินให้เหลือน้อยที่สุด ซึ่งจำเป็นสำหรับวิธีการดัชนีพืชดินที่ปรับเปลี่ยน ค่าเริ่มต้นคือ 0.08

ค่าต่ำหมายความว่าคุณจะไม่สนใจผลกระทบของดิน ค่าสูงหมายความว่าดินมีผลกระทบต่อผลลัพธ์มากขึ้น

ชื่อชั้นข้อมูลผลลัพธ์
