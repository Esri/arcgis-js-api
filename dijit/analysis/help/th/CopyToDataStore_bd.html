คัดลอกไป Datastore

เครื่องมือนี้จะให้ใส่ชั้นข้อมูลอินพุธและคัดลอกไปยัง data store ข้อมูลจะถูกคัดลอกไปยัง ArcGIS Data Store และจะเก็บไว้ในการจัดเก็บข้อมูลของคุณหรือความสัมพันธ์เชิงพื้นที่ชั่วคราว

ตัวอย่างเช่น, คุณสามารถคัดลอกฟีเจอร์ที่ถูกเก็บไว้ใน ข้อมูลขนาดใหญ่ใช้ไฟล์ร่วมกัน ไปยัง เก็บข้อมูลเชิงสัมพันธ์ และระบุว่ามีเพียงฟีเจอร์ในขอบเขตแผนที่ปัจจุบันเท่านั้นจะถูกคัดลอก สร้างฟีเจอร์เซอร์วิสหลักนี้กับฟีเจอร์เหล่านั้นที่อยู่ในขอบเขตแผนที่ที่กำหนด

เลือกชั้นที่จะคัดลอก

ชั้นข้อมูลหรือตารานั้นที่จะคัดลอก

นอกเหนือจากการเลือกชั้นข้อมูลจากแผนที่คุณสามารถเลือก เบราว์เลเยอร์ ที่ด้านล่างของรายการแบบเลื่อนลงเพื่อเรียกดูเนื้อหาของคุณสำหรับชุดข้อมูลชุดข้อมูลข้อมูลขนาดใหญ่หรือฟีเจอร์เลเยอร์

คุณอาจจะใช้ฟีเจอร์ทั้งหมดหรือเลือกบางฟีเจอร์ (เป็นตัวเลือก)

หรือใช้ปุ่มเลือกเพื่อระบุตำแหน่งที่จะคัดลอก

เลือก ArcGIS Data Store ที่จะคัดลอกไปยัง

Data Store เป็นการเก็บชั้นข้อมูลที่จะถูกคัดลอกไป พารามิเตอร์นี้จะลบล้างการเลือกใน การวิเคราะห์เชิงสิ่งแวดล้อม ตัวเลือกต่างๆ มีดังต่อไปนี้:

  • พื้นที่ชั่วคราว—นี้คือค่าตั้งต้น
  • สัมพันธ์.

ชื่อของชั้นข้อมูลผลลัพธ์

เป็นชื่อชั้นข้อมูลที่จะถูกสร้างใน เนื้อหาของฉัน และเพิ่มเข้าไปในแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุต ถ้าชั้นข้อมูลมีอยู่แล้วเครื่องมือจะล้มเหลว

ใช้ บันทึกผลลัพธ์ใน รายการตัวเลือก เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผล
