สร้างความผิดปกติหลายมิติ

คำนวณความผิดปกติของแต่ละรายการในแรสเตอร์หลายมิติที่มีเพื่อสร้างแรสเตอร์หลายมิติใหม่

ความผิดปกติคือการเบี่ยงเบนของการสังเกตจากค่ามาตรฐาน ค่าเฉลี่ย หรือค่ามัธยฐาน

เครื่องมือนี้จะคำนวณความผิดปกติตลอดช่วงเวลากับตัวแปรหนึ่งหรือหลายตัวในแรสเตอร์หลายมิติ หากคุณมีมิติที่ไม่ใช่เวลานอกเหนือจากมิติเวลาแล้ว ความผิดปกติจะถูกคำนวณในแต่ละขั้นตอนในส่วนของมิติเพิ่มเติม

ตัวอย่างเช่น คุณมีข้อมูลของอุณหภูมิมหาสมุทรรายเดือน ซึ่งรวบรวมที่ทุก ๆ ความ 1 เมตรไปจนลึกถึง 100 เมตร และคุณต้องการคำนวณความผิดปกติของอุณหภูมิเป็นค่าเบี่ยงเบนจากค่าเฉลี่ยรายปี เครื่องมือนี้จะกำหนดความผิดปกติของอุณหภูมิตามค่าเฉลี่ยรายปีหากคุณตั้งค่า รายปีเป็น ช่วงเวลาชั่วคราวเพื่อคำนวณค่าเฉลี่ยและจะคืนค่าความผิดปกติสำหรับแต่ละความลึกทั้ง 100 ชั้น

เครื่องมือนี้รองรับเฉพาะชุดข้อมูลแรสเตอร์หลายมิติที่มีมิติเวลา

หาก ใช้ขอบเขตแผนที่ปัจจุบัน ถูกเลือกไว้ ตัวแปรของ ชั้นข้อมูลที่เห็นอยู่ในปัจจุบันมาทำการวิเคราะห์เท่านั้น หากไม่เลือก ตัวแปรของชั้นข้อมูลเต็มจะถูกนำมาวิเคราะห์ ถึงแม้ว่าข้อมูลเหล่านั้นจะอยู่นอกขอบเขตแผนที่ปัจจุบัน

เลือกชั้นภาพหลายมิติเพื่อสร้างความผิดปกติ

ชั้นภาพอินพุทของชุดข้อมูลแรสเตอร์หลายมิติ

เลือกตัวแปรที่จะสร้างความผิดปกติ

ตัวแปรหรือตัวแปรที่จะคำนวณความผิดปกติ หากไม่มีการระบุตัวแปร ตัวแปรทั้งหมดที่มีมิติเวลาจะถูกวิเคราะห์

เลือกวิธีการสร้างความผิดปกติ

ระบุวิธีการที่จะใช้ในการคำนวณความผิดปกติ

  • ความแตกต่างจากค่าเฉลี่ย—ความแตกต่างระหว่างค่าพิกเซลและค่าเฉลี่ยของพิกเซลนั้นในแต่ละส่วนที่กำหนดโดยช่วงเวลาจะถูกคำนวณ นี้คือค่าตั้งต้น
  • เปอร์เซ็นต์ความแตกต่างจากค่าเฉลี่ย—ความแตกต่างระหว่างค่าพิกเซลและค่าเฉลี่ยของพิกเซลนั้นในแต่ละส่วนที่กำหนดโดยช่วงเวลาจะถูกคำนวณ
  • เปอร์เซ็นต์ของค่าเฉลี่ย—เปอร์เซ็นต์ของค่าเฉลี่ยจะถูกคำนวณ
  • Z Score—Z-Score สำหรับพิกเซลแต่ละรายการจะถูกคำนวณ Z-Score เท่ากับ 0 หมายถึงค่าของพิกเซลเหมือนกันกับค่าเฉลี่ย Z-Score เท่ากับ 1 หมายถึงค่าของพิกเซลคือ 1 ซึ่งเป็นส่วนเบี่ยงเบนมาตรฐานจากค่าเฉลี่ย หาก Z-Score เป็น 2 ค่าของพิกเซลคือ 2 ซึ่งเป็นส่วนเบี่ยงเบนมาตรฐานจากค่าเฉลี่ยและอื่น ๆ
  • ความแตกต่างจากค่ามัธยฐาน—ความแตกต่างระหว่างค่าพิกเซลและค่าเฉลี่ยทางคณิตศาสตร์ของค่าของพิกเซลนั้นในแต่ละส่วนที่กำหนดโดยช่วงเวลาจะถูกคำนวณ
  • เปอร์เซ็นต์ความแตกต่างจากค่ามัธยฐาน—เปอร์เซ็นต์ความแตกต่างระหว่างค่าพิกเซลและค่าเฉลี่ยทางคณิตศาสตร์ของค่าของพิกเซลนั้นในแต่ละส่วนที่กำหนดโดยช่วงเวลาจะถูกคำนวณ
  • เปอร์เซ็นต์ของค่ามัธยฐาน—เปอร์เซ็นต์ของค่ามัธยฐานทางคณิตศาสตร์จะถูกคำนวณ

วิธีการทางคณิตศาสตร์สำหรับการคำนวณค่าความผิดปกติมีการระบุไว้ด้านล่าง ความผิดปกติสามารถคำนวณได้โดยใช้ค่าเฉลี่ยหรือค่ามัธยฐานเป็นข้อกำหนดของค่าเฉลี่ย หากการกระจายข้อมูลไม่ตรง ค่าเฉลี่ยอาจได้รับส่งผลเป็นอย่างมากจากขอบเขต ดังนั้นค่ามัธยฐานหรือวิธี Z-score อาจเหมาะสมกว่าสำหรับข้อมูลประเภทนี้

  • ความแตกต่างจากค่าเฉลี่ย = x - µ
    • x = ค่าพิกเซลในแต่ละรายการ
    • µ = หมายถึงค่าของพิกเซลนั้นในช่วงเวลาที่กำหนด
  • เปอร์เซ็นต์แตกต่างจากค่าเฉลี่ย = |x - µ| / [(x + µ)/2]
    • x = ค่าพิกเซลในแต่ละรายการ
    • µ = หมายถึงค่าของพิกเซลนั้นในช่วงเวลาที่กำหนด
    • |x - µ| = ค่าสัมบูรณ์ของความแตกต่างระหว่างค่าและค่าเฉลี่ย
  • เปอร์เซ็นต์ของค่าเฉลี่ย = x / µ
    • x = ค่าพิกเซลในแต่ละรายการ
    • µ = หมายถึงค่าของพิกเซลนั้นในช่วงเวลาที่กำหนด
  • ความแตกต่างจากค่ามัธยฐาน = x - ß
    • x = ค่าพิกเซลในแต่ละรายการ
    • ß = ค่ามัธยฐานของค่าพิกเซลในช่วงเวลาที่กำหนด
  • เปอร์เซ็นต์ความแตกต่างจากค่ามัธยฐาน = |x - ß| / [(x + ß)/2]
    • x = ค่าพิกเซลในแต่ละรายการ
    • ß = ค่ามัธยฐานของค่าพิกเซลในช่วงเวลาที่กำหนด
    • |x - ß| = ค่าสัมบูรณ์ของความแตกต่างระหว่างค่าและค่ามัธยฐาน
  • เปอร์เซ็นต์ของค่ามัธยฐาน = x / ß
    • x = ค่าพิกเซลในแต่ละรายการ
    • ß = ค่ามัธยฐานของค่าพิกเซลในช่วงเวลาที่กำหนด
  • Z-score = (x - µ) / S
    • x = ค่าพิกเซลในแต่ละรายการ
    • µ = หมายถึงค่าของพิกเซลนั้นในช่วงเวลาที่กำหนด
    • S = ค่าเบี่ยงเบนมาตรฐานของค่าพิกเซลในช่วงเวลาที่กำหนด

เลือกช่วงเวลาชั่วคราวเพื่อคำนวณค่าเฉลี่ย

ระบุช่วงเวลาชั่วคราวที่จะใช้ในการคำนวณค่าเฉลี่ย

  • ทั้งหมด—คำนวณค่าเฉลี่ยของทุกส่วนสำหรับแต่ละพิกเซล
  • รายปี—คำนวณค่าเฉลี่ยรายปีสำหรับแต่ละพิกเซล
  • เกิดซ้ำรายเดือน—คำนวณค่าเฉลี่ยรายเดือนสำหรับแต่ละพิกเซล
  • เกิดซ้ำรายสัปดาห์—คำนวณค่าเฉลี่ยรายสัปดาห์สำหรับแต่ละพิกเซล
  • เกิดซ้ำรายวัน—คำนวณค่าเฉลี่ยรายวันสำหรับแต่ละพิกเซล
  • รายชั่วโมง—คำนวณค่าเฉลี่ยรายชั่วโมงสำหรับแต่ละพิกเซล
  • แรสเตอร์ภายนอก—ชุดข้อมูลแรสเตอร์ที่มีอยู่ซึ่งมีค่าเฉลี่ยหรือค่ามัธยฐานสำหรับพิกเซลมีการอ้างอิงแต่ละรายการ

เลือกชั้นข้อมูลค่าเฉลี่ยของรูปภาพเป็นข้อมูลอ้างอิง

ระบุชุดข้อมูลแรสเตอร์อ้างอิงที่มีค่าเฉลี่ยที่คำนวณไว้ก่อนหน้าสำหรับพิกเซลแต่ละรายการ ความผิดปกติจะถูกคำนวณโดยเปรียบเทียบกับค่าเฉลี่ยนี้

เพิกเฉยค่าที่หายไปในการคำนวณ

ระบุว่าค่าที่ไม่มีจะถูกละเว้นในการวิเคราะห์

  • ทำเครื่องหมาย—การวิเคราะห์จะรวมพิกเซลที่ถูกต้องทั้งหมดตามส่วนข้อมูลที่กำหนดและละเว้นพิกเซล NoData ใด ๆ นี้คือค่าตั้งต้น
  • ไม่ได้ทำเครื่องหมาย—การวิเคราะห์จะส่งผลให้เกิด NoData หากมีค่า NoData ใด ๆ สำหรับพิกเซลตามขนาดที่กำหนด

ชื่อชั้นข้อมูลผลลัพธ์

ชื่อของชั้นข้อมูลที่จะถูกสร้างขึ้นใน เนื้อหาของฉัน และเพิ่มไปยังแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุท ถ้าชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกขอให้ตั้งชื่อใหม่

คุณสามารถระบุชื่อโฟลเดอร์ใน My Content ที่ผลลัพธ์จะบันทึกโดยการใช้ กล่อง drop-down บันทึกผลใน หากคุณมีสิทธิ์ในการสร้างทั้งชั้นข้อมูลรูปภาพแบบไทล์และไดนามิก คุณจะสามารถระบุประเภทชั้นข้อมูลที่จะสร้างเอาท์พุทโดยใช้กล่องดรอปดาวน์ บันทึกผลลัพธ์เป็น ได้
