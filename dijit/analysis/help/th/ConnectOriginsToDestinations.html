เชื่อมต่อจุดเริ่มต้นไปยังจุดสิ้นสุด

การวัดระยะทางจากต้นทางไปยังปลายทาง โดยจะได้ระยะเวลาที่ใช้ในการเดินทาง หรือ ระยะทางระหว่างจุดทั้งสอง เครื่องมือนี้สามารถรายงานจากระยะแบบเส้นตรง หรือ ระยะทางจากเส้านทางที่ใช้เดินทางจริง ซึ่งสามารถกำหนดรูปแบบได้ เช่น การเดิน หรือการขับรถ หรือระยะเวลาตามรูปแบบการเดินทาง ด้วยการใช้งานเครื่องมือนี้ คุณสามารถทำสิ่งต่างๆ ได้ ดังนี้:

  • คำนวณระยะทางทั้งหมดหรือระยะเวลาในการเดินทางจากบ้านไปยังที่ทำงาน
  • วัดระยะห่างในการเดินทางของลูกค้าไปยังร้านค้าของคุณ ใช้ข้อมูลนี้ในการระบุกลุ่มตลาดที่ต้องการ โดยเฉพาะการสร้างแคมเปญน์โฆษณาหรือการเลือกตำแหน่งที่ตั้งร้านค้าใหม่
  • คำนวณระยะทางในการเดินทางสำหรับยานพาหนะของคุณหลังจากนั้น ทำการประมวลผลด้วยเครื่องมือสรุปผลด้วยระยะภายในเพื่อรายงานระยะทางภายในเมืองหรือภูมิภาคอื่นๆ

คุณกำหนดจุดเริ่มต้นและจุดสิ้นสุด แล้วเครื่องมือจะให้ผลลัพธ์เป็นชั้นข้อมูลที่มีเส้นทาง พร้อมด้วยระยะทางจากคู่ของจุดเริ่มต้น และจุดสิ้นสุด

ถ้า ใช้ขอบเขตแผนที่ปัจจุบัน ถูกเลือกไว้ จะมีการใช้ข้อมูลเพื่อการวิเคราะห์นี้เฉพาะข้อมูลที่อยู่ในพื้นที่จากขอบเขตแผนที่ที่เห็นอยู่ในปัจจุบันมาใช้ในการวิเคราะห์ ถ้าไม่เลือก ฟีทเจอร์ทั้งหมดในชั้นข้อมูลนำเข้าจะถูกพิจารณา แม้ว่าจะอยู่นอกขอบเขตแผนที่ปัจจุบัน

กำหนดชั้นข้อมูลจุด เพื่อแทนที่ตำแหน่งจุดเริ่มต้น

กำหนดชั้นข้อมูลจุด เพื่อแทนที่จุดเริ่มต้น เครื่องมือสนับสนุน 5,000 เริ่มต้น และ 5,000 สิ้นสุด

หากคุณเลือกชั้นข้อมูลจากจุดเริ่มต้นเดียว มันจะเชื่องโยงสู่จุดปลายทางได้ทั้งหมด

นอกจากนี้ยังสามารถเลือกชั้นข้อมูลจากแผนที่ของคุณ โดยคุณสามารถใช้ เลือกชั้นข้อมูลการวิเคราะห์ลิฟวิ่งแอตลาส หรือ เลือกชั้นข้อมูลการวิเคราะห์ ที่พบได้จากด้านล่างของรายการ การเปิดแกลเลอรี่ที่มีคอลเลกชันของชั้นข้อมูลที่มีประโยชน์สำหรับการวิเคราะห์หลายหลายอย่าง

เลือกชั้นข้อมูลปลายทาง

เลือกชั้นข้อมูลที่แสดงจุดหมายหรือปลายทางต่างๆ ที่เชื่อมโยงกับจุดเริ่มต้นเครื่องมือสนับสนุน 5,000 เริ่มต้น และ 5,000 สิ้นสุด

หากคุณเลือกชั้นข้อมูลที่มีเพียงจุดหมายเดียว, จุดเริ่มต้นทั้งหมดจะเชื่อมโยงไปยังจุดหมายปลายทางนั้น

หากมีจุดเริ่มต้นหนึ่งจุดและจุดหมายปลายทางหลายจุด เครื่องมือจะทำหน้าที่เชื่อมจุดเริ่มต้นไปยังจุดหมายปลายทางแต่ละจุด

นอกจากนี้ยังสามารถเลือกชั้นข้อมูลจากแผนที่ของคุณ โดยคุณสามารถใช้ เลือกชั้นข้อมูลการวิเคราะห์ลิฟวิ่งแอตลาส หรือ เลือกชั้นข้อมูลการวิเคราะห์ ที่พบได้จากด้านล่างของรายการ การเปิดแกลเลอรี่ที่มีคอลเลกชันของชั้นข้อมูลที่มีประโยชน์สำหรับการวิเคราะห์หลายหลายอย่าง

ถ้ามีจุดเริ่มต้น และจุดสิ้นสุดหลายจุด คุณจำเป็นต้องมีฟิลด์ที่ใช้บอกวิธีการจับคู่ระหว่างจุดเริ่มต้น และจุดสิ้นสุด ดังนี้: ฟิลด์ ID ในจุดเริ่มต้น และ ฟิลด์ ID ของจุดสิ้นสุดที่คู่กัน เครื่องมือทำหน้าที่เชื่อมโยงระหว่างจุดเริ่มต้นกับจุดหมายปลายทางระหว่างการวิเคราะห์ยกตัวอย่างเช่น ให้คิดว่าคุณมีจุดเริ่มต้นสองสุด มีค่าฟิลด์ ID เป็นคำว่า Rosie และ Harold ส่วนจุดสิ้นสุดต้องมีค่าในฟิลด์ ID เหมือนกัน จุดที่มีคำว่า Rosie ในข้อมูลจุดเริ่มต้นจะเชื่อมต่อไปยัง จุดที่มีคำว่า Rosie ในข้อมูลจุดสิ้นสุด และจุด Harold จะเชื่อมต่อไปยัง Harold ไม่มีการสร้างการเชื่อมโยงหรือการวัดระยะอื่นๆ

ค่า ID ต้องเป็นค่าที่ไม่ซ้ำกันในแต่ละชั้นข้อมูล ตัวอย่างเช่น ในชั้นข้อมูลจุดสิ้นสุด ห้ามมีคำว่า Rosie สองค่า ค่าที่อยู่ในชั้นข้อมูลจุดเริ่มต้น และจุดสิ้นสุด ต้องมีค่าตรงกัน Harold ไม่สามารถเชื่อมต่อกับค่าที่กรอกผิดอย่าง Harodlได้

การวัด

เลือกได้ว่าจะเชื่อมต่อระหว่างจุดเริ่มต้นกับจุดสิ้นสุดได้โดยใช้เส้นตรงหรือใช้เส้นทางตามการเดินทางจริง เส้นทางที่ได้จะทาบไปตามถนน หรือทางเดินเท้า ตามกฎการเดินทางที่มีการระบุไว้ เช่น ถนนที่อนุญาตให้เดินรถทางเดียว และแยกที่ไม่อนุญาตให้รถส่วนบุคคล หรือรถบรรทุกเลิ้ยวเข้าไปวิธีการในการเดินทาง กำหนดโดยผู้ดูแลระบบในองค์กรของคุณ

เมื่อคุณกำหนด เวลาในการขับรถ, คุณจะมีตัวเลือกในการวัดระยะของการขับรถ

  • เมื่อไม่เลือก ใช้ข้อมูลจราจร จะค้นหาเส้นทางที่เดินทางได้เร็วที่สุด โดยอิงจากความเร็วในการเดินทางแบบคงที่

    โดยตัวเลือกที่เลือกนี้ หากคุณต้องการสร้างพื้นที่ด้วยการกำหนดเวลาในการขับรถ ไม่ใช่ที่คุณจะคำนวณโดยใช้การเปลี่ยนสภาพการจราจรสำหรับกำหนดเวลาที่เริ่มขับรถออกมา

  • เมื่อเลือก ใช้ข้อมูลจราจร จะค้นหาเส้นทางที่เดินทางได้เร็วที่สุดโดยอิงจากเวลาเริ่มต้นและเปลี่ยนไปตามสภาพการจราจร เมื่อเลือก สภาพการจราจสด ตัวเลือกนี้ใช้เวลาที่ออกจากจุดเริ่มต้น เป็นเวลาปัจจุบัน และใช้ความเร็วการจราจรในปัจจุบัน และที่ประมาณค่าได้โดยอ้างอิงจากอุปกรณ์เซนเซอร์ที่ส่งข้อมูลมาสร้างเส้น การคาดการณ์ความเร็วในการจราจรถูกคำนวณโดยอ้างอิงจากความเร็วของสภาพการจราจรสด ประวัติความเร็ว และเหตุการณ์ปัจจุบัน เช่น สภาพอากาศ การคาดการณ์ความเร็วในการจราจร 12 ชั่วโมง ดังนั้น คุณสามารถปรับเปลี่ยนและตั้งเวลาที่จะออกเดินทางล่วงได้ถึง 12 ชั่วโมง

    ใช้การตั้งค่าการรายงานสดการจราจรเหล่านี้ เพื่อหาระยะเวลาในการขับขี่ สำหรับการออกเดินทางตอนนี้ การออกเดินทางในอีก 1 ชั่วโมงนับจากนี้ และอื่นๆ

  • เมื่อเลือก ใช้ข้อมูลจราจร และเลือก การจราจรอิงตามเงื่อนไขสำหรับ วัน และเวลา เครื่องมือจะหาเส้นทางที่เดินทางได้เร็วที่สุด โดยอิงจากประวัติความเร็วเฉลี่ยในทุกๆ ห้านาทีในช่วงสัปดาห์ ผลลัพธ์ที่ได้นั้น จะคำนวณจากการจราจรปกติ แต่ไม่ครอบคลุมถึงปัจจัยหรือสภาพการจราจรในปัจจุบัน และเหตุการณ์ต่างๆ ซึ่งอาจทำให้เกิดความแตกต่างไปจากปกติ

    คุณสามารถใช้การตั้งค่าการจราจรเพื่อค้นหา "พื้นที่ไหนที่ฉันสามารถไปถึงเมื่อเริ่มออกตอน 11:30 น. ในวันพุธ?

    เวลาที่คุณกำหนดอ้างอิงตามโซนเวลา ที่จุดเริ่มต้นของคุณตั้งอยู่ เช่น ถ้าคุณกำหนดเวลาเป็น 8 โมงเช้า มีจุดเริ่มต้นสองจุด ในนิวยอร์ค และลอสแองเจลิส เส้นทางที่ได้จะสร้างโดยเริ่มจาก แปดโมงเช้า เวลาฝั่งตะวันออก ตอนแปดโมงเช้า เฉพาะเวลาในเขตแปซิฟิก

หมายเหตุ เมื่อยานพาหนะขับออกจากจุด เวลาจะเปลี่ยนแปลงไปตามสภาพการจราจร เครื่องมือเชื่อมจุดเริ่มต้นไปจุดสิ้นสุดนี้ มีรายการให้หลากหลายเมื่อคุณเลือก ใช้ข้อมูลจราจร อย่างไรก็ตาม ข้อมูลจราจรไม่ได้มีทุกภูมิภาค กดที่ ดูความพร้อม เพื่อเชื่อมต่อไปหาดูว่าพื้นที่ศีกษาของคุณมีหรือไม่

ชื่อชั้นข้อมูลผลลัพธ์

เป็นชื่อชั้นข้อมูลที่จะถูกสร้างใน เนื้อหาของฉัน และเพิ่มเข้าไปในแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุท ถ้าชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกถามให้ตั้งชื่อใหม่

ใช้ บันทึกผลลัพธ์ใน รายการตัวเลือก เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผล
